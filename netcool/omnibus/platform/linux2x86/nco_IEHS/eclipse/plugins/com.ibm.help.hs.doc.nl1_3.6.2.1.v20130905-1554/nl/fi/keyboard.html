Tietojen määritys ja selaus näppäimistön avulla

Voit selata ohjejärjestelmää näppäimistön avulla.

Selaus näppäimistön avulla

Voit selata ohjejärjestelmää käyttämällä seuraavia näppäinyhdistelmiä:

  • Kun olet Aihe-ruudussa, voit siirtyä seuraavaan linkkiin painamalla sarkainnäppäintä.
  • Kun haluat laajentaa tai pienentää solmua rakenne-esityksessä, paina oikeaa tai vasenta nuolta.
  • Kun haluat siirtyä seuraavaan aihesolmuun, paina alanuolta.
  • Kun haluat siirtyä edelliseen aihesolmuun, paina ylänuolta.
  • Kun haluat siirtyä seuraavaan linkkiin, painikkeeseen tai aihesolmuun jossakin näkymässä, paina sarkainnäppäintä.
  • Kun haluat siirtyä ruudun ylä- tai alareunaan, paina Home- tai End-näppäintä.
  • Kun haluat siirtyä taaksepäin, paina näppäinyhdistelmää Alt + vasen nuoli. Kun haluat siirtyä eteenpäin, paina näppäinyhdistelmää Alt + oikea nuoli.
  • Kun haluat siirtyä seuraavaan ruutuun, paina F6-näppäintä.
  • Kun haluat siirtyä edelliseen ruutuun, paina näppäinyhdistelmää Vaihto + F6.
  • Kun haluat tulostaa aktiivisen ruudun, paina näppäinyhdistelmää Ctrl + P.
  • Kun haluat avata ponnahdusvalikon, paina näppäinyhdistelmää Alt + M.
  • Kun haluat siirtyä ponnahdusvalikon edelliseen vaihtoehtoon, paina ylänuolta.
  • Kun haluat siirtyä ponnahdusvalikon seuraavaan vaihtoehtoon, paina alanuolta.
  • Kun haluat valita ponnahdusvalikon vaihtoehdon, siirry vaihtoehdon kohdalle ja paina Enter-näppäintä.
  • Kun haluat sulkea ponnahdusvalikon, paina Esc-näppäintä.
  • Kun haluat siirtyä Sisältö-näkymään, paina näppäinyhdistelmää Alt + C (ja sitten Enter-näppäintä, jos käytössä on IE-ohjelma).
  • Kun haluat siirtyä hakutulosnäkymään, paina näppäinyhdistelmää Alt + R (ja sitten Enter-näppäintä, jos käytössä on IE-ohjelma).
  • Kun haluat siirtyä hakemistonäkymään, paina näppäinyhdistelmää Alt + I (ja sitten Enter-näppäintä, jos käytössä on IE-ohjelma).
  • Kun haluat siirtyä kirjanmerkkinäkymään, paina näppäinyhdistelmää Alt + Z (ja sitten Enter-näppäintä, jos käytössä on IE-ohjelma).

JAWS 7 -käyttäjille

  • Kun haluat laajentaa rakenne-esityksen solmun, paina näppäinyhdistelmää Alt + O. JAWS toistaa "Alt O".
  • Kun haluat pienentää rakenne-esityksen solmun, paina näppäinyhdistelmää Alt + B. JAWS toistaa "Alt B".
  • Kun haluat siirtyä eteenpäin rakenne-esityksen seuraavaan aihesolmuun, paina näppäinyhdistelmää Ctrl + vaihtonäppäin + alanuolinäppäin.
  • Kun haluat siirtyä taaksepäin rakenne-esityksen edelliseen aihesolmuun, paina näppäinyhdistelmää Ctrl + vaihtonäppäin + ylänuolinäppäin.
  • Kun haluat siirtyä ruudun ylä- tai alareunaan, paina PageUp- tai PageDown-näppäintä.
  • Kun haluat siirtyä edelliseen ponnahdusvalikon vaihtoehtoon, paina näppäinyhdistelmää Ctrl + vaihtonäppäin + ylänuolinäppäin.
  • Kun haluat siirtyä seuraavaan ponnahdusvalikon vaihtoehtoon, paina näppäinyhdistelmää Ctrl + vaihtonäppäin + alanuolinäppäin.
  • Kun haluat valita ponnahdusvalikon vaihtoehdon, siirry vaihtoehdon kohdalle ja paina Enter-näppäintä.
  • Kun haluat laajentaa päivitystoiminnossa näkyvän dokumentaatioluettelon, paina näppäinyhdistelmää Alt + O. JAWS toistaa "Alt O".
  • Kun haluat pienentää päivitystoiminnossa näkyvän dokumentaatioluettelon, paina näppäinyhdistelmää Alt + B. JAWS toistaa "Alt B".
  • Voit siirtyä Sisältö-, Hakutulokset-, Hakemisto- ja Kirjanmerkit-näkymien välillä siirtymällä ensin Navigointinäkymien valikoima -kehykseen ja käyttämällä sitten edellä kuvattuja näppäinyhdistelmiä.

Muiden näppäinyhdistelmien käytöstä on kuvaus kohdassa Selaus näppäimistön avulla.

Home Page Reader 3.4.1 -ohjelman käyttäjille

  • Kun haluat laajentaa tai pienentää rakenne-esityksen solmun, paina Enter-näppäintä.
  • Kun haluat avata ponnahdusvalikon, paina ensin näppäinyhdistelmää Ctrl + L, sitten End-näppäintä ja lopuksi Enter-näppäintä.
  • Kun haluat siirtyä edelliseen ponnahdusvalikon vaihtoehtoon, paina näppäinyhdistelmää vaihtonäppäin + sarkainnäppäin.
  • Kun haluat siirtyä ponnahdusvalikon seuraavaan vaihtoehtoon, paina sarkainnäppäintä.
  • Kun haluat valita ponnahdusvalikon vaihtoehdon, siirry vaihtoehdon kohdalle ja paina Enter-näppäintä.
  • Kun haluat sulkea ponnahdusvalikon, paina näppäinyhdistelmää Alt + vaihtonäppäin + H.
  • Voit siirtyä Sisältö-, Hakutulokset-, Hakemisto- ja Kirjanmerkit-näkymien välillä siirtymällä ensin Navigointinäkymien valikoima -kehykseen ja toimimalla seuraavasti:
    • Jos haluat siirtyä Sisältö-näkymään, paina näppäinyhdistelmää Alt + vaihtonäppäin + C.
    • Jos haluat siirtyä hakutulosnäkymään, paina näppäinyhdistelmää Alt + vaihtonäppäin + R.
    • Jos haluat siirtyä hakemistonäkymään, paina näppäinyhdistelmää Alt + vaihtonäppäin + I.
    • Jos haluat siirtyä kirjanmerkkinäkymään, paina näppäinyhdistelmää Alt + vaihtonäppäin + Z.

Muiden näppäinyhdistelmien käytöstä on kuvaus kohdassa Selaus näppäimistön avulla.
