Definició d'adreces d'interès al sistema d'ajuda

Per trobar un tema més fàcilment, definiu una adreça d'interès i deseu-la a la vista Adreces d'interès del sistema d'ajuda.

Procediment

  1. Per definir una adreça d'interès per al tema actual del panell Tema, feu clic a Adreça d'interès () a la barra d'eines.
  2. Per veure les vostres adreces d'interès, feu clic a la pestanya Adreces d'interès () a la barra d'eines.
    Nota:
    • Si el sistema d'ajuda s'executa en mode d'Information Center, el botó Adreça d'interès () només es visualitza quan navegueu per pàgines amb l'Internet Explorer. Podeu fer clic al menú Preferit de l'Internet Explorer per cercar les adreces d'interès que heu creat.
    • Si el sistema d'ajuda s'executa en mode autònom, feu clic a la pestanya Adreces d'interès () per cercar les vostres adreces d'interès.
  3. Per suprimir una adreça d'interès, feu clic a la pestanya Adreces d'interès (), seleccioneu el nom del tema al panell Tema i feu clic a Suprimeix adreça d'interès seleccionada () a la barra d'eines.
  4. Per suprimir totes les adreces d'interès, feu clic a la pestanya Adreces d'interès () i feu clic a Suprimeix totes les adreces d'interès () a la barra d'eines.
