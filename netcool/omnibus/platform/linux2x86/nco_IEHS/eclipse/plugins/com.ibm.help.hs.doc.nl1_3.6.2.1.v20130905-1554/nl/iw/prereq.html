דרישות מקדימות של מערכת העזרה

בעת הגדרת מערכת העזרה של IBM® Eclipse, ודאו שתחנת העבודה, מערכת ההפעלה, הדפדפן והחומרה שברשותכם עומדים בדרישות המתוארות בנושא זה.

דרישות דפדפן

כדי לעבוד עם מערכת העזרה של IBM Eclipse, ניתן להתקין את Microsoft® Internet Explorer גירסה 6.0 או 7.0, או את Firefox 2.0 או 3.0. כדי להבטיח שכל הפונקציות של גירסה זו של IBM Eclipse Help System ניתנות לשימוש, עליכם להפעיל קובצי Cookie ו-‎JavaScript™ בדפדפן ולהשבית את פונקציית הדפדפן שחוסמת חלונות מוקפצים.

דרישות של סביבת מערכת ההפעלה

כדי לעבוד עם IBM Eclipse Help System, עליכם להתקין את IBM Java™ Runtime Environment (JRE)‎ גירסה 6.0. עיינו בטבלה הבאה לקבלת דרישות מפורטות.

התאמה נתמכת

ניתן להתאים את מערכת העזרה באמצעות הקובץ plugin_customization.ini. הוא נמצא תחת תוסף המוצר הראשי. ניתן להגדיר אותו בקובץ ה-script של ההפעלה.

ניתן גם לשנות פרמטרי תצורה באמצעות עריכת הקובץ org.eclipse.help.base_<version number>/preferences.ini תחת המדריך eclipse/plugins, על אף שמומלץ להשתמש בתצורת ברירת המחדל.

ניתן להתאים את מערכת העזרה באמצעות שינוי הקובץ preferences.ini.

הערה: אל תשנו את הפרמטר שלהלן always_external_browser; אחרת מערכת העזרה עלולה לא לפעול כראוי.
always_external_browser
עבור מערכות ההפעלה Microsoft Windows, תצורת ברירת המחדל של פרמטר זה היא always_external_browser=true. בעת הפעלת מערכת העזרה במצב עצמאי, הגדרה זו עוזרת לכם לפתוח את דפדפן ברירת המחדל של מערכת ההפעלה במקום הדפדפן המוטבע במערכת העזרה. אין צורך להגדיר את always_external_browser=true עבור מערכות ההפעלה פרט ל-Windows.
