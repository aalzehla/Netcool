查看帮助系统中的信息

您可以了解如何使用工具栏上的各种按钮在帮助系统中进行浏览和搜索。

关于此任务

  • 帮助系统的必备要求
    在安装 IBM® Eclipse Help System 时，请确保您的工作站、操作系统、浏览器和硬件均满足本主题中所述的要求。
  • 浏览帮助系统
    使用帮助系统提供的图形用户界面来查找和查看产品文档。
  • 搜索帮助系统
    您可以搜索帮助系统中的所有主题，搜索单个主题或主题集合，或创建搜索范围以仅搜索某个信息子集。“搜索结果”视图最多可以显示 500 个结果（按相关性排序）。
  • 设置适当的范围
    您可以通过设置适当的过滤条件来缩小自己的工作范围。
  • 使用“索引”视图查找信息
    该主题描述了如何使用“索引”视图来查找帮助系统中的主题。
  • 在帮助系统中设置书签
    通过设置书签并将其保存到帮助系统的“书签”视图中可以更轻松地找到主题。
  • 查看不同语言版本的信息
    要查看另一种语言版本的帮助系统用户界面，必须将操作系统的区域设置更改为您选择的语言环境和语言。
  • 帮助系统的内容打印
    IBM Eclipse Help System 为您提供了多种打印系统中帮助内容的方式。
  • 帮助系统中的辅助功能部件
    辅助功能部件可以帮助身体不便（如：行动不便或有视力障碍）或具有其他特殊需求的人员顺利地使用信息技术产品。
  • 更新帮助系统中的信息
    您可以从帮助系统中下载文档内容，定期对此进行更新（如果可用）。Web 站点上的内容将定期进行更新。
