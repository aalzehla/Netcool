キーボード・ショートカットとキーボード・アクセラレーター

マウスで実行できる操作をキーまたはキーの組み合わせで実行することも可能です。

Microsoft Internet Explorer と Mozilla Firefox の Web ブラウザーでヘルプ・システムを操作するときに使用できるキーボード・ショートカットとキーボード・アクセラレーターを以下の表にまとめます。
注: Internet Explorer では、アクセラレーター・キーを使用してフォーカスをコントロール要素に移動し、Enter を押してそのコントロール要素をアクティブにします。 Firefox では、アクセラレーター・キーを使用して、フォーカスを移動する操作とコントロール要素をアクティブにする操作を同時に実行します。
表 1. Internet Explorer でヘルプ・システムをナビゲートするときに使用できるキーボード・ショートカットとキーボード・アクセラレーター
操作 Internet Explorer のショートカット
フレーム (ページ) の中から次のリンク、ボタン、トピック・ブランチに移動する操作 Tab の後に Enter を押してリンクをたどります
目次ビュー・フレームへの移動 Alt+C の後に Enter と Shift+F6 を押します
索引ビュー・フレームへの移動 Alt+I の後に Enter と Shift+F6 を押します
検索結果ビュー・フレームへの移動 Alt+R の後に Enter と Shift+F6 を押します
トピック・ビュー・フレームへの移動 Alt+K の後に Enter を押します
トピック・ブランチの展開または省略 右矢印または左矢印
次のトピック・ブランチへの移動 下矢印または Tab
前のトピック・ブランチへの移動 上矢印または Shift+Tab
上部へのスクロール Home
下部へのスクロール End
次のフレーム F6
前のフレーム Shift+F6
現行ページまたはアクティブ・フレームの印刷 Ctrl+P
表 2. Firefox でヘルプ・システムをナビゲートするときに使用できるキーボード・ショートカットとキーボード・アクセラレーター
操作 Firefox のショートカット
フレーム (ページ) の中から次のリンク、ボタン、トピック・ブランチに移動する操作 Tab の後に Enter を押してリンクをたどります
目次ビュー・フレームへの移動 Shift+Alt+C と Shift+F6
索引ビュー・フレームへの移動 Shift+Alt+I と Shift+F6
検索結果ビュー・フレームへの移動 Shift+Alt+R と Shift+F6
トピック・ビュー・フレームへの移動 Firefox では、この操作のための直接的なショートカットはありません。Tab または JAWS のナビゲーション機能を使用してください。
トピック・ブランチの展開または省略 右矢印または左矢印
次のトピック・ブランチへの移動 下矢印または Tab
前のトピック・ブランチへの移動 上矢印または Shift+Tab
上部へのスクロール Home
下部へのスクロール End
次のフレーム F6
前のフレーム Shift+F6
現行ページまたはアクティブ・フレームの印刷 Ctrl+P
