ヘルプ・システムの前提条件

IBM® Eclipse ヘルプ・システムをセットアップするときには、ワークステーション、オペレーション・システム、ブラウザー、ハードウェアがこのトピックで取り上げる要件を満たしていることを確認してください。

ブラウザー要件

IBM Eclipse ヘルプ・システムを使用する場合は、Microsoft Internet Explorer バージョン 6.0、7.0、Firefox 2.0、3.0 のいずれかをインストールできます。このバージョンの IBM Eclipse ヘルプ・システムのすべての機能を使用するには、ブラウザーで Cookie と JavaScript を有効にし、ポップアップ・ウィンドウをブロックするブラウザーの機能を無効にすることが必要です。

オペレーティング・システム環境要件

IBM Eclipse ヘルプ・システムを使用するには、IBM Java ランタイム環境 (JRE) バージョン 6.0 をインストールする必要があります。 詳しい要件については、以下の表を参照してください。

サポートされるカスタマイズ

plugin_customization.ini ファイルを使用して、ヘルプ・システムを構成できます。このファイルは、メインの製品プラグインの下にあります。開始スクリプトの中でそのファイルを設定することもできます。

eclipse/plugins ディレクトリーの下にある org.eclipse.help.base_<version number>/preferences.ini ファイルを編集することによって、構成パラメーターを変更することも可能ですが、デフォルトの構成を強くお勧めします。

preferences.ini ファイルに変更を加えることにより、ヘルプ・システムをカスタマイズできます。

注: always_external_browser パラメーターは変更しないでください。変更すると、ヘルプ・システムが正しく機能しなくなる可能性があります。
always_external_browser
Microsoft Windows オペレーティング・システムでは、 このパラメーターのデフォルト構成は always_external_browser=true です。 ヘルプ・システムをスタンドアロン・モードで始動する場合、 この設定により、ヘルプ・システムに組み込まれているブラウザーではなく、 オペレーティング・システムのデフォルトのブラウザーを開くことができます。 Windows 以外のオペレーティング・システムでは、always_external_browser=true を設定する必要はありません。
