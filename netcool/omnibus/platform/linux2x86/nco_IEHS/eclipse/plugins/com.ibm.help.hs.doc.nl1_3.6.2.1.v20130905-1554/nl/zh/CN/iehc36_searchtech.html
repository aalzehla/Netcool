高级搜索技术

您可以使用引号、通配符或布尔运算符来执行更多的高级搜索，以返回更小的搜索结果集

  • 在帮助系统中进行搜索时，您可以通过将某些词或短语加上双引号，从而将其标识为精确匹配的字符串。

    例如，当在搜索字段中输入 "log file" 时，将搜索已出现的所有 log file 字符串。如果不输入双引号，那么将返回包含 log 和 file 这两个词的实例的主题。

  • 您可以在搜索中使用诸如星号 (*) 和问号 (?) 之类的通配符以扩展搜索范围。
    • 使用星号 (*) 可以表示搜索查询中的多个未知或可变字符。例如，如果输入 par*，那么将返回包含 partly、participate、partial 以及其他以 par 开头的单词的主题。
      注： 搜索字符串有一个限制：在使用带有星号的搜索项时，它必须至少前跟三个字符，类似地，不带通配符的搜索词必须包含至少三个字符。
    • 使用问号可以表示搜索查询中的单个字符。 例如，如果输入 par?，那么将返回包含 part 的主题，但不会返回包含 partial 或 partly 的主题。
    注： 在使用带有星号 (*) 的搜索项时，它必须至少前跟三个字符。
  • 您可以使用布尔运算符 AND、OR 和 NOT 以使查询更加具体。例如：
    database AND "log file"
    将搜索结果缩小为同时包含搜索项 database 和短语 log file 的主题。
    database OR "log file"
    将搜索结果扩展为包含搜索项 database 或短语 log file 的主题。
    database NOT "log file"
    将搜索结果缩小为包含搜索项 database 但不包含短语 log file 的主题。
    database OR "data base" NOT "log file"
    搜索包含搜索项 database 或短语 data base 但不包含短语 log file 的主题。
