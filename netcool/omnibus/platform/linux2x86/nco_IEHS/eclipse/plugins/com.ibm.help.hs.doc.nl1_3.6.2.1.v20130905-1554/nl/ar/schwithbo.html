البحث باستخدام المعاملات البوولية(‏AND أو OR‏ أو NOT)‏

يمكنك ادراج معاملات Boolean، متضمنة AND، OR، NOT، في عبارة البحث الخاصة بك لجعل عملية البحث أكثر كفاءة.

نبذة عن هذه المهمة

يمكن أن يجعل استخدام معاملات Boolean عملية البحث الخاصة بك أكثر كفاءة من خلال تضييق أو امتداد نطاق البحث الخاص بك. على سبيل المثال:

database AND "log file"
قم بتضييق البحث بحيث يتضمن على الموضوعات التي تحتوي على المصطلح database والعبارة log file في أي موضع بالوثيقة.
database OR "log file"
يقوم بتوسيع البحث ليتضمن الموضوعات التي تتضمن على مصطلح database أو عبارة log file في أي موضع بالوثيقة.
database NOT "log file"
للبحث عن الموضوعات التي تتضمن database ولا تتضمن عبارة log file في أي موضع بالوثيقة.
database OR "data base" NOT "log file"
للبحث عن الوثائق التي تتضمن على أيا من المصطلح database أو العبارة data base ولا تتضمن على العبارة log file في أي موضع بالوثيقة.
