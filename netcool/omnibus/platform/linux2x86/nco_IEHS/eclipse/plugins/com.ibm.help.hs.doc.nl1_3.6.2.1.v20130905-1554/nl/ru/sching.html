Поиск в справочной системе

В этом разделе описана процедура поиска информации в IBM Eclipse Help System.

Для поиска разделов в справочной системе введите запрос в поле Поиск и нажмите кнопку Открыть. В окне результатов поиска будет показано до 500 результатов поиска, отсортированных по степени соответствия. При выборе результата поиска будет показан текст соответствующего раздела документации. В этом тексте слова из строки поиска будут выделены. Итоговая страница будет автоматически прокручена до первого вхождения искомой строки.

После выполнения поиска можно использовать функцию выделения. Для включения или выключения выделения искомого слова нажмите кнопку Выделить искомые термины ().

Для переключения между панелью навигации и панелью результатов поиска щелкните на вкладке Оглавление () или Результаты поиска ().

На панели Результаты поиска показаны разделы, отобранные в соответствии с критерием поиска без применения настроенных фильтров.

Прим.: Поиск выполняется без учета регистра символов.
  • Поиск слов и фраз целиком
    В справочной системе можно выполнять поиск всего слова или фразы.
  • Поиск с использованием символов подстановки
    Из этого раздела можно узнать о том, как расширить область поиска с помощью символов подстановки.
  • Поиск с использованием булевских операторов (AND, OR, NOT)
    Для повышения эффективности поиска в строке поиска можно указывать булевские операторы, в том числе AND, OR и NOT.
  • Сужение диапазона поиска
    Область поиска можно сузить до определенного набора разделов.
  • Быстрый поиск
    В этом разделе приведены инструкции по быстрому выполнению поиска.
  • Просмотр результатов поиска
    В справочной системе можно просмотреть результаты поиска по заданному критерию.
