Pré-requisitos para o sistema de ajuda

Quando você configurar o IBM® Eclipse Help System, certifique-se de que sua estação de trabalho, seu sistema operacional, navegador e hardware atendam aos requisitos descritos neste tópico.

Requisitos de navegador

Para trabalhar com o IBM Eclipse Help System, é possível instalar o Microsoft Internet Explorer Versão 6.0 ou 7.0, ou Firefox 2.0 ou 3.0. Para assegurar que todas as funções dessa versão do IBM Eclipse Help System sejam utilizáveis, você deve ativar cookies e JavaScript no navegador e desativar a função de bloqueio de janelas pop-up do navegador.

Requisitos do ambiente do sistema operacional

Para trabalhar com o IBM Eclipse Help System, você deve instalar um Java Runtime Environment (JRE) versão 6.0. Consulte a seguinte tabela para requisitos detalhados.

Customização suportada

É possível configurar o sistema de ajuda com o arquivo plugin_customization.ini. Ele está sob o plug-in do produto principal. É possível configurá-lo no script de início.

É possível também alterar os parâmetros de configuração por meio da edição do arquivo org.eclipse.help.base_<número da versão>/preferences.ini sob o diretório eclipse/plugins, embora a configuração padrão seja altamente recomendada.

Para obter informações detalhadas sobre a customização de seu sistema de ajuda, é possível modificar o arquivo preferences.ini.

Nota: Não modifique o seguinte parâmetro always_external_browser, caso contrário o sistema de ajuda poderá não funcionar adequadamente.
always_external_browser
Para sistemas operacionais Microsoft Windows, a configuração padrão deste parâmetro é always_external_browser=true. Quando você inicia o sistema de ajuda no modo independente, essa configuração o ajuda a abrir o navegador padrão do sistema operacional ao invés do navegador integrado no sistema de ajuda. Não é necessário configurar always_external_browser=true para os sistemas operacionais diferentes do Windows.
