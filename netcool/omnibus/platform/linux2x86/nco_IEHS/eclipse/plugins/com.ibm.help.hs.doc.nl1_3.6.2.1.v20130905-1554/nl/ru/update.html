Обновление информации в справочной системе

Документацию следует время от времени обновлять, загружая обновления из справочной системы. Документация на сайте регулярно обновляется.

Об этой задаче

Прим.: Функция обновления доступна только при работе в автономном режиме.

Для того чтобы проверить наличие обновлений и загрузить их, выполните следующие действия:

Процедура

  1. Нажмите кнопку Обновить () на панели инструментов. Будет показан список установленных наборов документации.
    Прим.: Значок Обновить доступен только при наличии обновлений информации. Кроме того, значок Обновить () можно скрыть на панели инструментов, настроив параметр -showUpdater в сценарии запуска.
  2. Нажмите кнопку Далее. Будут показаны Обновления для имеющихся наборов информации и Новые наборы информации. Доступные обновления хранятся на сайте обновления.
  3. Выберите документацию для установки и нажмите кнопку Установить обновления.
  4. После завершения установки нажмите Готово. Справочная система будет обновлена автоматически. Ее не требуется перезапускать вручную.
