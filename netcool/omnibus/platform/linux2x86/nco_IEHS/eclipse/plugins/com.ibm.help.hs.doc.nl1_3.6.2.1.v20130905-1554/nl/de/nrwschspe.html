Suchbereich eingrenzen

Sie können Ihren Suchbereich für bestimmte Themen eingrenzen.

Warum und wann dieser Vorgang ausgeführt wird

Standardmäßig werden bei der Suche in der Hilfefunktion alle Themen durchsucht, die in der Navigationssicht zu sehen sind. Sie können den Suchbereich jedoch auf eine bestimmte Themengruppe durch die Erstellung einer Suchliste eingrenzen und die Suchliste für eine spätere Verwendung speichern.

Sie können eine Suchliste durch Ausführen der folgenden Schritte erstellen:

  1. Klicken Sie auf den Link Suchbereich neben dem Feld Suchen. Sie gelangen zum Fenster 'Suchbereich auswählen'.
  2. Wählen Sie Nur in den folgenden Themen suchen aus und klicken Sie auf Neu. Das Fenster 'Neue Suchliste' wird angezeigt.
  3. Geben Sie im Feld Listenname einen Namen für Ihre neue Suchliste ein.
  4. Wählen Sie in der Liste Zu durchsuchende Themen die Verzweigungen der Navigationssicht aus, die Sie in Ihre Suche einschließen möchten. Sie können die Verzweigungen einblenden, um die Unterkategorien auf der zweiten Ebene der Überschrift auszuwählen.
  5. Klicken Sie auf OK. Ihre neue Suchliste wird nun in der Liste Nur in den folgenden Themen suchen angezeigt. Diese Option ist automatisch standardmäßig ausgewählt. Um diese Suchliste auf Ihre folgenden Suchabfragen anzuwenden, klicken Sie auf OK.
    Einschränkung: In einigen Browsern (zum Beispiel Konqueror und Safari) wird die Funktion zum Speichern von Suchlisten nicht unterstützt.
Nächste Maßnahme

Die momentan aktive Suchliste wird immer im oberen Teil des Browsers neben dem Link Suchbereich angezeigt.

Die zuletzt verwendete Suchliste, die Sie erstellt haben, bleibt bei jedem Start des Systems bestehen.
