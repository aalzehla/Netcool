說明系統的必備需求

設定 IBM® Eclipse Help System 時，請務必確定您的工作站、作業系統、瀏覽器及硬體皆符合本主題中所說明的需求。

瀏覽器需求

如果要使用 IBM Eclipse Help System，您可以安裝 Microsoft Internet Explorer 6.0 版或 7.0 版，或 Firefox 2.0 或 3.0。為了確保此 IBM Eclipse Help System 版本的所有功能皆能使用，您必須啟用瀏覽器中的 Cookie 及 JavaScript，並停用瀏覽器封鎖蹦現視窗的功能。

作業系統環境需求

如果要使用 IBM Eclipse Help System，您必須安裝「IBM Java Runtime Environment (JRE) 6.0 版」。請參閱下表以取得詳細需求。

表 1. 作業系統環境需求
IBM Eclipse Help System 版本 作業系統 作業系統及 JRE 需求
IBM Eclipse Help System（適用於 Microsoft Windows 32 位元）
  • Windows XP Professional Intel 32 位元
  • 此版本的 IBM Eclipse Help System 包含內嵌的 IBM JRE。如果您不想要保留內嵌的 JRE，可以將其從套件中移除。您的系統必須安裝 JRE 6.0 才能執行 IBM Eclipse Help System。
IBM Eclipse Help System（適用於 Linux）
  • Red Hat Linux Enterprise 5 Intel32 位元
  • 此版本的 IBM Eclipse Help System 包含內嵌的 IBM JRE。如果您不想要保留內嵌的 JRE，可以將其從套件中移除。您的系統上必須有 IBM JRE 6.0 才能執行 IBM Eclipse Help System。
  • 在這些作業系統上，您可以透過 Konqueror 瀏覽器以基本模式開啟說明系統。
  • 如果要在說明系統中顯示雙向字元、雙位元組字元或多位元組字元，請在系統上安裝對應的字型，並將該字型設為瀏覽器的顯示字型。
  • 如果要支援多種語言，請在啟動說明系統之前，將系統語言環境設成 UTF-8。

  • AIX 5.3 PPC 64 位元
  • 此版本的 IBM Eclipse Help System 包含內嵌的 IBM JRE。如果您不想要保留內嵌的 JRE，可以將其從套件中移除。您的系統上必須安裝 IBM JRE 6.0 才能執行 IBM Eclipse Help System。

支援的自訂作業

您可以使用 plugin_customization.ini 檔來配置說明系統。檔案位於主要產品外掛程式之下。您可以在啟動 Script 中加以設定。

您也可以編輯 eclipse/plugins 目錄下的 org.eclipse.help.base_<version number>/preferences.ini 檔案以變更配置參數，不過我們極力建議您使用預設配置。

您可以修改 preferences.ini 檔來自訂您的說明系統。

註： 請勿修改下列參數 always_external_browser；否則說明系統可能無法正常運作。
always_external_browser
對於 Microsoft Windows 作業系統，此參數的預設配置為 always_external_browser=true。 以獨立模式啟動說明系統時，這個設定會協助您開啟作業系統的預設瀏覽器，而非內嵌在說明系統中的瀏覽器。您不需要為 Windows 以外的作業系統設定 always_external_browser=true。
