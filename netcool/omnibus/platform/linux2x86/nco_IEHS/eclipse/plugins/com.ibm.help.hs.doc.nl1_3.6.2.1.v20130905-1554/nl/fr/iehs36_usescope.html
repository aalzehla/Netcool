Utilisation de la portée existante

Vous pouvez utiliser la portée existante pour réduire votre portée de travail.

Procédure

  1. Cliquez sur Portées : dans la barre d'outils.
  2. Dans la fenêtre en incrustation
    1. Cliquez sur Afficher toutes les rubriques pour définir la portée de l'ensemble du document.
    2. Cliquez sur Afficher uniquement les rubriques suivantes et choisissez une portée dans la liste.
  3. Cliquez sur OK pour activer votre portée de travail.
