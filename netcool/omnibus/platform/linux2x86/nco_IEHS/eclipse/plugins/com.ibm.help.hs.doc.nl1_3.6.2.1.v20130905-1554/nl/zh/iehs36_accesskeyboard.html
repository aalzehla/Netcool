键盘快捷键和加速键

您可以使用按键或按键组合来执行也可使用鼠标完成的操作。

下表列出了可在 Microsoft® Internet Explorer 和 Mozilla Firefox Web 浏览器帮助系统中使用的键盘快捷键和加速键。
注： 在 Internet Explorer 中，请使用加速键将焦点移至控件元素，然后按 Enter 键进行激活。在 Firefox 中，使用加速键可移动焦点并同时激活控件元素。
表 1. 用于使用 Internet Explorer 浏览帮助系统的键盘快捷键和加速键
操作 Internet Explorer 快捷键
转至框架（页面）中的下一个链接、按钮或主题分支 Tab 键，然后按 Enter 键以追踪链接
移至“目录视图”框架 Alt+C 键，然后按 Enter 键和 Shift+F6 键
移至“索引视图”框架 Alt+I 键，然后按 Enter 键和 Shift+F6 键
移至“搜索结果视图”框架 Alt+R 键，然后按 Enter 键和 Shift+F6 键
移至“主题视图”框架 Alt+K 键，然后按 Enter 键
展开或折叠主题分支 右箭头或左箭头
移至下一个主题分支 下箭头或 Tab 键
移至上一个主题分支 上箭头或 Shift+Tab 键
滚至顶部 Home 键
滚至底部 End 键
下一框架 F6 键
上一框架 Shift+F6 键
打印当前页面或激活框架 Ctrl+P 键
表 2. 用于使用 Firefox 浏览帮助系统的键盘快捷键和加速键
操作 Firefox 快捷键
转至框架（页面）中的下一个链接、按钮或主题分支 Tab 键，然后按 Enter 键以追踪链接
移至“目录视图”框架 Shift+Alt+C 键和 Shift+F6 键
移至“索引视图”框架 Shift+Alt+I 键和 Shift+F6 键
移至“搜索结果视图”框架 Shift+Alt+R 键和 Shift+F6 键
移至“主题视图”框架 在 Firefox 中，此操作没有直接快捷键。请使用 Tab 键或 JAWS 中的导航功能。
展开或折叠主题分支 右箭头或左箭头
移至下一个主题分支 下箭头或 Tab 键
移至上一个主题分支 上箭头或 Shift+Tab 键
滚至顶部 Home 键
滚至底部 End 键
下一框架 F6 键
上一框架 Shift+F6 键
打印当前页面或激活框架 Ctrl+P 键
