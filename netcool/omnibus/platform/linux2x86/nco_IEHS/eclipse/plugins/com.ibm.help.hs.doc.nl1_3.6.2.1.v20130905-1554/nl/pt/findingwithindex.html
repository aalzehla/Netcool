Localizar informações através da vista Índice remissivo

Este tópico descreve como trabalhar com a vista Índice remissivo para localizar tópicos no sistema de ajuda.

Sobre esta tarefa

Para trabalhar com a vista Índice remissivo de modo a obter as informações pretendidas, execute os seguintes passos:

Procedimento

  1. Faça clique no separador Índice remissivo () para apresentar a vista Índice remissivo.
  2. Escreva uma palavra-chave no campo de entrada de dados. A lista de índice remissivo deslocar-se-á automaticamente para as palavras-chave com maior correspondência.
    Nota:
    • Se uma palavra-chave estiver associada a mais do que um tópico, não será apresentada no Índice remissivo. Pode encontrar os títulos dos respectivos tópicos na vista Índice remissivo.
  3. Faça clique na palavra-chave, numa palavra-chave secundária ou num título de tópico, para apresentar o tópico na vista Índice.
    Nota: Os índices remissivos não são sensíveis a maiúsculas e minúsculas.

    Para alternar entre a vista Navegação e a vista Índice remissivo, faça clique no separador Índice () ou no separador Índice remissivo ().
