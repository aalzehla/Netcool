การตั้งค่าบุ๊กมาร์กในระบบวิธีใช้

คุณสามารถค้นหาหัวข้อได้อย่างง่ายดายโดยการตั้งค่าบุ๊กมาร์ก และบันทึกบุ๊กมาร์กนั้นลงในมุมมอง Bookmark ของระบบวิธีใช้

กระบวนการ

  1. หากต้องการตั้งค่าบุ๊กมาร์กสำหรับหัวข้อปัจจุบันในบานหน้าต่างหัวข้อ ให้คลิก Bookmark () บนแถบเครื่องมือ
  2. หากต้องการดูบุ๊กมาร์กของคุณ ให้คลิกแท็บ Bookmarks () บนแถบเครื่องมือ
    หมายเหตุ:
    • ถ้าระบบวิธีใช้กำลังรันอยู่ในโหมด information center, ปุ่ม Bookmark () จะปรากฏขึ้น ก็ต่อเมื่อคุณเรียกดูหน้าต่างๆ ด้วย Internet Explorer คุณสามารถคลิกเมนู Favorite ของ Internet Explorer เพื่อค้นหาบุ๊กมาร์กที่คุณสร้างไว้
    • ถ้าระบบวิธีใช้ทำงานอยู่ในโหมดสแตนด์อะโลน ให้คลิกแท็บ Bookmarks () เพื่อค้นหาบุ๊กมาร์กของคุณ
  3. หากต้องการลบบุ๊กมาร์ก ให้คลิกแท็บ Bookmarks () เลือกชื่อหัวข้อในบานหน้าต่างหัวข้อ และคลิก Delete Selected Bookmark () บนแถบเครื่องมือ
  4. หากต้องการลบบุ๊กมาร์กทั้งหมด ให้คลิกแท็บ Bookmarks () และคลิก Delete All Bookmarks () บนแถบเครื่องมือ
หัวข้อต้นทาง: การดูข้อมูลในระบบวิธีใช้
