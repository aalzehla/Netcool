Ver la información en el sistema de ayuda

Puede aprender a navegar y buscar en el sistema de ayuda utilizando los distintos botones de la barra de herramientas.

Acerca de esta tarea

  • Software de requisito previo para el sistema de ayuda
    Cuando configure el Sistema de ayuda de IBM® Eclipse, asegúrese de que la estación de trabajo, el sistema operativo, el navegador y el hardware cumplen los requisitos que se describen en este tema.
  • Navegar por el sistema de ayuda
    Utilice la interfaz gráfica de usuario proporcionada por el sistema de ayuda para localizar y ver documentación del producto.
  • Buscar en el sistema de ayuda
    Puede buscar en todos los temas del sistema de ayuda, buscar en un tema individual o una colección de temas, o crear un ámbito de búsqueda para buscar solamente en un subconjunto de la información. La vista Resultados de búsqueda puede mostrar hasta 500 resultados, ordenados por relevancia.
  • Establecer el ámbito adecuado
    Puede acotar el ámbito de trabajo estableciendo los criterios de filtro adecuado.
  • Buscar información con la vista Índice
    En este tema se describe cómo trabajar con la vista Índice para buscar temas en el sistema de ayuda.
  • Establecer favoritos en el sistema de ayuda
    Puede encontrar un tema más fácilmente estableciendo una marca de favorito y guardándola en la vista Favoritos del sistema de ayuda.
  • Ver la información en distintos idiomas
    Para ver la interfaz de usuario del sistema de ayuda en otro idioma, debe cambiar la configuración regional del sistema operativo para que se corresponda con el entorno local y el idioma que elija.
  • Imprimir en el sistema de ayuda
    El Sistema de ayuda de IBM Eclipse proporciona distintas formas de imprimir el contenido de la ayuda en el sistema.
  • Características de accesibilidad en el sistema de ayuda
    Las características de accesibilidad ayudan a las personas con discapacidades físicas, como las de movilidad restringida o visión limitada, o con otras necesidades especiales, a utilizar los productos de tecnologías de la información de manera satisfactoria.
  • Actualizar la información en el sistema de ayuda
    Puede actualizar regularmente el contenido de la documentación, si está disponible, descargándolo del sistema de ayuda. El contenido se actualizará periódicamente en el sitio web.
