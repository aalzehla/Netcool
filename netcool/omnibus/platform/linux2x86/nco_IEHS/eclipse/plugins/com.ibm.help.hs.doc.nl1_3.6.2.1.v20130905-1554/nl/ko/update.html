도움말 시스템에서 정보 업데이트

도움말 시스템에서 다운로드하여(지원되는 경우) 정기적으로 문서 컨텐츠를 업데이트할 수 있습니다. 웹 사이트의 컨텐츠는 정기적으로 업데이트됩니다.

이 태스크 정보

주: 업데이트 기능은 독립형 모드에서만 사용 가능합니다.

사용 가능한 업데이트 사항을 확인하고 다운로드하려면 다음 단계를 수행하십시오.

프로시저

  1. 도구 모음에서 업데이트()를 클릭하십시오. 설치된 문서 세트 목록이 표시됩니다.
    주: 정보 업데이트 사항이 없으면 업데이트 아이콘이 사용 불가능합니다. 시작 스크립트에서 -showUpdater 매개변수를 지정하여 업데이트 아이콘()을 컨텐츠 보기 도구 모음에 표시할 것인지 여부를 설정할 수 있습니다.
  2. 하단에서 다음을 클릭하여 기존 정보 콜렉션의 업데이트 사항 및 새 정보 콜렉션 목록을 표시하십시오. 사용 가능한 업데이트 사항이 업데이트 사이트에 저장되어 있습니다.
  3. 설치할 문서를 선택하고 업데이트 사항 설치를 클릭하십시오.
  4. 설치가 완료되면 완료를 클릭하십시오. 도움말 시스템은 자동으로 새로 고쳐지므로 도움말 시스템을 다시 시작하지 않아도 됩니다.
