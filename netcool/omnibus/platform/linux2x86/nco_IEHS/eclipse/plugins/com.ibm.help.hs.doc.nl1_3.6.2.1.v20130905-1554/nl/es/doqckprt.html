Realizar una impresión rápida

En este tema se muestra cómo imprimir uno o varios temas en el sistema de ayuda.

Acerca de esta tarea

En la vista Navegación, siga los pasos siguientes para imprimir uno o varios temas:

Procedimiento

  1. Seleccione un tema en la vista Navegación.
  2. Pulse Imprimir temas () en la barra de herramientas para visualizar el menú.
    • Si sólo desea imprimir el tema seleccionado, pulse Imprimir tema seleccionado.
    • Si desea imprimir el tema seleccionado y todos sus subtemas, pulse Imprimir tema seleccionado y todos los subtemas para abrir la ventana Vista previa de impresión.
    • Si desea cambiar la configuración de página antes de imprimir, seleccione Archivo > Configuración de página en la ventana de vista previa y modifique los valores.
  3. Pulse Imprimir en la ventana Imprimir.
    Restricción:

    MaxTopics y maxConnections son limitaciones de la impresión rápida. Son parámetros definidos para optimizar los recursos antes de iniciar el sistema.

    Cuando el número de temas que imprime supere el valor definido en maxTopics, visualizará una ventana Alerta.
    Figura 1. Ejemplo de ventana Alerta
    • Pulse Sí para continuar con la impresión, si está de acuerdo con el límite de maxTopics.
    • Pulse No para cancelar la impresión.
    Cuando el número de conexiones de impresora supere el valor definido en maxConnections, visualizará una ventana Error.
    Figura 2. Ejemplo de ventana Error
    • Pulse Aceptar. Inténtelo de nuevo más adelante.
