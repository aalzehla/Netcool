Entradas no teclado e navegação

É possível navegar no sistema de ajuda utilizando o teclado.

Navegando Através do Teclado

Utilize as seguintes combinações de teclas para navegar no sistema de ajuda através do teclado:

  • Na área de janela Tópico, vá para o próximo link e pressione Tab.
  • Para expandir e ocultar um nó na árvore de navegação, pressione as setas Direita e Esquerda.
  • Para mover-se para o próximo nó de tópico, pressione a seta para Baixo.
  • Para mover-se para o nó de tópico anterior, pressione a seta para Cima.
  • Para ir para o próximo link, botão ou nó de tópico de dentro de uma visualização, pressione a tecla Tab.
  • Para rolar completamente para cima ou para baixo em uma área de janela, pressione Home ou End.
  • Para voltar, pressione Alt+Seta Esquerda, para avançar, pressione Alt+Seta Direita.
  • Para ir para a próxima área de janela, pressione F6.
  • Para mover-se para a área de janela anterior, pressione Shift+F6.
  • Para imprimir a área de janela ativa, pressione Ctrl+P.
  • Para abrir o menu pop-up, pressione Alt+M.
  • Para mover-se para o item anterior no menu pop-up, pressione a seta para Cima.
  • Para mover-se para o próximo item no menu pop-up, pressione a seta para Baixo.
  • Para selecionar um item no menu pop-up, vá para o item e pressione Enter.
  • Para fechar o menu pop-up, pressione Esc.
  • Para mover-se para a visualização de conteúdo, pressione Alt+C. (então pressione Enter se estiver utilizando o IE)
  • Para mover-se para a visualização dos Resultados de Busca, pressione Alt+R. (então pressione Enter se estiver utilizando o IE)
  • Para mover-se para a visualização do Índice, pressione Alt+I. (então pressione Enter se estiver utilizando o IE)
  • Para ir para a visualização de Favoritos, pressione Alt+Z. (então pressione Enter se estiver utilizando o IE)

Para Usuários do JAWS 7

  • Para expandir um nó na árvore de navegação, pressione Alt+O. JAWS diz "Alt O".
  • Para ocultar um nó na árvore de navegação, pressione Alt+B. JAWS diz "Alt B".
  • Para mover-se para o próximo nó de tópico na árvore, pressione a Ctrl+Shift+Seta para Baixo.
  • Para mover-se para o nó de tópico anterior na árvore, pressione Ctrl+Shift+Seta para Cima.
  • Para rolar completamente para cima ou para baixo em uma área de janela, pressione PageUp ou PageDown.
  • Para mover-se para o item anterior no menu pop-up, pressione Ctrl+Shift+Seta para cima.
  • Para mover-se para o próximo item no menu pop-up, pressione Ctrl+Shift+Seta para Baixo.
  • Para selecionar um item no menu pop-up, vá para o item e pressione Enter.
  • Para expandir uma lista de documentação no atualizador, pressione Alt+O. JAWS diz "Alt O".
  • Para ocultar uma lista de documentação no atualizador, pressione Alt+B. JAWS diz "Alt B".
  • Para alternar entre visualização de Conteúdos, visualização de Resultados de Busca, visualização de Índice e de Favoritos, mova-se para a Seleção da estrutura de visualizações de navegação e então utilizar as combinações de teclas descritas na seção anterior.

Para outras combinações de teclado conforme documentado na Navegação pela seção teclado.

Para usuários do Home Page Reader 3.4.2

  • Para expandir e reduzir um nó na estrutura em árvore, pressione Enter.
  • Para abrir o menu pop-up, pressione Ctrl+L, pressione End, em seguida, pressione Enter.
  • Para ir para o item anterior no menu pop-up, pressione Shift+Tab.
  • Para ir para o próximo item no menu pop-up, pressione Tab.
  • Para selecionar um item no menu pop-up, vá para o item e pressione Enter.
  • Para fechar o menu pop-up, pressione Alt+Shift+H.
  • Para alternar entre visualização de Conteúdos, visualização de Resultados de Busca, visualização de Índice e de Favoritos, mova-se para a Seleção da estrutura de visualizações de navegação e então:
    • Para ir para a visualização Conteúdo, pressione Alt+Shift+C.
    • Para ir para a visualização Resultados da Procura, pressione Alt+Shift+R.
    • Para ir para a visualização Índice, pressione Alt+Shift+I.
    • Para ir para a visualização Favoritos, pressione Alt+Shift+Z.

Para outras combinações de teclado conforme documentado na Navegação pela seção teclado.
