Nach exakten Wörtern oder Ausdrücken suchen

Sie können in der Hilfefunktion nach exakten Wörtern oder Ausdrücken suchen, indem Sie eine genaue Zeichenfolge eingeben.

Warum und wann dieser Vorgang ausgeführt wird

Wenn Sie die Hilfefunktion durchsuchen, können Sie einige Wörter oder Ausdrücke als exakte Zeichenfolge eingeben, indem Sie sie in Anführungszeichen setzen.

Beispiel: Wenn Sie im Feld "log file" anstelle von log file in das Feld Suchen eingeben, suchen Sie nach der Zeichenfolge log file. Ohne die Anführungszeichen wird der Suchausdruck so interpretiert, dass in den Themen nach den Wörtern log und file gesucht wird.

Anmerkung: Nur in Englisch und in Deutsch berücksichtigt die Suchmaschine auch Wortstämme anderer Formen eines einzelnen Suchbegriffs. Bei einer Suchoperation für das Wort challenge beispielsweise wird auch das Wort challenging gefunden. Schließen Sie den Begriff, den Sie in das Feld Suchen eingeben, in doppelte Anführungszeichen ein, wenn die Suchergebnisse solche Variationen des Begriffs nicht enthalten sollen.
