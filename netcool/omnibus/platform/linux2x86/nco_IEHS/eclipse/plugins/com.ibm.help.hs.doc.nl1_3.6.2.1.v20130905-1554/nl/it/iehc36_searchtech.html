Tecniche di ricerca avanzate

È possibile eseguire ricerche più avanzate utilizzando le virgolette, i caratteri jolly o gli operatori booleani al fine di restituire un serie più ridotta di risultati.

  • Quando si effettua la ricerca nel sistema della guida, è possibile identificare alcune parole o frasi come corrispondenza esatta, racchiudendole tra virgolette.

    Ad esempio, quando si immette "file log" nel campo Ricerca, viene effettuata la ricerca di tutte le ricorrenze della stringa file log. Se non vengono utilizzate le virgolette, vengono restituiti gli argomenti contenenti istanze sia della parola file che della parola log.

  • È possibile utilizzare i caratteri jolly, quali l'asterisco (*) e il punto interrogativo (?) , nella ricerca per ampliarne l'ambito.
    • Utilizzare l'asterisco (*) per più caratteri sconosciuti o variabili nella query di ricerca. Ad esempio, se si immette par*, vengono restituiti gli argomenti che contengono le parole parzialmente, partecipare, parziale ed altre che iniziano con par.
      Nota: Esiste una limitazione per la stringa di ricerca: quando si utilizza un termine di ricerca con un asterisco, esso deve essere preceduto da almeno tre caratteri, allo stesso modo un termine di ricerca senza un carattere jolly deve contenere almeno tre caratteri.
    • Utilizzare il punto interrogativo per un carattere singolo nella query di ricerca. Ad esempio, se si immette par?, vengono restituiti gli argomenti che contengono la parola pari, ma non parziale o parzialmente.
    Nota: quando si utilizza un termine di ricerca con un asterisco (*), esso deve essere preceduto da almeno tre caratteri
  • È possibile utilizzare gli operatori booleani AND, OR e NOT per rendere più specifica la query. Ad esempio:
    database AND "file log"
    Restringe i risultati della ricerca agli argomenti che contengono sia il termine database che la frase file log.
    database OR "file log"
    Espande i risultati della ricerca agli argomenti che contengono o il termine database o la frase file log.
    database NOT "file log"
    Restringe i risultati della ricerca agli argomenti che contengono il termine database ma non la frase file log.
    database OR "data base" NOT "file log"
    Ricerca gli argomenti che contengono il termine database o la frase data base, ma non la frase file log.
