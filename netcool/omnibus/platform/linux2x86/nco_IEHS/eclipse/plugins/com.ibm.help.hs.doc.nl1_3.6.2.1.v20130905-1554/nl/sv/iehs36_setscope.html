Ange omfång

Du kan begränsa arbetsomfånget genom att ange ett filtervillkor.

Du kan filtrera innehåll i informationscentret med hjälp av olika villkor, som plattform, produktnamn och version. Om du vill arbeta med en mindre uppsättning med avsnitt kan du filtrera innehåll.

De avsnitt som inte uppfyller villkoren eller som inte tillhör avsnittsuppsättningen för omfånget visas inte i navigeringsträdet när du synkroniserar visningen.

  • Använda det befintliga omfånget
    Du kan använda det befintliga omfånget till att begränsa arbetsomfånget.
  • Ändra det befintliga omfånget
    Om du vill kan du ändra det ursprungliga omfånget.
  • Skapa ett omfång
    Om du vill kan du skapa ett omfång.
