Προηγμένες τεχνικές αναζήτησης

Μπορείτε να εκτελέσετε πιο σύνθετες αναζητήσεις χρησιμοποιώντας διπλά εισαγωγικά, χαρακτήρες υποκατάστασης ή λογικούς τελεστές για να περιορίσετε τον αριθμό των θεμάτων που περιλαμβάνονται στα αποτελέσματα της αναζήτησης.

  • Όταν κάνετε αναζήτηση στο σύστημα βοήθειας, μπορείτε να περικλείσετε μια φράση σε διπλά εισαγωγικά ώστε να γίνει αναζήτηση της ακριβούς φράσης.

    Για παράδειγμα, εάν καταχωρήσετε τη φράση "αρχείο καταγραφής" στο πεδίο Αναζήτηση, θα αναζητηθούν όλες οι εμφανίσεις της φράσης αρχείο καταγραφής. Εάν παραλείψετε τα διπλά εισαγωγικά, θα επιστραφούν τα θέματα που περιέχουν και τις δύο λέξεις αρχείο και καταγραφής.

  • Μπορείτε να χρησιμοποιήσετε χαρακτήρες υποκατάστασης όπως π.χ. ο αστερίσκος (*) και το λατινικό ερωτηματικό (?) στην αναζήτηση για να επεκτείνετε την εμβέλεια της αναζήτησης.
    • Χρησιμοποιεήστε τον αστερίσκο (*) για περισσότερους από έναν άγνωστους ή μεταβλητούς χαρακτήρες στο ερώτημα αναζήτησης. Για παράδειγμα, αν καταχωρήσετε τους χαρακτήρες συμ*, θα επιστραφούν τα θέματα που περιέχουν τις λέξεις συμμετοχή, συμμετέχω, συμμετρικό και άλλες λέξεις που αρχίζουν από συμ.
      Σημείωση: Υπάρχει ένας περιορισμός για τους όρους αναζήτησης: όταν χρησιμοποιείτε έναν όρο αναζήτησης με αστερίσκο (*), θα πρέπει να υπάρχουν τουλάχιστον τρεις χαρακτήρες πριν τον αστερίσκο. Παρομοίως, ένας όρος αναζήτησης χωρίς χαρακτήρα υποκατάστασης πρέπει να αποτελείται από τουλάχιστον τρεις χαρακτήρες.
    • Χρησιμοποιήστε ένα λατινικό ερωτηματικό (?) για ένα μεμονωμένο χαρακτήρα στο ερώτημα αναζήτησης. Για παράδειγμα, αν καταχωρήσετε τους χαρακτήρες παρ?, θα επιστραφούν τα θέματα που περιέχουν τη λέξη παρά, αλλά όχι τα θέματα που περιέχουν τις λέξεις παράρτημα ή παράκαμψη.
    Σημείωση: Όταν χρησιμοποιείτε έναν όρο αναζήτησης με αστερίσκο (*), θα πρέπει να υπάρχουν τουλάχιστον τρεις χαρακτήρες πριν τον αστερίσκο.
  • Μπορείτε να χρησιμοποιήσετε τους λογικούς τελεστές AND, OR και NOT για να κάνετε το ερώτημα αναζήτησης πιο συγκεκριμένο. Για παράδειγμα:
    "βάση δεδομένων" AND "αρχείο καταγραφής"
    Περιορίζει την αναζήτηση έτσι ώστε στα αποτελέσματά της να περιλαμβάνονται μόνο τα θέματα που περιέχουν τόσο τη φράση βάση δεδομένων όσο και τη φράση αρχείο καταγραφής.
    "βάση δεδομένων" OR "αρχείο καταγραφής"
    Διευρύνει την αναζήτηση έτσι ώστε στα αποτελέσματά της να περιλαμβάνονται τα θέματα που περιέχουν είτε τη φράση βάση δεδομένων είτε τη φράση αρχείο καταγραφής.
    "βάση δεδομένων" NOT "αρχείο καταγραφής"
    Περιορίζει την αναζήτηση έτσι ώστε στα αποτελέσματά της να περιλαμβάνονται μόνο τα θέματα που περιέχουν τη φράση βάση δεδομένων αλλά όχι τη φράση αρχείο καταγραφής.
    database OR "βάση δεδομένων" NOT "αρχείο καταγραφής"
    Αναζητούνται τα θέματα που περιέχουν είτε τον όρο database είτε τη φράση βάση δεδομένων, αλλά όχι τη φράση αρχείο καταγραφής.
