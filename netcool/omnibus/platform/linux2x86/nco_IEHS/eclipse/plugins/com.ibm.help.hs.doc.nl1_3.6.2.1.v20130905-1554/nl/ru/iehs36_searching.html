Поиск в справочной системе

Можно искать во всех разделах справочной системы, конкретном разделе или наборе разделов; можно создать область поиска для поиска только в определенном подмножестве информации. На панели Результаты поиска показывается до 500 результатов, упорядоченных по степени соответствия.

Об этой задаче

Для переключения между панелью Оглавление и панелью Результаты поиска нажмите вкладку Оглавление () или Результаты поиска ().

Прим.: Поиск выполняется без учета регистра символов.

Процедура

  • Для поиска во всех разделах справочной системы введите запрос в поле Поиск и нажмите кнопку Выполнить. Можно также создать область поиска для регулярного поиска в определенном подмножестве разделов.
  • Для поиска в определенном разделе или наборе разделов выберите их в дереве навигации и нажмите Разделы для поиска () на панели инструментов панели Оглавление. Выберите Поиск в выбранном разделе или Поиск в выбранном разделе и его подразделах, введите свой запрос в поле Поиск и нажмите OK.

Результаты

На панели Результаты поиска будут показаны разделы, удовлетворяющие поисковому запросу. Для просмотра раздела выберите его в списке результатов поиска.

Разделы с одинаковым заголовком и аннотацией считаются похожими страницами. Если один из разделов в результатах поиска имеет похожие страницы, внизу раздела зеленым цветом показывается общее число похожих страниц. Также можно нажать тег Похожие страницы, чтобы развернуть и просмотреть все похожие страницы.

Дальнейшие действия

  1. Откройте вкладку Результаты поиска () для просмотра панели результатов поиска. По умолчанию результаты поиска не сгруппированы.
  2. Нажмите Показать категории результатов () на панели инструментов для группировки результатов поиска.
Для управления результатами поиска можно использовать значки панели инструментов:
  • Для показа/скрытия обзора под каждым результатом поиска нажмите Показать описания результатов ().
  • Для включения/выключения выделения нажмите Выделение ().
  • Приемы расширенного поиска
    Поддерживается расширенный поиск с помощью цитат, шаблонов и булевских операторов, позволяющий уменьшить количество результатов поиска
  • Быстрый поиск
    С помощью функции Быстрый поиск можно выполнять поиск в разделах и наборах разделов на панели навигации.
