Навигация с помощью клавиатуры

Для навигации по справочной системе можно применять следующие клавиши:

  • Для перехода к следующей ссылке на панели информации нажмите клавишу Tab.
  • Для развертывания или свертывания узла в дереве навигации нажмите клавишу со стрелкой вправо или влево.
  • Для перемещения к следующему узлу раздела нажмите клавишу со стрелкой вниз.
  • Для перемещения к предыдущему узлу раздела нажмите клавишу со стрелкой вверх.
  • Для перехода к следующей ссылке, кнопке или узлу раздела в окне справочной системы нажмите клавишу Tab.
  • Для прокрутки всего содержимого экрана вверх или вниз нажмите Home или End.
  • Для перемещения назад нажмите клавиши Alt+стрелка влево; для перемещения вперед - клавиши Alt+стрелка вправо.
  • Для перехода к следующей панели нажмите клавишу F6.
  • Для перехода к предыдущей панели нажмите клавиши Shift+F6.
  • Для печати содержимого активной панели нажмите Ctrl+P.
  • Для открытия всплывающего меню нажмите клавиши Alt+M.
  • Для перемещения к предыдущему пункту всплывающего меню нажмите клавишу со стрелкой вверх.
  • Для перемещения к следующему пункту всплывающего меню нажмите клавишу со стрелкой вниз.
  • Для выбора пункта всплывающего меню выделите этот пункт и нажмите клавишу Enter.
  • Для закрытия всплывающего меню нажмите Esc.
  • Для перехода к панели содержимого нажмите Alt+C (в IE после этого нажмите Enter).
  • Для перехода к панели результатов поиска нажмите Alt+R (в IE после этого нажмите Enter).
  • Для перехода к панели указателя нажмите Alt+I (в IE после этого нажмите Enter).
  • Для перехода к панели закладок нажмите Alt+Z (в IE после этого нажмите Enter).

Информация для пользователей JAWS 7

  • Для того чтобы развернуть узел в дереве навигации, нажмите Alt+O. JAWS произнесет "Alt O".
  • Для того чтобы свернуть узел в дереве навигации, нажмите Alt+B. JAWS произнесет "Alt B".
  • Для перемещения к следующему узлу раздела в дереве нажмите Ctrl+Shift+стрелка вниз.
  • Для перемещения к предыдущему узлу раздела в дереве нажмите Ctrl+Shift+стрелка вверх.
  • Для прокрутки содержимого до конца панели используйте клавиши PageUp или PageDown.
  • Для перехода к предыдущему пункту всплывающего меню нажмите Ctrl+Shift+стрелка вверх.
  • Для перехода к следующему пункту всплывающего меню нажмите Ctrl+Shift+стрелка вниз.
  • Для выбора пункта всплывающего меню выделите этот пункт и нажмите клавишу Enter.
  • Для того чтобы развернуть узел в дереве навигации, нажмите Alt+O. JAWS произнесет "Alt O".
  • Для того чтобы свернуть список документации в модуле обновления нажмите Alt+B. JAWS произнесет "Alt B".
  • Для переключения между панелями Оглавление, Результаты поиска, Индекс и Закладки перейдите к фрейму Выбранные панели навигации и используйте сочетания клавиш, указанные в предыдущем разделе.

Остальные клавиши работают так, как описано в разделе Навигация с помощью клавиатуры.

Для пользователей Home Page Reader 3.4.1

  • Для того чтобы развернуть или свернуть узел дерева, нажмите Enter.
  • Для того чтобы открыть всплывающее меню, нажмите Ctrl+L, затем - End, а затем - Enter.
  • Для перехода к предыдущему пункту всплывающего меню нажмите Shift+Tab.
  • Для перехода к следующему пункту всплывающего меню нажмите Tab.
  • Для выбора пункта всплывающего меню выделите этот пункт и нажмите клавишу Enter.
  • Для того чтобы закрыть всплывающее меню, нажмите Alt+Shift+H.
  • Для переключения между панелями Оглавление, Результаты поиска, Индекс и Закладки перейдите к фрейму Выбранные панели навигации, а затем:
    • Для перехода к панели Оглавление нажмите Alt+Shift+C.
    • Для перехода к панели Результаты поиска нажмите Alt+Shift+R.
    • Для перехода к панели Индекс нажмите Alt+Shift+I.
    • Для перехода к панели Закладки нажмите Alt+Shift+Z.

Остальные клавиши работают так, как описано в разделе Навигация с помощью клавиатуры.
