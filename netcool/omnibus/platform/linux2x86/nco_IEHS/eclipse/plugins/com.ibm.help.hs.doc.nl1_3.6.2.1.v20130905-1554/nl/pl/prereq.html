Wymagania wstępne dla systemu pomocy

Przed zainstalowaniem komponentu IBM® Eclipse Help System należy sprawdzić, czy stacja robocza, system operacyjny, przeglądarka i sprzęt spełniają wymagania opisane w tym temacie.

Wymagania dotyczące przeglądarki

W celu pracy z komponentem IBM Eclipse Help System można zainstalować przeglądarkę Microsoft® Internet Explorer 6.0 lub 7.0 albo przeglądarkę Firefox 2.0 lub 3.0. Aby umożliwić korzystanie ze wszystkich funkcji tej wersji komponentu IBM Eclipse Help System, konieczne jest włączenie informacji cookie i skryptów JavaScript™ w przeglądarce oraz wyłączenie funkcji blokowania okien wywoływanych.

Wymagania dotyczące systemu operacyjnego

Aby móc korzystać z komponentu IBM Eclipse Help System, należy zainstalować środowisko IBM Java™ Runtime Environment (JRE) 6.0. Szczegółowe wymagania zawiera poniższa tabela.

Dostosowywanie systemu pomocy

System pomocy można skonfigurować za pomocą pliku plugin_customization.ini. Ten plik znajduje się we wtyczce głównego produktu. Ustawić można go w skrypcie uruchamiania.

Parametry konfiguracyjne można także zmienić, edytując plik org.eclipse.help.base_<numer wersji/preferences.ini w katalogu eclipse/plugins, chociaż stanowczo zalecane jest użycie konfiguracji domyślnej.

System pomocy można dostosować, modyfikując plik preferences.ini.

Uwaga: Nie należy modyfikować parametru always_external_browser, ponieważ w przeciwnym razie system pomocy może nie działać poprawnie.
always_external_browser
W systemach operacyjnych Microsoft Windows wartość domyślna tego parametru to always_external_browser=true. Po uruchomieniu systemu pomocy w trybie autonomicznym to ustawienie spowoduje otwarcie domyślnej przeglądarki systemu operacyjnego zamiast wbudowanej przeglądarki systemu pomocy. W przypadku systemów operacyjnych innych niż Windows nie ma potrzeby określania ustawienia always_external_browser=true.
