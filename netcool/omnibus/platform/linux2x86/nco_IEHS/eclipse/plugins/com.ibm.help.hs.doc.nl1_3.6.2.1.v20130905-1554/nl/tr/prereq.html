Yardım sistemine ilişkin önkoşul gereksinimleri

IBM® Eclipse Help System uygulamasını kurarken, iş istasyonunuzun, işletim sisteminizin, tarayıcınızın ve donanımınızın bu konuda açıklanan gereksinimleri karşıladığından emin olmanız gerekir.

Tarayıcı gereksinimleri

IBM Eclipse Help System ile çalışmak için Microsoft® Internet Explorer Sürüm 6.0 ya da 7.0 ya da Firefox 2.0 ya da 3.0 düzeyini kurabilirsiniz. IBM Eclipse Help System olanağının bu sürümünün tüm işlevlerinin kullanılabilir olduğundan emin olmak için, tanımlama bilgilerini ve tarayıcınızdaki JavaScript™'i etkinleştirmeli ve açılan pencereleri engelleme işlevini devre dışı bırakmalısınız.

İşletim sistemi ortamı gereksinimleri

IBM Eclipse Help System ile çalışmak için, IBM Java™ Runtime Environment (JRE) sürüm 6.0'yı kurmalısınız. Ayrıntılı gereksinimler için aşağıdaki tabloya bakın.

Desteklenen özelleştirme

Yardım sistemini plugin_customization.ini dosyasıyla yapılandırabilirsiniz. Bu, ana ürün eklentisinin altındadır. Başlangıç komutunda bunu ayarlayabilirsiniz.

Yapılanış parametrelerini eclipse/plugins dizinindeki org.eclipse.help.base_<sürüm numarası>/preferences.ini dosyasını düzenleyerek yapılandırabilirsiniz; ancak, varsayılan yapılanışı kullanmanız önerilir.

Yardım sisteminizi, preferences.ini dosyasını düzenleyerek özelleştirebilirsiniz.

Not: Aşağıdaki parametreleri değiştirmeyin: always_external_browser; aksi takdirde yardım sistemi düzgün çalışmayabilir.
always_external_browser
Microsoft Windows işletim sistemlerinde, bu parametrenin varsayılan yapılanışı always_external_browser=true değerindedir. Yardım sistemini bağımsız modda başlattığınızda, bu ayar, yardım sistemindeki yerleşik tarayıcı yerine işletim sisteminin varsayılan tarayıcısını açmanıza yardımcı olur. Windows dışındaki işletim sistemlerinde always_external_browser=true parametresini ayarlamanıza gerek yoktur.
