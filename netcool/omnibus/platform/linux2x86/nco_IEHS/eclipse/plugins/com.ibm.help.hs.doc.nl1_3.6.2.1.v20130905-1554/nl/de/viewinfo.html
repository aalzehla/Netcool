Informationen in der Hilfefunktion anzeigen

Hier finden Sie Informationen zum Navigieren in der Hilfefunktion und zum Durchsuchen der Hilfefunktion mithilfe der verschiedenen Schaltflächen in der Symbolleiste.

Informationen zu diesem Vorgang

  • Vorausgesetzte Anforderungen für die Hilfefunktion
    Beim Einrichten von IBM® Eclipse Help System müssen Sie sicherstellen, dass Ihre Workstation, Ihr Betriebssystem, Ihr Browser sowie Ihre Hardware die in diesem Thema beschriebenen Anforderungen erfüllen.
  • In der Hilfefunktion navigieren
    Sie können die Produktdokumentation mithilfe der Benutzerschnittstelle der Hilfefunktion durchsuchen und anzeigen.
  • Hilfefunktion durchsuchen
    Sie können in der Hilfefunktion wahlweise alle Themen, einzelne Themen oder eine Themensammlung durchsuchen. Zusätzlich können Sie einen Suchbereich erstellen und so nur einen Teil der angebotenen Informationen durchsuchen. In der Sicht 'Suchergebnisse' werden maximal 500 Ergebnisse sortiert nach deren Relevanz angezeigt.
  • Geeigneten Bereich festlegen
    Sie können den Arbeitsbereich mithilfe von Filterkriterien eingrenzen.
  • Informationen mit der Indexsicht suchen
    In diesem Thema wird das Arbeiten mit der Indexsicht zum Suchen nach Themen in der Hilfefunktion beschrieben.
  • Lesezeichen in der Hilfefunktion setzen
    Sie können ein Thema schneller finden, wenn Sie ein Lesezeichen setzen und es in der Lesezeichensicht des Hilfesystems speichern.
  • Informationen in unterschiedlichen Sprachen anzeigen
    Wenn Sie die Benutzerschnittstelle der Hilfefunktion in einer anderen Sprache anzeigen möchten, müssen Sie die Ländereinstellungen Ihres Betriebssystems in die gewünschte Ländereinstellung und Sprache ändern.
  • In der Hilfefunktion drucken
    IBM Eclipse Help System stellt Ihnen unterschiedliche Möglichkeiten zum Drucken des Inhalts der Hilfe in dem System bereit.
  • Eingabehilfefunktion und Funktion zur behindertengerechten Bedienung in der Hilfefunktion
    Die Eingabehilfefunktion unterstützt Benutzer mit körperlichen Behinderungen wie z. B. eingeschränkter Beweglichkeit, eingeschränktem Sehvermögen oder anderen Einschränkungen beim erfolgreichen Einsatz von IT-Produkten.
  • Informationen in der Hilfefunktion aktualisieren
    Sie können den Dokumentationsinhalt regelmäßig aktualisieren, sofern er verfügbar ist; Sie können ihn von der Hilfefunktion herunterladen. Der Inhalt wird auf der Website in bestimmten zeitlichen Abständen aktualisiert.
