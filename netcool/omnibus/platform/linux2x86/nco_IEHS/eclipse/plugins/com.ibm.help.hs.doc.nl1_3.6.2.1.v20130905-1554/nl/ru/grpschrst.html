Просмотр результатов поиска

В справочной системе можно просмотреть результаты поиска по заданному критерию.

Если разделы документации помечены, как отвечающие различным критериям, таким как версия, имя продукта, платформа или категория, то результаты поиска можно просматривать в соответствии с такими критериями.

  1. Щелкните на вкладке Результаты поиска (). По умолчанию будут показаны результаты поиска, не разбитые на группы.
  2. Нажмите кнопку Сгруппировать результаты поиска () на панели инструментов.
  3. Выберите критерий группировки в меню.
    • GroupByNone

      Когда выбрано это значение по умолчанию, результаты поиска не организованы в группы.

    • GroupByVersion
    • GroupByProdname
    • GroupByPlatform
    • Показать категории результатов

      Объединяет результаты поиска в группы по названию книги.

  4. Результаты поиска будут показаны в соответствии с выбранным критерием.
Прим.: Для добавления или удаления описаний результатов на панели результатов поиска нажмите Показать описания результатов ().
