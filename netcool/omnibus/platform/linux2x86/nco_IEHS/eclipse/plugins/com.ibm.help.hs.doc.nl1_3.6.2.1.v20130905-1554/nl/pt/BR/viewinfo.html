Visualizando Informações no Sistema de Ajuda

É possível aprender como navegar e procurar no sistema de ajuda utilizando os diversos botões na barra de ferramentas.

Sobre Esta Tarefa

  • Acessibilidade e Atalhos do Teclado no Sistema de Ajuda
    Os recursos de acessibilidade ajudam os usuários que possuem deficiências físicas, como mobilidade restrita ou visão limitada, a trabalhar com os produtos de software corretamente.
  • Pré-requisitos para o sistema de ajuda
    Quando você configurar o IBM® Eclipse Help System, certifique-se de que sua estação de trabalho, seu sistema operacional, navegador e hardware atendam aos requisitos descritos neste tópico.
  • Navegando no Sistema de Ajuda
    Utilize a interface gráfica com o usuário fornecida pelo sistema de ajuda para localizar e visualizar a documentação do produto.
  • Procurando no Sistema de Ajuda
    É possível procurar todos os tópicos no sistema de ajuda, procurar um tópico individual ou uma coleção de tópicos ou criar um escopo de procura para procurar apenas um subconjunto das informações. A visualização Resultados da Procura pode exibir até 500 resultados, classificados por relevância.
  • Configurando o Escopo Adequado
    É possível limitar seu escopo de trabalho pela configuração dos critérios de filtro adequados.
  • Localizando Informações com a Visualização do Índice
    Este tópico descreve como trabalhar com a visualização do Índice para localizar tópicos no sistema de ajuda.
  • Configuração de indicadores no sistema de ajuda
    É possível localizar um tópico mais facilmente configurando um marcador e salvando-o na visualização Favoritos do sistema de ajuda.
  • Visualizando Informações em Diferentes Idiomas
    Para visualizar a interface de usuário do sistema de ajuda em outro idioma, você deve alterar as configurações regionais de seu sistema operacional de acordo com o local e idioma de sua preferência.
  • Imprimindo no sistema de ajuda
    O IBM Eclipse Help System oferece a você diferentes formas de imprimir os conteúdos da ajuda no sistema.
  • Atualizando informações no sistema de ajuda
    É possível atualizar regularmente os conteúdos de documentos, se disponíveis, fazendo o download de tais documentos no sistema de ajuda. Os conteúdos serão periodicamente atualizados no Web site.
