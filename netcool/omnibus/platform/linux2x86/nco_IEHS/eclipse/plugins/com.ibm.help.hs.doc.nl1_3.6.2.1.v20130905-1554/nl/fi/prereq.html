Ohjetoiminnon ennakkoedellytykset

Kun otat IBM Eclipse -ohjetoiminnon käyttöön, varmista, että työasema, käyttöjärjestelmä, selain ja laitteisto vastaavat tässä aiheessa kuvattuja edellytyksiä.

Selainta koskevat edellytykset

IBM Eclipse -ohjetoiminnon käyttö edellyttää, että asennettuna on Microsoft Internet Explorer -ohjelman versio 6.0 tai 7.0 tai Firefox-ohjelman versio 2.0 tai 3.0. Evästeet ja JavaScript on otettava käyttöön selaimessa ja kohoikkunoiden esto poistettava käytöstä, jotta IBM Eclipse -ohjetoiminnon tämän version kaikki toiminnot ovat käytettävissä.

Käyttöjärjestelmää koskevat edellytykset

IBM Eclipse -ohjetoiminnon käyttö edellyttää, että asennettuna on Java Runtime Environment (JRE) -käyttöympäristön versio 6.0. Katso tarkemmat vaatimukset seuraavasta taulukosta.

Tuettu mukautus

Voit määrittää ohjetoiminnon kokoonpanon tiedoston plugin_customization.ini avulla. Se sijaitsee päätuotteen lisäosan alapuolella. Voit asettaa sen aloituskomentosarjassa.

Voit myös muuttaa kokoonpanoparametreja muokkaamalla hakemistossa eclipse/plugins olevaa tiedostoa org.eclipse.help.base_<version numero>/preferences.ini, vaikka oletuskokoonpanon säilyttäminen on suositeltavaa.

Voit mukauttaa ohjetoimintoa muokkaamalla preferences.ini-tiedostoa.

Huomautus: Älä muuta parametria always_external_browser. Jos sitä muutetaan, ohjetoiminto ei ehkä toimi oikein.
always_external_browser
Microsoft Windows -käyttöjärjestelmissä tämän parametrin oletusasetus on always_external_browser=true. Kun ohjetoiminto aloitetaan erillistilassa, tämä asetus auttaa aloittamaan käyttöjärjestelmän oletusselaimen ohjetoiminnossa valmiina olevan selaimen asemesta. Asetusta always_external_browser=true ei tarvitse määrittää muissa kuin Windows-käyttöjärjestelmissä.
