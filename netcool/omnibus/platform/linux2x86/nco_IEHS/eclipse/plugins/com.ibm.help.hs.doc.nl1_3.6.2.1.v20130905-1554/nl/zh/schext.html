搜索精确匹配的词或短语

您可以在帮助系统中搜索精确匹配的词或短语（作为精确匹配的字符串）。

在帮助系统中进行搜索时，您可以通过将某些词或短语加上双引号，从而将其标识为精确匹配的字符串。

例如，当在搜索字段中输入 "log file" 而不是 log file 时，即搜索字符串 log file。如果没有双引号，那么就认为是在主题中搜索 log 和 file 这两个词。

注： 仅在英语和德语版本中，搜索引擎才会搜索单个搜索词的“词干”的其他变化形式。 例如，搜索词 challenge 还将查找词 challenging。如果不希望搜索结果包括搜索项的这种变体，请将您在搜索字段中输入的搜索项用双引号括起来。
