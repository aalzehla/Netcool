Visualizando Informações em Diferentes Idiomas

Para visualizar a interface de usuário do sistema de ajuda em outro idioma, você deve alterar as configurações regionais de seu sistema operacional de acordo com o local e idioma de sua preferência.

O sistema exibirá a documentação traduzida que foi fornecida pelo proprietário do produto. Se o proprietário do produto não forneceu a documentação no idioma escolhido, mas forneceu a versão no idioma inglês, o sistema exibirá a versão no idioma inglês, por padrão, mesmo que a opção de idioma do navegador esteja definida para outro idioma.
