Informationen in der Hilfefunktion aktualisieren

Sie können den Dokumentationsinhalt regelmäßig aktualisieren, sofern er verfügbar ist; Sie können ihn von der Hilfefunktion herunterladen. Der Inhalt wird auf der Website in bestimmten zeitlichen Abständen aktualisiert.

Informationen zu diesem Vorgang

Anmerkung: Die Aktualisierungsfunktion ist nur im Standalone-Modus verfügbar.

Sie können prüfen, ob Aktualisierungen vorhanden sind und verfügbare Aktualisierungen herunterladen, indem Sie die folgenden Schritte ausführen:

Vorgehensweise

  1. Klicken Sie in der Symbolleiste auf Aktualisieren (). Daraufhin wird eine Liste installierter Dokumentationsgruppen angezeigt.
    Anmerkung: Das Symbol 'Aktualisieren' ist nicht verfügbar, wenn keine Informationsaktualisierungen verfügbar sind. Es ist außerdem möglich, das Symbol 'Aktualisieren' () in der Symbolleiste der Inhaltssicht ein- oder auszublenden; hierfür muss der Parameter -showUpdater im Startscript angegeben werden.
  2. Klicken Sie unten auf die Schaltfläche Weiter, um eine Liste der Aktualisierungen für vorhandene Informationssammlungen und für neue Informationssammlungen anzuzeigen. Die verfügbaren Aktualisierungen werden auf einer Update-Site gespeichert.
  3. Wählen Sie die Dokumentation aus, die Sie installieren möchten, und klicken Sie anschließend auf Aktualisierungen installieren.
  4. Klicken Sie auf Fertig stellen, wenn die Installation abgeschlossen ist. Sie müssen die Hilfefunktion nicht erneut starten, da Sie automatisch aktualisiert angezeigt wird.
