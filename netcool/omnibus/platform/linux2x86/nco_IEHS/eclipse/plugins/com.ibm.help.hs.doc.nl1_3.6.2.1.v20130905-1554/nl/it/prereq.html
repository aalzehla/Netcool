Requisiti per il sistema della guida

Quando si configura IBM® Eclipse Help System, assicurarsi che la stazione di lavoro, il sistema operativo, il browser e l'hardware soddisfino i requisiti descritti in questo argomento.

Requisiti del browser

Per utilizzare IBM Eclipse Help System, è possibile installare Microsoft® Internet Explorer Versione 6.0 o 7.0 oppure Firefox 2.0 o 3.0. Per assicurarsi che tutte le funzioni di questa versione di IBM Eclipse Help System siano utilizzabili, è necessario abilitare i cookie e JavaScript™ nel browser e disabilitare la funzione del browser che blocca le finestre a comparsa (pop-up).

Personalizzazione supportata

È possibile configurare il sistema della guida con il file plugin_customization.ini. È contenuto sotto il plug-in del prodotto principale. È possibile impostarlo nello script di avvio.

Inoltre, è possibile modificare i parametri di configurazione, modificando il file org.eclipse.help.base_<numero versione/preferences.ini nella directory eclipse/plugins, ma si consiglia di utilizzare la configurazione predefinita.

È possibile personalizzare il sistema della guida modificando il file preferences.ini.

Nota: non modificare il seguente parametro always_external_browser; altrimenti, il sistema della guida potrebbe non funzionare correttamente.
always_external_browser
Per i sistemi operativi Microsoft Windows, la configurazione predefinita di questo parametro è always_external_browser=true. Quando si avvia il sistema della guida in modalità autonoma, questa impostazione consente all'utente di aprire il browser predefinito del sistema operativo piuttosto che il browser integrato nel sistema della guida. Non è necessario impostare always_external_browser=true per i sistemi operativi diversi da Windows.
