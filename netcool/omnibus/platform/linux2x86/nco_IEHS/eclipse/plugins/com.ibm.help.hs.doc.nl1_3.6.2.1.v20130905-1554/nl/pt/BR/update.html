Atualizando informações no sistema de ajuda

É possível atualizar regularmente os conteúdos de documentos, se disponíveis, fazendo o download de tais documentos no sistema de ajuda. Os conteúdos serão periodicamente atualizados no Web site.

Sobre Esta Tarefa

Nota: A função de atualização está disponível apenas no modo independente.

É possível verificar e fazer o download das atualizações disponíveis executando as seguintes etapas:

Procedimento

  1. Clique em Atualização () na barra de ferramentas. Será exibida uma lista de conjuntos de documentos instalados.
    Nota: O ícone Atualizar não estará disponível se não houver informações sobre atualizações. E é possível configurar o ícone Atualizar () como visível ou invisível na barra de ferramentas de visualização Conteúdo especificando o parâmetro -showUpdater no script de início.
  2. Clique em Próximo na parte inferior para visualizar uma lista de Atualizações para as coleções de informações existentes e Novas coleções de informações. As atualizações disponíveis são armazenadas em um site de atualização.
  3. Selecione a documentação que você pretende instalar, e então clique em Instalar Atualizações.
  4. Clique em Finalizar quando a instalação estiver concluída. Você não precisa reiniciar o sistema de ajuda, pois ele será atualizado automaticamente.
