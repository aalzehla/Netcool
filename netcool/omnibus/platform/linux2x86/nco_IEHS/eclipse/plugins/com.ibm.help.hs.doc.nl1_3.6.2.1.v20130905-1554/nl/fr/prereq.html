Prérequis pour le système d'aide

Lorsque vous configurez le système d'aide IBM® Eclipse, assurez-vous que votre poste de travail, votre système d'exploitation, votre navigateur et votre matériel satisfont bien aux conditions détaillées dans la présente rubrique.

Navigateur

Pour utiliser le système d'aide IBM Eclipse, vous pouvez installer Microsoft Internet Explorer version 6.0 ou 7.0, ou Firefox 2.0 ou 3.0. Pour pouvoir utiliser toutes les fonctionnalités de cette version du système d'aide IBM Eclipse, vous devez activer les cookies et JavaScript dans le navigateur et désactiver dans ce dernier le blocage des fenêtres en incrustation.

Pour pouvoir utiliser le système d'aide IBM Eclipse, vous devez installer IBM Java Runtime Environment (JRE) version 6.0. Voir le tableau suivant pour le détail des conditions requises.

Personnalisation prise en charge

Vous pouvez configurer le système d'aide avec le fichier plugin_customization.ini. Celui-ci se trouve sous le plugin principal du produit. Vous pouvez le définir dans le script de démarrage.

Vous pouvez également modifier les paramètres de configuration en éditant le fichier org.eclipse.help.base_<n° version>/preferences.ini dans le répertoire eclipse/plugins, bien que la configuration par défaut soit fortement recommandée.

Vous pouvez personnaliser votre système d'aide en modifiant le fichier preferences.ini.

Remarque : Ne modifiez pas le paramètre always_external_browser, car le système d'aide pourrait alors ne plus fonctionner correctement.
always_external_browser
Dans le cas des systèmes d'exploitation Microsoft Windows, la configuration par défaut de ce paramètre est always_external_browser=true. Lorsque vous démarrez le système d'aide en mode autonome, grâce à ce paramètre, c'est le navigateur par défaut du système d'exploitation qui s'ouvre, et non celui intégré au système d'aide. Il n'est pas nécessaire de définir le paramètre always_external_browser=true pour les autres systèmes d'exploitation que Windows.
