Навигация в справочной системе

Поиск и просмотр документации продукта выполняются через графический пользовательский интерфейс справочной системы.

На вкладке Оглавление () внизу панели показываются заголовки разделов в справочной системе. Заголовки показываются в виде иерархического списка или в виде обычного оглавления. Щелкнув на заголовке раздела в дереве навигации, можно просмотреть его содержимое.

Кнопки Назад () и Вперед () позволяют перемещаться между уже просмотренными разделами.

Нажмите кнопку Домашняя страница (), чтобы открыть начальную страницу справочной системы.

Откройте вкладку Оглавление () внизу панели для возврата в оглавление после выполнения поиска.

При переходе по ссылке в текущем разделе открывается новый раздел, и дерево навигации автоматически изменяется, показывая расположение нового раздела на основе параметров по умолчанию.

Если это изменение автоматически не произошло, можно выполнить следующие действия.
  • Найдите текущий раздел в дереве навигации, нажмите кнопку Показать в оглавлении () на панели инструментов панели Оглавление.
  • Для того чтобы дерево навигации было всегда синхронизировано с панелью раздела, нажмите Связать с оглавлением () на панели инструментов панели Оглавление. Для выключения этой функции нажмите кнопку еще раз.

Дополнительная информация о навигации с помощью клавиатуры приведена в разделе Навигация с помощью клавиатуры.
