Быстрая печать

В этом разделе приведены инструкции по печати разделов справочной системы.

Об этой задаче

Для печати одного или нескольких разделов выполните следующие действия на панели навигации:

Процедура

  1. Выберите раздел на панели навигации.
  2. Нажмите кнопку Печать разделов() на панели инструментов, чтобы открыть меню.
    • Для того чтобы напечатать только выбранный раздел, выберите пункт Печать выбранного раздела.
    • Для того чтобы напечатать выбранный раздел и все вложенные в него разделы, выберите пункт Печать выбранного раздела и подразделов. Будет показано окно предварительного просмотра.
    • Для изменения конфигурации страницы перед печатью откройте в окне предварительного просмотра меню Файл > Настройка страницы и измените необходимые параметры.
  3. Нажмите кнопку Печать в окне Печать.
    Ограничение:

    При быстрой печати действуют ограничения MaxTopics и maxConnections. Эти ограничения задаются в параметрах для оптимизации использования ресурсов перед запуском системы.

    Если число выбранных для печати разделов превышает значение maxTopics, то появляется окно Предупреждение.
    Рис. 1. Пример окна Предупреждение
    • Нажмите кнопку Да, чтобы выполнить печать без превышения ограничения maxTopics.
    • Нажмите кнопку Нет, чтобы отменить печать.
    Если число соединений с принтером превышает значение maxConnections, то появляется окно Ошибка.
    Рис. 2. Пример окна Ошибка
    • Нажмите OK. Повторите операцию позднее.
