Предварительные требования для работы со справочной системой

Перед установкой IBM® Eclipse Help System убедитесь, что рабочая станция, операционная система, браузер и аппаратное обеспечение отвечают требованиям, описанным в этом разделе.

Требования к браузеру

Для работы с IBM Eclipse Help System можно установить браузер Microsoft Internet Explorer версии 6.0 или 7.0 либо Firefox версии 2.0 или 3.0. Для того чтобы работали все функции IBM Eclipse Help System, в браузере необходимо включить поддержку cookie и JavaScript и выключить функцию блокировки всплывающих окон.

Требования к среде операционной системы

Для работы с IBM Eclipse Help System необходимо установить IBM Java Runtime Environment (JRE) версии 6.0. Подробное описание требований приведено в следующей таблице.

Поддерживаемые способы настройки

Справочную систему можно настроить с помощью файла plugin_customization.ini. Он находится в основном модуле продукта. Его можно указать в сценарии запуска.

Параметры конфигурации также можно настроить в файле org.eclipse.help.base_<номер-версии>/preferences.ini в каталоге eclipse/plugins, однако настоятельно рекомендуется использовать конфигурацию по умолчанию.

Можно настроить справочную систему, изменив файл preferences.ini.

Прим.: Параметр always_external_browser изменять нельзя; в противном случае справочная система будет работать неправильно.
always_external_browser
В операционных системах Microsoft Windows по умолчанию применяется параметр always_external_browser=true. При запуске справочной системы в автономном режиме этот параметр указывает, что должно быть открыто окно браузера по умолчанию, а не встроенного браузера справочной системы. Параметр always_external_browser=true не следует устанавливать в операционных системах, отличных от Windows.
