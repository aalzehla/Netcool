Definir marcadores no sistema de ajuda

Pode localizar um tópico mais facilmente definindo um marcador e guardando o mesmo na vista Marcadores do sistema de ajuda.

Procedimento

  1. Para definir um marcador para o tópico actual no painel de tópicos, faça clique em Marcador () na barra de ferramentas.
  2. Para visualizar os marcadores, faça clique no separador Marcadores () na barra de ferramentas.
    Nota:
    • Se o sistema de ajuda estiver a ser executado no modo de Information Center, o botão Marcador () é apresentado apenas quando pesquisa páginas com o Internet Explorer. Pode fazer clique no menu Favorites (Favoritos) do Internet Explorer para localizar os marcadores criados.
    • Se o sistema de ajuda estiver a ser executado no modo autónomo, faça clique no separador Marcadores () para localizar os marcadores.
  3. Para eliminar um marcador, faça clique no separador Marcadores (), seleccione o nome do tópico no painel de tópicos e faça clique em Eliminar marcador seleccionado () na barra de ferramentas.
  4. Para eliminar todos os marcadores, faça clique no separador Marcadores () e faça clique em Eliminar todos os marcadores () na barra de ferramentas.
