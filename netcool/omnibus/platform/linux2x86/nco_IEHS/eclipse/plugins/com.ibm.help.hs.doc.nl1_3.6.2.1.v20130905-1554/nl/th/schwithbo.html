การค้นหาด้วยตัวดำเนินการบูลีน (AND, OR, NOT)

คุณสามารถแทรกตัวดำเนินการบูลีนซึ่งประกอบด้วย AND, OR และ NOT ในสตริงการค้นหาของคุณเพื่อทำให้การค้นหาของคุณมีประสิทธิภาพมากขึ้น

ตัวดำเนินการบูลีนสามารถทำให้การค้นหาของคุณมีประสิทธิภาพมากขึ้นโดยจำกัดขอบเขตการค้นหาของคุณให้แคบลงหรือกว้างขึ้น ตัวอย่างเช่น:

database AND "log file"
จำกัดการค้นหาหัวข้อที่มีทั้งคำว่า database และวลี log file ในเอกสาร
database OR "log file"
ขยายการค้นหาหัวข้อที่มีคำว่า database หรือวลี log file ในเอกสาร
database NOT "log file"
ค้นหาหัวข้อที่มีคำว่า database และไม่มีวลี log file ในเอกสาร
database OR "data base" NOT "log file"
ค้นหาเอกสารที่มีคำว่า database หรือวลี data base และไม่มีวลี log file ในเอกสาร
หัวข้อต้นทาง: การค้นหาข้อมูลในระบบวิธีใช้
