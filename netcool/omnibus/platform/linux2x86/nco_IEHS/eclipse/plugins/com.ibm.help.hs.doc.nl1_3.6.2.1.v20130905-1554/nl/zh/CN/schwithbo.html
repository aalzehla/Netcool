使用布尔运算符（AND、OR 或 NOT）进行搜索

可以在搜索字符串中插入布尔运算符（包括 AND、OR 和 NOT），以便更有效地进行搜索。

布尔运算符可以通过缩小或扩大搜索范围，从而更有效地进行搜索。例如：

database AND "log file"
将搜索范围缩小为在文档任何位置同时包含搜索项 database 和短语 log file 的主题。
database OR "log file"
将搜索范围扩大为在文档任何位置包含搜索项 database 或短语 log file 的主题。
database NOT "log file"
搜索在文档任何位置包含 database 但不包含短语 log file 的主题。
database OR "data base" NOT "log file"
搜索在其中任何位置包含搜索项 database 或短语 data base 但不包含短语 log file 的文档。
