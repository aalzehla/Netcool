빠른 검색 수행

빠른 검색 기능을 사용하여 탐색에서 주제 또는 주제 세트를 검색할 수 있습니다.

빠른 검색을 수행하려면 탐색 보기에서 다음 단계를 수행하십시오.

  1. 탐색 보기의 도구 모음에서 주제 검색()을 클릭하여 메뉴를 표시하십시오.
    1. 열린 주제에서만 검색하려면 선택된 주제 검색을 클릭하십시오.
    2. 해당 주제와 모든 하위 주제까지 검색하려면 선택된 주제 및 모든 하위 주제 검색을 클릭하십시오.
  2. 팝업 검색 대화 상자의 검색 필드에 조회할 사항을 입력하고 확인을 클릭하여 검색을 시작하십시오.

검색 단어가 일치하면 검색 결과 보기에서 리턴되는 주제와 검색 결과 수를 알리는 메시지를 표시합니다.

검색 단어가 일치하지 않으면 조회에 대해 주제에서 0개의 결과를 찾았음을 알리는 메시지가 표시됩니다.

상위 주제: 도움말 시스템 검색
