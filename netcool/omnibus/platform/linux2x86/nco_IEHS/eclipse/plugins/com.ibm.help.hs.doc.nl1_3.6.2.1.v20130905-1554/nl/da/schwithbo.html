Søgning med booleske operatorer (AND, OR, NOT)

For at gøre din søgning mere effektiv kan du indsætte booleske operatorer, f.eks. AND, OR og NOT, i søgeudtrykket.

Om denne opgave

De booleske operatorer kan gøre søgeomfanget mere effektivt ved enten af øge eller begrænse det. Eksempel:

database AND "log file"
Begrænser søgningen til emner, der indeholder både udtrykket database og log file i hele dokumentet.
database OR "log file"
Udvider søgningen til at inkludere emner, der indeholder udtrykket database eller log file i hele dokumentet.
database NOT "log file"
Søger efter emner, der indeholder udtrykket database, men ikke udtrykketlog file i hele dokumentet.
database OR "data base" NOT "log file"
Søger efter dokumenter, der indeholder enten udtrykket database eller udtrykket data base, og som ikke indeholder udtrykket log file i hele dokumentet.
