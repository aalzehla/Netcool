Software de requisito previo para el sistema de ayuda

Cuando configure el Sistema de ayuda de IBM® Eclipse, asegúrese de que la estación de trabajo, el sistema operativo, el navegador y el hardware cumplen los requisitos que se describen en este tema.

Requisitos del navegador

Para trabajar con el Sistema de ayuda de IBM Eclipse, puede instalar Microsoft® Internet Explorer Versión 6.0 o 7.0, o bien Firefox 2.0 o 3.0. Para asegurarse de que se pueden utilizar todas las funciones de esta versión del Sistema de ayuda de IBM Eclipse, debe habilitar las cookies y JavaScript™ en el navegador e inhabilitar la función de bloqueo de ventanas emergentes del navegador.

Requisitos del entorno del sistema operativo

Para trabajar con el Sistema de ayuda de IBM Eclipse, debe instalar IBM Java™ Runtime Environment (JRE) versión 6.0. Consulte la tabla siguiente para ver los requisitos detallados.

Personalización soportada

Puede configurar el sistema de ayuda con el archivo plugin_customization.ini. Se encuentra bajo el plug-in de producto principal. Puede establecerlo en el script de inicio.

También puede cambiar parámetros de configuración editando el archivo org.eclipse.help.base_<número de versión>/preferences.ini bajo el directorio eclipse/plugins, aunque se recomienda mantener la configuración predeterminada.

Puede personalizar su sistema de ayuda modificando el archivo preferences.ini.

Nota: No modifique el parámetro siguiente always_external_browser; de lo contrario es posible que el sistema de ayuda no funcione correctamente.
always_external_browser
Para los sistemas operativos Microsoft Windows, la configuración predeterminada de este parámetro es always_external_browser=true. Cuando inicia el sistema de ayuda en modalidad autónoma, este valor le ayuda a abrir el navegador predeterminado del sistema operativo en lugar del navegador incorporado en el sistema de ayuda. No es necesario que establezca always_external_browser=true para los sistemas operativos que no sean Windows.
