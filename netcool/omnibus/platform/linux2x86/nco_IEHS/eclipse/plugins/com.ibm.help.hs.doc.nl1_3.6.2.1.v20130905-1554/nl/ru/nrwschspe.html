Сужение диапазона поиска

Область поиска можно сузить до определенного набора разделов.

По умолчанию поиск данных в справочной системе выполняется во всех разделах, показанных на панели навигации. Однако область поиска можно сузить до определенного набора разделов, создав список поиска и сохранив его для использования в будущем.

Для создания списка поиска выполните следующие действия:

  1. Щелкните на ссылке Область поиска, показанной рядом с полем Поиск. Будет показано окно Выбрать область поиска.
  2. Включите переключатель Поиск только в следующих разделах и нажмите кнопку Создать. Будет показано окно Создать список поиска.
  3. В поле Имя списка укажите имя создаваемого списка поиска.
  4. В списке Разделы для поиска выберите ветви дерева навигации, которые должны просматриваться при поиске. Для выбора подкатегорий на более низком уровне, разверните ветвь.
  5. Нажмите OK. Новый список поиска будет показан в области Поиск только в следующих разделах. Соответствующая опция будет автоматически выбрана. Нажмите OK, чтобы этот список поиска применялся во всех последующих операциях поиска.
    Ограничение: В некоторых браузерах (в том числе Konqueror и Safari) функция сохранения списка поиска не поддерживается.

Активный список поиска всегда показан в верхней области окна браузера рядом со ссылкой Область поиска.

Последний созданный список поиска будет применяться при каждом обращении к справочной системе.
