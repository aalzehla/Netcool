Buscar con caracteres comodín

Este tema incluye información acerca de cómo realizar búsquedas con caracteres comodín para expandir el ámbito de búsqueda.

Puede utilizar caracteres comodín, como por ejemplo un asterisco (*) y un signo de interrogación (?), para buscar un término en el sistema de ayuda.

  • Asterisco (*): para múltiples caracteres desconocidos o variables en el término. Por ejemplo, si el término de búsqueda es par*, las palabras parte, participar, parcial, así como otras palabras que empiecen por par, figurarían entre los resultados.
  • Signo de interrogación (?): hace las veces de un solo carácter desconocido o variable, o de ninguno. Por ejemplo, si el termino de búsqueda es part?, la palabra parte sería un resultado de la búsqueda, pero no las palabras partir ni partición.
