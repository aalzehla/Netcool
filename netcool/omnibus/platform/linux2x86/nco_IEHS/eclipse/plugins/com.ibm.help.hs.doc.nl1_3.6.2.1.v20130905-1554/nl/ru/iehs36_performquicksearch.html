Быстрый поиск

С помощью функции Быстрый поиск можно выполнять поиск в разделах и наборах разделов на панели навигации.

Для выполнения быстрого поиска выполните следующие действия на панели навигации:

  1. Откройте меню, нажав Разделы для поиска () на панели инструментов панели навигации.
    1. Для поиска только в открытом разделе выберите команду Поиск в выбранном разделе.
    2. Для поиска в разделе и всех его подразделах выберите команду Поиск в выбранном разделе и подразделах.
  2. Во всплывающем окне поиска введите запрос в поле Поиск и нажмите OK, чтобы начать поиск.

Если искомые слова найдены, откроется панель результатов поиска со списком разделов и сообщением о количестве результатов поиска.

Если искомые слова не найдены, появится сообщение о том, что по запросу ничего не найдено.
