Recherche d'informations à l'aide de la vue Index

Cette rubrique explique comment utiliser la vue Index pour rechercher des rubriques du système d'aide.

Pourquoi et quand exécuter cette tâche

Pour rechercher les informations souhaitées dans la vue Index, procédez comme suit :

Procédure

  1. Cliquez sur l'onglet Index () pour passer en vue Index.
  2. Entrez un mot clé dans la zone de saisie. La liste d'index défile automatiquement jusqu'au mot clé le plus approchant.
    Remarque :
    • Si un mot clé est associé à plusieurs rubriques, il ne s'affiche pas dans l'index. Mais la vue Index affichera les titres des rubriques contenant ce mot clé.
  3. Cliquez sur le mot clé, sur un mot clé secondaire, ou sur un titre de rubrique afin d'afficher dans la vue Sommaire la rubrique correspondante.
    Remarque : Les index ne font pas de distinction entre les majuscules et les minuscules.

    Pour basculer entre la vue de navigation et la vue Index, cliquez sur l'onglet Sommaire () ou sur l'onglet Index ().
