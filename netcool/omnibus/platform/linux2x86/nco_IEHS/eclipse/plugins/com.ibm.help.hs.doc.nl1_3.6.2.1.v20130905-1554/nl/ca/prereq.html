Requisits de prerequisit per al sistema d'ajuda

Quan configureu l'IBM® Eclipse Help System (IEHS), assegureu-vos que la vostra estació de treball, sistema operatiu, navegador i maquinari compleixen els requisits que es descriuen en aquest tema.

Requisits del navegador

Per treballar amb l'IBM Eclipse Help System (IEHS), podeu instal·lar el Microsoft® Internet Explorer Versió 6.0 ó 7.0, o bé el Firefox 2.0 ó 3.0. Per tal d'assegurar-vos que totes les funcions d'aquesta versió de l'IBM Eclipse Help System (IEHS) es poden utilitzar, heu d'habilitar les galetes i el JavaScript™ al navegador i inhabilitar la funció del navegador que bloqueja les finestres emergents.

Requisits de l'entorn de sistema operatiu

Per treballar amb l'IBM Eclipse Help System (IEHS), heu d'instal·lar un entorn de temps d'execució IBM Java™ Runtime Environment (JRE) versió 6.0. Consulteu la taula següent per veure els requisits detallats.

Personalització suportada

Podeu configurar el sistema d'ajuda amb el fitxer plugin_customization.ini. Es troba sota el connector del producte principal. Podeu definir-lo a l'script d'inici.

També podeu canviar els paràmetres de configuració editant el fitxer org.eclipse.help.base_<número versió/preferences.ini sota el directori eclipse/plugins, malgrat que es recomana molt utilitzar la configuració per defecte.

Podeu personalitzar el vostre sistema d'ajuda modificant el fitxer preferences.ini.

Nota: No modifiqueu el següent paràmetre always_external_browser; si ho feu, és possible que el sistema d'ajuda no funcioni correctament.
always_external_browser
Per als sistemes operatius Microsoft Windows, la configuració per defecte d'aquest paràmetre és always_external_browser=true. Quan inicieu el sistema d'ajuda en mode autònom, aquest valor us ajuda a obrir el navegador per defecte del sistema operatiu en comptes del navegador incrustat al sistema d'ajuda. No cal que definiu always_external_browser=true per als sistemes operatius que no siguin Windows.
