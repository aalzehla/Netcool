เทคนิคการค้นหาขั้นสูง

คุณสามารถดำเนินการค้นหาขั้นสูงเพิ่มเติมได้โดยใช้เครื่องหมายอัญประกาศ ไวด์การ์ด หรือตัวดำเนินการบูลีนเพื่อจำกัดผลการค้นหา

  • เมื่อคุณค้นหาข้อมูลในระบบวิธีใช้ คุณสามารถระบุคำ หรือวลีที่เป็นสตริงที่แน่นอนได้โดยใส่เครื่องหมายอัญประกาศ คู่

    ตัวอย่างเช่น เมื่อคุณพิมพ์ "log file" ลงในฟิลด์ ค้นหา คุณกำลังค้นหาสตริง log file ที่มีอยู่ทั้งหมด แต่ถ้าไม่ได้พิมพ์เครื่องหมายอัญประกาศคู่ลงไป หัวข้อที่มีอินสแตนซ์ของทั้งคำว่า log และ คำว่า file จะถูกส่งคืน

  • คุณสามารถใช้ไวด์การ์ด เช่น เครื่องหมายดอกจัน (*) และเครื่องหมายคำถาม (?) ในการค้นหาของคุณเพื่อขยายขอบเขตการค้นหา
    • ใช้เครื่องหมายดอกจัน (*) สำหรับอักขระที่ไม่รู้จักหรือตัวแปรใน เคียวรีการค้นหา ตัวอย่างเช่น ถ้าคุณพิมพ์ par* หัวข้อที่มี partly, participate, partial และคำอื่นๆ ที่ขึ้นต้นด้วย par จะถูกส่งคืน
      หมายเหตุ: มี ข้อจำกัดอย่างหนึ่งสำหรับสตริงการค้นหา: เมื่อใช้เงื่อนไขการค้นหา ที่มีเครื่องหมายดอกจัน ต้องมีอักขระนำหน้าอย่างน้องสามตัว ในลักษณะคล้ายกัน เงื่อนไขการค้นหาที่ไม่มีอักขระไวด์การ์ดต้องมีอักขระอย่างน้อย สามตัว
    • ใช้เครื่องหมายคำถามสำหรับอักขระเดี่ยวในเคียวรีการค้นหา ตัวอย่างเช่น ถ้าคุณพิมพ์ par? หัวข้อที่มีคำว่า part แต่ไม่ใช่ partial หรือ partly จะถูกส่งคืน
    หมายเหตุ: เมื่อใช้เงื่อนไขการค้นหาด้วยเครื่องหมายดอกจัน (*) ต้องนำหน้า ด้วยอักขระอย่างน้อย 3 ตัว
  • คุณสามารถใช้ตัวดำเนินการบูลีน AND, OR และ NOT เพื่อเจาะจงเคียวรีของคุณ ตัวอย่างเช่น:
    database AND "log file"
    จำกัดผลการค้นหาในหัวข้อที่มีทั้งคำว่า database และ วลี log file
    database OR "log file"
    ขยายผลการค้นหาในหัวข้อที่มีคำว่า database หรือ วลี log file
    database NOT "log file"
    จำกัดผลการค้นหาในหัวข้อที่มีคำว่า database แต่ไม่มีวลี log file
    database OR "data base" NOT "log file"
    ค้นหาหัวข้อที่มีคำว่า database หรือ วลี data base แต่ไม่มีวลี log file
หัวข้อต้นทาง: การค้นหาของระบบวิธีใช้
