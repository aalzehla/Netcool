빠른 검색 수행

이 주제에서는 빠른 검색을 수행하는 방법을 설명합니다.

빠른 검색을 수행하려면 탐색 보기에서 다음 단계를 수행하십시오.

  1. 탐색 보기의 도구 모음에서 주제 검색()을 클릭하여 메뉴를 표시하십시오.
    • 연 주제에서만 검색하려면 이 주제 검색을 클릭하십시오.
    • 이 주제와 하위 주제까지 모두 검색하려면 이 주제 및 모든 하위 주제 검색을 클릭하십시오.
      1. 검색 대화 상자의 검색 필드에 조회할 사항을 입력하십시오.
      2. 확인을 클릭하여 검색을 시작하십시오.
  2. 팝업 검색 대화 상자의 검색 필드에 조회할 사항을 입력하고 확인을 클릭하여 검색을 시작하십시오.

빠른 메뉴를 연 주제에서만 검색하고 검색 문자열이 일치하는 경우 주제가 자동으로 표시됩니다. 주제에 일치하는 항목이 없으면 찾을 수 없음 메시지가 표시됩니다.

주제와 하위 주제를 모두 검색하고 검색 용어가 일치하는 경우 검색 결과 보기에 상위 500개의 결과가 표시됩니다. 검색 결과를 선택하면 해당 주제가 표시됩니다. 조회한 단어는 표시된 주제에서 강조표시됩니다. 주제에 일치하는 항목이 없으면 찾을 수 없음 메시지가 표시됩니다.

참고:
  • 이 주제 검색 대화 상자는 검색 결과에 관계없이 자동으로 닫힙니다.
  • 이 주제 및 모든 하위 주제 검색 대화 상자는 취소를 클릭하거나 대화 상자 창을 닫아야 닫힙니다.
