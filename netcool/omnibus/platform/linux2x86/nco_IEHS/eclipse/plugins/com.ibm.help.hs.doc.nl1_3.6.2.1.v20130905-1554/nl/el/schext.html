Ακριβής αναζήτηση λέξεων και φράσεων

Μπορείτε να αναζητήσετε λέξεις ή ολόκληρες φράσεις στο σύστημα βοήθειας.

Όταν κάνετε αναζήτηση στο σύστημα βοήθειας, μπορείτε να περικλείσετε μια φράση σε διπλά εισαγωγικά ώστε να γίνει αναζήτηση τις ακριβούς φράσης.

Για παράδειγμα, εάν καταχωρήσετε τη φράση "αρχείο καταγραφής" στο πεδίο Αναζήτηση αντί για αρχείο καταγραφής, θα γίνει αναζήτηση της φράσης αρχείο καταγραφής. Χωρίς τα διπλά εισαγωγικά, η φράση αναζήτησης ερμηνεύεται ως αίτηση εντοπισμού τόσο της λέξης αρχείο όσο και της λέξης καταγραφής στα θέματα βοήθειας.

Σημείωση: Στα Αγγλικά και στα Γερμανικά μόνο, η μηχανή αναζήτησης αναζητεί επίσης τις εναλλακτικές γραμματικές μορφές μιας λέξης αναζήτησης. Για παράδειγμα, αν αναζητήσετε τη λέξη challenge, η μηχανή αναζήτησης θα εντοπίσει επίσης τη λέξη challenging. Τοποθετήστε σε διπλά εισαγωγικά τις λέξεις που καταχωρείτε στο πεδίο Αναζήτηση αν δεν θέλετε να περιλαμβάνονται τέτοιες γραμματικές παραλλαγές των λέξεων στα αποτελέσματα της αναζήτησης.
Γονικό θέμα: Αναζήτηση πληροφοριών στο σύστημα βοήθειας
