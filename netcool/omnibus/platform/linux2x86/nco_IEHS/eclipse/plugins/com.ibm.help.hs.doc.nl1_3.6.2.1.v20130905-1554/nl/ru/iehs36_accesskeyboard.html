Сочетания клавиш и клавиши быстрого доступа

Операции, выполняемые мышью, можно выполнять с помощью клавиш и сочетаний клавиш.

В следующей таблице перечислены сочетания клавиш и клавиши быстрого доступа, которые поддерживаются в справочной системе для браузеров Microsoft Internet Explorer и Mozilla Firefox.
Прим.: В Internet Explorer необходимо выбрать управляющий элемент клавишами быстрого доступа и нажать Enter для его активации. В Firefox клавиши быстрого доступа позволяют выбрать управляющий элемент и сразу его активировать.
Табл. 1. Сочетания клавиш и клавиши быстрого доступа для перемещения по справочной системе с помощью Internet Explorer
Действие Сочетание клавиш для Internet Explorer
Перейти на следующую ссылку, кнопку или ветвь раздела из фрейма (со страницы) Tab, затем Enter для перехода по ссылке
Перейти во фрейм панели Оглавление Alt+C, затем Enter и Shift+F6
Перейти во фрейм панели Индекс Alt+I, затем Enter и Shift+F6
Перейти во фрейм панели Результаты поиска Alt+R, затем Enter и Shift+F6
Перейти во фрейм панели раздела Alt+K, затем Enter
Свернуть/развернуть ветвь раздела Стрелка вправо/стрелка влево
Перейти на следующую ветвь раздела Стрелка вниз или Tab
Перейти на предыдущую ветвь раздела Стрелка вверх или Shift+Tab
Прокрутить вверх Home
Прокрутить вниз End
Следующий фрейм F6
Предыдущий фрейм Shift+F6
Печать текущей страницы или активного фрейма Ctrl+P
Табл. 2. Сочетания клавиш и клавиши быстрого доступа для перемещения по справочной системе с помощью Firefox
Действие Сочетание клавиш для Firefox
Перейти на следующую ссылку, кнопку или ветвь раздела из фрейма (со страницы) Tab, затем Enter для перехода по ссылке
Перейти во фрейм панели Оглавление Shift+Alt+C и Shift+F6
Перейти во фрейм панели Индекс Shift+Alt+I и Shift+F6
Перейти во фрейм панели Результаты поиска Shift+Alt+R и Shift+F6
Перейти во фрейм панели раздела В Firefox нет прямого сочетания клавиш для этого действия. Необходимо использовать Tab или функции навигации JAWS.
Свернуть/развернуть ветвь раздела Стрелка вправо/стрелка влево
Перейти на следующую ветвь раздела Стрелка вниз или Tab
Перейти на предыдущую ветвь раздела Стрелка вверх или Shift+Tab
Прокрутить вверх Home
Прокрутить вниз End
Следующий фрейм F6
Предыдущий фрейм Shift+F6
Печать текущей страницы или активного фрейма Ctrl+P
