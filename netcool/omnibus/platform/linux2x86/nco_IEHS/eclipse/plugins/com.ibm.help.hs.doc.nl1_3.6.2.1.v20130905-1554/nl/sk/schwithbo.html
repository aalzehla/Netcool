Vyhľadávanie pomocou boolovských operátorov (AND, OR, NOT)

Na zefektívnenie vyhľadávania môžete vo vyhľadávacích reťazcoch používať boolovské operátory, ako AND, OR a NOT.

Informácie o úlohe

Boolovské operátory môžu zvýšiť efektivitu vášho vyhľadávania zúžením alebo rozšírením rozsahu vyhľadávania. Napríklad:

database AND "log file"
Zúži vyhľadávanie tak, aby zahŕňalo témy, ktoré obsahujú výraz database a spojenie log file kdekoľvek v dokumente.
database OR "log file"
Rozšíri vyhľadávanie tak, aby zahŕňalo témy, ktoré obsahujú výraz database alebo spojenie log file kdekoľvek v dokumente.
database NOT "log file"
Vyhľadá témy, ktoré obsahujú výraz database a neobsahujú spojenie log file kdekoľvek v dokumente .
database OR "data base" NOT "log file"
Vyhľadá dokumenty, ktoré obsahujú buď výraz database alebo spojenie data base, ale neobsahujú spojenie log file kdekoľvek v dokumente.
