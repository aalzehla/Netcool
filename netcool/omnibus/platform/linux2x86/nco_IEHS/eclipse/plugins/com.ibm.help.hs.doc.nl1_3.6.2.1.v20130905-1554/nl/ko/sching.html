도움말 시스템 검색

이 주제에서는 IBM Eclipse Help System에서 검색하는 방법을 설명합니다.

도움말 시스템에서 주제를 검색하려면 검색 필드에 조회할 사항을 입력하고 이동을 클릭하십시오. 검색 결과 보기에 관련성이 높은 상위 500개의 결과가 표시됩니다. 검색 결과를 선택하면 해당 주제가 표시됩니다. 조회한 단어는 표시된 주제에서 강조표시됩니다. 해당 단어가 처음 나타나는 부분을 표시하도록 결과 페이지가 자동으로 스크롤됩니다.

검색한 후에 강조표시 기능을 사용할 수 있습니다. 조회하는 단어에 강조표시 효과를 사용하거나 사용하지 않으려면 검색 용어 강조표시()를 클릭하십시오.

탐색 보기와 검색 결과 보기 사이를 전환하려면 컨텐츠 탭() 또는 검색 결과 탭()을 클릭하십시오.

필터 설정에 상관없이 검색 조회열과 일치하는 주제가 검색 결과 보기에 표시됩니다.

주: 검색 문자열은 대소문자를 구분하지 않습니다.
  • 정확한 단어 또는 구문 검색
    도움말 시스템에서 입력한 단어 또는 구문만을 정확하게 검색할 수 있습니다.
  • 와일드 카드로 검색
    이 주제에서는 검색 범위를 넓히기 위해 와일드카드로 검색하는 방법을 설명합니다.
  • 부울 연산자(AND, OR, NOT)를 사용하여 검색
    검색 문자열에 AND, OR, NOT 등의 부울 연산자를 삽입하여 검색 정확도를 높일 수 있습니다.
  • 검색 범위 좁히기
    검색 범위를 특정 주제로 좁힐 수 있습니다.
  • 빠른 검색 수행
    이 주제에서는 빠른 검색을 수행하는 방법을 설명합니다.
  • 검색 결과 표시
    도움말 시스템에서 다양한 기준별로 검색 결과를 표시할 수 있습니다.
