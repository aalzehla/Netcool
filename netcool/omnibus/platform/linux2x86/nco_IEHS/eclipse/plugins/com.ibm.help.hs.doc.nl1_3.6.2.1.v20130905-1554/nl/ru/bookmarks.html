Создание закладок в справочной системе

Для упрощения доступа к разделу можно создать для его закладку и сохранить на панели Закладки справочной системы.

Процедура

  1. Для создания закладки на открытый в данный момент раздел нажмите кнопку Создать закладку () на панели инструментов.
  2. Для просмотра закладок откройте вкладку Закладки () на панели инструментов.
    Прим.:
    • Если справочная система работает в режиме Information Center, кнопка Закладка () показывается только в Internet Explorer. Созданные закладки можно просмотреть в меню Избранное браузера Internet Explorer.
    • Если справочная система работает в автономном режиме, откройте вкладку Закладки () для просмотра закладок.
  3. Для удаления закладки откройте вкладку Закладки (), выберите имя раздела на панели раздела и нажмите кнопку Удалить выбранную закладку () на панели инструментов.
  4. Для удаления всех закладок откройте вкладку Закладки () и нажмите кнопку Удалить все закладки () на панели инструментов.
