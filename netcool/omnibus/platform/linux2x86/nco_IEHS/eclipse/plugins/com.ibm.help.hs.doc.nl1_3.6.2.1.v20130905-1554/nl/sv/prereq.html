Krav för hjälpsystemet

När du installerar IBM Eclipse Help System måste du se till att datorn, operativsystemet, webbläsaren och maskinvaran uppfyller kraven i det här avsnittet.

Krav för webbläsare

Om du vill arbeta med IBM Eclipse Help System måste du installera Microsoft Internet Explorer version 6.0 eller 7.0, eller Mozilla Firefox version 2.0 eller 3.0. Om du vill se till att det går att använda alla funktioner i den här versionen av IBM Eclipse Help System måste du aktivera cookier och JavaScript i webbläsaren, och avaktivera funktioner för blockering av fönster i webbläsaren.

Krav för operativsystemmiljön

Om du vill arbeta med IBM Eclipse Help System måste du installera en IBM-JRE-miljö (Java Runtime Environment) med version 6.0. Mer information om kraven finns i följande tabell.

Tabell 1. Krav för operativsystemmiljön

IEHS för 32-bitarsversioner av Microsoft Windows

  • Om du vill att dubbelriktade tecken, dubbelbytetecken eller flerbytetecken ska visas i hjälpsystemet måste du installera motsvarande teckensnitt och ange det som visningsteckensnitt i webbläsaren.

Anpassningar som kan utföras

Du kan använda filen plugin_customization.ini till att konfigurera hjälpsystemet. Den här filen finns i insticksprogramskatalogen för huvudprodukten. Du kan ange den i startskriptet.

Du kan även ändra konfigurationsparametrarna genom att redigera filen org.eclipse.help.base_<version>/preferences.ini i mappen "eclipse/plugins", men du bör använda standardkonfigurationen.

Du kan anpassa hjälpsystemet genom att ändra filen preferences.ini.

Obs! Du bör inte ändra parametern always_external_browser. Om du gör det kommer eventuellt inte hjälpsystemet att fungera på rätt sätt.
always_external_browser
För Microsoft Windows är standardkonfigurationen för den här parametern always_external_browser=true. När du startar hjälpsystemet i fristående läge gör den här inställningen att du kan öppna standardwebbläsaren för operativsystemet i stället för den webbläsare som är inbäddad i hjälpsystemet. Du behöver inte ange always_external_browser=true för andra operativsystem än Windows.
