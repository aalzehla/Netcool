Características de accesibilidad en el sistema de ayuda

Las características de accesibilidad ayudan a las personas con discapacidades físicas, como las de movilidad restringida o visión limitada, o con otras necesidades especiales, a utilizar los productos de tecnologías de la información de manera satisfactoria.

Las características de accesibilidad de este Information Center permiten a los usuarios hacer lo siguiente:

  • Utilizar software de lector de pantalla y sintetizadores de voz digitales para oír lo que aparece en la pantalla. Consulte la documentación del producto de la tecnología de asistencia para obtener detalles sobre el uso de tecnologías de asistencia con información basada en HTML.
  • Utilizar lupas de pantalla para aumentar lo visualizado en la pantalla.
  • Operar características específicas o equivalentes utilizando solamente el teclado.
  • Atajos y aceleradores de teclado
    Puede utilizar claves o combinaciones de claves para realizar operaciones que también pueden realizarse utilizando un ratón.
