利用布林運算子（AND、OR、NOT）搜尋

您可以在搜尋字串中插入布林運算子，包括 AND、OR 和 NOT，使您的搜尋更有效率。

布林運算子會縮小或擴大搜尋範圍，藉以讓您的搜尋更有效率。比方說：

database AND "log file"
縮小搜尋範圍，所包括的主題必須同時在文件的任何位置中含有 database 詞彙和 log file 詞組。
database OR "log file"
擴大搜尋範圍，所包括的主題必須在文件的任何位置含有 database 詞彙或 log file 詞組。
database NOT "log file"
搜尋文件中包含 database、但任何位置都不含 log file 詞組的主題。
database OR "data base" NOT "log file"
搜尋文件中包含 database 詞彙或 data base 詞組、但任何位置都不含 log file 詞組的文件。
