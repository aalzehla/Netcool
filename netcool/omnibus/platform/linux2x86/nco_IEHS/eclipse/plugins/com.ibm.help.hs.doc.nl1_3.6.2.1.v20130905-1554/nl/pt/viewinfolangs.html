Visualizar informações em diferentes idiomas

Para visualizar a interface de utilizador do sistema de ajuda noutro idioma, tem de alterar as definições regionais do sistema operativo para o locale e o idiomas pretendidos.

O sistema apresentará a documentação traduzida que tenha sido fornecida pelo proprietário do produto. Se o proprietário do produto não tiver fornecido a documentação no idioma seleccionado pelo utilizador, mas tiver disponibilizado a versão em inglês, o sistema apresentará, por predefinição, a versão em inglês, mesmo se a opção de idioma do navegador tiver sido definida para outro idioma.
