การจำกัดขอบเขตการค้นหาของคุณให้แคบลง

คุณสามารถจำกัดขอบเขตการค้นหาหัวข้อเฉพาะของคุณให้แคบลง

เมื่อคุณค้นหาข้อมูลในระบบวิธีใช้ หัวข้อทั้งหมดที่แสดงในมุมมองนำทางจะถูกค้นหาตามค่าดีฟอลต์ อย่างไรก็ตาม คุณสามารถจำกัดขอบเขตการค้นหาหัวข้อเฉพาะของคุณให้แคบลงได้โดยสร้างรายการค้นหา และคุณสามารถบันทึกรายการค้นหานั้นเพื่อใช้ในอนาคตได้

คุณสามารถสร้างรายการค้นหาได้โดยทำตามขั้นตอนต่อไปนี้:

  1. คลิกลิงก์ Search scope ที่อยู่ถัดจากฟิลด์ Search คุณจะเข้าถึงหน้าต่าง เลือกขอบเขตการค้นหา
  2. เลือก Search only the following topics และคลิก New คุณจะมองเห็นหน้าต่าง รายการค้นหาใหม่
  3. ในฟิลด์ List name ให้ใส่ชื่อรายการค้นหาใหม่ของคุณ
  4. ในรายการ Topics to search ให้เลือกกิ่งของมุมมองนำทางที่คุณต้องการให้รวมอยู่ในการค้นหาของคุณ คุณสามารถขยายกิ่งเพื่อเลือกหมวดหมู่ย่อยที่ระดับรองของส่วนหัว
  5. คลิก OK ถึงตอนนี้ รายการค้นหาใหม่ของคุณจะถูกแสดงในรายการ Search only the following topics อ็อพชันนี้จะถูกเลือกโดยอัตโนมัติตามค่าดีฟอลต์ คุณสามารถคลิก OK เพื่อใช้รายการค้นหานี้กับการค้นหาถัดมาของคุณ
    ข้อจำกัด: ในบางบราวเซอร์ (ตัวอย่างเช่น Konqueror และ Safari) ไม่สนับสนุนฟังก์ชันการบันทึกรายการค้นหา

รายการค้นหาที่แอ็คทีฟในปัจจุบันจะแสดงอยู่ที่ด้านบนของบราวเซอร์เสมอ ซึ่งอยู่ถัดจากลิงก์ Search scope

รายการค้นหาล่าสุดที่คุณสร้างขึ้นจะยังคงอยู่ในแต่ละครั้งที่คุณเรียกใช้ระบบ

หัวข้อต้นทาง: การค้นหาข้อมูลในระบบวิธีใช้
