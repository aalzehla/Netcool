ข้อกำหนดสิ่งที่จำเป็นต้องมีสำหรับระบบวิธีใช้

เมื่อคุณตั้งค่าระบบวิธีใช้ IBM® Eclipse ทำให้แน่ใจว่าเวิร์กสเตชัน ระบบปฏิบัติการ เบราว์เซอร์ และฮาร์ดแวร์ของคุณเป็นไปตามข้อกำหนดที่ได้กล่าวไว้ในหัวข้อนี้

ข้อกำหนดเกี่ยวกับเบราว์เซอร์

ในการทำงานกับระบบวิธีใช้ IBM Eclipse คุณสามารถติดตั้ง Microsoft® Internet Explorer เวอร์ชัน 6.0 หรือ 7.0 หรือ Firefox 2.0 หรือ 3.0 เพื่อให้มั่นใจว่า ฟังก์ชันทั้งหมดของระบบวิธีใช้ IBM Eclipse เวอร์ชันนี้สามารถใช้งานได้ คุณต้องเปิดใช้งานคุกกี้และ JavaScript™ ในเบราว์เซอร์ และปิดใช้งานฟังก์ชันบล็อกหน้าต่างป็อปอัพ ของเบราว์เซอร์

ข้อกำหนดเกี่ยวกับสภาวะแวดล้อมระบบปฏิบัติการ

ในการ ทำงานกับระบบวิธีใช้ IBM Eclipse คุณต้องติดตั้ง IBM Java™ Runtime Environment (JRE) เวอร์ชัน 6.0 โปรดดูตารางต่อไปนี้สำหรับข้อกำหนดโดยละเอียด

ตารางที่ 1. ข้อกำหนดเกี่ยวกับสภาวะแวดล้อมระบบปฏิบัติการ
เวอร์ชันของระบบวิธีใช้ IBM Eclipse ระบบปฏิบัติการ ข้อกำหนดเกี่ยวกับระบบปฏิบัติการและ JRE
ระบบวิธีใช้ IBM Eclipse สำหรับ Microsoft Windows® แบบ 32 บิต
  • Windows XP Professional Intel แบบ 32 บิต
  • ระบบวิธีใช้ IBM Eclipse เวอร์ชันนี้ประกอบด้วย IBM JRE แบบฝัง คุณสามารถลบ JRE ออกจากแพ็กเกจได้ถ้าคุณไม่ต้องการเก็บ JRE แบบฝังไว้ คุณต้องมี JRE 6.0 ติดตั้งไว้บนระบบของคุณเพื่อรันระบบวิธีใช้ IBM Eclipse
ระบบวิธีใช้ IBM Eclipse สำหรับ Linux®
  • Red Hat Linux Enterprise 5 Intel® แบบ 32 บิต
  • ระบบวิธีใช้ IBM Eclipse เวอร์ชันนี้ประกอบด้วย IBM JRE แบบฝัง คุณสามารถลบ JRE ออกจากแพ็กเกจได้ถ้าคุณไม่ต้องการเก็บ JRE แบบฝังไว้ คุณต้องมี IBM JRE 6.0 บนระบบของคุณเพื่อรันระบบวิธีใช้ IBM Eclipse
  • สำหรับระบบปฏิบัติการเหล่านี้ คุณสามารถเปิดระบบวิธีใช้ในโหมดพื้นฐานผ่านเบราว์เซอร์ Konqueror
  • หากต้องการแสดงอักขระแบบสองทิศทาง อักขระแบบสองไบต์ หรืออักขระแบบมัลติไบต์ ในระบบวิธีใช้ ให้ติดตั้งฟอนต์ที่สอดคล้องกันบนระบบ และตั้งค่าฟอนต์ให้เป็นฟอนต์สำหรับแสดงผลของเบราว์เซอร์
  • หากต้องการสนับสนุนหลายภาษา ให้ตั้งค่าโลแคลของระบบให้เป็น UTF-8 ก่อนเริ่มต้นระบบวิธีใช้
ระบบวิธีใช้ IBM Eclipse สำหรับ AIX
  • AIX 5.3 PPC แบบ 64 บิต
  • ระบบวิธีใช้ IBM Eclipse เวอร์ชันนี้ประกอบด้วย IBM JRE แบบฝัง คุณสามารถลบ JRE ออกจากแพ็กเกจได้ถ้าคุณไม่ต้องการเก็บ JRE แบบฝังไว้ คุณต้องมี IBM JRE 6.0 ติดตั้งไว้บนระบบของคุณเพื่อรันระบบวิธีใช้ IBM Eclipse

การปรับแต่งค่าที่ได้รับการสนับสนุน

คุณสามารถกำหนดคอนฟิก ระบบวิธีใช้ได้ด้วยไฟล์ plugin_customization.ini ไฟล์ดังกล่าว อยู่ภายใต้ปลั๊กอินหลักของผลิตภัณฑ์ คุณสามารถตั้งค่าได้ในสคริปต์เริ่มต้น

คุณยังสามารถ เปลี่ยนแปลงพารามิเตอร์ของคอนฟิกูเรชันโดยแก้ไขไฟล์ org.eclipse.help.base_<version number>/preferences.ini ภายใต้ไดเร็กทอรี eclipse/plugins แม้ว่าจะแนะนำให้ใช้ดีฟอลต์คอนฟิกูเรชันก็ตาม

คุณ สามารถปรับแต่งค่าระบบวิธีใช้โดยการแก้ไขไฟล์ preferences.ini

หมายเหตุ: ห้ามแก้ไขพารามิเตอร์ ต่อไปนี้ always_external_browser; มิฉะนั้นระบบวิธีใช้อาจทำงานไม่ถูกต้อง
always_external_browser
สำหรับระบบปฏิบัติการ Microsoft Windows คอนฟิกูเรชันที่เป็นดีฟอลต์ของพารามิเตอร์นี้คือ always_external_browser=true เมื่อคุณเริ่มต้นระบบวิธีใช้ในโหมดสแตนด์อะโลน ค่าติดตั้งนี้ช่วยคุณเปิดดีฟอลต์ของเบราว์เซอร์ของระบบปฏิบัติการแทนเบราว์เซอร์ที่ฝังอยู่ในระบบวิธีใช้ คุณไม่จำเป็นต้องตั้งค่า always_external_browser=true สำหรับระบบปฏิบัติการที่ไม่ใช่ Windows
หัวข้อต้นทาง: การดูข้อมูลในระบบวิธีใช้
