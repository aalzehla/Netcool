검색 결과 표시

도움말 시스템에서 다양한 기준별로 검색 결과를 표시할 수 있습니다.

문서 세트의 주제가 다른 기준(버전, 제품 이름, 플랫폼 또는 카테고리)으로 구분된 경우, 해당 기준별로 주제의 검색 결과를 분류할 수 있습니다.

  1. 검색 결과 탭()을 클릭하십시오. 기본적으로 검색 결과는 구분되어 있지 않습니다.
  2. 도구 모음에서 검색 결과 그룹화()를 클릭하십시오.
  3. 메뉴에서 그룹화 기준을 선택하십시오.
    • GroupByNone

      이 기본 섹션을 사용하면 검색 결과가 정렬되지 않은 상태로 표시됩니다.

    • GroupByVersion
    • GroupByProdname
    • GroupByPlatform
    • 결과 범주 표시

      책갈피 이름별로 검색 결과를 그룹화하여 표시합니다.

  4. 그러면 선택한 기준에 따라 나열된 검색 결과를 얻을 수 있습니다.
주: 검색 결과 보기에 결과 설명을 추가하거나 제거하려면 결과 설명 표시()를 클릭하십시오.
상위 주제: 도움말 시스템 검색
