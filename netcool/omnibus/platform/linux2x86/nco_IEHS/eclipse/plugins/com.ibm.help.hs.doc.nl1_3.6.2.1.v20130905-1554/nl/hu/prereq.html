Súgórendszer használatához szükséges előfeltételek

Amikor beállítja az IBM® Eclipse Help System rendszert, akkor győződjön meg róla, hogy a munkaállomás, az operációs rendszer, a böngésző és a hardver megfelel az ebben a témakörben ismertetett követelményeknek.

Böngészőre vonatkozó követelmények

Az IBM Eclipse Help System kezeléséhez telepítse a Microsoft® Internet Explorer 6.0 vagy 7.0 változatát, illetve a Firefox 2.0 vagy 3.0 változatát. Ahhoz, hogy az IBM Eclipse Help System ezen változatának minden funkciója használható legyen, engedélyezze a cookie-kat és a JavaScript™-et a böngészőben, és tiltsa le az előugró ablakok blokkolása böngészőfunkciót.

Támogatott személyre szabás

A Help System a plugin_customization.ini fájllal szabható személyre. Ez a fő termék bedolgozó alatt található. Az indítási parancsfájlban állíthatja be.

A konfigurációs paramétereket a org.eclipse.help.base_<version number>/preferences.ini fájl szerkesztésével is módosíthatja, amely az eclipse/plugins könyvtárban található, habár ajánlott az alapértelmezett konfiguráció megtartása.

A súgórendszert a preferences.ini fájl módosításával szabhatja személyre.

Megjegyzés: Ne módosítsa az always_external_browser paramétert; máskülönben előfordulhat, hogy a súgórendszer nem működik megfelelően.
always_external_browser
Microsoft Windows operációs rendszerek esetén a paraméter alapértelmezett beállítása always_external_browser=true. Amikor elindítja a súgórendszert az önálló módban, akkor a beállítás miatt az operációs rendszer alapértelmezett böngészőjét fogja megnyitni a súgórendszerbe beágyazott böngésző helyett. Az always_external_browser=true paramétert nem kell beállítani a Windows rendszertől eltérő operációs rendszerek esetén.
