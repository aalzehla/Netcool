Просмотр информации в справочной системе

Инструкции по навигации и поиску в справочной системе с помощью кнопок панели инструментов.

Об этой задаче

  • Предварительные требования для работы со справочной системой
    Перед установкой IBM® Eclipse Help System убедитесь, что рабочая станция, операционная система, браузер и аппаратное обеспечение отвечают требованиям, описанным в этом разделе.
  • Навигация в справочной системе
    Поиск и просмотр документации продукта выполняются через графический пользовательский интерфейс справочной системы.
  • Поиск в справочной системе
    Можно искать во всех разделах справочной системы, конкретном разделе или наборе разделов; можно создать область поиска для поиска только в определенном подмножестве информации. На панели Результаты поиска показывается до 500 результатов, упорядоченных по степени соответствия.
  • Настройка подходящей области
    Рабочую область можно сузить, указав соответствующие критерии фильтрации.
  • Поиск информации на панели Индекс
    В этом разделе приведена информация о поиске разделов справки на панели Индекс.
  • Создание закладок в справочной системе
    Для упрощения доступа к разделу можно создать для его закладку и сохранить на панели Закладки справочной системы.
  • Просмотр информации на разных языках
    Для изменения языка пользовательского интерфейса справочной системы необходимо указать в параметрах операционной системы соответствующие значения локали и языка.
  • Печать в справочной системе
    В IBM Eclipse Help System предусмотрены различные способы печати справочной информации.
  • Специальные функции доступа в справочной системе
    Специальные функции доступа позволяют людям с ограниченными возможностями, например с нарушениями двигательного аппарата или ослабленным зрением, полноценно работать с программными продуктами.
  • Обновление информации в справочной системе
    Документацию следует время от времени обновлять, загружая обновления из справочной системы. Документация на сайте регулярно обновляется.
