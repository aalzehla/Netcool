Funzioni di accessibilità nel sistema della guida

Le funzioni di accessibilità aiutano gli utenti con disabilità fisiche, come ad esempio limitata mobilità o vista, o con altre speciali esigenze, a utilizzare con successo i prodotti IT (information technology).

Le funzioni di accessibilità in questo sistema di guida consentono agli utenti di effettuare le seguenti operazioni:

  • Utilizzare il software per la lettura dei dati visualizzati sullo schermo e i sintetizzatori vocali digitali per ascoltare una riproduzione audio dei dati visualizzati. Consultare la documentazione del prodotto di tecnologia assistita per avere ulteriori informazioni sull'utilizzo di tali tecnologie con informazioni basate sul linguaggio HTML.
  • Utilizzare i programmi di ingrandimento dello schermo per ingrandire i dati visualizzati sullo schermo.
  • Usufruire di funzioni specifiche o equivalenti con il solo utilizzo della tastiera.
  • Tasti di scelta rapida e acceleratori
    È possibile utilizzare i tasti o le combinazioni di tasti per eseguire le operazioni che possono essere eseguite anche mediante il mouse.
