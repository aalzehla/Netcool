Visualizar informações no sistema de ajuda

Pode obter informações sobre como navegar e efectuar procuras no sistema de ajuda, utilizando os vários botões da barra de ferramentas.

Sobre esta tarefa

  • Pré-requisitos para o sistema de ajuda
    Ao configurar o IBM® Eclipse Help System, certifique-se de que a estação de trabalho, o sistema operativo, o navegador e o hardware cumprem os requisitos descritos neste tópico.
  • Navegar no sistema de ajuda
    Utilize a interface gráfica do utilizador fornecida pelo sistema de ajuda, para localizar e visualizar a documentação do produto.
  • Procurar no sistema de ajuda
    Pode procurar todos os tópicos do sistema de ajuda, procurar um tópico individual ou conjunto de tópicos ou criar um âmbito de procura, para procurar apenas um subconjunto das informações. A vista Resultados da procura pode apresentar até 500 resultados, ordenados por relevância.
  • Definir o âmbito adequado
    Pode limitar o âmbito de trabalho, definindo os critérios de filtragem adequados.
  • Localizar informações através da vista Índice remissivo
    Este tópico descreve como trabalhar com a vista Índice remissivo para localizar tópicos no sistema de ajuda.
  • Definir marcadores no sistema de ajuda
    Pode localizar um tópico mais facilmente definindo um marcador e guardando o mesmo na vista Marcadores do sistema de ajuda.
  • Visualizar informações em diferentes idiomas
    Para visualizar a interface de utilizador do sistema de ajuda noutro idioma, tem de alterar as definições regionais do sistema operativo para o locale e o idiomas pretendidos.
  • Imprimir no sistema de ajuda
    O IBM Eclipse Help System disponibiliza várias formas para imprimir o conteúdo da ajuda dentro do sistema.
  • Funções de acessibilidade no sistema de ajuda
    As funções de acessibilidade ajudam as pessoas com incapacidade física, tal como mobilidade restrita ou visão limitada, ou que tenham outras necessidades especiais, a utilizar produtos de tecnologias de informação com êxito.
  • Actualizar informações no sistema de ajuda
    Pode actualizar com regularidade o conteúdo da documentação, caso esteja disponível, descarregando o mesmo a partir do sistema de ajuda. O conteúdo será actualizado periodicamente no sítio da Web.
