Acessibilidade e Atalhos do Teclado no Sistema de Ajuda

Os recursos de acessibilidade ajudam os usuários que possuem deficiências físicas, como mobilidade restrita ou visão limitada, a trabalhar com os produtos de software corretamente.

IBM® Eclipse Help System (IEHS) é um Componente Comum IBM. Os produtos da IBM adotam o IEHS para apresentar documentação on-line para clientes externos.

No Banco de Dados de Acessibilidade de Produto do Accessibility Center (AC) da IBM, o IEHS é categorizado como um Componente Reutilizável, pois é um componente reutilizado nos produtos IBM, ao invés de ser um componente vendido separadamente para clientes externos.
