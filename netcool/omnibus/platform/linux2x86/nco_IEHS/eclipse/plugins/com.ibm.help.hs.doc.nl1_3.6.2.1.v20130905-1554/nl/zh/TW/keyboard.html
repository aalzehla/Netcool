鍵盤輸入及導覽

您可以利用鍵盤來導覽說明系統。

鍵盤導覽

請使用下列按鍵組合，以鍵盤來導覽說明系統：

  • 在「主題」窗格中，如果要移至下一鏈結，請按 Tab 鍵。
  • 如果要展開和收合導覽樹狀結構中的某個節點，請按「右移鍵」和「左移鍵」。
  • 如果要移到下一個主題節點，請按「下移鍵」。
  • 如果要移到上一個主題節點，請按「上移鍵」。
  • 如果要從其中一個視圖內移至下一個鏈結、按鈕或主題節點，請按 Tab 鍵。
  • 如果要在窗格中向上或向下捲動到底，請按 Home 或 End 鍵。
  • 如果要向後，請按 Alt+左移鍵；如果要向前，請按 Alt+右移鍵。
  • 如果要移至下一個窗格，請按 F6。
  • 如果要移至上一個窗格，請按 Shift+F6。
  • 如果要列印作用中的窗格，請按 Ctrl+P。
  • 如果要開啟蹦現功能表，請按 Alt+M。
  • 如果要移至蹦現功能表的上一個項目，請按「上移鍵」。
  • 如果要移至蹦現功能表的下一個項目，請按「下移鍵」。
  • 如果要選取蹦現功能表上的項目，請移至項目，然後按 Enter 鍵。
  • 如果要關閉蹦現功能表，請按 ESC。
  • 如果要移至內容視圖，請按 Alt+C。（然後按 Enter 鍵，如果您使用 IE 的話）
  • 如果要移至「搜尋結果」視圖，請按 Alt+R。（然後按 Enter 鍵，如果您使用 IE 的話）
  • 如果要移至「索引」視圖，請按 Alt+I。（然後按 Enter 鍵，如果您使用 IE 的話）
  • 如果要移至「書籤」視圖，請按 Alt+Z。（然後按 Enter 鍵，如果您使用 IE 的話）

對於 JAWS 7 使用者

  • 如果要展開導覽樹狀結構中的節點，請按 Alt+O。JAWS 會說 "Alt O"。
  • 如果要收合導覽樹狀結構中的節點，請按 Alt+B。JAWS 會說 "Alt B"。
  • 如果要往下移至樹狀結構中的下一個主題節點，請按 Ctrl+Shift+下移鍵。
  • 如果要往上移至樹狀結構中的上一個主題節點，請按 Ctrl+Shift+上移鍵。
  • 如果要在窗格中向上或向下捲動到底，請按 PageUp 或 PageDown 鍵。
  • 如果要移至蹦現功能表的上一個項目，請按 Ctrl+Shift+上移鍵。
  • 如果要移至蹦現功能表的下一個項目，請按 Ctrl+Shift+下移鍵。
  • 如果要選取蹦現功能表上的項目，請移至項目，然後按 Enter 鍵。
  • 如果要展開更新程式中的文件清單，請按 Alt+O。JAWS 會說 "Alt O"。
  • 如果要收合更新程式中的文件清單，請按 Alt+B。JAWS 會說 "Alt B"。
  • 如果要在「內容」視圖、「搜尋結果」視圖、「索引」視圖及「書籤」視圖間切換，請移至導覽視圖頁框的「選項」，然後再使用上一節所說明的按鍵組合。

請使用「鍵盤導覽」一節中所記載的其他按鍵組合。

對於 Home Page Reader 3.4.1 的使用者

  • 如果要展開和收合樹狀結構中的某個節點，請按 Enter 鍵。
  • 如果要開啟蹦現功能表，請按 Ctrl+L，然後按 End 鍵，再按 Enter 鍵。
  • 如果要移至蹦現功能表的上一個項目，請按 Shift+Tab。
  • 如果要移至蹦現功能表的下一個項目，請按 Tab 鍵。
  • 如果要選取蹦現功能表上的項目，請移至項目，然後按 Enter 鍵。
  • 如果要關閉蹦現功能表，請按 Alt+Shift +H。
  • 如果要在「內容」視圖、「搜尋結果」視圖、「索引」視圖及「書籤」視圖間切換，請移至導覽視圖頁框的「選項」，然後︰
    • 如果要移至「內容」視圖，請按 Alt+Shift+C。
    • 如果要移至「搜尋結果」視圖，請按 Alt+Shift+R。
    • 如果要移至「索引」視圖，請按 Alt+Shift+I。
    • 如果要移至「書籤」視圖，請按 Alt+Shift+Z。

請使用「鍵盤導覽」一節中所記載的其他按鍵組合。
