Pré-requisitos para o sistema de ajuda

Ao configurar o IBM® Eclipse Help System, certifique-se de que a estação de trabalho, o sistema operativo, o navegador e o hardware cumprem os requisitos descritos neste tópico.

Requisitos de navegador

Para utilizar o IBM Eclipse Help System, pode instalar o Microsoft® Internet Explorer Versão 6.0 ou 7.0 ou o Firefox 2.0 ou 3.0. Para garantir que todas as funções desta versão do IBM Eclipse Help System podem ser utilizadas, tem de activar cookies e JavaScript™ no navegador e desactivar a função de bloqueio de janelas emergentes do navegador.

Requisitos do ambiente de sistema operativo

Para utilizar o IBM Eclipse Help System, tem de instalar um IBM Java™ Runtime Environment (JRE) versão 6.0. Consulte a tabela seguinte para obter uma descrição detalhada dos requisitos.

Personalização suportada

Pode configurar o sistema de ajuda com o ficheiro plugin_customization.ini. Este encontra-se no suplemento principal do produto. Pode definir o mesmo no script inicial.

Pode também alterar os parâmetros de configuração, editando o ficheiro org.eclipse.help.base_<version number>/preferences.ini que se encontra no directório eclipse/plugins, embora seja recomendada a utilização da configuração predefinida.

Pode personalizar o sistema de ajuda modificando o ficheiro preferences.ini.

Nota: Não modifique o seguinte parâmetro always_external_browser. Caso contrário, o sistema de ajuda poderá não funcionar correctamente.
always_external_browser
Para os sistemas operativos Microsoft Windows, a configuração predefinida deste parâmetro é always_external_browser=true. Ao iniciar o sistema de ajuda no modo autónomo, esta definição permite abrir o navegador predefinido do sistema operativo, em vez do navegador incorporado no sistema de ajuda. Não é necessário definir always_external_browser=true para os sistemas operativos diferentes de Windows.
