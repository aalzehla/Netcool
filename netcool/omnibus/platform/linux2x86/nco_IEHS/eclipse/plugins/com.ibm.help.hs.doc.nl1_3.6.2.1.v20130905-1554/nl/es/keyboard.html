Entrada de datos y navegación mediante el teclado

Puede navegar por el sistema de ayuda utilizando el teclado.

Navegar mediante el teclado

Utilice las siguientes combinaciones de teclas para navegar por el sistema de ayuda mediante el teclado:

  • En el panel de temas, para ir al próximo enlace, pulse la tecla de tabulación.
  • Para expandir y contraer un nodo en el árbol de navegación, pulse las teclas de flecha hacia la derecha y hacia la izquierda.
  • Para pasar al nodo del próximo tema, pulse la tecla de flecha hacia abajo.
  • Para pasar al nodo del tema anterior, pulse la tecla de flecha hacia arriba.
  • Para ir al próximo enlace, botón o nodo desde dentro de una de las vistas, pulse la tecla de tabulación.
  • Para desplazarse hasta el principio o hasta el final de un panel, pulse la tecla Inicio o la tecla Fin.
  • Para retroceder, pulse las teclas Alt+Flecha izquierda; para avanzar, pulse las teclas Alt+Flecha derecha.
  • Para ir al próximo panel, pulse F6.
  • Para pasar al panel anterior, pulse las teclas Mayús+F6.
  • Para imprimir el panel activo, pulse las teclas Control+P.
  • Para abrir el menú emergente, pulse Alt+M.
  • Para pasar al elemento anterior del menú emergente, pulse la tecla de flecha hacia arriba.
  • Para pasar al elemento siguiente del menú emergente, pulse la tecla de flecha hacia abajo.
  • Para seleccionar un elemento del menú emergente, desplácese al elemento y pulse Intro.
  • Para cerrar el menú emergente, pulse Esc.
  • Para pasar a la vista de contenido, pulse las teclas Alt+C (a continuación, si utiliza IE, pulse Intro).
  • Para pasar a la vista Resultados de la búsqueda, pulse las teclas Alt+R (a continuación, si utiliza IE, pulse Intro).
  • Para pasar a la vista Índice, pulse las teclas Alt+I (a continuación, si utiliza IE, pulse Intro).
  • Para pasar a la vista Favoritos, pulse las teclas Alt+Z (a continuación, si utiliza IE, pulse Intro).

Para los usuarios de JAWS 7

  • Para expandir un nodo del árbol de navegación, pulse Alt+O. En JAWS es "Alt O".
  • Para contraer un nodo del árbol de navegación, pulse Alt+B. En JAWS es "Alt B".
  • Para desplazarse hacia abajo al próximo nodo de tema del árbol, pulse Control+Mayús+Flecha abajo.
  • Para desplazarse hacia arriba al nodo de tema anterior del árbol, pulse Control+Mayús+Flecha arriba.
  • Para desplazarse hasta el principio o hasta el final de un panel, pulse la tecla Retroceso página o la tecla Avance página.
  • Para pasar al elemento anterior del menú emergente, pulse las teclas Control+Mayús+Flecha arriba.
  • Para pasar al elemento siguiente del menú emergente, pulse las teclas Control+Mayús+Flecha abajo.
  • Para seleccionar un elemento del menú emergente, desplácese al elemento y pulse Intro.
  • Para expandir una lista de documentación en el actualizador, pulse Alt+O. En JAWS es "Alt O".
  • Para contraer una lista de documentación en el actualizador, pulse Alt+B. En JAWS es "Alt B".
  • Para conmutar entre las vistas Contenido, Resultados de la búsqueda, Índice y Favoritos, vaya al marco Selección de vistas de navegación y utilice las combinaciones de teclas indicadas en la sección anterior.

Utilice las demás combinaciones de teclas según se indica en la sección Navegar mediante el teclado.

Para los usuarios de Home Page Reader 3.4.1

  • Para expandir o contraer un nodo en la estructura en árbol, pulse Intro.
  • Para abrir el menú emergente, primero pulse Control+L, después Fin y por último Intro.
  • Para pasar al elemento anterior del menú emergente, pulse Mayús+Tabulador.
  • Para pasar al elemento siguiente del menú emergente, pulse Tabulador.
  • Para seleccionar un elemento del menú emergente, desplácese al elemento y pulse Intro.
  • Para cerrar el menú emergente, pulse Alt+Mayús+H.
  • Para conmutar entre las vistas Contenido, Resultados de la búsqueda, Índice y Favoritos, vaya al marco Selección de vistas de navegación y, a continuación:
    • Para pasar a la vista Contenido, pulse las teclas Alt+Mayús+C.
    • Para pasar a la vista Resultados de la búsqueda, pulse las teclas Alt+Mayús+R.
    • Para pasar a la vista Índice, pulse las teclas Alt+Mayús+I.
    • Para pasar a la vista Favoritos, pulse las teclas Alt+Mayús+Z.

Utilice las demás combinaciones de teclas según se indica en la sección Navegar mediante el teclado.
