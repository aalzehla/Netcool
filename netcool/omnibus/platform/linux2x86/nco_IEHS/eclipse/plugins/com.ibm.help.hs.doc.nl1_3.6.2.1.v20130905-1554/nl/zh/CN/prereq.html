帮助系统的必备要求

在安装 IBM® Eclipse Help System 时，请确保您的工作站、操作系统、浏览器和硬件均满足本主题中所述的要求。

浏览器要求

要使用 IBM Eclipse Help System，您可以安装 Microsoft® Internet Explorer V6.0 或 V7.0 或安装 Firefox 2.0 或 3.0。为了确保可以使用该版本 IBM Eclipse Help System 的所有功能，您必须在浏览器中启用 cookie 和 JavaScript™，并且禁用浏览器的阻止弹出窗口功能。

操作系统环境要求

要使用 IBM Eclipse Help System，必须安装 IBM Java™ Runtime Environment (JRE) V6.0。 请参阅下表，了解具体要求。

表 1. 操作系统环境要求
IBM Eclipse Help System 版本 操作系统 操作系统和 JRE 要求
适用于 Microsoft Windows®（32 位）的 IBM Eclipse Help System
  • Windows XP Professional（Intel 32 位）
  • 该版本的 IBM Eclipse Help System 包含嵌入式 IBM JRE。如果不想保留嵌入式 JRE，可以将其从软件包中除去。必须在系统上安装 JRE 6.0，才能运行 IBM Eclipse Help System。
适用于 Linux® 的 IBM Eclipse Help System
  • Red Hat Linux Enterprise 5（Intel® 32 位）

  • 在这些操作系统上，您可以通过 Konqueror 浏览器在基本模式下打开帮助系统。
  • 要在帮助系统中显示双向字符、双字节字符或多字节字符，必须在系统上安装相应的字体，并将其设置为浏览器的显示字体。

支持定制

您可以使用 plugin_customization.ini 文件来配置该帮助系统。它在主产品插件下。您可以在启动脚本中对它进行设置。

虽然强烈建议使用缺省配置，但您也可以通过编辑 eclipse/plugins 目录下的 org.eclipse.help.base_<version number>/preferences.ini 文件来更改配置参数。

您可以通过修改 preferences.ini 文件定制帮助系统。

注： 请勿修改参数 always_external_browser；否则，帮助系统可能会无法正常工作。
always_external_browser
对于 Microsoft Windows 操作系统，该参数的缺省配置是 always_external_browser=true。 在单机模式下启动帮助系统时，该设置可以帮助您打开操作系统的缺省浏览器，而不是帮助系统中嵌入的浏览器。对于除 Windows 之外的其他操作系统，您都无需设置 always_external_browser=true。
