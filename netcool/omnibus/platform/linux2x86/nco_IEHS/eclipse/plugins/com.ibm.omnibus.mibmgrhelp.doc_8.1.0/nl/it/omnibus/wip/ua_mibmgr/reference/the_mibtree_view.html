La vista Moduli MIB

La vista Moduli MIB fornisce una rappresentazione di tutti i moduli MIB importati e consente di ricercare moduli MIB, di eliminare moduli MIB e di ricercare valori MIB specifici.

È possibile spostarsi nella vista Moduli MIB espandendo e riducendo i nodi all'interno della struttura ad albero MIB. le diramazioni di primo livello della struttura ad albero MIB contengono i nomi dei moduli MIB e, contenuti all'interno di ciascuna diramazione di modulo MIB, si travano gli altri elementi che costituiscono il MIB.

È possibile filtrare i dati nella vista Moduli MIB selezionando una delle seguenti opzioni dall'elenco a discesa Visualizza:
  • Tutti
  • Moduli
  • Enumerazioni
  • Convenzioni testuali
  • Sintassi
  • Importazioni
  • Esportazioni
  • Colonne
  • Descrizioni

Per ricercare nella vista Moduli MIB un valore specifico, immettere il testo nel campo Cerca e premere Invio oppure fare clic su (Cerca).

MIB Manager inizia la ricerca dall'elemento selezionato oppure, se nessun elemento è attualmente selezionato, dal primo elemento nell'elenco. Per trovare l'elemento successivo corrispondente nell'elenco, continuare a fare clic su (Cerca).

Quando viene individuato un elemento, questo elemento viene evidenziato automaticamente e visualizzato nella vista Moduli MIB.

Nel campo Cerca, specificare un'espressione regolare valida o immettere un testo semplice per rappresentare un nome oggetto completo o parziale.

Per eliminare uno o più moduli MIB, fare clic con il pulsante destro del mouse sui moduli MIB selezionati, quindi, dal menu a discesa, selezionare Elimina. Il modulo MIB selezionato viene eliminato da MIB Manager. Tuttavia, i file di origine MIB originali, forniti per l'importazione MIB, non vengono eliminati.

Nota: è necessario eliminare un modulo MIB se contiene errori ed esiste una nuova versione di sostituzione del modulo MIB. L'eliminazione di moduli MIB utilizzando questo metodo può determinare dipendenze MIB mancanti. File delle regole incompleti possono essere inoltre generati durante il processo di esportazione insieme a messaggi di avvertenza o di errore nel file di log.
