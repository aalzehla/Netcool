Establecimiento de las preferencias generales

Utilice las preferencias generales para especificar las opciones de importación de MIB.

Para establecer las preferencias de directorio, pulse Preferencias o, desde el menú principal, pulse Archivo > Preferencias y, a continuación, en el panel de la izquierda de la ventana Preferencias, pulse General.

  • Si se selecciona el recuadro de selección Recorrer subdirectorios en la importación, se explorarán todos los subdirectorios durante el proceso de importación para localizar los archivos MIB.
  • En el campo Número máximo de hebras de divisor de archivo, especifique el número de hebras computacionales que se utilizarán para detectar las MIB durante el proceso de importación. Esto puede mejorar el rendimiento al importar un gran número de MIB. El valor predeterminado es 5.
  • En el campo Número máximo de hebras de analizador MIB, especifique el número de hebras computacionales que se utilizarán para analizar los archivos MIB durante el proceso de importación. Esto puede mejorar el rendimiento al importar un gran número de MIB. El valor predeterminado es 10.
  • Para cambiar el color de resaltado de las selecciones realizadas en la ventana de MIB Manager, pulse , seleccione un color de la paleta de colores y, a continuación, pulse Aceptar.
