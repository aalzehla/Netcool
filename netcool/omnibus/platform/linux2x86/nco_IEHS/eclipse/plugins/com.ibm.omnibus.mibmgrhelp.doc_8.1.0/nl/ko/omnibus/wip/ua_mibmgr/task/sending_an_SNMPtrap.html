IBM Tivoli Netcool/OMNIbus 버전 8.1

SNMP 트랩 생성

SNMP 트랩을 생성하는 방법입니다. MIB Manager에서 생성된 규칙 파일을 테스트하는 데에도 이 기능을 사용할 수 있습니다.

프로시저

SNMP 트랩을 생성하려면 다음을 수행하십시오.

  1. 트랩을 클릭하십시오.
  2. 대상 주소 필드에서 SNMP Trapd 서비스를 실행하는 컴퓨터의 호스트 이름 또는 IP 주소를 지정하십시오.
  3. 대상 포트 필드에서 SNMP Trapd 서비스가 인식하는 UDP 포트를 지정하십시오.
  4. 커뮤니티 필드에서 Trapd 관리자에 대해 유효한 문자열을 지정하십시오.
  5. 생성하는 트랩의 유형에 따라, 가변 바인딩 섹션이 SNMP 트랩 생성 창에 표시될 수 있습니다. 가변 바인딩 섹션에서 사용 가능한 필드도 생성하는 트랩 유형에 따라 달라집니다.

    필드 값 구문에 대한 세부사항을 보려면 마우스를 가변 바인딩 필드 이름 위에 두십시오. 쉼표로 구분하여 여러 값을 지정할 수 있습니다. 이 값은 트랩을 빌드할 때 랜덤으로 선택됩니다.

  6. 반복 지표 필드에서는 생성될 수 있는 트랩 수(1 - 1000)를 지정하십시오.

    큰 수를 지정하면, 신속하게 연속해서 트랩이 생성되므로 Trapd 관리자에서 알람 스톰이 발생할 수 있습니다. 값이 1보다 크거나(> 1) 세미콜론으로 구분된 목록으로 지정되는 경우 MIB Manager가 지정된 목록에서 무작위로 선택하고 임의 값을 varbind로 표시하여 트랩을 생성합니다.

  7. 실행을 클릭하여 트랩을 생성하십시오.

결과

SNMP 트랩 생성 창의 결과 필드가 트랩이 성공적으로 전송되었는지 여부를 표시합니다.
