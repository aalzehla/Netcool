Задание предпочтений экспорта

В этом разделе описывается, как задать предпочтения экспорта файла правил.

Для задания предпочтений каталогов щелкните по значку Предпочтения или выберите пункт главного меню Файл > Предпочтения, а затем на левой панели окна Предпочтения щелкните по заголовку Экспорт.

  • В поле Путь включения укажите положение правил, установленных с пакетами файлов правил Netcool Include Library (NCiL) (если они были установлены). Эта опция требуется только в том случае, если данные экспортируются в формате файлов Netcool Include Library и путь каталога должен указывать на включенные в пакет правила. В результате новые файлы правил не нужно изменять вручную. Значение по умолчанию - ДОМАШНИЙ_КАТАЛОГ_NC/omnibus/probes/архитектура.
  • В поле Формат файлов экспорта по умолчанию выберите для файла вывода или формат Windows/DOS (CRLF), или формат Unix (LF). Это свойство позволяет создавать правила на компьютере Windows и затем копировать их на компьютер UNIX, не запуская специальную утилиту, такую как dos2unix.
    Прим.: Этот выбор применяется ко всем файлам вывода и не ограничен только файлами правил. Значение по умолчанию - формат Unix.
  • В поле Типы оповещений укажите опции выпадающих списков, показываемых при редактировании поля прерываний @Type в представлении Подробности.

Следующие опции доступны только в том случае, если ранее был выбран формат экспорта NCKL:

  • Если включен переключатель Включить описания varbind, описания varbind добавляются как документация комментариев. Значение по умолчанию - enabled.
  • Если включен переключатель Вычислять лучшее значение для @AlertKey, значение @AlertKey генерируется из родительского объекта varbind. Значение по умолчанию - enabled.
  • Если включен переключатель Задать @Agent для каждого предприятия, поле @Agent задается для каждого предприятия. Поле @Agent основано на именах модулей MIB, обрабатываемых для файла правил. Значение по умолчанию - enabled.
  • Если включен переключатель Добавить шаблон @Class, задается поле @Class. Если включен переключатель Добавить шаблон @Class, в поле @Class задается значение 300. Значение по умолчанию - enabled.
