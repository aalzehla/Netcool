IBM Tivoli Netcool/OMNIbus, מהדורה 7.4

הגדרת העדפות מדריך

השתמשו בהעדפות המדריך כדי לציין את המדריכים המשמשים לאחסון קובצי MIB שיובאו, נתוני MIB, וקובצי כללים שיובאו.

כדי להגדיר את העדפות המדריך, לחצו על העדפות ולחילופין, מהתפריט, לחצו על קובץ > העדפות, ולאחר מכן מהחלונית השמאלית בחלון העדפות, לחצו על מדריך.

  • בשדה מדריך נתונים, ציינו את המיקום שבו רכיבי ה-MIB המיובאים יאוחסנו בתור קובצי XML. מיקום ברירת המחדל הוא $NCHOME/omnibus/platform/arch/mibmanager/workspace/data.
  • בשדה מדריך יצוא, ציינו את מקום הפקת הקבצים שיוצאו. תת-מדריך ייווצר בהתאם לסוג הייצוא. מיקום ברירת המחדל הוא $NCHOME/omnibus/platform/arch/mibmanager/workspace/export.
  • בשדה מדריך יבוא, ציינו את המיקום שבו ממוקמים קובצי ה-MIB שיובאו. מיקום ברירת המחדל הוא $NCHOME/omnibus/platform/arch/mibmanager/workspace/mibs.
