SNMP-Traps bearbeiten

Vorgehensweise beim Bearbeiten von SNMP-Traps.

Informationen zu diesem Vorgang

In der Sicht Details können Sie die folgenden Felder eines SNMP-Traps bearbeiten:
  • @Severity
  • @Type
  • @ExpireTime
  • CODEBLOCK
Anmerkung: SNMP umfasst keine Standardmechanismen für die Angabe von SET- (@Type) oder CLEAR-Informationen (@Severity). Diese Informationen müssen manuell angegeben werden.

Doppelklicken Sie in der Sicht Details auf einen Feldwert, um ihn bearbeiten zu können. Klicken Sie nach dem Vornehmen einer Änderung an einem Feld auf , um die Änderung zu speichern, oder auf , um sie abzubrechen.

Vorgehensweise

So bearbeiten Sie einen SNMP-Trap:

  1. Wählen Sie in der Sicht MIB-Module oder OID-Baumstruktur einen Trap aus.

    Detaillierte Informationen zu dem Trap werden in der Sicht Details angezeigt.

  2. Doppelklicken Sie zum Bearbeiten des Felds @Severity auf den Feldwert und wählen Sie aus der Dropdown-Liste einen neuen Wert aus.
  3. Doppelklicken Sie zum Bearbeiten des Felds @Type auf den Feldwert und wählen Sie aus der Dropdown-Liste einen neuen Wert aus.
  4. Doppelklicken Sie zum Bearbeiten des Felds @ExpireTime auf den Feldwert und geben Sie einen neuen Zeitraum (in Sekunden, Minuten, Stunden oder Tagen) ein.
  5. Doppelklicken Sie zum Bearbeiten des Felds CODEBLOCK auf den Feldwert (auch wenn das Feld leer ist) und geben Sie einen neuen Codeblock ein.
