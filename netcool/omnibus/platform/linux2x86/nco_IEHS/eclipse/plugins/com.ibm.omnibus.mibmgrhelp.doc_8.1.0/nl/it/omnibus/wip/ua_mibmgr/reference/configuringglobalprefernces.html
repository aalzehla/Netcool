Configurazione di preferenze globali

È possibile configurare diverse preferenze globali di MIB Manager dalla finestra Preferenze.

Per configurare le preferenze globali di MIB Manager, fare clic su Preferenze oppure, dal menu, fare clic su File > Preferenze.

Nella finestra Preferenze, è possibile configurare preferenze di directory, esportazione, generali, registrazione e ricerca.

  • Impostazione di preferenze di directory
    Utilizzare le preferenze di directory per specificare le directory utilizzate per memorizzare i MIB importati, i dati MIB e i file delle regole esportati.
  • Impostazione di preferenze di esportazione
    Questa sezione descrive come impostare le preferenze di esportazione dei file delle regole.
  • Impostazione di preferenze generali
    Utilizzare le preferenze generali per specificare le opzioni di importazione MIB.
  • Impostazione di preferenze di registrazione
    Utilizzare le preferenze di registrazione per specificare la directory dei file di log MIB Manager e per impostare il livello dei messaggi predefinito.
  • Impostazione di preferenze di ricerca
    Questa sezione descrive la procedura di definizione delle preferenze MIB Manager utilizzate per ricercare i file MIB.
