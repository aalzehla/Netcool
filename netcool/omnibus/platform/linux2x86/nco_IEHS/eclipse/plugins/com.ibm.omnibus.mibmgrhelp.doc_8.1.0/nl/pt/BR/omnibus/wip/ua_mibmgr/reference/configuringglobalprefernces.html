Configurando preferências globais

É possível configurar diversas preferências globais do MIB Manager a partir da janela Preferências.

Para configurar as preferências globais do MIB Manager, clique em Preferências ou, no menu, clique em Arquivo > Preferências.

Na janela Preferências, é possível configurar as preferências do diretório, exportação, geral, criação de log e procura.

  • Configurando as preferências de diretório
    Use as preferências do diretório para especificar os diretórios usados para armazenar MIBs importadas, dados da MIB e arquivos de regras exportados.
  • Configurando as preferências de exportação
    Esta seção descreve como configurar as preferências para a exportação do arquivo de regras.
  • Configurando Preferências Gerais
    Use as preferências gerais para especificar as opções de importação do MIB.
  • Configurando as preferências de criação de log
    Use as preferências de criação de log para especificar o diretório de arquivo de log do MIB Manager e para configurar o nível de mensagem padrão.
  • Configuração de preferências de procura
    Esta seção descreve como definir as preferências do MIB Manager que são usadas para procurar os arquivos MIB.
