IBM Tivoli Netcool/OMNIbus 버전 8.1

글로벌 환경 설정 구성

환경 설정 창에서 여러 MIB Manager 글로벌 환경 설정을 구성할 수 있습니다.

MIB Manager 글로벌 환경 설정을 구성하려면 환경 설정을 클릭하거나 메뉴에서 파일 > 환경 설정을 클릭하십시오.

환경 설정 창에서 디렉토리, 내보내기, 일반, 로깅, 검색 환경 설정을 구성할 수 있습니다.

  • 디렉토리 환경 설정 설정
    디렉토리 환경 설정을 사용하여 가져온 MIB, MIB 데이터, 내보낸 규칙 파일을 저장하는 데 사용되는 디렉토리를 지정합니다.
  • 내보내기 환경 설정 설정
    이 절에서는 규칙 파일 내보내기 환경 설정의 설정 방법에 대해 설명합니다.
  • 일반 환경 설정 설정
    일반 환경 설정을 사용하여 MIB 가져오기 옵션을 지정합니다.
  • 로깅 환경 설정 설정
    로깅 환경 설정을 사용하여 MIB Manager 로그 파일 디렉토리를 지정하고 기본 메시지 레벨을 설정합니다.
  • 검색 환경 설정 설정
    이 절에서는 MIB 파일을 검색하는 데 사용되는 MIB Manager 환경 설정을 정의하는 방법을 설명합니다.
상위 주제: Netcool MIB Manager
