Impostazione di preferenze di ricerca

Questa sezione descrive la procedura di definizione delle preferenze MIB Manager utilizzate per ricercare i file MIB.

Per impostare le preferenze di ricerca, fare clic su Preferenze oppure, dal menu, fare clic su File > Preferenze, quindi dal pannello di sinistra della finestra Preferenze, fare clic su Cerca.

Nel campo Percorso di ricerca MIB, impostare il percorso di ricerca in cui si trovano i file MIB. Le directory devono essere separate da un carattere di punto e virgola (;). In tutte le sottodirectory viene effettuata una ricerca automatica per individuare i file MIB dipendenti.

Il percorso predefinito include l'ubicazione dei file MIB di base e dei file MIB RFC forniti con MIB Manager. È possibile specificare qualsiasi nuova directory alla fine del percorso file predefinito. Ciò garantisce che nella directory di base vengano ricercate le dipendenze contenenti i MIB più comunemente utilizzati e nella directory rfc vengano ricercati i MIB meno comunemente utilizzati.
