Impostazione di preferenze di directory

Utilizzare le preferenze di directory per specificare le directory utilizzate per memorizzare i MIB importati, i dati MIB e i file delle regole esportati.

Per impostare le preferenze di directory, fare clic su Preferenze oppure, dal menu, fare clic su File > Preferenze, quindi dal pannello di sinistra della finestra Preferenze, fare clic su Directory.

  • Nel campo Directory di dati, specificare il percorso in cui i MIB importati verranno memorizzati come file XML. L'ubicazione predefinita è NCHOME/omnibus/var/mibmanager/data.
  • Nel campo Directory di esportazione, specificare il percorso in cui vengono generati i file esportati. Viene creata una sottodirectory basata sul tipo di esportazione. L'ubicazione predefinita è NCHOME/omnibus/var/mibmanager/export.
  • Nel campo Directory di importazione, specificare il percorso in cui vengono inseriti i file MIB importati. L'ubicazione predefinita è NCHOME/omnibus/var/mibmanager/mibs.
