設定搜尋喜好設定

本節說明如何定義用於搜尋 MIB 檔案的 MIB Manager 喜好設定。

若要設定搜尋喜好設定，請按一下 喜好設定，或從主功能表中按一下檔案 > 喜好設定，然後從「喜好設定」視窗的左窗格中，按一下搜尋。

在 MIB 搜尋路徑欄位中，設定 MIB 檔案所在位置的搜尋路徑。目錄必須用分號 (;) 區隔。 系統會自動搜尋所有子目錄以尋找相依的 MIB 檔案。

預設路徑包含隨 MIB Manager 一起包裝的基本 MIB 檔案及 RFC MIB 檔案的位置。 可以在預設檔案路徑結尾指定任何新的目錄。如此可確保會起始搜尋基本目錄中包含最常用 MIB 的相依關係，然後搜尋 rfc 目錄中不常用的 MIB。
