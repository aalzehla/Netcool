Benutzervorgaben für Verzeichnisse festlegen

Mithilfe der Benutzervorgaben für Verzeichnisse können Sie die Verzeichnisse zum Speichern importierter MIBs, von MIB-Daten und exportierter Regeldateien angeben.

Klicken Sie zum Festlegen der Benutzervorgaben für Verzeichnisse auf Benutzervorgaben oder klicken Sie im Menü auf Datei > Benutzervorgaben und dann im linken Bereich des Fensters Benutzervorgaben auf Verzeichnis.

  • Geben Sie im Feld Datenverzeichnis an, an welcher Position die importierten MIBs als XML-Dateien gespeichert werden. Die Standardposition ist NCHOME/omnibus/var/mibmanager/data.
  • Geben Sie im Feld Exportverzeichnis an, an welcher Position die exportierten Dateien generiert werden. Daraufhin wird ein Unterverzeichnis auf der Basis des Exporttyps erstellt. Die Standardposition ist NCHOME/omnibus/var/mibmanager/export.
  • Geben Sie im Feld Importverzeichnis an, an welcher Position die importierten MIB-Dateien gespeichert werden. Die Standardposition ist NCHOME/omnibus/var/mibmanager/mibs.
