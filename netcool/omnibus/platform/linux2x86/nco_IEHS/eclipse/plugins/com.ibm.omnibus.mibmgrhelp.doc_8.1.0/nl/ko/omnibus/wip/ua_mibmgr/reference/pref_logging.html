로깅 환경 설정 설정

로깅 환경 설정을 사용하여 MIB Manager 로그 파일 디렉토리를 지정하고 기본 메시지 레벨을 설정합니다.

디렉토리 환경 설정을 설정하려면, 환경 설정을 클릭하거나 기본 메뉴에서 파일 > 환경 설정을 클릭한 후 환경 설정 창의 왼쪽 분할창에서 로깅을 클릭하십시오.

  • 로그 디렉토리 필드에서 애플리케이션 로그 파일이 기록되는 위치를 지정하십시오. 기본 위치는 NCHOME/omnibus/log입니다.
  • 기본 메시지 레벨 필드에서 애플리케이션의 기본 메시지 레벨을 지정하십시오. 또한 편집 메뉴를 사용하여 기본 메시지 로깅 레벨을 변경할 수도 있습니다. 메시지 레벨은 편집 메뉴를 사용하여 변경하는 경우가 아니면 애플리케이션이 처음 시작될 때부터 애플리케이션이 종료할 때까지 유효합니다.
    참고: 기본 메시지 레벨 필드에서 메시지 레벨을 변경해도 애플리케이션의 현재 메시지 레벨은 변경되지 않습니다. 애플리케이션이 처음(기본 메시지 레벨) 시작될 때 초기에 설정된 메시지 레벨에만 적용됩니다. 대형 로그 파일이 생성되지 않도록 하기 위해 기본 메시지 레벨을 경고로 설정하십시오.
  • 로깅 레벨 설정
    로깅 레벨을 변경하여 MIB Manager를 디버깅하는 데 유용한 자세한 로그 또는 추적 파일을 제공할 수 있습니다. 로깅 레벨을 올려 제공되는 기타 정보는 MIB 구문 분석 시 특정 문제점을 디버깅할 때 도움이 될 수 있습니다.
