配置廣域喜好設定

您可以從「喜好設定」視窗配置數個 MIB Manager 廣域喜好設定。

若要配置 MIB Manager 廣域喜好設定，請按一下 喜好設定，或從功能表中按一下檔案 > 喜好設定。

在「喜好設定」視窗中，您可以配置目錄、匯出、一般、記載及搜尋喜好設定。

  • 設定目錄喜好設定
    使用目錄喜好設定來指定目錄，用來儲存匯入的 MIB、MIB 資料以及匯出的規則檔案。
  • 設定匯出喜好設定
    本節說明如何設定規則檔匯出喜好設定。
  • 設定一般喜好設定
    使用一般喜好設定來指定 MIB 匯入選項。
  • 設定記載喜好設定
    使用記載喜好設定來指定 MIB Manager 日誌檔目錄，以及設定預設訊息層次。
  • 設定搜尋喜好設定
    本節說明如何定義用於搜尋 MIB 檔案的 MIB Manager 喜好設定。
