Tivoli Netcool/OMNIbus MIB Manager

Tivoli Netcool/OMNIbus MIB Manager הוא יישום המבוסס על IBM® Eclipse שיכול לשמש אתכם בניתוח קבצים של בסיס מידע ניהולי (MIB) של פרוטוקול ניהול רשת פשוט (Simple Network Management‏,SNMP), שמהם תוכלו להפיק קובצי כללים של Netcool. הוא אמור להחליף את תוכנית השירות mib2rules.

Tivoli Netcool/OMNIbus MIB Manager הוא בעל התכונות הבאות:
  • הוא מייבא קובצי SNMP MIB ומפענח יחסי תלות של MIB כדי לבנות עץ מזהה אובייקטים (OID).
  • הוא יכול לייצא חלק מהאובייקטים של SNMP שיובאו או את כולם לקובצי הכללים של Netcool ולמבני קבצים אחרות.
  • הוא כולל את רכיבי ה-MIB הבסיסיים וכן את רכיבי ה-RFC MIB הנפוצים ביותר בשימוש על-ידי קובצי MIB אחרים.
  • הוא כולל הגדרות התקן המאפשרות קיבוץ ממורכז התקן של מודולים של MIB שיובאו.
  • הוא יכול לסנן ולבצע חיפוש במודולים של MIB ובעץ ה-OID באמצעות שם אובייקט או OID.
  • הוא יכול להפיק לכידות SNMP לשליחה לבדיקת SNMP‏ (nco_p_mttrapd) או לסוכני SNMP אחרים.
  • הפעלת MIB Manager
    ניתן להפעיל את MIB Manager משורת הפקודה. הוא לא אמור לרוץ תחת בקרת תהליך או בתור שירות של Windows.
  • Tivoli Netcool/OMNIbus MIB Manager
    Tivoli Netcool/OMNIbus MIB Manager הוא יישום המבוסס על IBM Eclipse שיכול לשמש אתכם בניתוח קבצים של בסיס מידע ניהולי (MIB) של פרוטוקול ניהול רשת פשוט (Simple Network Management‏,SNMP), שמהם תוכלו להפיק קובצי כללים של Netcool. הוא אמור להחליף את תוכנית השירות mib2rules.
  • שימוש ב-MIB Manager
    ניתן להשתמש ב-MIB Manager לניתוח מודולים של MIB וליצירת קובצי כללים של Netcool.
  • שימוש בהתקנים של MIB Manager
    התקן של MIB Manager משמש לקיבוץ מודולים משויכים של MIB ולהגדרת כל האובייקטים הדרושים לייצוא.
  • קביעת תצורה של העדפות גלובליות
    ניתן לקבוע תצורה של מספר העדפות גלובליות של MIB Manager מחלון העדפות.
  • אודות SNMP
    מקטע זה מספק מידע נוסף אודות הארכיטקטורה של Simple Network Management Protocol ‏(SNMP), בסיס המידע הניהולי של SNMP‏ (MIB), ומושגים נוספים בנושא MIB.
