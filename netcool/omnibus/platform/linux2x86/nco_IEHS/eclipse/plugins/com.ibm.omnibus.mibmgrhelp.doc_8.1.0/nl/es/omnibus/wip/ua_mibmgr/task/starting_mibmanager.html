IBM Tivoli Netcool/OMNIbus Versión 8.1

Inicio de MIB Manager

Inicie la interfaz gráfica de usuario de MIB Manager desde la línea de mandatos. No está diseñado para ejecutarse bajo el control de procesos o como un servicio de Windows.

Acerca de esta tarea

Consulte la documentación sobre los sistemas operativos soportados para obtener información sobre los requisitos para ejecutar MIB Manager en los sistemas operativos AIX y Solaris.

Asegúrese de que la variable de entorno $DISPLAY se establece en un servidor de X11 en funcionamiento antes de iniciar MIB Manager.

Procedimiento

  1. Utilice el mandato siguiente para iniciar la interfaz gráfica de usuario de MIB Manager:

    NCHOME/omnibus/bin/nco_mibmanager

  2. Utilice el mandato siguiente para iniciar el programa de utilidad de línea de mandatos de MIB Manager:

    NCHOME/omnibus/bin/nco_mibmanager_batch [opciones]

  • Opciones de línea de mandatos de MIB Manager
    MIB Manager tiene un programa de utilidad de línea de mandatos que se puede utilizar para importar datos de MIB y exportar archivos de reglas. Puede emitir mandatos manualmente o llamarlos de forma programática.
