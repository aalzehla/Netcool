Задание предпочтений поиска

В этом разделе описывается, как определить предпочтения MIB Manager, используемые для поиска файлов MIB.

Для задания предпочтений поиска щелкните по значку Предпочтения или выберите пункт главного меню Файл > Предпочтения, а затем на левой панели окна Предпочтения щелкните по заголовку Поиск.

В поле Путь поиска MIB задайте путь поиска, где располагаются файлы MIB. Каталоги нужно разделять символами точки с запятой (;). Поиск всех подкаталогов производится автоматически для обнаружения зависимых файлов MIB.

Путь по умолчанию включает в себя положение основных файлов MIB и файлов RFC MIB, включенных в пакет MIB Manager. В конце пути к файлу по умолчанию можно указать любые новые каталоги. Это обеспечивает первоначальный поиск в основном каталоге зависимостей, содержащих наиболее часто используемые MIB, а затем поиск в каталоге rfc реже используемых MIB.
