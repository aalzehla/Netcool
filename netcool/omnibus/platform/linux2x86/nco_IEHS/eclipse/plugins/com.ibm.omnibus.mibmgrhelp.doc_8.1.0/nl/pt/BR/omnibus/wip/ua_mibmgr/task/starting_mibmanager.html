Iniciando o MIB Manager

Inicie a GUI do MIB Manager a partir da linha de comandos. Não é projetado para ser executado sob um controle de processo ou como um serviço do Windows.

Sobre Esta Tarefa

Consulte a documentação de Sistemas Operacionais Suportados para obter informações sobre os requisitos para executar o MIB Manager nos sistemas operacionais AIX e Solaris.

Assegure-se de que a variável de ambiente $DISPLAY esteja configurada para um servidor X11 em funcionamento antes de iniciar o MIB Manager.

Procedimento

  1. Use o comando a seguir para iniciar a GUI do MIB Manager:

    NCHOME/omnibus/bin/nco_mibmanager

  2. Use o comando a seguir para iniciar o utilitário de linha de comandos do MIB Manager:

    NCHOME/omnibus/bin/nco_mibmanager_batch [options]

  • Opções da linha de comandos do MIB Manager
    O MIB Manager possui um utilitário de linha de comandos que pode ser usado para importar dados MIB e exportar arquivos de regras. É possível emitir comandos manualmente ou chamá-los programaticamente.
