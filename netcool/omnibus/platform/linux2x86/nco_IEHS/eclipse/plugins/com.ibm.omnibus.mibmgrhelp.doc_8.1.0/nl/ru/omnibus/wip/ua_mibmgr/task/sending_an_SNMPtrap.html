Генерирование прерываний SNMP

Как генерировать прерывания SNMP. Эта функциональность может использоваться также для проверки файлов правил, сгенерированных MIB Manager.

Процедура

Чтобы сгенерировать прерывание SNMP:

  1. Выберите Прерывание.
  2. В поле Адрес назначения задайте имя хоста или IP-адрес компьютера, где запущена служба SNMP Trapd.
  3. В поле Порт назначения задайте порт UDP, на котором ведет прием служба SNMP Trapd.
  4. В поле Сообщество задайте допустимую строку для менеджера Trapd.
  5. В зависимости от типа генерируемого прерывания в окне Генерировать прерывание SNMP может появиться раздел Связывания переменных. Доступные поля в разделе Связывания переменных также зависят от типа генерируемого прерывания.

    Наводите указатель мыши на имена полей в разделе Связывания переменных, чтобы узнать подробности синтаксиса для переменной в каждом поле. Можно задавать несколько значений, разделенных запятыми. Эти значения будут выбираться случайным образом для построения прерывания.

  6. В поле Показатель повторения задайте количество прерываний (от 1 до 1000), которые можно сгенерировать.

    Если указать большое число, прерывания будут генерироваться в быстрой последовательности, что может вызвать наплыв оповещений менеджера Trapd. Для значений, больших единицы (>1), и для значений, заданных в разделенных точками с запятой списках, MIB Manager произвольным образом выбирает значения из заданных списков и генерирует прерывания со связываниями переменных, представленными произвольными значениями.

  7. Нажмите кнопку Выполнить, чтобы сгенерировать прерывание.

Результаты

В поле Результат в окне Генерировать прерывание SNMP будет указано, было ли прерывание успешно сгенерировано.
