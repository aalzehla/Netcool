Задание предпочтений каталогов

Используйте предпочтения каталогов для задания каталогов, применяемых для хранения импортированных MIB, данных MIB и экспортируемых файлов правил.

Для задания предпочтений каталогов щелкните по значку Предпочтения или выберите пункт меню Файл > Предпочтения, а затем на левой панели окна Предпочтения щелкните по заголовку Каталог.

  • В поле Каталог данных укажите, где будут храниться MIB, импортированные в виде файлов XML. Положение по умолчанию - ДОМАШНИЙ_КАТАЛОГ_NC/omnibus/var/mibmanager/data.
  • В поле Каталог экспорта задайте, где генерируются экспортируемые файлы. Подкаталог основан на типе созданного экспорта. Положение по умолчанию - ДОМАШНИЙ_КАТАЛОГ_NC/omnibus/var/mibmanager/export.
  • В поле Каталог импорта задайте, где расположены импортируемые файлы MIB. Положение по умолчанию - ДОМАШНИЙ_КАТАЛОГ_NC/omnibus/var/mibmanager/mibs.
