Establecimiento del nivel de registro cronológico

Puede cambiar el nivel de registro cronológico para proporcionar archivos de registro o de rastreo más detallados para facilitar la depuración de MIB Manager. La información adicional que se proporciona aumentando los niveles de registro cronológico puede ayudarle en la depuración de problemas específicos al analizar las MIB.

Para cambiar el nivel de registro cronológico, pulse Editar > Depurar en la barra de menú de la ventana principal. Los archivos de registro tienen seis niveles de mensaje por los que se puede pasar para aumentar el nivel del detalle que se captura: ninguno, error, aviso, info, depuración y detalle.

En la tabla siguiente se proporciona una descripción breve de cada nivel de registro cronológico:

Tabla 1. Lista de niveles de registro cronológico con su descripción asociada
Nivel de registro Descripción
ninguno

Especifica un nivel de registro en el que no se da salida a mensajes de depuración.

Se mejora el rendimiento de la aplicación pero no se muestran mensajes de aviso ni de error durante el análisis.
error

Especifica un nivel de registro en el que sólo se registran los errores que afectan a los resultados.

Esto incluye los errores identificados por MIB Manager que harían que se mostraran datos incorrectos en la ventana principal.
aviso

Especifica un nivel de registro en el que se registran mensajes de aviso y de error.

Un aviso indica una condición que podría originar datos erróneos y, por tanto, se debe verificar. Sin embargo, la condición no causará que se corrompan los datos.
info

Especifica un nivel de registro en el que se registran mensajes informativos, de aviso y críticos.

Los mensajes informativos contienen detalles relativos al progreso general de las tareas. Por ejemplo, proporcionan el texto completo de los objetos a medida que se analizan.
depuración
Especifica un nivel de registro en el que se registran todos los mensajes.
Nota: Ejecute MIB Manager en la modalidad de depuración sólo cuando intente resolver una condición de error. El volumen de datos que se produce al analizar sólo un número moderado de MIB llenará rápidamente su sistema de archivos.
detalle

Especifica un nivel de registro en el que se registran todos los mensajes de información.

Un mensaje detallado incluye el estado de las variables, matrices y hashes a medida que cambian el valor y el estado.
