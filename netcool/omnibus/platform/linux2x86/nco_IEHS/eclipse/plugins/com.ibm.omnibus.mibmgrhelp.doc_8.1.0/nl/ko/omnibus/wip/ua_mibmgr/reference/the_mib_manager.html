IBM Tivoli Netcool/OMNIbus 버전 8.1

Netcool MIB Manager

Netcool MIB Manager는 SNMP(Simple Network Management Protocol) MIB(Management Information Base) 파일을 구문 분석하는 데 사용할 수 있는 IBM® Eclipse 기반 GUI 애플리케이션으로 여기에서 Netcool 규칙 파일을 생성할 수 있습니다. MIB Manager는 mib2rules 유틸리티를 대체합니다.

Tivoli Netcool/OMNIbus에는 수동으로 또는 프로그래밍 방식으로 명령을 실행하는데 사용할 수 있는 Netcool MIB Manager 명령행 유틸리티(nco_mibmanager_batch)도 포함됩니다.

Netcool MIB Manager에는 다음과 같은 기능이 있습니다.
  • SNMP MIB 파일을 가져와서 MIB 종속성을 분석하여 OID(Object Identifier) 트리를 빌드합니다.
  • 가져온 SNMP 오브젝트의 전부 또는 일부를 Netcool 규칙 파일과 기타 파일 형식으로 내보낼 수 있습니다.
  • 기본 MIB와 기타 MIB 파일에서 가장 일반적으로 요구되는 RFC MIB가 포함되어 있습니다.
  • 가져온 MIB 모듈의 디바이스 중심 그룹화를 사용하기 위한 디바이스 정의가 포함되어 있습니다.
  • 오브젝트 이름 또는 OID를 기준으로 MIB 모듈과 OID 트리를 필터링하고 검색할 수 있습니다.
  • SNMP 프로브(nco_p_mttrapd) 또는 기타 SNMP 에이전트에 전송할 SNMP 트랩을 생성할 수 있습니다.
  • MIB 데이터를 가져오고 규칙 파일을 내보내는 데 사용할 수 있는 명령행 인터페이스가 있습니다. 명령을 수동으로 실행하거나 프로그래밍 방식으로 호출할 수 있습니다.
  • MIB Manager 시작
    명령행에서 MIB Manager GUI를 시작할 수 있습니다. MIB Manager는 프로세스 제어 하에 실행되거나 Windows 서비스로 실행되도록 설계되지 않았습니다.
  • Netcool MIB Manager 사용
    MIB Manager를 사용하여 MIB 모듈을 구문 분석하고 Netcool 규칙 파일을 작성할 수 있습니다.
  • 글로벌 환경 설정 구성
    환경 설정 창에서 여러 MIB Manager 글로벌 환경 설정을 구성할 수 있습니다.
  • SNMP에 대한 정보
    이 절에서는 SNMP(Simple Network Management Protocol) 아키텍처, SNMP MIB(Management Information Base), 일부 추가 MIB 개념에 대한 추가 정보를 제공합니다.
