Редактирование прерываний SNMP

Как редактировать прерывания SNMP.

Об этой задаче

В представлении Подробности можно редактировать следующие поля прерывания SNMP:
  • @Severity
  • @Type
  • @ExpireTime
  • CODEBLOCK
Прим.: SNMP не предоставляет стандартных механизмов для предоставления информации о наборе (@Type ) или очистке (@Severity). Эту информацию надо задать вручную.

В представлении Подробности дважды щелкните по значению в поле, чтобы сделать его изменяемым. После внесения изменения в поле щелкните по значку для сохранения изменений или по значку для их отмены.

Процедура

Чтобы отредактировать прерывание SNMP:

  1. Выберите прерывание в одном из представлений - Модули MIB или Дерево OID.

    Подробная информация о прерывании будет показана в представлении Подробности.

  2. Для изменения поля @Severity дважды щелкните по значению в поле и выберите новое значение из выпадающего списка.
  3. Для изменения поля @Type дважды щелкните по значению в поле и выберите новое значение из выпадающего списка.
  4. Для изменения поля @ExpireTime дважды щелкните по значению в поле и введите новый период времени (в секундах, минутах, часах или днях).
  5. Для изменения поля CODEBLOCK дважды щелкните по значению в поле (даже если оно пустое) и введите новый блок кода.
