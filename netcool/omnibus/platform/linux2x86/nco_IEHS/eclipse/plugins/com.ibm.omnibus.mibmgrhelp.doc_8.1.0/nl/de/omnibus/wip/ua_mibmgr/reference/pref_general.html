Allgemeine Benutzervorgaben festlegen

Mithilfe der allgemeinen Benutzervorgaben können Sie die MIB-Importoptionen angeben.

Klicken Sie zum Festlegen der Benutzervorgaben für Verzeichnisse auf Benutzervorgaben oder klicken Sie im Hauptmenü auf Datei > Benutzervorgaben und dann im linken Bereich des Fensters Benutzervorgaben auf Allgemein.

  • Wenn das Kontrollkästchen Bei Import Unterverzeichnisse durchsuchen ausgewählt wird, werden beim Importprozess alle Unterverzeichnisse durchsucht, um alle MIB-Dateien zu ermitteln.
  • Geben Sie im Feld Maximale Threadanzahl für Dateiverteiler die Anzahl der Rechenthreads an, die während des Importprozesses zum Ermitteln von MIBs verwendet werden. Dadurch kann beim Import einer großen Anzahl MIBs die Leistung verbessert werden. Der Standardwert ist 5.
  • Geben Sie im Feld Maximale Threadanzahl für MIB-Parser die Anzahl der Rechenthreads an, die während des Importprozesses zum Parsing von MIB-Dateien verwendet werden. Dadurch kann beim Import einer großen Anzahl MIBs die Leistung verbessert werden. Der Standardwert ist 10.
  • Um die Hervorhebungsfarbe der Auswahl zu ändern, die im MIB Manager-Fenster vorgenommen wurde, klicken Sie auf , wählen Sie aus der Farbpalette eine Farbe aus und klicken Sie dann auf OK.
