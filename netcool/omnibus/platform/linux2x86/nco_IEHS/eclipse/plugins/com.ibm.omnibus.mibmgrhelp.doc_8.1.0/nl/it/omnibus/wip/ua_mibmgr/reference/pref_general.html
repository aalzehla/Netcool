Impostazione di preferenze generali

Utilizzare le preferenze generali per specificare le opzioni di importazione MIB.

Per impostare le preferenze di directory, fare clic su Preferenze oppure, dal menu, fare clic su File > Preferenze, quindi dal pannello di sinistra della finestra Preferenze, fare clic su Generale.

  • Se è selezionata la casella di spunta Attraversa sottodirectory in importazione, durante il processo di importazione per individuare tutti i file MIB, la ricerca viene eseguita in tutte le sottodirectory.
  • Nel campo Conteggio massimo thread di divisione file, specificare il numero di thread di calcolo utilizzati per rilevare i MIB durante il processo di importazione. Questo valore può migliorare le prestazioni durante l'importazione di un numero elevato di MIB. Il valore predefinito è 5.
  • Nel campo Conteggio massimo thread di divisione file, specificare il numero di thread di calcolo utilizzati per rilevare i file MIB durante il processo di importazione. Questo valore può migliorare le prestazioni durante l'importazione di un numero elevato di MIB. Il valore predefinito è 10.
  • Per modificare il colore di evidenziazione delle selezioni effettuate nella finestra MIB Manager, fare clic su , selezionare un colore dalla tavolozza dei colori e fare clic su OK.
