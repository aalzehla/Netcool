Netcool MIB マネージャー の使用

MIB マネージャーを使用して、MIB モジュールを構文解析し、Netcool ルール・ファイルを作成できます。

MIB マネージャーには、基本となる MIB および RFC MIB のセットが組み込まれており、これらは、NCHOME/omnibus/var/mibmanager/mibs ディレクトリーにあります。 これらの MIB は、追加の MIB をインポートできるようにするために提供されています。MIB マネージャーは、これらのバンドルされた MIB を自動的に使用して、新しい MIB のインポート時に 依存関係を解決します。バンドルされている MIB モジュールは、 MIB マネージャーを最初に開始する際に「MIB モジュール」ビューに 表示されます。

MIB マネージャーには、バンドルされている基本 MIB のインポートを表す XML データ・ファイルも組み込まれています。MIB マネージャーは、これらの XML ファイルを使用して、 ロー・テキスト MIB ファイルからインポートする MIB データを保管します。

MIB を、MIB マネージャーが開始されるサーバーからアクセス可能な ディレクトリーに追加します。標準ベースのすべての MIB は、別々のディレクトリーにインストールする必要があり、これらのディレクトリーは、依存関係を 容易に解決するために検索パスに配置できます。デフォルトで、基本の SNMP MIB は、NCHOME/omnibus/var/mibmanager/mibs/base ディレクトリーにインストールされ、すべての RFC MIB は、NCHOME/omnibus/var/mibmanager/mibs/rfc ディレクトリーにインストールされます。

「Netcool MIB マネージャー」ウィンドウ

MIB マネージャーを 開始すると、以下のエレメントが「MIB マネージャー」初期ウィンドウに表示されます。複数のエレメントが、「MIB マネージャー」初期ウィンドウに表示されます。次の図は、それらのエレメントを示しています。

図 1. 「MIB マネージャー」初期ウィンドウ
 1  ツールバー
使用頻度が最も高い MIB マネージャーの機能にアクセスできます。
 2  パースペクティブ
MIB データは、2 つのパースペクティブ (「MIB マネージャー」または「mib2rules」) のいずれかで表示できます。「MIB マネージャー」パースペクティブは、デフォルトのパースペクティブです。「mib2rules」パースペクティブは、 オブジェクト ID (OID) に焦点を当てており、mib2rules ユーティリティーに精通した ユーザーのために提供されています。
 3  「デバイス」ビュー
このビューを使用して、 MIB マネージャー・デバイスの作成、更新、および削除を行います。
 4  「詳細」ビューと「コンソール」ビュー
「詳細」ビューには、選択された MIB エレメントの詳細が表示されます。「コンソール」ビューには、現在のインスタンスの実行中に MIB マネージャーにより生成される、すべてのログ・メッセージが表示されます。
 5  「MIB モジュール」
このビューを使用して、 MIB モジュールの検索、削除、および特定の MIB 値の検索を行います。
 6  「OID ツリー」ビュー
このビューには、インポートされた MIB モジュールに存在する、すべての MIB オブジェクトが表示されます。
  • 「MIB モジュール」ビュー
    「MIB モジュール」ビューは、インポートされた すべての MIB モジュールを表示します。これを使用して、MIB モジュールの検索、 MIB モジュールの削除、および特定の MIB 値の検索を行うことができます。
  • 「OID ツリー」ビュー
    「オブジェクト ID (OID) ツリー」ビューは、インポートされた MIB モジュールに 存在するすべての MIB オブジェクトを表示します。
  • MIB データのインポート
    SNMP MIB データを MIB マネージャーへインポートする方法です。
  • MIB データのエクスポート
    MIB マネージャーから SNMP MIB データをさまざまなフォーマットに エクスポートする方法です。
  • SNMP トラップの編集
    SNMP トラップの編集方法です。
  • SNMP トラップの生成
    SNMP トラップの生成方法です。この機能は、MIB マネージャーで生成される ルール・ファイルのテストにも使用できます。
  • デバイスの作成および削除
    MIB マネージャー・デバイスは、関連する MIB モジュールのグループ化、およびエクスポートに必要なすべてのオブジェクトの定義に使用されます。デバイスの作成、更新、削除は MIB マネージャー GUI で行えます。
