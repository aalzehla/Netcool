Configurando as preferências de criação de log

Use as preferências de criação de log para especificar o diretório de arquivo de log do MIB Manager e para configurar o nível de mensagem padrão.

Para configurar as preferências de diretório, clique no Preferências ou a partir do menu principal, clique em Arquivo > Preferênciase em seguida a partir da área da janela esquerda da janela Preferências, clique em Criação de log.

  • No campo Diretório de log, especifique onde o arquivo de log do aplicativo será gravado. O local padrão é NCHOME/omnibus/log.
  • No campo Nível de Mensagens Padrão, especifique o nível de mensagens padrão do aplicativo. Também é possível usar o menu Editar para alterar o nível de log de mensagens padrão. O nível de mensagens será válido a partir do momento em que o aplicativo é iniciado até quando o aplicativo for encerrado, a menos que ele seja alterado usando o menu Editar.
    Nota: Alterar o nível de mensagens no campo Nível de Mensagens Padrão não altera o nível de mensagens padrão do aplicativo. Só irá afetar o nível de mensagens que foi configurado inicialmente quando o aplicativo foi primeiro iniciado (o nível de mensagens padrão). Para evitar gerar grandes arquivos de log, configure o nível de mensagens padrão como avisar.
  • Configurando o nível de criação de log
    É possível alterar o nível de criação de log para fornecer um log mais detalhado ou rastrear arquivos para ajudar na depuração do MIB Manager. As informações adicionais fornecidas por meio do aumento dos níveis de criação de log poderão ajudá-lo quando da depuração de problemas específicos ao analisar MIBs.
