Использование Netcool MIB Manager

MIB Manager можно использовать для синтаксического анализа модулей MIB и создания файлов правил Netcool.

MIB Manager содержит набор базовых MIB и MIB RFC в каталоге ДОМАШНИЙ_КАТАЛОГ_NC/omnibus/var/mibmanager/mibs. Эти MIB поставляются, чтобы обеспечить возможность импорта дополнительных MIB. MIB Manager автоматически использует эти включенные в пакет поставки MIB для разрешения зависимостей при импорте новых MIB. Включенные в поставку модули MIB можно увидеть в представлении Модули MIB при первом запуске MIB Manager.

MIB Manager включает в себя также файлы данных XML, представляющие импорт входящих в поставку базовых MIB. MIB Manager использует эти файлы XML для хранения данных MIB, которые он импортирует из исходных текстовых файлов MIB.

Добавьте ваши MIB в каталог, доступный с сервера, на котором запущен компонент MIB Manager. Все стандартные MIB должны быть установлены в отдельном каталоге, который может быть расположен по пути поиска для простоты разрешения зависимостей. По умолчанию базовые SNMP MIB установлены в каталоге ДОМАШНИЙ_КАТАЛОГ_NC/omnibus/var/mibmanager/mibs/base, а все RFC MIB установлены в каталоге ДОМАШНИЙ_КАТАЛОГ_NC/omnibus/var/mibmanager/mibs/rfc.

Окно Netcool MIB Manager

При запуске MIB Manager перечисленные ниже элементы появляются в исходном окне MIB Manager: несколько элементов появляются в исходном окне MIB Manager. Эти элементы показаны на приведенном ниже рисунке.

Рис. 1. Исходное окно MIB Manager
 1  Панель инструментов
Обеспечивает доступ к наиболее часто используемым функциям MIB Manager.
 2  Перспективы
Просматривать данные MIB можно в одной из двух перспектив - MIB Manager или mib2rules. Перспектива MIB Manager используется как перспектива по умолчанию. Перспектива mib2rules основана на идентификаторах объектов (Object Identifiers, OID) и предоставляется для пользователей, знакомых с утилитой mib2rules.
 3  Представление Устройство
Используйте это представление, чтобы создавать, обновлять и удалять устройства MIB Manager.
 4  Представление Подробности и представление Консоль
В представлении Подробности показаны подробные сведения о выбранных элементах MIB. В представлении Консоль показаны все сообщения журналов, сгенерированные компонентом MIB Manager во время работы текущего экземпляра.
 5  Модули MIB
Используйте это представление для поиска и удаления модулей MIB и для поиска конкретных значений MIB.
 6  Представление Дерево OID
В этом представлении показаны все объекты MIB, находящиеся в импортированных модулях MIB.
  • Представление Модули MIB
    Представление Модули MIB содержит все импортированные модули MIB и предоставляет возможность их поиска и удаления, а также поиска конкретных значений MIB.
  • Представление Дерево OID
    Представление Дерево OID (Object Identifier, идентификатор объектов) показывает все объекты MIB, которые расположены в импортированных модулях MIB.
  • Импорт данных MIB
    Как импортировать данные SNMP MIB в MIB Manager.
  • Экспорт данных MIB
    Как экспортировать данные SNMP MIB из MIB Manager в различных форматах.
  • Редактирование прерываний SNMP
    Как редактировать прерывания SNMP.
  • Генерирование прерываний SNMP
    Как генерировать прерывания SNMP. Эта функциональность может использоваться также для проверки файлов правил, сгенерированных MIB Manager.
  • Создание и удаление устройств
    Устройство MIB Manager используется для группировки связанных модулей MIB и для определения всех объектов, требуемых для экспорта. Вы можете создавать, обновлять и удалять устройства в графическом пользовательском интерфейсе MIB Manager.
