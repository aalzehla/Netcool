MIB 模組視圖

「MIB 模組」視圖可呈現所有匯入的 MIB 模組，並可讓您搜尋 MIB 模組、刪除 MIB 模組以及搜尋特定的 MIB 值。

您可以透過展開及收合 MIB 樹狀結構中的節點，來瀏覽「MIB 模組」視圖。MIB 樹狀結構的最上層分支包含 MIB 模組的名稱，且每個 MIB 模組分支中包含的是組成 MIB 的其他元素。

您可以從視圖下拉清單中選取下列其中一個選項，來過濾「MIB 模組」視圖中的資料：
  • 全部(A)
  • 模組
  • 列舉
  • 文字使用慣例
  • 語法
  • 匯入項目
  • 匯出項目
  • 直欄
  • 說明

若要搜尋「MIB 模組」視圖中的特定值，請在搜尋欄位中輸入文字，然後按 Enter 或按一下 （搜尋）。

MIB Manager 從所選項目開始搜尋，或者如果目前沒有選取任何項目，則從清單中的第一個項目開始搜尋。若要從清單中尋找下一相符項，請繼續按一下 （搜尋）。

當找到一個項目時，系統會將它自動強調顯示並顯示在「MIB 模組」視圖中。

在搜尋欄位中指定有效的正規表示式或輸入純文字，來代表完整或局部的物件名稱。

若要刪除一個以上 MIB 模組，請在選取的 MIB 模組上按一下滑鼠右鍵，然後從蹦現功能表中選取刪除。選取的 MIB 模組即會從 MIB Manager 中刪除。但是，不會刪除提供給 MIB 匯入項目的原始 MIB 原始檔。

註： 您必須只刪除含有錯誤且有新的 MIB 模組替換版本的 MIB 模組。使用這個方法刪除 MIB 模組可導致遺漏 MIB 相依關係。在匯出處理程序期間，也會產生不完整的規則檔案以及在日誌檔中產生警告或錯誤訊息。
