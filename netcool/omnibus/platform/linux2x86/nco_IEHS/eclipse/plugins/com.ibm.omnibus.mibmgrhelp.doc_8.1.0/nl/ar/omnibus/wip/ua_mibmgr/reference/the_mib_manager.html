IBM Tivoli Netcool/OMNIbus النسخة 8.1

Netcool MIB Manager

Netcool MIB Manager هو تطبيق IBM® Eclipse-based GUI يمكنك استخدامه لاجراء تحليل لغوى على ملفات قاعدة معلومات الادارة (MIB) والتى تخص Simple Network Management Protocol (SNMP)، والتى يمكنك من خلالها تكوين ملفات قواعد Netcool. فهو مقصود كاستبدال لوظيفة mib2rules.

كما Tivoli Netcool/OMNIbus يتضمن أيضا وظيفة سطر أمر Netcool MIB Manager (nco_mibmanager_batch) والتى يمكنك استخدامها لاصدار الأوامر يدويا أو برمجيا.

Netcool MIB Manager به الخصائص التالية:
  • يقوم باستقبال ملفات SNMP MIB وحل ارتباطات MIB لبناء تسلسل Object Identifier (OID).
  • يمكن تصدير بعض أو كل عناصر SNMP التي تم استقبالها في ملفات قواعد Netcool ونسق الملف الأخرى.
  • يتضمن MIBs الأساسية و RFC MIBs الأكثر طلبا بواسطة ملفات MIB الأخرى.
  • يتضمن تعريفات الجهاز لاتاحة مجموعة مركزية-للجهاز لوحدات برامج MIB التي تم استقبالها.
  • يمكنه ترشيح بيانات وبحث وحدات برامج MIB وتسلسل OID بواسطة اسم العنصر أو OID.
  • كما يمكنه تكوين صفحات SNMP الشرطية لارسالها الى برنامج اختبار SNMP (nco_p_mttrapd) أو أدوات SNMP البرمجية الأخرى.
  • يوجد به واجهة تعامل لسطر الأمر يمكنك استخدامها لاستقبال بيانات MIB وارسال ملفات القواعد. يمكنك اصدار الأوامر يدويا أو استدعائها برمجيا.
  • بدء MIB Manager
    قم ببدء MIB Manager GUI من سطر الأمر. لم يتم تصميمه لكى يتم تشغيله تحت سيطرة العملية أو كخدمة Windows.
  • استخدام Netcool MIB Manager
    يمكنك استخدام MIB Manager لاجراء تحليل لغوى على وحدات برامج MIB وتكوين ملفات قواعد Netcool.
  • توصيف التفضيلات الشاملة
    يمكنك توصيف العديد من تفضيلات MIB Manager الشاملة من نافذة Preferences.
  • نبذة عن SNMP
    يقدم هذا القسم معلومات اضافية عن أسلوب بناء Simple Network Management Protocol (SNMP)، و management information base (MIB) الى SNMP وبعض مفاهيم MIB الاضافية.
