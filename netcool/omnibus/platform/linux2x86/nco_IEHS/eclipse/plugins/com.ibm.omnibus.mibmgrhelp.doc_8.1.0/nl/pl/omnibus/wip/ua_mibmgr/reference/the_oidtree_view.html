Widok Drzewo OID

Widok Drzewo identyfikatora obiektu (OID) reprezentuje wszystkie obiekty MIB, które znajdują się w zaimportowanych modułach MIB.

W widoku Drzewo OID można filtrować dane, wybierając jedną z poniższych opcji z listy rozwijanej Widok:
  • Wszystkie
  • Pułapki/Powiadomienia
  • Obiekty
  • Moduły

Aby przeszukać widok Drzewo OID w celu znalezienia określonej wartości, należy wpisać tekst w polu Szukaj i nacisnąć klawisz Enter lub kliknąć ikonę (Szukaj).

Program MIB Manager zaczyna wyszukiwanie od wybranego elementu lub, jeśli żaden element nie jest zaznaczony, od pierwszej pozycji na liście. Aby znaleźć następną pasującą pozycję na liście, klikaj ponownie ikonę (Szukaj).

Jeśli element jest zlokalizowany, jest on automatycznie podświetlony i wyświetlony w widoku Drzewo OID.

W polu Szukaj podaj poprawne wyrażenie regularne lub wpisz zwykły tekst, stanowiący pełną lub częściową nazwę obiektu, lub wpisz identyfikator OID. Liczba wyświetlana w każdej gałęzi węzła jest ostatnią cyfrą identyfikatora OID dla danego obiektu.
