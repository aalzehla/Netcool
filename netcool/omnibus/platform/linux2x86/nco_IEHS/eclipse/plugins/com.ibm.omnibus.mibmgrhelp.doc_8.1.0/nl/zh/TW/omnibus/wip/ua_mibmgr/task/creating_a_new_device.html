建立及刪除裝置

MIB Manager 裝置用來分組關聯的 MIB 模組及定義需要匯出的所有物件。您可以在 MIB Manager GUI 中建立、更新及刪除裝置。

關於這項作業

MIB Manager 裝置顯示在「裝置」視圖中。 從「裝置」樹狀結構中選取裝置，以顯示裝置的製造商、型號及作業系統版本。

程序

建立裝置

  1. 若要建立新裝置，請執行下列動作：
    1. 在「MIB 模組」視圖中，至少選取一個 MIB 模組，然後按一下 （建立裝置）。
    2. 在製造商欄位中，從下拉清單中選取裝置製造商。

      模型和作業系統版本欄位中會移入可用的選項。

    3. 在模型欄位中，從下拉清單中選取選項。

      如果沒有模型可用，則輸入敘述性模型名稱。

    4. 在作業系統版本欄位中，從下拉清單中選取選項。

      如果沒有作業系統版本可用，則輸入敘述性作業系統版本名稱。

    5. 在對映的 MIB 模組欄位中，選取要與新裝置建立關聯的 MIB 模組。
    6. 按一下確定以建立裝置。

      新裝置會顯示在「裝置」視圖中。

更新裝置

  1. 若要更新裝置，請執行下列動作：
    1. 在「裝置」視圖中，選取裝置並展開其子節點，直到看到作業系統版本為止。
    2. 選取作業系統版本，然後按一下 （編輯裝置）。
    3. 在製造商欄位中，從下拉清單中選取裝置製造商。

      模型和作業系統版本欄位中會移入可用的選項。

    4. 在模型欄位中，從下拉清單中選取選項。

      如果沒有模型可用，則輸入敘述性模型名稱。

    5. 在作業系統版本欄位中，從下拉清單中選取選項。

      如果沒有作業系統版本可用，則輸入敘述性作業系統版本名稱。

    6. 在對映的 MIB 模組欄位中，選取要與新裝置建立關聯的 MIB 模組。
    7. 按一下確定以更新裝置詳細資料。

      更新的裝置詳細資料會顯示在「裝置」視圖中。

刪除裝置

  1. 在「裝置」視圖中，選取裝置然後按一下 （刪除裝置）。

    若要刪除多個裝置，請按住 Ctrl+Alt 鍵，使用左滑鼠按鈕選取元素，然後按一下 （刪除裝置）。
