MIB Manager indítása

Indítsa el a MIB Manager grafikus felületet a parancssorból. Nem úgy tervezték, hogy folyamatvezérlés alatt vagy Windows szolgáltatásként fusson.

Erről a feladatról

A MIB Manager futtatásához AIX és Solaris operációs rendszereken, a követelményekért tekintse meg a Támogatott operációs rendszerek című dokumentációt.

Győződjön meg róla, hogy a $DISPLAY környezeti változó működő X11 kiszolgálóra van beállítva, mielőtt elindítja a MIB Manager terméket.

Eljárás

  1. A MIB Manager grafikus felület indításához adja ki a következő parancsot:

    NCHOME/omnibus/bin/nco_mibmanager

  2. A MIB Manager parancssori segédprogram indításához adja ki a következő parancsot:

    NCHOME/omnibus/bin/nco_mibmanager_batch [options]

  • MIB Manager parancssori paraméterek
    A MIB Manager olyan segédprogrammal rendelkezik, amely segítségével importálhatja a MIB adatokat és exportálhatja a szabályfájlokat. A parancsokat kiadhatja kézzel vagy meghívhatja őket programozással.
