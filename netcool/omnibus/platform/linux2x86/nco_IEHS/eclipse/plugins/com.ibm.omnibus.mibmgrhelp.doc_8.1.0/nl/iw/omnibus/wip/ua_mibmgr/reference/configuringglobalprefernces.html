IBM Tivoli Netcool/OMNIbus, מהדורה 7.4

קביעת תצורה של העדפות גלובליות

ניתן לקבוע תצורה של מספר העדפות גלובליות של MIB Manager מחלון העדפות.

כדי לקבוע את התצורה של העדפות הגלובליות של MIB Manager, לחצו על העדפות ולחילופין, מהתפריט, לחצו על קובץ > העדפות.

בחלון העדפות, ניתן לקבוע תצורה של העדפות מדריך, העדפות יצוא, העדפות כלליות, העדפות תיעוד והעדפות חיפוש.

  • הגדרת העדפות מדריך
    השתמשו בהעדפות המדריך כדי לציין את המדריכים המשמשים לאחסון קובצי MIB שיובאו, נתוני MIB, וקובצי כללים שיובאו.
  • הגדרת העדפות יצוא
    מקטע זה מתאר כיצד להגדיר את העדפות הייצוא של קובצי כללים.
  • הגדרת העדפות כלליות
    השתמש בהעדפות הכלליות כדי לציין את אפשרויות הייבוא עבור MIB.
  • הגדרת העדפות תיעוד
    השתמשו בהעדפות התיעוד כדי לציין את מדריך קובץ היומן של MIB Manager, וכדי להגדיר את רמת ההודעה המוגדרת כברירת מחדל.
  • הגדרת העדפות חיפוש
    מקטע זה מתאר כדי להגדיר את ההעדפות של MIB Manager המשמשות לחיפוש של קובצי MIB.
נושא אב: Tivoli Netcool/OMNIbus MIB Manager
