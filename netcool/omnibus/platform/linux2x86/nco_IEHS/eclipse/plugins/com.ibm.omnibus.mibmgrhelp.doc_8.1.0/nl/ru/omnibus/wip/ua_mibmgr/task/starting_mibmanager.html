IBM Tivoli Netcool/OMNIbus, Версия 8.1

Запуск MIB Manager

Запустите графический пользовательский интерфейс MIB Manager из командной строки. Он не рассчитан на запуск под управлением процесса или в качестве службы Windows.

Об этой задаче

Посмотрите в списке поддерживаемых систем в документации информацию о требованиях для работы MIB Manager в операционных системах AIX и Solaris.

Прежде чем запускать MIB Manager, убедитесь, что для переменной среды $DISPLAY задан функционирующий сервер X11.

Процедура

  1. Чтобы запустить графический пользовательский интерфейс MIB Manager, введите следующую команду:

    ДОМАШНИЙ_КАТАЛОГ_NC/omnibus/bin/nco_mibmanager

  2. Чтобы запустить утилиту командной строки MIB Manager, введите следующую команду:

    ДОМАШНИЙ_КАТАЛОГ_NC/omnibus/bin/nco_mibmanager_batch [опции]

  • Опции командной строки MIB Manager
    В MIB Manager есть утилита командной строки, при помощи которой можно импортировать данные MIB и файлы правил экспорта. Вы можете вводить команды вручную или можете вызывать их программным образом.
