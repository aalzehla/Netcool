編輯 SNMP 設陷

如何編輯 SNMP 設陷。

關於這項作業

在「詳細資料」視圖中，可以編輯下列 SNMP 設陷欄位：
  • @Severity
  • @Type
  • @ExpireTime
  • CODEBLOCK
註： SNMP 不提供用來指定設定 (@Type ) 或清除 (@Severity) 資訊的標準機制。您必須手動指定這項資訊。

在「詳細資料」視圖中，按兩下欄位值讓它可編輯。變更欄位之後，按一下 來儲存變更，或按一下 來取消變更。

程序

若要編輯 SNMP 設陷，請執行下列動作：

  1. 在「MIB 模組」或「OID 樹狀結構」視圖中，選取設陷。

    設陷的相關詳細資訊會顯示在「詳細資料」視圖中。

  2. 若要編輯 @Severity 欄位，請按兩下欄位值，並從下拉清單中選擇新值。
  3. 若要編輯 @Type 欄位，請按兩下欄位值，並從下拉清單中選擇新值。
  4. 若要編輯 @ExpireTime 欄位，請按兩下欄位值並輸入新時段（以秒、分鐘、小時或天數為單位）。
  5. 若要編輯 CODEBLOCK 欄位，請按兩下欄位值（即使欄位是空的也是如此），並輸入新的程式碼區塊。
