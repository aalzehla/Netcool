IBM Tivoli Netcool/OMNIbus Version 8.1

Starting MIB Manager

Start the MIB Manager GUI from the command line. It is not designed to be run under process control or as a Windows service.

About this task

See the Supported Operating Systems documentation for information about the requirements for running MIB Manager on the AIX and Solaris operating systems.

Ensure that the $DISPLAY environment variable is set to a functioning X11 server before starting MIB Manager.

Procedure

  1. Use the following command to start the MIB Manager GUI:

    NCHOME/omnibus/bin/nco_mibmanager

  2. Use the following command to start the MIB Manager command-line utility:

    NCHOME/omnibus/bin/nco_mibmanager_batch [options]

  • MIB Manager command-line options
    MIB Manager has a command-line utility that you can use to import MIB data and export rules files. You can issue commands manually or call them programmatically.
