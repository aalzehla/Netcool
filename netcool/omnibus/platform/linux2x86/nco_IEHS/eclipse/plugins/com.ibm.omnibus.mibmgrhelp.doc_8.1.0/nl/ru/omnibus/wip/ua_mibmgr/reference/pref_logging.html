Задание предпочтений записи в журнал

Используйте предпочтения записи в журнал для указания каталога файлов журналов MIB Manager и задания уровня сообщений по умолчанию.

Для задания предпочтений каталогов щелкните по значку Предпочтения или выберите пункт главного меню Файл > Предпочтения, а затем на левой панели окна Предпочтения щелкните по заголовку Запись в журнал.

  • В поле Каталог журналов укажите, куда записывается файл журнала прикладных программ. Положение по умолчанию - ДОМАШНИЙ_КАТАЛОГ_NC/omnibus/log.
  • В поле Уровень сообщений по умолчанию задайте уровень сообщений по умолчанию для прикладной программы. Можно использовать также меню Изменить, чтобы изменить уровень записи в журнал сообщений по умолчанию. Этот уровень сообщений используется от запуска до остановки прикладной программы, пока он не будет изменен в меню Изменить.
    Прим.: Изменение уровня сообщений в поле Уровень сообщений по умолчанию не изменяет текущего уровня сообщений прикладной программы. Такое изменение воздействует только на уровень сообщений, который задается первоначально при первом запуске прикладной программы (уровень сообщений по умолчанию). Чтобы исключить генерирование больших файлов журналов, задайте для уровня сообщений по умолчанию значение warn.
  • Задание уровня записи в журнал
    Уровень записи в журнал можно изменить для создания более подробных файлов журналов или трассировок, что помогает при отладке MIB Manager. Дополнительная информация, предоставляемая при увеличении уровня записи в журнал, может помочь в отладке конкретных ошибок при синтаксическом анализе MIB.
