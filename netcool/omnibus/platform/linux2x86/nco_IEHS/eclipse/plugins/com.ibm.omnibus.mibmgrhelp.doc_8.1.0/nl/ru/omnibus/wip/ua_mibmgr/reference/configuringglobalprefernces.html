Конфигурирование глобальных предпочтений

Сконфигурировать некоторые глобальные предпочтения MIB Manager можно в окне Предпочтения.

Чтобы сконфигурировать глобальные предпочтения MIB Manager, щелкните по значку Предпочтения или в меню выберите Файл > Предпочтения.

В окне Предпочтения можно сконфигурировать предпочтения каталогов, экспорта, записи в журнал, поиска и общие предпочтения.

  • Задание предпочтений каталогов
    Используйте предпочтения каталогов для задания каталогов, применяемых для хранения импортированных MIB, данных MIB и экспортируемых файлов правил.
  • Задание предпочтений экспорта
    В этом разделе описывается, как задать предпочтения экспорта файла правил.
  • Задание общих предпочтений
    Используйте общие предпочтения для задания опций импорта MIB.
  • Задание предпочтений записи в журнал
    Используйте предпочтения записи в журнал для указания каталога файлов журналов MIB Manager и задания уровня сообщений по умолчанию.
  • Задание предпочтений поиска
    В этом разделе описывается, как определить предпочтения MIB Manager, используемые для поиска файлов MIB.
