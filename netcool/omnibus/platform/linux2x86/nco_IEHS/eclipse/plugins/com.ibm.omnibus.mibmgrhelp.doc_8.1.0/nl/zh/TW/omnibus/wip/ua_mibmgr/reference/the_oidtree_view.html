OID 樹狀結構視圖

「物件 ID (OID) 樹狀結構」視圖會呈現所有位於匯入 MIB 模組中的 MIB 物件。

您可以從視圖下拉清單中選取下列其中一個選項，來過濾「OID 樹狀結構」視圖中的資料：
  • 全部(A)
  • 設陷/通知
  • 物件
  • 模組

若要搜尋「OID 樹狀結構」視圖中的特定值，請在搜尋欄位中輸入文字，然後按 Enter 或按一下 （搜尋）。

MIB Manager 從所選項目開始搜尋，或者如果目前沒有選取任何項目，則從清單中的第一個項目開始搜尋。若要從清單中尋找下一相符項，請繼續按一下 （搜尋）。

當找到一個項目時，系統會將它自動強調顯示並顯示在「OID 樹狀結構」視圖中。

在搜尋欄位中指定有效的正規表示式或輸入純文字，來代表完整或局部的物件名稱或 OID。顯示在每個分支節點上的號碼是該特定物件之 OID 的最後一位數。
