產生 SNMP 設陷

如何產生 SNMP 設陷。此功能還可用於測試 MIB Manager 產生的規則檔案。

程序

若要產生 SNMP 設陷，請執行下列動作：

  1. 按一下 設陷。
  2. 在目的地位址欄位中，指定執行「SNMP 設陷」服務之電腦的主機名稱或 IP 位址。
  3. 在目的地埠欄位中，指定「SNMP 設陷」服務接聽所在的 UDP 埠。
  4. 在社群欄位中，指定設陷管理程式的有效字串。
  5. 視產生的設陷類型而定，變數連結區段可能在「產生 SNMP 設陷」視窗中可見。變數連結區段中可用的欄位也視產生的設陷類型而定。

    將滑鼠指標移至變數連結欄位名稱上方，以取得欄位值語法的詳細資料。您可以指定多個值，以逗點區隔。這些值將隨機選擇用來建置設陷。

  6. 在重複度量欄位中，指定可產生的設陷數（從 1 到 1000）。

    如果指定的數目很大，則會快速連續產生設陷並可導致設陷管理程式發生警示風暴。若為大於 1 (> 1) 的值以及指定為分號區隔清單的值，MIB Manager 會隨機從指定清單中選取，然後將任意值呈現為 varbind 來產生設陷。

  7. 按一下執行以產生設陷。

結果

「產生 SNMP 設陷」視窗中的結果欄位將指出設陷是否順利傳送。
