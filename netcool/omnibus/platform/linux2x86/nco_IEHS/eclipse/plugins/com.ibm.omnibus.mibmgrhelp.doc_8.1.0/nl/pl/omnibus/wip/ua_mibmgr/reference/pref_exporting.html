Ustawianie preferencji eksportu

W tej sekcji opisano sposób ustawiania preferencji eksportu plików reguł.

Aby ustawić preferencje katalogu, należy kliknąć opcję Preferencje lub wybrać z menu głównego opcję Plik > Preferencje, a następnie z panelu po lewej stronie okna Preferencje kliknąć opcję Eksport.

  • W polu Dołącz ścieżkę określ położenie reguł, które zostały zainstalowane z pakietami plików reguł Netcool Include Library (NCiL), jeśli zostały zainstalowane. Ta opcja jest wymagana tylko wtedy, jeśli eksportujesz dane w formacie pliku Netcool Include Library i gdy ścieżka katalogu musi wskazywać włączone reguły. W rezultacie nowe pliki reguł nie muszą być edytowane ręcznie. Wartością domyślną jest NCHOME/omnibus/probes/arch.
  • W polu Domyślny format pliku eksportu wybierz format Windows/DOS (CRLF) lub UNIX (LF) dla pliku wyjściowego. Ta właściwość umożliwia tworzenie reguł na komputerze z systemem Windows oraz kopiowanie ich do komputera z systemem UNIX bez uruchamiania programu narzędziowego takiego jak dos2unix.
    Uwaga: Ta opcja ma zastosowanie do wszystkich plików wyjściowych i nie jest ograniczona do plików reguł. Domyślną wartością jest format UNIX.
  • W polu Typy alertów określ opcje listy rozwijanej, które są wyświetlane, gdy pole @Type pułapki jest edytowane w widoku Szczegóły.

Następujące opcje są dostępne tylko wtedy, gdy format eksportu NCKL został wcześniej wybrany:

  • Jeśli pole wyboru Dołącz opisy powiązań zmiennych jest zaznaczone, opisy powiązań zmiennych są dodawane jako komentarz. Domyślnie ta opcja jest włączona.
  • Jeśli pole wyboru Oblicz lepszą wartość dla @AlertKey jest zaznaczone, wartość @AlertKey jest generowana z obiektu nadrzędnego powiązań zmiennych. Domyślnie ta opcja jest włączona.
  • Jeśli pole wyboru Ustaw @Agent dla każdego zarządzanego urządzenia jest zaznaczone, pole @Agent jest ustawiane dla każdego typu zarządzanego urządzenia. Pole @Agent opiera się na nazwach modułów MIB, które są przetwarzane dla plików reguł. Domyślnie ta opcja jest włączona.
  • Jeśli pole wyboru Dodaj szablon @Class jest zaznaczone, pole @Class jest ustawione. Jeśli pole wyboru Dodaj szablon @Class jest zaznaczone, pole @Class jest ustawione na 300. Domyślnie ta opcja jest włączona.
