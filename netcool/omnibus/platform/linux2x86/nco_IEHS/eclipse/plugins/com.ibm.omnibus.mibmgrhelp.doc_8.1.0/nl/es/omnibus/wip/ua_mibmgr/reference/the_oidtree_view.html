La vista Árbol de OID

La vista Árbol de identificador de objeto (OID) proporciona una representación de todos los objetos MIB que se encuentran en los módulos MIB importados.

Puede filtrar los datos de la vista Árbol de OID seleccionando una de las siguientes opciones de la lista desplegable Ver:
  • Todas
  • Interrupciones/Notificaciones
  • Objetos
  • Módulos

Para explorar la vista Árbol de OID para buscar un valor concreto, escriba el texto en el campo Buscar y pulse Intro o pulse (Buscar).

MIB Manager inicia la búsqueda desde el elemento seleccionado o, si no hay ningún elemento seleccionado, desde el primer elemento de la lista. Para encontrar el siguiente elemento coincidente de la lista, siga pulsando (Buscar).

Cuando se localiza un elemento, se resalta automáticamente y se muestra en la vista Árbol OID.

En el campo Buscar, especifique una expresión regular válida o texto sin formato para representar un nombre de objeto completo o parcial, u OID. El número que se muestra en cada nodo de ramificación es el último dígito del OID para ese determinado objeto.
