ディレクトリー設定のセットアップ

ディレクトリー設定を使用して、インポートされた MIB、MIB データ、およびエクスポートされたルール・ファイルの保管に使用するディレクトリーを指定します。

ディレクトリー設定をセットアップするには、 「設定」をクリックするか、「メニュー」から、「ファイル」 > 「設定」をクリックして、「設定」ウィンドウの左側のペインから 「ディレクトリー」をクリックします。

  • 「データ・ディレクトリー」フィールドに、インポートされた MIB が XML ファイルとして保管される場所を指定します。デフォルトの場所は NCHOME/omnibus/var/mibmanager/data です。
  • 「エクスポート・ディレクトリー」フィールドに、 エクスポート・ファイルが生成される場所を指定します。エクスポートのタイプに基づいたサブディレクトリーが作成されます。デフォルトの場所は NCHOME/omnibus/var/mibmanager/export です。
  • 「インポート・ディレクトリー」フィールドに、 インポートされた MIB ファイルが配置される場所を指定します。デフォルトの場所は NCHOME/omnibus/var/mibmanager/mibs です。
