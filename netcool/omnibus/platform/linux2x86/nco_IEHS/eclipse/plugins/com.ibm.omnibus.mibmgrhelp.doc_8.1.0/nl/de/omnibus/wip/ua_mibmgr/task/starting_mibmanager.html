IBM Tivoli Netcool/OMNIbus Version 8.1

MIB Manager starten

Starten Sie die MIB Manager-GUI über die Befehlszeile. Die Komponente ist nicht dafür vorgesehen, innerhalb einer Prozesssteuerung oder als Windows-Dienst ausgeführt zu werden.

Informationen zu diesem Vorgang

In der Dokumentation zu den unterstützten Betriebssystemen finden Sie Informationen zu den Voraussetzungen, die zur Ausführung von MIB Manager unter AIX und Solaris erfüllt sein müssen.

Stellen Sie sicher, dass die Umgebungsvariable $DISPLAY mit einem funktionsfähigen X11-Server definiert ist, bevor Sie MIB Manager starten.

Vorgehensweise

  1. Starten Sie die MIB Manager-GUI mit dem folgenden Befehl:

    NCHOME/omnibus/bin/nco_mibmanager

  2. Starten Sie das MIB Manager-Befehlszeilendienstprogramm mit dem folgenden Befehl:

    NCHOME/omnibus/bin/nco_mibmanager_batch [optionen]

  • Befehlszeilenoptionen für MIB Manager
    MIB Manager beinhaltet ein Befehlszeilendienstprogramm, mit dessen Hilfe Sie MIB-Daten importieren und Regeldateien exportieren können. Sie können Befehle manuell ausgeben oder programmgesteuert aufrufen.
