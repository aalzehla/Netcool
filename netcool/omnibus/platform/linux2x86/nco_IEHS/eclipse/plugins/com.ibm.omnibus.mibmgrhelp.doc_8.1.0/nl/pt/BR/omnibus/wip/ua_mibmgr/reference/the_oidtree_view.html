A Visualização OID Tree

A visualização Object Identifier (OID) Tree fornece uma representação de todos objetos MIB localizados nos módulos MIB importados.

É possível filtrar os dados na visualização do OID Tree selecionando uma das opções a seguir a partir da lista suspensa Visualizar:
  • Tudo
  • Traps/Notificações
  • Objetos
  • Módulos

Para fazer a procura na visualização do OID Tree por um valor específico, digite o texto no campo Procurar e pressione Enter ou clique no (Procurar).

O MIB Manager inicia a procura a partir do item selecionado ou, se não houver nenhum item selecionado atualmente, a partir do primeiro item da lista. Para localizar o próximo item correspondente da lista, contunue a clicar no (Procurar).

Ao localizar um item, ele será automaticamente destacado e exibido na visualização OID Tree.

No campo Procurar, especifique uma expressão regular válida ou insira um texto simples para representar um nome do objeto parcial ou completo ou OID. O número exibido em cada nó de ramificação é o último dígito do OID de um objeto específico.
