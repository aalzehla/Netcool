Netcool MIB Manager

Netcool MIB Manager - это прикладная программа с графическим пользовательским интерфейсом на основе IBM® Eclipse, которую можно использовать для синтаксического анализа файлов информационной базы управления (management information base, MIB) протокола SNMP; из них вы можете генерировать файлы правил Netcool. Она предназначена в качестве замены утилиты mib2rules.

Tivoli Netcool/OMNIbus также содержит утилиту командной строки Netcool MIB Manager (nco_mibmanager_batch), при помощи которой можно вводить команды вручную или программным образом.

В компоненте Netcool MIB Manager есть следующие функции:
  • Он импортирует файлы MIB SNMP и разрешает зависимости MIB для построения дерева идентификаторов объектов (Object Identifier, OID).
  • Он может экспортировать некоторые или все импортированные объекты SNMP в файлы правил Netcool и файлы других форматов.
  • Он включает в себя базовые MIB и MIB RFC, чаще всего требуемые в других файлах MIB.
  • Он включает в себя определения устройств, чтобы сделать возможной группировку импортируемых модулей MIB по устройствам.
  • Он допускает фильтрацию модули MIB и дерева OID и выполнение поиска в них по именам объектов или по OID.
  • Он может генерировать прерывания SNMP для отправки зонду SNMP (nco_p_mttrapd) или другим агентам SNMP.
  • У него есть интерфейс командной строки, при помощи которого можно импортировать данные MIB и файлы правил экспорта. Вы можете вводить команды вручную или можете вызывать их программным образом.
  • Запуск MIB Manager
    Запустите графический пользовательский интерфейс MIB Manager из командной строки. Он не рассчитан на запуск под управлением процесса или в качестве службы Windows.
  • Использование Netcool MIB Manager
    MIB Manager можно использовать для синтаксического анализа модулей MIB и создания файлов правил Netcool.
  • Конфигурирование глобальных предпочтений
    Сконфигурировать некоторые глобальные предпочтения MIB Manager можно в окне Предпочтения.
  • Об архитектуре SNMP
    В этом разделе изложена дополнительная информация об архитектуре Simple Network Management Protocol (SNMP), базе управляющей информации (management information base, MIB) SNMP и некоторых дополнительных понятиях MIB.
