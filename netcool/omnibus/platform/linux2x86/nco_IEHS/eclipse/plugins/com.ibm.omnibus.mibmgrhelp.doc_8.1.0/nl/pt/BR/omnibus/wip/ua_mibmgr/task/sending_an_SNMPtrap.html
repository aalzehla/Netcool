Gerando traps SNMP

Como gerar traps SNMP. Esta funcionalidade também pode ser usada para testar arquivos de regras gerados pelo MIB Manager.

Procedimento

Para gerar um trap SNMP:

  1. Clique no Trap.
  2. No campo Endereço de Destino, especifique o nome do host ou endereço IP de um computador executando um serviço SNMP Trapd.
  3. No campo Porta de Destino, especifique a porta UDP que o serviço SNMP Trapd estiver recebendo.
  4. No campo Comunidade, especifique uma sequência válida para o gerenciador Trapd.
  5. Dependendo do tipo de trap que estiver sendo gerado, uma seção Ligações de Variável poderá estar visível na janela Gerar um SNMP Trap. Os campos disponíveis na seção Ligações de Variável também dependem do tipo de trap sendo gerado.

    Passar o mouse sobre o campo Ligações de Variável nomeia detalhes da sintaxe do valor do campo. É possível especificar diversos valores, separados por vírgulas. Estes valores serão escolhidos aleatoriamente para construir o trap.

  6. No campo Repetir Métrica, especifique o número de traps (de 1 a 1000) que possam ser gerados.

    Se você especificar um número grande e como os traps são gerados em sucessão rápida, poderá causar um grande alarme no gerenciador do Trapd. Para valores maiores que um (> 1) e para valores especificados como listas separadas por ponto e vírgula, o MIB Manager seleciona aleatoriamente a partir das listas especificadas e gera traps com os valores arbitrários apresentados como varbinds.

  7. Clique em Executar para gerar o trap.

Resultados

O campo Resultado na janela Gerar um Trap SNMP irá indicar se o trap foi enviado com sucesso.
