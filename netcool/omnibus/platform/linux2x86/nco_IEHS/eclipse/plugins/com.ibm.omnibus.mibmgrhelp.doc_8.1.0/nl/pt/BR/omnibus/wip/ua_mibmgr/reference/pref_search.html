Configuração de preferências de procura

Esta seção descreve como definir as preferências do MIB Manager que são usadas para procurar os arquivos MIB.

Para configurar preferências de procura, clique em Preferências ou, no menu principal, clique em Arquivo > Preferências e então, na área de janela esquerda da janela Preferências, clique em Procurar.

No campo Caminho da procura MIB, configure o caminho da procura para onde os arquivos MIB estiverem localizados. Os diretórios deverão estar separados por um ponto e vírgula (;). Os subdiretórios serão procurados automaticamente para localizar arquivos MIB dependentes.

O caminho padrão inclui o local dos arquivos MIB de base e dos arquivos MIB de RFC que são empacotados com o MIB Manager. É possível especificar os novos diretórios no término do caminho de arquivo padrão. O que assegura que o diretório base seja inicialmente pesquisado a procura de dependências contendo os MIBs mais comumente usados e que em seguida o diretório rfc seja pesquisado a procura de MIBs usados com menos frequência.
