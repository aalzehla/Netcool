IBM Tivoli Netcool/OMNIbus النسخة 8.1

استخدام Netcool MIB Manager

يمكنك استخدام MIB Manager لاجراء تحليل لغوى على وحدات برامج MIB وتكوين ملفات قواعد Netcool.

MIB Manager يتضمن مجموعة من MIBs و RFC MIBs الأساسية، فى الدليل NCHOME/omnibus/var/mibmanager/mibs. يتم تقديم MIBs لتتمكن من استقبال MIBs اضافية. MIB Manager يستخدم MIBs المجمعة آليا لحل الارتباطات أثناء استقبال MIBs جديدة. وحدات برامج MIB المجمعة تكون مرئية فى مشاهدة وحدات برامج MIB عندما تقوم ببدء MIB Manager للمرة الأولى.

MIB Manager يتضمن أيضا ملفات بيانات XML التى تمثل استقبال لحزم برامج MIBs الأساسية. يقوم MIB Manager باستخدام ملفات XML لتخزين بيانات MIB التى يقوم باستقبالها من ملفات MIB للنص الصريح.

قم باضافة واحدات MIBs الخاصة بك الى الدليل الذى يمكن التوصل له من وحدة الخدمة التى تم بدء MIB Manager منها. يجب تركيب MIBs القياسي في دليل منفصل يمكن أن يتم وضعه في مسار البحث لتحليل الارتباط بسهولة. بصفة افتراضية، يتم تركيب SNMP MIBs الأساسية فى الدليل NCHOME/omnibus/var/mibmanager/mibs/base، ويتم تركيب كل RFC MIBs فى الدليل NCHOME/omnibus/var/mibmanager/mibs/rfc.

نافذة Netcool MIB Manager

عندما تقوم ببدء MIB Manager، يتم عرض العناصر التالية فى نافذة MIB Manager المبدئية: يتم عرض عناصر متعددة فى نافذة MIB Manager المبدئية. الشكل التالي يعرض هذه العناصر.

شكل 1. نافذة MIB Manager المبدئية
 1  خط الأدوات
يوفر امكانية التوصل الى أكثر وظائف MIB Manager استخداما.
 2  المشاهدات الرئيسية
يمكنك مشاهدة بيانات MIB فى مشاهدة رئيسية أو أكثر، MIB Manager أو mib2rules. تعد مشاهدة MIB Manager هي المشاهدة الرئيسية المفترضة. تركز المشاهدة الرئيسية mib2rules على أكواد تعريف العناصر (OIDs) ويتم تقديمها للمستخدمين الذين هم على دراية بوظيفة mib2rules.
 3  مشاهدة الجهاز
قم باستخدام هذه المشاهدة لتكوين وتحديث وحذف أجهزة MIB Manager.
 4  مشاهدة التفاصيل ومشاهدة شاشة التحكم الرئيسية
تعرض مشاهدة التفاصيل تفاصيل عناصر MIB المحددة. تعرض مشاهدة شاشة التحكم الرئيسية كل رسائل السجل التى يتم تكونها بواسطة MIB Manager بينما تكون النسخة الحالية قيد التشغيل.
 5  وحدات برامج MIB
قم باستخدام هذه المشاهدة للبحث عن وحذف وحدات برامج MIB، وللبحث عن قيم MIB محددة.
 6  مشاهدة تسلسل OID
تقدم هذه المشاهدة توضيح لكل عناصر MIB الموجودة فى وحدات برامج MIB التى تم استقبالها.
  • مشاهدة وحدات برامج MIB
    تقدم مشاهدة وحدات برامج MIB تمثيل لكل وحدات برامج MIB التي تم استقبالها وتسمح لك بالبحث عن وحدات برامج MIB، وحذف وحدات برامج MIB والبحث عن قيم MIB محددة.
  • مشاهدة تسلسل OID
    تقدم مشاهدة تسلسل كود تعريف العنصر (OID) تمثيل لكل عناصر MIB التي توجد في وحدات برامج MIB التي تم استقبالها.
  • استقبال بيانات MIB
    كيف يتم استقبال ملفات تعقب مشاكل SNMP في MIB Manager.
  • تصدير بيانات MIB
    كيفية ارسال بيانات SNMP MIB من MIB Manager الى تنسيقات متنوعة.
  • تحرير ملفات تعقب مشاكل SNMP
    كيف يتم تحرير ملفات تعقب مشاكل SNMP.
  • تكوين ملفات تعقب مشاكل SNMP
    كيف يتم تكوين ملفات تعقب مشاكل SNMP. كما يمكن استخدام هذه الوظيفة لاختبار القواعد التى يتم تكوينها بواسطة MIB Manager.
  • تكوين وحذف الأجهزة
    يتم استخدام جهاز MIB Manager لجمع وحدات برامج MIB المجمعة ولتعريف كل العناصر اللازمة للارسال. يمكنك تكوين وتحديث وحذف الأجهزة فى MIB Manager GUI.
