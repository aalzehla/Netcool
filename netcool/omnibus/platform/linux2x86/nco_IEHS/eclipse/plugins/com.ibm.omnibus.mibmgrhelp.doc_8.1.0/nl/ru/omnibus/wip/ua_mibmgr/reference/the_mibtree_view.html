Представление Модули MIB

Представление Модули MIB содержит все импортированные модули MIB и предоставляет возможность их поиска и удаления, а также поиска конкретных значений MIB.

Перемещаться по представлению Модули MIB можно, раскрывая и сворачивая узлы на дереве MIB. Ветви высшего уровня дерева MIB содержат имена модулей MIB, а ветви, содержащиеся внутри каждого модуля MIB, представляют другие элементы, составляющие этот MIB.

Фильтровать данные в представлении Модули MIB можно, выбрав одну из следующих опций из выпадающего списка Представление:
  • Все
  • Модули
  • Перечисления
  • Текстовые соглашения
  • Синтаксис
  • Импорты
  • Экспорты
  • Столбцы
  • Описания

Для поиска в представлении Модули MIB конкретного значения введите текст в поле Поиск, а после этого или нажмите клавишу Enter, или щелкните по значку (Поиск).

MIB Manager начинает поиск с выбранного элемента или, если никакой элемент в данный момент не выбран, с первого элемента в списке. Чтобы найти в списке следующий подходящий элемент, снова щелкните по значку (Поиск).

Если элемент найден, он автоматически выделяется и выводится в представлении Модули MIB.

В поле Поиск задайте допустимые регулярные выражения или введите простой текст, представляющий полное или частичное имя объекта.

Для удаления одного или нескольких модулей MIB щелкните правой кнопкой мыши по выбранному модулю или модулям MIB, а затем во всплывающем меню выберите Удалить. Выбранный модуль MIB удаляется из MIB Manager. Однако исходные файлы MIB, подготовленные для импорта, удалить нельзя.

Прим.: Такой модуль MIB нужно удалить только в том случае, если он содержит ошибки и для этого модуля MIB существует новая замещающая версия. Удаление модулей MIB таким способом может привести к отсутствию зависимостей MIB. В процессе экспорта могут быть сгенерированы также неполные файлы правил, что будет сопровождаться предупреждениями или сообщениями об ошибках в файле журнала.
