Configurando o nível de criação de log

É possível alterar o nível de criação de log para fornecer um log mais detalhado ou rastrear arquivos para ajudar na depuração do MIB Manager. As informações adicionais fornecidas por meio do aumento dos níveis de criação de log poderão ajudá-lo quando da depuração de problemas específicos ao analisar MIBs.

Para alterar o nível de criação de log, clique em Editar > depurar a partir barra de menus da janela principal. Os arquivos de log têm seis níveis de mensagem com as quais é possível fazer ciclos para aumentar o nível de detalhamento que for capturado: nenhum, erro, aviso, informações, depuração e detalhamento.

A tabela a seguir fornece uma breve descrição de cada nível de criação de log:

Tabela 1. Uma lista de níveis de criação de log com as descrições associadas
Nível de log Descrição
nenhum

Especifica um nível de criação de log no qual nenhuma mensagem de derpuração é de saída.

O desempenho do aplicativo é melhorado mas nenhuma mensagem de erro ou aviso é exibido durante a análise.
erro

Especifica um nível de criação de log no qual somente erros que afetam os resultados são registrados.

Isso inclui qualquer erro identificado pelo MIB Manager que resultaria em dados incorretos sendo exibidos na janela principal.

aviso

Especifica um nível de criação de log no qual avisos e mensagens de erro são registrados.

Um aviso indica uma condoção que poderia resultar em dados errados e portanto devem ser verificados. Entretanto, a condição não fará com que os dados sejam corrompidos.
informações

Especificar um nível de criação de log no qual mensagens críticas, informativas e de aviso são registradas.

Mensagens informativas contém detalhes concernentes de progresso de tarefas em geral. Por exemplo, fornece o texto completo de objetos à medida em que estão sendo analisados.
depuração
Especifica um nível de log na qual todas as mensagens estão registradas.
Nota: Somente execute o MIB Manager no modo de depuração quando você estiver tentando resolver uma condição de erro. O volume de dados produzido quando da análise de apenas um número moderado de MIBs irá preencher rapidamente o sistema de arquivos.
detalhado

Especifica um nível de log na qual todas as mensagens de informações estão registradas.

Uma mensagem detalhada inclui o estado das variáveis, matrizes e hashes à medida em que mudam o valor e estado.
