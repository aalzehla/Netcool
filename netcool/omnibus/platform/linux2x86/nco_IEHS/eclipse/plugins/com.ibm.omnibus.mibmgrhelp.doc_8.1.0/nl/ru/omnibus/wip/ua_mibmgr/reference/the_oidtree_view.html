Представление Дерево OID

Представление Дерево OID (Object Identifier, идентификатор объектов) показывает все объекты MIB, которые расположены в импортированных модулях MIB.

Фильтровать данные в представлении Дерево OID можно, выбрав одну из следующих опций из выпадающего списка Представление:
  • Все
  • Прерывания/Уведомления
  • Объекты
  • Модули

Для поиска в представлении Дерево OID конкретного значения введите текст в поле Поиск, а после этого или нажмите клавишу Enter, или щелкните по значку (Поиск).

MIB Manager начинает поиск с выбранного элемента или, если никакой элемент в данный момент не выбран, с первого элемента в списке. Чтобы найти в списке следующий подходящий элемент, снова щелкните по значку (Поиск).

Если элемент найден, он автоматически выделяется и выводится в представлении Дерево OID.

В поле Поиск задайте допустимые регулярные выражения или введите простой текст, представляющий полное или частичное имя объекта или OID. Число, показанное на каждом узле ветвей, представляет собой последнюю цифру OID для данного конкретного объекта.
