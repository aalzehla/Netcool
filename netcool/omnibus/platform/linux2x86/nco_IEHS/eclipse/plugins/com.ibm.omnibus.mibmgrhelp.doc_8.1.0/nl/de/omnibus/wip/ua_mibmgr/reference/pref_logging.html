IBM Tivoli Netcool/OMNIbus Version 8.1

Benutzervorgaben für die Protokollierung festlegen

Mithilfe der Benutzervorgaben für die Protokollierung können Sie das Verzeichnis für die Protokolldatei von MIB Manager angeben und die Standardnachrichtenstufe festlegen.

Klicken Sie zum Festlegen der Benutzervorgaben für Verzeichnisse auf Benutzervorgaben oder klicken Sie im Hauptmenü auf Datei > Benutzervorgaben und dann im linken Bereich des Fensters Benutzervorgaben auf Protokollierung.

  • Geben Sie im Feld Protokollverzeichnis an, an welcher Position die Anwendungsprotokolldatei gespeichert wird. Die Standardposition ist NCHOME/omnibus/log.
  • Geben Sie im Feld Standardnachrichtenstufe die Standardnachrichtenstufe der Anwendung an. Die Standardnachrichtenstufe für die Protokollierung kann auch über das Menü Bearbeiten geändert werden. Die Nachrichtenstufe ist ab dem Zeitpunkt, an dem die Anwendung gestartet wurde, bis zu dem Zeitpunkt gültig, an dem sie beendet wird, wenn sie nicht über das Menü Bearbeiten geändert wird.
    Anmerkung: Das Ändern der Nachrichtenstufe im Feld Standardnachrichtenstufe führt nicht zur Änderung der aktuellen Nachrichtenstufe der Anwendung. Es wirkt sich lediglich auf die Nachrichtenstufe aus, die anfangs festgelegt wurde, als die Anwendung gestartet wurde (die Standardnachrichtenstufe). Um die Generierung umfangreicher Protokolldateien zu vermeiden, müssen Sie die Standardnachrichtenstufe auf den Wert Warnung festlegen.
  • Protokollierungsstufe festlegen
    Sie können die Protokollierungsstufe ändern, um ein detaillierteres Protokoll oder Tracedateien als Unterstützung beim Debugging von MIB Manager bereitzustellen. Die zusätzlichen Informationen, die durch Erhöhen der Protokollierungsstufen bereitgestellt werden, können das Debugging bestimmter Probleme während des Parsing von MIBs erleichtern.
