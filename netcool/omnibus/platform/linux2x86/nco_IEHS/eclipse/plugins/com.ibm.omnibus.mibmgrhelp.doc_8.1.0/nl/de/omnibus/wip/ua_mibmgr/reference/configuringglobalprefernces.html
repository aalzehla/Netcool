IBM Tivoli Netcool/OMNIbus Version 8.1

Globale Benutzervorgaben konfigurieren

Über das Fenster Benutzervorgaben können Sie mehrere globale MIB Manager-Benutzervorgaben konfigurieren.

Klicken Sie zum Konfigurieren der globalen MIB Manager-Benutzervorgaben auf Benutzervorgaben oder klicken Sie im Menü auf Datei > Benutzervorgaben.

Im Fenster Benutzervorgaben können Sie Benutzervorgaben für Verzeichnisse, Export, Allgemeines, Protokollierung und Suche konfigurieren.

  • Benutzervorgaben für Verzeichnisse festlegen
    Mithilfe der Benutzervorgaben für Verzeichnisse können Sie die Verzeichnisse zum Speichern importierter MIBs, von MIB-Daten und exportierter Regeldateien angeben.
  • Benutzervorgaben für den Export festlegen
    In diesem Abschnitt wird beschrieben, wie Sie die Benutzervorgaben für den Export für Regeldateien festlegen.
  • Allgemeine Benutzervorgaben festlegen
    Mithilfe der allgemeinen Benutzervorgaben können Sie die MIB-Importoptionen angeben.
  • Benutzervorgaben für die Protokollierung festlegen
    Mithilfe der Benutzervorgaben für die Protokollierung können Sie das Verzeichnis für die Protokolldatei von MIB Manager angeben und die Standardnachrichtenstufe festlegen.
  • Benutzervorgaben für die Suche festlegen
    In diesem Abschnitt wird beschrieben, wie die Benutzervorgaben von MIB Manager für die Suche nach den MIB-Dateien definiert werden.
