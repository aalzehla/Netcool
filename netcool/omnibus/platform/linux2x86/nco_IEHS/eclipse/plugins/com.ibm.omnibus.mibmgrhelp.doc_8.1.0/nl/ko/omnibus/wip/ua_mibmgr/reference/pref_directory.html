IBM Tivoli Netcool/OMNIbus 버전 8.1

디렉토리 환경 설정 설정

디렉토리 환경 설정을 사용하여 가져온 MIB, MIB 데이터, 내보낸 규칙 파일을 저장하는 데 사용되는 디렉토리를 지정합니다.

디렉토리 환경 설정을 설정하려면, 환경 설정을 클릭하거나 메뉴에서 파일 > 환경 설정을 클릭한 후 환경 설정 창의 왼쪽 분할창에서 디렉토리를 클릭하십시오.

  • 데이터 디렉토리 필드에서 가져온 MIB를 XML 파일로 저장할 위치를 지정하십시오. 기본 위치는 NCHOME/omnibus/var/mibmanager/data입니다.
  • 내보내기 디렉토리 필드에서는 내보낸 파일이 생성되는 위치를 지정하십시오. 내보내기 유형을 기본으로 하는 서브디렉토리가 작성됩니다. 기본 위치는 NCHOME/omnibus/var/mibmanager/export입니다.
  • 가져오기 디렉토리 필드에서 가져온 MIB 파일의 위치를 지정하십시오. 기본 위치는 NCHOME/omnibus/var/mibmanager/mibs입니다.
상위 주제: 글로벌 환경 설정 구성
