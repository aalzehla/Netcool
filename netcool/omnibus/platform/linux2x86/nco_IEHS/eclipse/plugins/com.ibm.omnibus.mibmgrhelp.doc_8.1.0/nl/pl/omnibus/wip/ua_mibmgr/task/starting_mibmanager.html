IBM Tivoli Netcool/OMNIbus wersja 8.1

Uruchamianie programu MIB Manager

Interfejs GUI programu MIB Manager można uruchomić za pomocą wiersza komend. Program ten nie jest przystosowany do uruchamiania z użyciem funkcji kontroli procesów ani do uruchamiania jako usługa Windows.

O tym zadaniu

Informacje na temat wymagań dotyczących uruchamiania programu MIB Manager w systemach operacyjnych AIX i Solaris zawiera dokumentacja dotycząca obsługiwanych systemów operacyjnych.

Przed uruchomieniem programu MIB Manager należy upewnić się, że dla zmiennej środowiskowej $DISPLAY ustawiono działający serwer X11.

Procedura

  1. Użyj następującej komendy, aby uruchomić interfejs GUI programu MIB Manager:

    NCHOME/omnibus/bin/nco_mibmanager

  2. Użyj następującej komendy, aby uruchomić program narzędziowy wiersza komend programu MIB Manager:

    NCHOME/omnibus/bin/nco_mibmanager_batch [opcje]

  • Opcje wiersza komend programu MIB Manager
    Program MIB Manager zawiera program narzędziowy wiersza komend, którego można użyć do importowania danych bazy MIB i eksportowania plików reguł. Komendy można wprowadzać ręcznie lub wywoływać je programowo.
