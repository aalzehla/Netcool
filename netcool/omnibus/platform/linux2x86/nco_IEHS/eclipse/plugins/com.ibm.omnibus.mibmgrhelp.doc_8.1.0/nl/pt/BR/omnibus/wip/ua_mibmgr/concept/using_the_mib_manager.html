Usando o Netcool MIB Manager

É possível usar o MIB Manager para analisar módulos MIB e criar arquivos de regras do Netcool.

O MIB Manager inclui um conjunto de MIBs de base e MIBs de RFC no diretório NCHOME/omnibus/var/mibmanager/mibs. Esses MIBs são fornecidos para permitir que você importe MIBs adicionais. O MIB Manager usa esses MIBs empacotados automaticamente para resolver dependências durante a importação de novos MIBs. Os módulos MIB empacotados ficam visíveis na visualização Módulos MIB quando o MIB Manager é iniciado pela primeira vez.

O MIB Manager também inclui arquivos de dados XML que representam uma importação dos MIBs de base empacotados. O MIB Manager usa esses arquivos XML para armazenar os dados MIB que ele importa dos arquivos MIB de texto bruto.

Inclua seus MIBs em um diretório que possa ser acessado no servidor a partir do qual o MIB Manager é iniciado. Todos os MIBs baseados no padrão deverão estar instalados em um diretório separado que poderá ser colocado no caminho da procura para fácil resolução de dependência. Por padrão, os MIBs básicos de SNMP são instalados no diretório NCHOME/omnibus/var/mibmanager/mibs/base e todos os MIBs de RFC são instalados no diretório NCHOME/omnibus/var/mibmanager/mibs/rfc.

A janela Netcool MIB Manager

Quando você inicia o MIB Manager, os elementos a seguir são exibidos na janela inicial MIB Manager: vários elementos são exibidos na janela inicial MIB Manager. A figura a seguir mostra esses elementos.

Figura 1. Janela inicial MIB Manager
 1  Barra de ferramentas
Fornece acesso às funções do MIB Manager usadas mais frequentemente.
 2  Perspectivas
É possível visualizar dados MIB em uma de duas perspectivas, MIB Manager ou mib2rules. A perspectiva do MIB Manager é a perspectiva padrão. A perspectiva mib2rules concentra-se em Identificadores de Objeto (OIDs) e é fornecida para os usuários que estão familiarizados com o utilitário mib2rules.
 3  Visualização Dispositivo
Use essa visualização para criar, atualizar e excluir dispositivos do MIB Manager.
 4  Visualização Detalhes e visualização Console
A visualização Detalhes exibe os detalhes dos elementos MIB selecionados. A visualização Console mostra todas as mensagens de log que são geradas pelo MIB Manager enquanto a instância atual está em execução.
 5  Módulos MIB
Use essa visualização para procurar e excluir módulos MIB e para procurar valores de MIB específicos.
 6  Visualização Árvore de OID
Essa visualização fornece uma representação de todos os objetos MIB que estão nos módulos MIB importados.
  • A visualização do MIB Modules
    A visualização do MIB Modules fornece uma representação de todos os módulos MIB importados e permite procurar módulos MIB, excluir módulos MIB e procurar valores específicos MIB.
  • A Visualização OID Tree
    A visualização Object Identifier (OID) Tree fornece uma representação de todos objetos MIB localizados nos módulos MIB importados.
  • Importando dados MIB
    Como importar dados MIB SNMP para o MIB Manager.
  • Exportando dados MIB
    Como exportar dados MIB de SNMP do MIB Manager para uma variedade de formatos.
  • Editando traps SNMP
    Como editar traps SNMP.
  • Gerando traps SNMP
    Como gerar traps SNMP. Esta funcionalidade também pode ser usada para testar arquivos de regras gerados pelo MIB Manager.
  • Criando e excluindo dispositivos
    Um dispositivo do MIB Manager é usado para agrupar Módulos MIB associados e definir todos os objetos necessários para exportação. É possível criar, atualizar e excluir dispositivos na GUI do MIB Manager.
