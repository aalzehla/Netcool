Netcool MIB Manager

A Netcool MIB Manager egy IBM® Eclipse alapú grafikus felületi alkalmazás, amely az Egyszerű hálózatkezelési protokoll (SNMP) felügyeleti információs alapfájlok (MIB) értelmezéséhez használhat, amelyekből Netcool szabályfájlokat állathat elő. A mib2rules segédprogram helyettesítőjének szánták.

A Tivoli Netcool/OMNIbus a Netcool MIB Manager parancssori segédprogramot (nco_mibmanager_batch) is tartalmazza, amely segítségével kiadhatja a parancsokat kézzel vagy programozással.

A Netcool MIB Manager az alábbi szolgáltatásokat biztosítja:
  • Az objektumazonosító (OID) fa összeállításához importálja az SNMP MIB fájlokat és feloldja a MIB függőségeket.
  • Az összes vagy néhány importált SNMP objektumot Netcool szabályfájlokba és más fájlformátumokba exportálja.
  • Tartalmazza azokat a MIB fájlokat és RFC MIB fájlokat, amelyeket a többi MIB fájl a leggyakrabban igényel.
  • Eszközmeghatározásokat tartalmaz az importált MIB modulok eszközcentrikus csoportosításának lehetővé tételéhez.
  • A MIB modulokat és az OID fát objektumnév vagy OID alapján képes szűrni és keresni.
  • Az SNMP vizsgálónak (nco_p_mttrapd) vagy más SNMP ügynököknek küldendő SNMP trapeket állítja elő.
  • Rendelkezik egy olyan parancssori felülettel, amelyet a MIB adatok importálására és a szabályfájlok exportálására használhat. A parancsokat kiadhatja kézzel vagy meghívhatja őket programozással.
  • MIB Manager indítása
    Indítsa el a MIB Manager grafikus felületet a parancssorból. Nem úgy tervezték, hogy folyamatvezérlés alatt vagy Windows szolgáltatásként fusson.
  • Netcool MIB Manager használata
    A MIB Manager eszközt a MIB modulok értelmezésére és a Netcool szabályfájlok létrehozásához használhatja.
  • Globális beállítások megadása
    Több MIB Manager globális beállítást is megadhat a Beállítások ablakban.
  • SNMP bemutatása
    Ez a szakasz kiegészítő információkat nyújt az Egyszerű hálózatkezelési protokoll (SNMP) architektúráról, az SNMP management information base (MIB) alkalmazásról és néhány további MIB fogalomról.
