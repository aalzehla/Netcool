Establecimiento de las preferencias de búsqueda

En esta sección se describe cómo definir las preferencias de MIB Manager que se utilizan para buscar los archivos MIB.

Para establecer las preferencias de búsqueda, pulse Preferencias o, desde el menú principal, pulse Archivo > Preferencias y, a continuación, en el panel de la izquierda de la ventana Preferencias, pulse Buscar.

En el campo Vía de acceso de búsqueda de MIB, establezca la vía de acceso de búsqueda de la ubicación de los archivos MIB. Los directorios deben estar separados por punto y coma (;). Se buscan automáticamente los subdirectorios para localizar archivos MIB dependientes.

La vía de acceso predeterminada incluye la ubicación de los archivos MIB base y los archivos MIB RFC empaquetados con MIB Manager. Puede especificar los nuevos directorios que desee al final de la vía de acceso de archivo predeterminada. De este modo se garantiza que inicialmente se explora el directorio base para buscar las dependencias que contienen las MIB utilizadas más comúnmente y, a continuación, se explora el directorio rfc para buscar las MIB utilizadas con menor frecuencia.
