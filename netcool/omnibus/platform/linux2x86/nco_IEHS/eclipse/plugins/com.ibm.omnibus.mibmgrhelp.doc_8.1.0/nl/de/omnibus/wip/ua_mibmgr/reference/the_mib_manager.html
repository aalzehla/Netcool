IBM Tivoli Netcool/OMNIbus Version 8.1

Netcool MIB Manager

Bei Netcool MIB Manager handelt es sich um eine Eclipse-basierte IBM® GUI-Anwendung, die Sie zum Parsen von SNMP-MIB-Dateien (MIB = Management Information Base) verwenden können, aus denen Sie Netcool-Regeldateien generieren können. Die Anwendung dient als Ersatz für das Dienstprogramm mib2rules.

Tivoli Netcool/OMNIbus beinhaltet auch ein Netcool MIB Manager-Befehlszeilendienstprogramm (nco_mibmanager_batch), mit dem Sie Befehle manuell oder programmgesteuert ausgeben können.

Netcool MIB Manager umfasst folgende Features:
  • Importieren von SNMP-MIB-Dateien und Auflösen von MIB-Abhängigkeiten zum Aufbauen einer OID-Baumstruktur.
  • Exportieren eines Teils der oder aller importierten SNMP-Objekte in Netcool-Regeldateien und weitere Dateiformate.
  • Enthält die Basis-MIBs und RFC-MIBs, die häufig von anderen MIB-Dateien benötigt werden.
  • Enthält Gerätedefinitionen, die eine gerätebezogene Gruppierung der importierten MIB-Module ermöglichen.
  • Filtern und Suchen von MIB-Modulen und der OID-Baumstruktur nach Objektname oder OID.
  • Generieren von SNMP-Traps zum Senden an den SNMP-Test (nco_p_mttrapd) oder an andere SNMP-Agenten.
  • Eine Befehlszeilenschnittstelle ist enthalten, mit deren Hilfe Sie MIB-Daten importieren und Regeldateien exportieren können. Sie können Befehle manuell ausgeben oder programmgesteuert aufrufen.
  • MIB Manager starten
    Starten Sie die MIB Manager-GUI über die Befehlszeile. Die Komponente ist nicht dafür vorgesehen, innerhalb einer Prozesssteuerung oder als Windows-Dienst ausgeführt zu werden.
  • Netcool MIB Manager verwenden
    Sie können MIB Manager für das Parsing von MIB-Modulen und die Erstellung von Netcool-Regeldateien verwenden.
  • Globale Benutzervorgaben konfigurieren
    Über das Fenster Benutzervorgaben können Sie mehrere globale MIB Manager-Benutzervorgaben konfigurieren.
  • Informationen zu SNMP
    Dieser Abschnitt enthält zusätzliche Informationen zur SNMP-Architektur, zur Management Information Base (MIB) von SNMP sowie zu weiteren MIB-Konzepten.
