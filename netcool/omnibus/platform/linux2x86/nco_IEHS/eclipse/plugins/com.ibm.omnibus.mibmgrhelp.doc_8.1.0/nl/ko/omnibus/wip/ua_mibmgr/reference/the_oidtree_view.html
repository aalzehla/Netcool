OID 트리 보기

OID(Object Identifier) 트리 보기는 가져온 MIB 모듈에 있는 모든 MIB 오브젝트의 표시를 제공합니다.

보기 드롭 다운 목록에서 다음 옵션 중 하나를 선택하여 OID 트리 보기에서 데이터를 필터링할 수 있습니다.
  • 모두
  • 트랩/알림
  • 오브젝트
  • 모듈

OID 트리 보기에서 특정 값을 검색하려면, 검색 필드에 텍스트를 입력하고 Enter를 누르거나 (검색)을 클릭하십시오.

MIB Manager는 선택한 항목부터 또는 현재 선택한 항목이 없는 경우 목록의 첫 번째 항목부터 검색을 시작합니다. 목록에서 다음으로 일치하는 항목을 찾으려면, 계속해서 (검색)을 클릭하십시오.

항목을 찾으면, OID 트리 보기에서 자동으로 강조되어 표시됩니다.

검색 필드에서는 유효한 정규식을 지정하거나 전체 또는 일부 오브젝트 이름이나 OID를 표시하는 평문을 입력하십시오. 각 브랜치 노드에 표시되는 숫자는 해당 오브젝트 OID의 마지막 숫자입니다.
