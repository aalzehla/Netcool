Sicht "OID-Baumstruktur"

Die Sicht "OID-Baumstruktur" (OID = Objekt-ID) bietet eine Darstellung aller MIB-Objekte, die in den importierten MIB-Modulen enthalten sind.

Sie können die Daten in der Sicht OID-Baumstruktur filtern, indem Sie aus der Dropdown-Liste Sicht eine der folgenden Optionen auswählen:
  • Alles
  • Traps/Benachrichtigungen
  • Objekte
  • Module

Um in der Sicht OID-Baumstruktur einen bestimmten Wert zu suchen, müssen Sie den Text in das Feld Suchen eingeben und entweder die Eingabetaste drücken oder auf ("Suchen") klicken.

MIB Manager beginnt beim ausgewählten Element oder, falls momentan kein Element ausgewählt ist, beim ersten Element in der Liste mit der Suche. Klicken Sie zum Suchen des nächsten übereinstimmenden Elements in der Liste nochmals auf ("Suchen").

Wenn ein Element gefunden wird, wird es automatisch hervorgehoben und in der Sicht OID-Baumstruktur angezeigt.

Geben Sie im Feld Suchen einen gültigen regulären Ausdruck an oder geben Sie einfachen Text ein, der einen vollständigen Objektnamen (bzw. eine OID) oder einen Teil davon darstellt. Die Zahl, die in jedem Verzweigungsknoten angezeigt wird, ist die letzte Ziffer der OID für das betreffende Objekt.
