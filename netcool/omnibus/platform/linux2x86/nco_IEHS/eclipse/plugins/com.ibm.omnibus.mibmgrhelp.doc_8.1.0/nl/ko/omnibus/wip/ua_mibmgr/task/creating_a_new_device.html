IBM Tivoli Netcool/OMNIbus 버전 8.1

디바이스 작성과 삭제

MIB Manager 디바이스는 연관된 MIB 모듈을 그룹화하고 내보내기에 필요한 모든 오브젝트를 정의하는 데 사용됩니다. MIB Manager GUI에서 디바이스를 작성하고 업데이트하며 삭제할 수 있습니다.

이 태스크 정보

MIB Manager 디바이스는 디바이스 보기에 표시됩니다. 디바이스 트리에서 디바이스를 선택하여 디바이스의 제조업체, 모델 번호, OS 버전을 표시하십시오.

프로시저

디바이스 작성

  1. 새 디바이스를 작성하려면 다음을 수행하십시오.
    1. MIB 모듈 보기에서 최소 하나 이상의 MIB 모듈을 선택한 후 (디바이스 작성)을 클릭하십시오.
    2. 제조업체 필드의 드롭 다운 목록에서 디바이스 제조업체를 선택하십시오.

      모델 및 OS 버전 필드는 사용 가능한 옵션으로 채워집니다.

    3. 모델 필드의 드롭 다운 목록에서 옵션을 선택하십시오.

      사용 가능한 모델이 없는 경우 설명하는 모델 이름을 입력하십시오.

    4. OS 버전 필드의 드롭 다운 목록에서 옵션을 선택하십시오.

      사용 가능한 OS 버전이 없는 경우 설명하는 OS 버전 이름을 입력하십시오.

    5. 맵핑된 MIB 모듈 필드에서는 새 디바이스와 연관시키려는 MIB 모듈을 선택하십시오.
    6. 확인을 클릭하여 디바이스를 작성하십시오.

      새 디바이스가 디바이스 보기에 표시됩니다.

디바이스 업데이트

  1. 디바이스를 업데이트하려면 다음을 수행하십시오.
    1. 디바이스 보기에서 디바이스를 선택한 후 OS 버전이 표시될 때까지 서브노드를 펼치십시오.
    2. OS 버전을 선택한 후 (디바이스 편집)을 클릭하십시오.
    3. 제조업체 필드의 드롭 다운 목록에서 디바이스 제조업체를 선택하십시오.

      모델 및 OS 버전 필드는 사용 가능한 옵션으로 채워집니다.

    4. 모델 필드의 드롭 다운 목록에서 옵션을 선택하십시오.

      사용 가능한 모델이 없는 경우 설명하는 모델 이름을 입력하십시오.

    5. OS 버전 필드의 드롭 다운 목록에서 옵션을 선택하십시오.

      사용 가능한 OS 버전이 없는 경우 설명하는 OS 버전 이름을 입력하십시오.

    6. 맵핑된 MIB 모듈 필드에서는 새 디바이스와 연관시키려는 MIB 모듈을 선택하십시오.
    7. 확인을 클릭하여 디바이스 세부사항을 업데이트하십시오.

      업데이트된 디바이스 세부사항이 디바이스 보기에 표시됩니다.

디바이스 삭제

  1. 디바이스 보기에서는 디바이스를 선택한 후 (디바이스 삭제)을 클릭하십시오.

    여러 디바이스를 삭제하려면, Ctrl+Alt 키를 누른 상태에서 왼쪽 마우스 단추를 사용하여 요소를 선택한 후 (디바이스 삭제)을 클릭하십시오.
