Creación y supresión de dispositivos

Los dispositivos de MIB Manager se utilizan para agrupar módulos MIB asociados y definir todos los objetos requeridos para la exportación. Puede crear, actualizar y suprimir dispositivos en la interfaz gráfica de usuario de MIB Manager.

Acerca de esta tarea

Los dispositivos de MIB Manager se muestran en la vista Dispositivo. Seleccione un dispositivo del árbol Dispositivo para visualizar el fabricante, el número de modelo y la versión del sistema operativo del dispositivo.

Procedimiento

Creación de un dispositivo

  1. Para crear un nuevo dispositivo:
    1. En la vista Módulos MIB, seleccione como mínimo un módulo MIB y, a continuación, pulse (Crear dispositivo).
    2. En el campo Fabricante, seleccione un fabricante de dispositivos de la lista desplegable.

      Los campos Modelo y Versión de sistema operativo se rellenan con las opciones disponibles.

    3. En el campo Modelo, seleccione una opción de la lista desplegable.

      Si no hay modelos disponibles, indique un nombre de modelo descriptivo.

    4. En el campo Versión de sistema operativo, seleccione una opción de la lista desplegable.

      Si no hay versiones de sistema operativo disponibles, indique un nombre de versión de sistema operativo descriptivo.

    5. En el campo Módulos MIB correlacionados, seleccione los módulos MIB que desea asociar con el nuevo dispositivo.
    6. Pulse Aceptar para crear el dispositivo.

      El nuevo dispositivo se muestra en la vista Dispositivo.

Actualización de un dispositivo

  1. Para actualizar un dispositivo:
    1. En la vista Dispositivo, seleccione un dispositivo y expanda sus subnodos hasta que se visualice la versión de sistema operativo.
    2. Seleccione la versión de sistema operativo y, a continuación, pulse (Editar dispositivo).
    3. En el campo Fabricante, seleccione un fabricante de dispositivos de la lista desplegable.

      Los campos Modelo y Versión de sistema operativo se rellenan con las opciones disponibles.

    4. En el campo Modelo, seleccione una opción de la lista desplegable.

      Si no hay modelos disponibles, indique un nombre de modelo descriptivo.

    5. En el campo Versión de sistema operativo, seleccione una opción de la lista desplegable.

      Si no hay versiones de sistema operativo disponibles, indique un nombre de versión de sistema operativo descriptivo.

    6. En el campo Módulos MIB correlacionados, seleccione los módulos MIB que desea asociar con el nuevo dispositivo.
    7. Pulse Aceptar para actualizar los detalles del dispositivo.

      Los detalles del dispositivo actualizado se visualizarán en la vista Dispositivo.

Supresión de un dispositivo

  1. En la vista Dispositivo, seleccione el dispositivo y pulse (Suprimir dispositivo).

    Para suprimir varios dispositivos, mantenga pulsadas las teclas Control+Alt, seleccione los elementos con el botón izquierdo del ratón y, a continuación, pulse (Suprimir dispositivo).
