Globális beállítások megadása

Több MIB Manager globális beállítást is megadhat a Beállítások ablakban.

A MIB Manager globális beállításainak megadásához kattintson a Beállítások ikonra, vagy a menüben kattintson a Fájl > Beállítások menüpontra.

A Beállítások ablakban megadhatja a könyvtár, exportálás, általános, naplózási és keresési beállításokat.

  • Könyvtár beállítások megadása
    A könyvtár beállításokkal adhatja meg az importált MIB-ek, MIB adatok és az exportált szabályfájlok tárolásához használandó könyvtárakat.
  • Exportálási beállítások megadása
    Ez a rész a szabályfájl exportálási beállításainak megadását írja le.
  • Általános beállítások megadása
    Az általános beállításokkal adhatja meg az MIB importálási paramétereket.
  • Naplózási beállítások megadása
    A naplózási beállításokkal adhatja meg a MIB Manager naplófájl könyvtárát, és állíthatja be az alapértelmezett üzenetszintet.
  • Keresési beállítások megadása
    Ez a rész az MIB fájlok keresésére használt MIB Manager beállítások megadásának módját írja le.
Szülőtéma: Netcool MIB Manager
