Establecimiento de las preferencias de registro cronológico

Utilice las preferencias de registro cronológico para especificar el directorio de archivos de registro de MIB Manager y establecer el nivel de mensaje predeterminado.

Para establecer las preferencias de directorio, pulse Preferencias o, desde el menú principal, pulse Archivo > Preferencias y, a continuación, en el panel de la izquierda de la ventana Preferencias, pulse Registro cronológico.

  • En el campo Directorio de registro, especifique donde se grabará el archivo de registro de la aplicación. La ubicación predeterminada es NCHOME/omnibus/log.
  • En el campo Nivel de mensaje predeterminado, especifique el nivel de mensaje predeterminado de la aplicación. También puede utilizar el menú Editar para cambiar el nivel de registro cronológico de mensaje predeterminado. El nivel de mensaje es válido desde que se inicia la aplicación por primera vez hasta que se cierra la aplicación, a menos que se cambie utilizando el menú Editar.
    Nota: Cambiando el nivel de mensaje en el campo Nivel de mensaje predeterminado no se cambia el nivel de mensaje actual de la aplicación. Sólo afecta al nivel de mensaje que se ha establecido inicialmente al iniciar la aplicación por primera vez (el nivel de mensaje predeterminado). Para evitar generar archivos de registro muy grandes, establezca el nivel de mensaje predeterminado en aviso.
  • Establecimiento del nivel de registro cronológico
    Puede cambiar el nivel de registro cronológico para proporcionar archivos de registro o de rastreo más detallados para facilitar la depuración de MIB Manager. La información adicional que se proporciona aumentando los niveles de registro cronológico puede ayudarle en la depuración de problemas específicos al analizar las MIB.
