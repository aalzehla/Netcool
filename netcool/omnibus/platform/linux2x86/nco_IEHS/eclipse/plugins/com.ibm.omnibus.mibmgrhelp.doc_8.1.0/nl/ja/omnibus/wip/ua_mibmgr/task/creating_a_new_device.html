IBM Tivoli Netcool/OMNIbus バージョン 8.1

デバイスの作成および削除

MIB マネージャー・デバイスは、関連する MIB モジュールのグループ化、およびエクスポートに必要なすべてのオブジェクトの定義に使用されます。デバイスの作成、更新、削除は MIB マネージャー GUI で行えます。

このタスクについて

MIB マネージャー・デバイスは、「デバイス」ビューに表示されます。 デバイスを「デバイス」ツリーから選択して、デバイスの 製造メーカー、型式番号、および OS バージョンを表示します。

手順

デバイスの作成

  1. 新規デバイスを作成するには、以下の手順を実行します。
    1. 「MIB モジュール」ビューで、 1 つ以上の MIB モジュールを選択してから (デバイスの 作成) をクリックします。
    2. 「製造メーカー」フィールドで、ドロップダウン・リストから デバイスの製造メーカーを選択します。

      「モデル」フィールドおよび「OS バージョン」フィールドに、使用可能なオプションが取り込まれます。

    3. 「モデル」フィールドで、ドロップダウン・リストから オプションを選択します。

      使用可能なモデルがない場合、モデルの記述名を 入力します。

    4. 「OS バージョン」フィールドで、ドロップダウン・リストから オプションを選択します。

      使用可能な OS バージョンがない場合、 OS バージョンの記述名を入力します。

    5. 「マップ済み MIB モジュール」フィールドで、 新規デバイスと関連付ける MIB モジュールを選択します。
    6. 「OK」をクリックして、デバイスを作成します。

      新規デバイスが「デバイス」ビューに表示されます。

デバイスの更新

  1. デバイスを更新するには、以下の手順を更新します。
    1. 「デバイス」ビューで、デバイスを選択してから、 デバイスのサブノードを、OS バージョンが表示されるまで展開します。
    2. OS バージョンを選択してから、 (デバイスの 編集) をクリックします。
    3. 「製造メーカー」フィールドで、ドロップダウン・リストから デバイスの製造メーカーを選択します。

      「モデル」フィールドおよび「OS バージョン」フィールドに、使用可能なオプションが取り込まれます。

    4. 「モデル」フィールドで、ドロップダウン・リストから オプションを選択します。

      使用可能なモデルがない場合、モデルの記述名を 入力します。

    5. 「OS バージョン」フィールドで、ドロップダウン・リストから オプションを選択します。

      使用可能な OS バージョンがない場合、 OS バージョンの記述名を入力します。

    6. 「マップ済み MIB モジュール」フィールドで、 新規デバイスと関連付ける MIB モジュールを選択します。
    7. 「OK」をクリックして、デバイスの詳細を更新します。

      更新されたデバイスの詳細が、「デバイス」ビューに表示されます。

デバイスの削除

  1. 「デバイス」ビューで、デバイスを選択してから、 (デバイスの削除) をクリックします。

    複数のデバイスを削除するには、 Ctrl キーと Alt キーを同時に押したまま左マウス・ボタンを使用して エレメントを選択してから、 (デバイスの削除) をクリックします。
