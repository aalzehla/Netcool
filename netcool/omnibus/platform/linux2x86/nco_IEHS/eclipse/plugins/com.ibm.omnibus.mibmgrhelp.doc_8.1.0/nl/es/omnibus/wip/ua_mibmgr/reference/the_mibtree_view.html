La vista Módulos MIB

La vista Módulos MIB proporciona una representación de todos los módulos MIB importados y permite buscar módulos MIB, suprimir módulos MIB y buscar valores MIB concretos.

Para desplazarse por la vista Módulos MIB, expanda y contraiga los nodos del árbol MIB. Las ramificaciones del nivel superior del árbol MIB contienen los nombres de los módulos MIB, y dentro de cada ramificación de módulo MIB se encuentran los otros elementos que componen la MIB.

Puede filtrar los datos de la vista Módulos MIB seleccionando una de las siguientes opciones de la lista desplegable Ver:
  • Todas
  • Módulos
  • Enumeraciones
  • Convenciones textuales
  • Sintaxis
  • Importaciones
  • Exportaciones
  • Columnas
  • Descripciones

Para explorar la vista Módulos MIB para buscar un valor concreto, escriba el texto en el campo Buscar y pulse Intro o pulse (Buscar).

MIB Manager inicia la búsqueda desde el elemento seleccionado o, si no hay ningún elemento seleccionado, desde el primer elemento de la lista. Para encontrar el siguiente elemento coincidente de la lista, siga pulsando (Buscar).

Cuando se localiza un elemento, se resalta automáticamente y se muestra en la vista Módulos MIB.

En el campo Buscar, especifique una expresión regular válida o texto sin formato para representar un nombre de objeto completo o parcial.

Para suprimir uno o varios módulos MIB, pulse con el botón derecho del ratón el módulo o módulos MIB seleccionados y, a continuación, en el menú emergente, seleccione Suprimir. El módulo MIB seleccionado se suprimirá de MIB Manager. Sin embargo, los archivos de origen MIB originales, suministrados para la importación MIB, no se suprimen.

Nota: Sólo debe suprimir el módulo MIB si contiene errores y si existe una nueva versión de sustitución del módulo MIB. La supresión de módulos MIB mediante este método puede causar que falten dependencias MIB. También se pueden generar archivos de reglas incompletos durante el proceso de exportación junto con mensajes de aviso o error en el archivo de registro.
