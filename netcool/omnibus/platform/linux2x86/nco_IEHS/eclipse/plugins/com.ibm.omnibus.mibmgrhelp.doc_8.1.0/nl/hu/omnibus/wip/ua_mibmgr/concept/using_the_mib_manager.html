IBM Tivoli Netcool/OMNIbus 8.1. változat

Netcool MIB Manager használata

A MIB Manager eszközt a MIB modulok értelmezésére és a Netcool szabályfájlok létrehozásához használhatja.

A MIB Manager alap MIB-ek és RFC MIB-ek készletét tartalmazza az NCHOME/omnibus/var/mibmanager/mibs könyvtárban. Ezek a MIB-ek további MIB-ek engedélyezéséhez és importálásához lettek megadva. A MIB Manager automatikusan ezeket a kötegelt MIB-eket használja a függőségek feloldására az új MIB-ek importálása közben. A kötegelt MIB modulok a MIB Modulok nézetben jelennek meg, a MIB Manager első indításakor.

A MIB Manager olyan XML adatfájlokat is tartalmaz, amelyek a kötegelt alap MIB-ek importálását ábrázolják. A MIB Manager ezeket az XML fájlokat használja a MIB adatok tárolására, amelyeket a nyers szöveges MIB fájlokból importál.

Adja hozzá a MIB-eket ahhoz a könyvtárhoz, amely elérhető a kiszolgálóról, amelyből a MIB Manager el lett indítva. Minden szabvány alapú MIB-t különálló könyvtárba kell telepíteni, melyet a keresési útvonalra helyezhet a függőség feloldás megkönnyítése érdekében. Alapértelmezésben az alap SNMP MIB-ek a NCHOME/omnibus/var/mibmanager/mibs/base könyvtárban vannak telepítve, és az összes RFC MIB a NCHOME/omnibus/var/mibmanager/mibs/rfc könyvtárban van telepítve.

A Netcool MIB Manager ablak

A MIB Manager indításakor a következő elemek jelennek meg a kezdeti MIB Manager ablakban: számos elem a MIB Manager ablakban jelenik meg. A következő ábra ezeket az elemeket ábrázolja.

1. Ábra MIB Manager kezdeti ablak
 1  Eszköztár
Hozzáférést biztosít a leggyakrabban használt MIB Manager funkciókhoz.
 2  Perspektívák
A MIB adatok a két perspektíva egyikében jeleníthetők meg, MIB Manager és mib2rules. A MIB Manager nézet az alapértelmezett nézet. A mib2rules perspektíva az Objektumazonosítókra (OID-k) fókuszál és olyan felhasználók számára kerül megadásra, akik ismerik a mib2rules segédprogramot.
 3  Eszköz nézet
Ebben a nézetben hozhatja létre, frissítheti és törölheti a MIB Manager eszközöket.
 4  Részletek nézet és Konzol nézet
A Részletek nézet a MIB elemek részleteit jeleníti meg. A Konzol nézet a MIB Manager által előállított összes naplóüzenetet megjeleníti, miközben az aktuális példány fut.
 5  MIB modulok
Ebben a nézetben keresheti ki és törölheti a MIB modulokat és itt keresheti ki az adott MIB értékeket.
 6  OID fa nézet
Ez a nézet jeleníti meg az importált MIB modulokban lévő összes MIB objektumot.
  • MIB modulok nézet
    Az MIB modulok nézet az összes importált MIB modult ábrázolja, és lehetővé teszi az MIB modulok keresését, törlését, és adott MIB értékek keresését.
  • OID fa nézet
    Az Objektumazonosító (OID) fa nézet az importált MIB modulokban található összes MIB objektumot ábrázolja.
  • MIB adatok importálása
    SNMP MIB adatok importálása az MIB Manager termékbe.
  • MIB adatok exportálása
    SNMP MIB adatok exportálása a MIB Manager termékből különböző formátumokban.
  • SNMP trapek szerkesztése
    SNMP trapek szerkesztésének módja.
  • SNMP trapek előállítása
    SNMP trapek előállításának módja. Ez a funkcionalitás a MIB Manager által előállított szabályfájlok tesztelésére is használható.
  • Eszközök létrehozása és törlése
    A MIB Manager eszközökkel lehet csoportosítani a társított MIB modulokat és meghatározni az exportáláshoz szükséges objektumokat. A MIB Manager grafikus felületén eszközöket hozhat létre, frissíthet és törölhet.
