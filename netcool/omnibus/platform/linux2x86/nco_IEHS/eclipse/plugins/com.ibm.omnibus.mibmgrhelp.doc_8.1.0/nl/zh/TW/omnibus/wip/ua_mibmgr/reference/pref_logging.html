設定記載喜好設定

使用記載喜好設定來指定 MIB Manager 日誌檔目錄，以及設定預設訊息層次。

若要設定目錄喜好設定，請按一下 喜好設定，或從主功能表中按一下檔案 > 喜好設定，然後從「喜好設定」視窗的左窗格中，按一下記載。

  • 在日誌目錄欄位中，指定寫入應用程式日誌檔的位置。預設位置為 NCHOME/omnibus/log。
  • 在預設訊息層次欄位中，指定應用程式的預設訊息層次。您也可以使用編輯功能表來變更預設訊息記載層次。訊息層次生效時間是從應用程式首次啟動直到應用程式關閉為止，除非使用編輯功能表來變更。
    註： 在預設訊息層次欄位中變更訊息層次，不會變更應用程式的現行訊息層次。它只會影響應用程式首次啟動時起始設定的訊息層次（預設訊息層次）。若要避免產生大的日誌檔，請將預設訊息層次設為警告。
  • 設定記載層次
    您可以變更記載層次以提供更詳細的日誌或追蹤檔案，來協助對 MIB Manager 進行除錯。透過遞增記載層次提供的額外資訊可協助您在剖析 MIB 時除錯特定問題。
