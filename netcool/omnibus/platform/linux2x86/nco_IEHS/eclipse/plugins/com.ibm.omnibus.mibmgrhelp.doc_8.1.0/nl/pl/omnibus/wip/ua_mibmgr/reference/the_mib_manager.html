IBM Tivoli Netcool/OMNIbus wersja 8.1

Netcool MIB Manager

Program Netcool MIB Manager jest aplikacją z interfejsem GUI firmy IBM® opartą na technologii Eclipse, która może być używana do analizy plików bazy informacji zarządzania (MIB) protokołu Simple Network Management Protocol (SNMP), z których można wygenerować pliki reguł programu Netcool. Ma ona zastąpić program narzędziowy mib2rules.

Program Tivoli Netcool/OMNIbus zawiera także program narzędziowy wiersza komend programu Netcool MIB Manager (nco_mibmanager_batch), którego można używać do ręcznego lub programowego wprowadzania komend.

Program Netcool MIB Manager ma następujące funkcje:
  • Importuje pliki MIB protokołu SNMP i rozwiązuje zależności bazy MIB w celu zbudowania drzewa identyfikatora obiektu OID (Object Identifier).
  • Może wyeksportować niektóre lub wszystkie zaimportowane obiekty SNMP do plików reguł Netcool i plików w innych formatach.
  • Zawiera podstawowe bazy MIB i bazy MIB RFC najczęściej wymagane przez inne pliki MIB.
  • Zawiera definicje urządzeń umożliwiające grupowanie zaimportowanych modułów MIB wg urządzeń.
  • Może filtrować i wyszukiwać moduły MIB oraz filtrować i przeszukiwać drzewo OID wg nazwy bądź identyfikatora obiektu.
  • Może generować pułapki SNMP i przesyłać je do sondy SNMP (nco_p_mttrapd) lub innych agentów SNMP.
  • Zawiera interfejs wiersza komend, którego można używać do importowania danych bazy MIB i eksportowania plików reguł. Komendy można wprowadzać ręcznie lub wywoływać je programowo.
  • Uruchamianie programu MIB Manager
    Interfejs GUI programu MIB Manager można uruchomić za pomocą wiersza komend. Program ten nie jest przystosowany do uruchamiania z użyciem funkcji kontroli procesów ani do uruchamiania jako usługa Windows.
  • Używanie programu Netcool MIB Manager
    Za pomocą programu MIB Manager można analizować moduły MIB i tworzyć pliki reguł programu Netcool.
  • Konfigurowanie preferencji globalnych
    Można skonfigurować kilka preferencji globalnych programu MIB Manager za pomocą okna Preferencje.
  • Na temat protokołu SNMP
    Niniejsza sekcja zawiera dodatkowe informacje na temat architektury protokołu SNMP (Simple Network Management Protocol), bazy MIB protokołu SNMP oraz niektórych dodatkowych pojęć dotyczących bazy MIB.
