编辑 SNMP 陷阱

如何编辑 SNMP 陷阱。

关于此任务

在“详细信息”视图中，可以编辑 SNMP 陷阱的以下字段：
  • @Severity
  • @Type
  • @ExpireTime
  • CODEBLOCK
注： SNMP 不提供用于指定设置 (@Type ) 或清除 (@Severity) 信息的标准机制。您必须手动执行此信息。

在“详细信息”视图中，双击字段值以使其变为可编辑状态。对字段进行更改后，单击 以保存更改，或单击 取消更改。

过程

要编辑 SNMP 陷阱：

  1. 在“MIB 模块”视图或“OID 树”视图中，选择一个陷阱。

    有关该陷阱的详细信息将显示在“详细信息”视图中。

  2. 要编辑 @Severity 字段，请双击字段值并从下拉列表中选择新值。
  3. 要编辑 @Type 字段，请双击字段值并从下拉列表中选择新值。
  4. 要编辑 @ExpireTime 字段，请双击字段值并输入新的时间段（秒数、分钟数、小时数或天数）。
  5. 要编辑 CODEBLOCK 字段，请双击字段值（即使字段为空）并输入新的代码块。
