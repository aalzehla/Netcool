Establecimiento de las preferencias de directorio

Utilice las preferencias de directorio para especificar los directorios que se utilizarán para almacenar las MIB importadas, los datos de MIB y los archivos de reglas exportados.

Para establecer las preferencias de directorio, pulse Preferencias o, desde el menú, pulse Archivo > Preferencias y, a continuación, en el panel de la izquierda de la ventana Preferencias, pulse Directorio.

  • En el campo Directorio de datos, especifique dónde se almacenarán las MIB importadas como archivos XML. La ubicación predeterminada es NCHOME/omnibus/var/mibmanager/data.
  • En el campo Directorio de exportación, especifique donde se generan los archivos exportados. Se crea un subdirectorio basado en el tipo de exportación. La ubicación predeterminada es NCHOME/omnibus/var/mibmanager/export.
  • En el campo Directorio de importación, especifique donde se ubican los archivos MIB importados. La ubicación predeterminada es NCHOME/omnibus/var/mibmanager/mibs.
