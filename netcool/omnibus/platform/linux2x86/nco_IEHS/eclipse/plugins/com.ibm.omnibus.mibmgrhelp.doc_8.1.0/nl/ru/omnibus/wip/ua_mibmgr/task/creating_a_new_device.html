IBM Tivoli Netcool/OMNIbus, Версия 8.1

Создание и удаление устройств

Устройство MIB Manager используется для группировки связанных модулей MIB и для определения всех объектов, требуемых для экспорта. Вы можете создавать, обновлять и удалять устройства в графическом пользовательском интерфейсе MIB Manager.

Об этой задаче

Устройства MIB Manager показаны в представлении Устройство. Выберите устройство в дереве Устройство, чтобы вывести производителя, номер модели и версию ОС этого устройства.

Процедура

Создание устройства

  1. Чтобы создать новое устройство:
    1. В представлении Модули MIB выберите по крайней мере один модуль MIB, а затем щелкните по значку (Создать устройство).
    2. В поле Производитель выберите производителя устройства из выпадающего списка.

      В полях Модель и Версия ОС содержатся допустимые опции выбора модели и операционной системы.

    3. В поле Модель выберите допустимую опцию из выпадающего списка.

      Если никакие модели не допустимы, введите описательное имя модели.

    4. В поле Версия ОС выберите допустимую опцию из выпадающего списка.

      Если никакие версии ОС не допустимы, введите описательное имя версии ОС.

    5. В поле Отображенные модули MIB выберите модули MIB, которые вы хотите связать с новым устройством.
    6. Нажмите кнопку OK, чтобы создать устройство.

      Новое устройство выводится в представлении Устройство.

Изменение устройства

  1. Чтобы изменить устройство:
    1. В представлении Устройство выберите устройство и раскройте его подузлы, пока не появится версия операционной системы.
    2. Выберите версию ОС и щелкните по значку (Изменить устройство).
    3. В поле Производитель выберите производителя устройства из выпадающего списка.

      В полях Модель и Версия ОС содержатся допустимые опции выбора модели и операционной системы.

    4. В поле Модель выберите допустимую опцию из выпадающего списка.

      Если никакие модели не допустимы, введите описательное имя модели.

    5. В поле Версия ОС выберите допустимую опцию из выпадающего списка.

      Если никакие версии ОС не допустимы, введите описательное имя версии ОС.

    6. В поле Отображенные модули MIB выберите модули MIB, которые вы хотите связать с новым устройством.
    7. Нажмите кнопку OK, чтобы изменить подробности устройства.

      Подробности измененного устройства выводятся в представлении Устройство.

Удаление устройства

  1. В представлении Устройство выберите устройство, а затем щелкните по значку (Удалить устройство).

    Чтобы удалить несколько устройств, удерживайте нажатыми клавиши Ctrl+Alt, выберите элементы при помощи левой кнопки мыши, а затем щелкните по значку (Удалить устройство).
