שימוש ב-MIB Manager

ניתן להשתמש ב-MIB Manager לניתוח מודולים של MIB וליצירת קובצי כללים של Netcool.

MIB Manager כולל ערכה של רכיבי MIB בסיסיים ורכיבי RFC MIB, הממוקמים במדריך $NCHOME/omnibus/platform/arch/mibmanager/workspace/mibs. רכיבים אלה ניתנים במטרה לאפשר לכם לייבא רכיבי MIB נוספים, ו-MIB Manager משתמש במודולים מצורפים אלה של MIB באופן אוטומטי כדי לפענח יחסי תלות במהלך יבוא של רכיבי MIB חדשים. ניתן לראות את מודולי ה-MIB המצורפים בתצוגה מודולים של MIB בעת הפעלת MIB Manager בפעם הראשונה.

כמו כן, MIB Manager כולל קובצי נתונים של XML המייצגים יבוא של רכיבי ה-MIB המצורפים. MIB Manager משתמש בקובצי XML אלה לאחסון נתוני ה-MIB שהוא מייבא מקובצי תמליל גולמי של MIB.

הוסיפו את רכיבי ה-MIB שלכם למדריך שניתן לגשת אליו מהמחשב שבו מופעל היישום של MIB Manager. יש להתקין את כל רכיבי ה-MIB המבוססים על תקנים במדריך נפרד שניתן למקם אותו בנתיב החיפוש כדי לפתור יחסי תלות בקלות. כברירת מחדל, הרכיבים הבסיסיים של SNMP MIB מותקנים במדריך $NCHOME/omnibus/platform/arch/mibmanager/workspace/mibs/base, וכל רכיבי ה-RFC MIB מותקנים במדריך $NCHOME/omnibus/platform/arch/mibmanager/workspace/mibs/rfc.

החלון של Netcool MIB Manager

בעת הפעלת MIB Manager, המרכיבים הבאים מוצגים בחלון הראשון של MIB Manager:
  • תפריט
  • סרגל כלים
  • התצוג התקן: השתמשו בתצוגה זו כדי ליצור, לעדכן ולמחוק התקנים של MIB Manager.
  • מודולים של MIB: השתמשו בתצוגה זו כדי לחפש ולמחוק מודולי MIB, וכדי לחפש אחר ערכי MIB ספציפיים.
  • התצוגה עץ OID: תצוגה זו מספקת ייצוג של כל אובייקטי ה-MIB הממוקמים במודולי ה-MIB שיובאו.
  • התצוגה פרטים: תצוגה זו מציגה את הפרטים של מרכיבי ה-MIB שנבחרו.
  • קונסול: תצוגה זו מציגה את כל הודעות היומן המופקות על-ידי MIB Manager כאשר המופע הנוכחי רץ.

תוכלו להציג את נתוני ה-MIB באמצעות אחת משתי הפרספקטיבות, MIB Manager או mib2rules. הפרספקטיבה של MIB Manager היא הפרספקטיבה המוגדרת כברירת מחדל. הפרספקטיבה mib2rules מתמקדת בזיהויי אובייקטים (רכיבי OID) והיא ניתנת לנוחיותם של המשתמשים המכירים את תוכנית השירות mib2rules.

  • תצוגת המודולים של MIB
    התצוגה מודולים של MIB מספקת ייצוג של כל המודולים של MIB שיובאו ומאפשרת לכם לחפש אחר מודולים של MIB, למחוק מודולים של MIB ולחפש אחר ערכים ספציפיים של MIB.
  • התצוגה 'עץ OID'
    התצוגה של עץ זיהוי האובייקט (OID) מספקת ייצוג של כל האובייקטים של MIB שגם ממוקמים במודולים של MIB שיובאו.
  • יבוא נתונים של MIB
    כיצד לייבא נתונים של SNMP MIB לתוך MIB Manager.
  • יצוא נתונים של MIB
    כיצד לייצא נתונים של SNMP MIB data מ-MIB Manager למגוון תבניות.
  • עריכת לכידות SNMP
    כיצד לערוך לכידות של SNMP.
  • הפקת לכידות SNMP‏
    כיצד ליצור לכידות של SNMP. ניתן להשתמש בתיפקודיות זו גם לבדיקת קובצי כללים שהופקו באמצעות MIB Manager.
