IBM Tivoli Netcool/OMNIbus 버전 8.1

MIB Manager 시작

명령행에서 MIB Manager GUI를 시작할 수 있습니다. MIB Manager는 프로세스 제어 하에 실행되거나 Windows 서비스로 실행되도록 설계되지 않았습니다.

이 태스크 정보

AIX 운영 체제와 Solaris 운영 체제에서 MIB Manager를 실행하기 위한 요구사항에 대한 정보는 지원되는 운영 체제 문서를 참조하십시오.

MIB Manager를 시작하기 전에 $DISPLAY 환경 변수가 작동하는 X11 서버에 설정되었는지 확인하십시오.

프로시저

  1. 다음 명령을 사용하여 MIB Manager GUI를 시작하십시오.

    NCHOME/omnibus/bin/nco_mibmanager

  2. 다음 명령을 사용하여 MIB Manager 명령행 유틸리티를 시작하십시오.

    NCHOME/omnibus/bin/nco_mibmanager_batch [options]

  • MIB Manager 명령행 옵션
    MIB Manager에는 MIB 데이터를 가져오고 규칙 파일을 내보내는 데 사용할 수 있는 명령행 유틸리티가 있습니다. 명령을 수동으로 실행하거나 프로그래밍 방식으로 호출할 수 있습니다.
