IBM Tivoli Netcool/OMNIbus V8.1

Netcool MIB Manager

Netcool MIB Manager 是一个基于 IBM® Eclipse 的 GUI 应用程序，可用于解析简单网络管理协议 (SNMP) 管理信息库 (MIB) 文件，根据这些文件可生成 Netcool 规则文件。它用于取代 mib2rules 实用程序。

Tivoli Netcool/OMNIbus 还包含 Netcool MIB Manager 命令行实用程序 (nco_mibmanager_batch)，可供您用于手动或者通过编程方式发出命令。

Netcool MIB Manager 具有以下功能：
  • 它可以导入 SNMP MIB 文件并解析 MIB 依赖关系以构建对象标识 (OID) 树。
  • 可以将一部分或所有已导入的 SNMP 对象导出为 Netcool 规则文件以及其他文件格式。
  • 它包含基本 MIB 以及其他 MIB 文件最常用的 RFC MIB。
  • 它包含设备定义以支持对导入的 MIB 模块进行以设备为中心的分组。
  • 它可以按名称或 OID 来过滤和搜索 MIB 模块和 OID 树。
  • 它可以生成 SNMP 陷阱以发送给 SNMP 探测器 (nco_p_mttrapd) 或其他 SNMP 代理。
  • 它具有命令行界面，可供您用于导入 MIB 数据和导出规则文件。您可以手动发出命令或者通过编程方式调用这些命令。
  • 启动 MIB 管理器
    从命令行启动 MIB 管理器 GUI。此管理器未设计成通过进程控制运行或作为 Windows 服务运行。
  • 使用Netcool MIB Manager
    可以使用 MIB 管理器 来解析 MIB 模块并创建 Netcool 规则文件。
  • 配置全局首选项
    可以从“首选项”窗口配置多个 MIB 管理器 全局首选项。
  • 关于 SNMP
    本节提供了关于简单网络管理协议 (SNMP) 体系结构、SNMP 管理信息库 (MIB) 以及其他一些 MIB 概要的附加信息。
