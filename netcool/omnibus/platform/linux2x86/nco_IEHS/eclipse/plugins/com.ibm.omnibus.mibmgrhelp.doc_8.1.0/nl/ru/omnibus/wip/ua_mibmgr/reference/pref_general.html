Задание общих предпочтений

Используйте общие предпочтения для задания опций импорта MIB.

Для задания общих предпочтений щелкните по значку Предпочтения или выберите пункт главного меню Файл > Предпочтения, а затем на левой панели окна Предпочтения щелкните по заголовку Общие.

  • Если включен переключатель Просмотреть подкаталоги для импорта, во время процесса импорта просматриваются все подкаталоги для поиска всех файлов MIB.
  • В поле Максимальное число потоков разветвителя файлов укажите количество потоков вычислений для обнаружения MIB в процессе импорта. Это может повысить производительность при импорте большого числа MIB. Значение по умолчанию - 5.
  • В поле Максимальное число потоков анализатора MIB укажите количество потоков вычислений для синтаксического анализа файлов MIB в процессе импорта. Это может повысить производительность при импорте большого числа MIB. Значение по умолчанию - 10.
  • Для изменения цвета выделения любого произведенного в окне MIB Manager выбора щелкните по значку , выберите цвет из палитры цветов, а затем нажмите кнопку OK.
