Netcool MIB Manager

Netcool MIB Manager è un'applicazione basata su IBM® Eclipse GUI che è possibile utilizzare per analizzare i file MIB (management information base) di SNMP (Simple Network Management Protocol), da cui è possibile generare i file delle regole Netcool.Non è indicato come sostituzione del programma di utilità mib2rules.

Tivoli Netcool/OMNIbus inoltre include un programma di utilità di riga comandi Netcool MIB Manager (nco_mibmanager_batch) che è possibile utilizzare per eseguire comandi manualmente o programmaticamente.

Netcool MIB Manager fornisce le seguenti funzioni:
  • Importa i file MIB SNMP e risolve le dipendenze MIB per generare una struttura ad albero OID (Object Identifier).
  • Può esportare alcuni i tutti gli oggetti SNMP importati nei file delle regole Netcool e in altri formati file.
  • Include i MIB di base e i MIB RFC più comunemente richiesti da altri file MIB.
  • Include le definizioni di dispositivi per abilitare il raggruppamento centratosui dispositivi dei moduli MIB importati.
  • Consente di filtrare e ricercare moduli MIB e la struttura ad albero OID per nome oggetto o OID.
  • Può generare trap SNMP da inviare al probe SNMP (nco_p_mttrapd) o ad altri agent SNMP.
  • Dispone di un'interfaccia di riga comandi che è possibile utilizzare per importare i dati MIB ed esportare i file delle regole. È possibile immettere i comandi manualmente o richiamarli in modo programmatico.
  • Avvio di MIB Manager
    Avviare la GUI di MIB Manager dalla riga comandi. Non è progettato per essere eseguito sotto il controllo processi o come servizio Windows.
  • Utilizzo di Netcool MIB Manager
    È possibile utilizzare MIB Manager per analizzare i moduli MIB e creare file delle regole Netcool.
  • Configurazione di preferenze globali
    È possibile configurare diverse preferenze globali di MIB Manager dalla finestra Preferenze.
  • Informazioni su SNMP
    Questa sezione fornisce informazioni aggiuntive sull'architettura SNMP (Simple Network Management Protocol), sul MIB (management information base) SNMP e su altri concetti relativi al MIB.
