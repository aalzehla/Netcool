IBM Tivoli Netcool/OMNIbus 8.1 版

啟動 MIB Manager

從指令行啟動 MIB Manager GUI。它未設計為受程序控制執行或作為 Windows 服務執行。

關於這項作業

如需在 AIX 和 Solaris 作業系統上執行 MIB Manager 的需求相關資訊，請參閱「支援的作業系統」說明文件。

請確保 $DISPLAY 環境變數設定為能發揮作用的 X11 伺服器，然後再啟動 MIB Manager。

程序

  1. 使用下列指令來啟動 MIB Manager GUI：

    NCHOME/omnibus/bin/nco_mibmanager

  2. 使用下列指令來啟動 MIB Manager 指令行公用程式：

    NCHOME/omnibus/bin/nco_mibmanager_batch [options]

  • MIB Manager 指令行選項
    MIB Manager 有一個指令行公用程式，您可以使用它來匯入 MIB 資料及匯出規則檔案。您可以手動發出指令，或以程式化方式呼叫指令。
