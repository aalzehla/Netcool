Configuración de las preferencias globales

Puede configurar varias preferencias globales de MIB Manager desde la ventana Preferencias.

Para configurar las preferencias globales de MIB Manager, pulse Preferencias o, desde el menú, pulse Archivo > Preferencias.

En la ventana Preferencias, puede configurar las preferencias de directorio, de exportación, generales, de registro cronológico y de búsqueda.

  • Establecimiento de las preferencias de directorio
    Utilice las preferencias de directorio para especificar los directorios que se utilizarán para almacenar las MIB importadas, los datos de MIB y los archivos de reglas exportados.
  • Establecimiento de las preferencias de exportación
    En esta sección se describe cómo establecer las preferencias de exportación de archivos de reglas.
  • Establecimiento de las preferencias generales
    Utilice las preferencias generales para especificar las opciones de importación de MIB.
  • Establecimiento de las preferencias de registro cronológico
    Utilice las preferencias de registro cronológico para especificar el directorio de archivos de registro de MIB Manager y establecer el nivel de mensaje predeterminado.
  • Establecimiento de las preferencias de búsqueda
    En esta sección se describe cómo definir las preferencias de MIB Manager que se utilizan para buscar los archivos MIB.
