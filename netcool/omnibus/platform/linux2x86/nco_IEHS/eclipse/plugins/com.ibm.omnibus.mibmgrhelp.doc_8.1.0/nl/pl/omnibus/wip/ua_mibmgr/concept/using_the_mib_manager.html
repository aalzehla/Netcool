IBM Tivoli Netcool/OMNIbus wersja 8.1

Używanie programu Netcool MIB Manager

Za pomocą programu MIB Manager można analizować moduły MIB i tworzyć pliki reguł programu Netcool.

Program MIB Manager zawiera zestaw podstawowych baz MIB i baz MIB RFC, które znajdują się w katalogu NCHOME/omnibus/var/mibmanager/mibs. Bazy te umożliwiają zaimportowanie dodatkowych baz MIB. Program MIB Manager automatycznie używa tych pakietowych baz MIB, aby rozwiązać zależności podczas importowania nowych baz MIB. Pakietowe moduły MIB są widoczne w widoku Moduły MIB podczas pierwszego uruchamiania programu MIB Manager.

Program MIB Manager zawiera również pliki danych XML, które reprezentują import pakietów podstawowych baz MIB. Program MIB Manager używa tych plików XML do zapisywania danych MIB, które importuje z surowych plików tekstowych MIB.

Należy dodać swoje bazy MIB do katalogu dostępnego z serwera, z którego uruchamiany jest program MIB Manager. Wszystkie standardowe bazy MIB muszą być zainstalowane w oddzielnym katalogu, który może być umieszczony w ścieżce wyszukiwania, aby ułatwić rozwiązywanie zależności. Domyślnie wszystkie podstawowe bazy MIB SNMP są zainstalowane w katalogu NCHOME/omnibus/var/mibmanager/mibs/base, a wszystkie bazy MIB RFC są zainstalowane w katalogu NCHOME/omnibus/var/mibmanager/mibs/rfc.

Okno programu Netcool MIB Manager

Po uruchomieniu programu MIB Manager wyświetlone są następujące elementy w początkowym oknie programu MIB Manager. Poniższy rysunek ilustruje te elementy.

Rysunek 1. Początkowe okno programu MIB Manager
 1  Pasek narzędzi
Umożliwia dostęp do najczęściej używanych funkcji programu MIB Manager.
 2  Perspektywy
Dane bazy MIB można wyświetlać za pomocą jednej z dwóch perspektyw: MIB Manager lub mib2rules. Perspektywa MIB Manager jest domyślną perspektywą. Perspektywa mib2rules skupia się na identyfikatorach obiektów (OID) i jest udostępniania dla użytkowników, którzy znają program narzędziowy mib2rules.
 3  Widok Urządzenie
Ten widok umożliwia tworzenie, aktualizowanie i usuwanie urządzeń MIB Manager.
 4  Widoki Szczegóły i Konsola
Widok Szczegóły wyświetla szczegóły dotyczące wybranych elementów bazy MIB. Widok Konsola wyświetla wszystkie komunikaty dziennika generowane przez program MIB Manager podczas działania bieżącej instancji.
 5  Moduły MIB
Ten widok służy do wyszukiwania i usuwania modułów MIB oraz do wyszukiwania określonych wartości MIB.
 6  Widok Drzewo OID
Ten widok reprezentuje wszystkie obiekty MIB, które znajdują się w zaimportowanych modułach MIB.
  • Widok Moduły MIB
    Widok Moduły MIB stanowi reprezentację wszystkich zaimportowanych modułów MIB i umożliwia wyszukiwanie oraz usuwanie modułów MIB, a także wyszukiwanie określonych wartości MIB.
  • Widok Drzewo OID
    Widok Drzewo identyfikatora obiektu (OID) reprezentuje wszystkie obiekty MIB, które znajdują się w zaimportowanych modułach MIB.
  • Importowanie danych MIB
    Sposób importowania danych MIB SNMP do programu MIB Manager.
  • Eksportowanie danych MIB
    Informacje na temat sposobu eksportowania danych MIB SNMP z programu MIB Manager do różnych formatów.
  • Edytowanie pułapek SNMP
    Sposób edytowania pułapek protokołu SNMP.
  • Generowanie pułapek protokołu SNMP
    Sposób generowania pułapek protokołu SNMP. Funkcja ta może również być używana do testowania plików reguł, które zostały wygenerowane przez program MIB Manager.
  • Tworzenie i usuwanie urządzeń
    Urządzenie programu MIB Manager jest używane do grupowania powiązanych modułów MIB oraz definiowania wszystkich obiektów wymaganych dla eksportowania. W interfejsie GUI programu MIB Manager można tworzyć, aktualizować i usuwać urządzenia.
