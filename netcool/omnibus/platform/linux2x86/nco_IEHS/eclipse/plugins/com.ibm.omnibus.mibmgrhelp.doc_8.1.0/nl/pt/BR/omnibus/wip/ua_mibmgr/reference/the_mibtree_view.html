A visualização do MIB Modules

A visualização do MIB Modules fornece uma representação de todos os módulos MIB importados e permite procurar módulos MIB, excluir módulos MIB e procurar valores específicos MIB.

É possível transitar pela visualização dos MIB Modules expandindo e reduzindo nós na árvore MIB. As ramificações de nível superior da árvore MIB contém os nomes dos módulos MIB e, contidos estão em cada ramificação do módulo MIB os outros elementos que englobam o MIB.

É possível filtrar os dados na visualização dos MIB Modules selecionando uma das opções a seguir a partir da lista suspensa Visualizar:
  • Tudo
  • Módulos
  • Enumerações
  • Convenções Textuais
  • Sintaxes
  • Importações
  • Exportações
  • Colunas
  • Descrições

Para fazer a procura na visualização dos MIB Modules por um valor específico, digite o texto no campo Procurar e pressione Enter ou clique no (Procurar).

O MIB Manager inicia a procura a partir do item selecionado ou, se não houver nenhum item selecionado atualmente, a partir do primeiro item da lista. Para localizar o próximo item correspondente da lista, continue a clicar no (Procurar).

Ao localizar um item, ele será automaticamente destacado e exibido na visualização MIB Modules.

No campo Procurar, especifique uma expressão regular válida ou insira um texto simples para representar um nome do objeto parcial ou completo.

Para excluir um ou mais módulos MIB, clique com o botão direito do mouse no(s) módulo(s) MIB e em seguida a partir do menu pop-up, selecione Excluir. O módulo MIB selecionado será excluído do MIB Manager. Entretanto, os arquivos de origem MIB originais, fornecidos para a importação MIB não serão excluídos.

Nota: Você deverá apenas excluir um módulo MIB se ele contiver erros e se houver uma nova versão de substituição do módulo MIB. Excluir módulos MIB usando este método poderá resultar na perda de dependências MIB. Arquivos de regras incompletos também podem ser gerados durante o processo de exportação juntamente com mensagens de erro ou avisos no arquivo de log.
