Edición de interrupciones SNMP

Como editar interrupciones SNMP.

Acerca de esta tarea

En la vista Detalles, puede editar los siguientes campos de una interrupción SNMP:
  • @Severity
  • @Type
  • @ExpireTime
  • CODEBLOCK
Nota: SNMP no proporciona mecanismos estándar para especificar información set (@Type ) o clear (@Severity). Esta información se debe especificar manualmente.

En la vista Detalles, pulse dos veces el valor de un campo para hacerlo editable. Tras realizar un cambio en un campo, pulse para guardar el cambio o pulse para cancelar el cambio.

Procedimiento

Para editar una interrupción SNMP:

  1. En la vista Módulos MIB o en la vista Árbol de OID, seleccione una interrupción.

    Se visualizará información detallada de la interrupción en la vista Detalles.

  2. Para editar el campo @Severity, pulse dos veces el valor del campo y seleccione un nuevo valor de la lista desplegable.
  3. Para editar el campo @Type, pulse dos veces el valor del campo y seleccione un nuevo valor de la lista desplegable.
  4. Para editar el campo @ExpireTime, pulse dos veces el valor del campo e indique un nuevo período de tiempo (en segundos, minutos, horas o días).
  5. Para editar el campo CODEBLOCK, pulse dos veces el valor del campo (aunque el campo esté vacío) e indique un nuevo bloque de código.
