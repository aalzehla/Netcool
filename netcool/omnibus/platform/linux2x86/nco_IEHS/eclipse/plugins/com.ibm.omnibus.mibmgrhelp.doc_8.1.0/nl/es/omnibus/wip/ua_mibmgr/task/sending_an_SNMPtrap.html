Generación de interrupciones SNMP

Cómo generar interrupciones SNMP. Esta funcionalidad también se puede utilizar para probar los archivos de reglas generados por MIB Manager.

Procedimiento

Para generar una interrupción SNMP:

  1. Pulse Interrupción.
  2. En el campo Dirección de destino, especifique el nombre de host o la dirección IP de un sistema que ejecute un servicio de Trapd SNMP.
  3. En el campo Puerto de destino, especifique el puerto UDP en el que el servicio Trapd SNMP está a la escucha.
  4. En el campo Comunidad, especifique una serie válida para el gestor de Trapd.
  5. Según el tipo de interrupción que se genera, es posible que se visualice la sección Enlaces variables en la ventana Generar una interrupción SNMP. Los campos que están disponibles de la sección Enlaces variables también dependen del tipo de interrupción que se genera.

    Deslice el ratón sobre los nombres del campo Enlaces variables para obtener detalles sobre la sintaxis del valor del campo. Puede especificar varios valores, separados por comas. Estos valores se elegirán aleatoriamente para compilar la interrupción.

  6. En el campo Repetir medida, especifique el número de interrupciones (de 1 a 1000) que se pueden generar.

    Si especifica un número grande, las interrupciones se generarán en una sucesión rápida y pueden provocar una tormenta de alarmas en el gestor de Trapd. Para los valores mayores de uno (> 1), y para los valores especificados como listas separadas por punto y coma, MIB Manager realiza una selección aleatoria de las listas especificadas y genera interrupciones con los valores arbitrarios presentados como enlaces de variable (varbinds).

  7. Pulse Ejecutar para generar la interrupción.

Resultados

El campo Resultado de la ventana Generar una interrupción SNMP indicará si la interrupción se ha enviado correctamente.
