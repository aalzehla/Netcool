MIB マネージャーの始動

MIB マネージャー GUI は、コマンド行から始動します。プロセス制御の下で、または Windows サービスとして実行されるようには設計されていません。

このタスクについて

AIX および Solaris オペレーティング・システム上で MIB マネージャーを実行するための要件について詳しくは、サポートされるオペレーティング・システムの資料を参照してください。

MIB マネージャーを始動する前に、$DISPLAY 環境変数が、機能している X11 サーバーに設定されていることを確認する必要があります。

手順

  1. 次のコマンドを使用して MIB マネージャー GUI を始動します。

    NCHOME/omnibus/bin/nco_mibmanager

  2. 次のコマンドを使用して MIB マネージャー・コマンド行ユーティリティーを開始します。

    NCHOME/omnibus/bin/nco_mibmanager_batch [options]

  • MIB マネージャー コマンド行オプション
    MIB マネージャー には、MIB データのインポートおよびルール・ファイルのエクスポートに使用できるコマンド行ユーティリティーが備わっています。コマンドを手動で発行したり、あるいはプログラムで呼び出したりすることができます。
