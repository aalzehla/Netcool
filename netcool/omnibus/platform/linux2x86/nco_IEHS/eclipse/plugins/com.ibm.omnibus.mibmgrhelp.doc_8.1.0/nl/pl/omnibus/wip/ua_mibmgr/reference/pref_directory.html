Ustawianie preferencji katalogu

Należy użyć preferencji katalogu do określenia katalogów wykorzystywanych do przechowywania zaimportowanych baz MIB, danych MIB oraz wyeksportowanych plików reguł.

Aby ustawić preferencje katalogu, należy kliknąć opcję Preferencje lub wybrać z menu opcję Plik > Preferencje, a następnie z panelu po lewej stronie okna Preferencje kliknąć opcję Katalog.

  • W polu Katalog danych należy określić miejsce, gdzie będą zapisywane zaimportowane bazy MIB jako pliki XML. Położeniem domyślnym jest NCHOME/omnibus/var/mibmanager/data.
  • W polu Katalog eksportu należy określić miejsce, gdzie są generowane wyeksportowane pliki. Utworzony zostaje podkatalog w oparciu o typ eksportowania. Położeniem domyślnym jest NCHOME/omnibus/var/mibmanager/export.
  • W polu Katalog importu należy określić miejsce, gdzie znajdują się zaimportowane pliki MIB. Położeniem domyślnym jest NCHOME/omnibus/var/mibmanager/mibs.
