Netcool MIB Manager

Netcool MIB Manager es una aplicación de interfaz gráfica de usuario basada en IBM® Eclipse que puede utilizar para analizar archivos de base de información de gestión (MIB) de protocolo simple de gestión de red (SNMP), a partir de los cuales se pueden generar archivos de reglas de Netcool. Está pensado como sustituto del programa de utilidad mib2rules.

Tivoli Netcool/OMNIbus incluye también un programa de utilidad de línea de mandatos de Netcool MIB Manager (nco_mibmanager_batch) que puede utilizar para emitir mandatos manualmente o de forma programática.

Netcool MIB Manager tiene las características siguientes:
  • Importa archivos MIB de SNMP y resuelve dependencias para construir un árbol identificador de objeto (OID).
  • Puede exportar algunos o todos los objetos SNMP importados a los archivos de reglas de Netcool y a otros formatos de archivo.
  • Incluye las MIB base y las MIB de RFC que se necesitan más a menudo en otros archivos MIB.
  • Incluye definiciones de dispositivo para habilitar la agrupación centrada en el dispositivo de módulos MIB importados.
  • Puede filtrar y buscar módulos MIB y el árbol OID por nombre de objeto u OID.
  • Puede generar condiciones de excepción SNMP para enviar al analizador SNMP (nco_p_mttrapd) u otros agentes de SNMP.
  • Tiene una interfaz de línea de mandatos que puede utilizar para importar datos de MIB y exportar archivos de reglas. Puede emitir mandatos manualmente o llamarlos de forma programática.
  • Inicio de MIB Manager
    Inicie la interfaz gráfica de usuario de MIB Manager desde la línea de mandatos. No está diseñado para ejecutarse bajo el control de procesos o como un servicio de Windows.
  • Utilización de Netcool MIB Manager
    Puede utilizar MIB Manager para analizar módulos MIB y crear archivos de reglas de Netcool.
  • Configuración de las preferencias globales
    Puede configurar varias preferencias globales de MIB Manager desde la ventana Preferencias.
  • Acerca de SNMP
    En esta sección se proporciona información adicional sobre la arquitectura de protocolo simple de gestión de red (SNMP), la base de información de gestión (MIB) de SNMP y algunos conceptos de MIB adicionales.
