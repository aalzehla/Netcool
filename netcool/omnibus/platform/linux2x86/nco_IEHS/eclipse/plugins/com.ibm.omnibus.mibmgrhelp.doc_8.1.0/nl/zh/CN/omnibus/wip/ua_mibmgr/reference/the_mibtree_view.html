“MIB 模块”视图

“MIB 模块”视图显示了所有已导入的 MIB 模块，并允许您搜索 MIB 模块、删除 MIB 模块，以及搜索特定的 MIB 值。

展开和折叠 MIB 树中的节点可在整个“MIB 模块”视图中移动。MIB 树的顶级分支包含 MIB 模块的名称，而包含在每个 MIB 模块分支内的是组成 MIB 的其他元素。

从视图下拉列表中选择以下一个选项可过滤“MIB 模块”视图中的数据：
  • 全部
  • 模块
  • 枚举
  • 文本约定
  • 语法
  • 导入
  • 导出
  • 列
  • 描述

要在“MIB 模块”视图中搜索特定值，请在搜索字段中输入您要搜索的文本，然后按 Enter 键或单击 （搜索）。

MIB 管理器 将从选定的项开始搜索，如果当前未选择任何项，那么将从列表中的第一项开始搜索。要在列表中查找下一个匹配项，请继续单击 （搜索）。

找到匹配项后，会自动突出显示该项并将其显示在“MIB 模块”视图中。

在搜索字段中，指定一个有效的正则表达式或输入纯文本来表示完整或部分的对象名称。

要删除一个或多个 MIB 模块，请右键单击选定的 MIB 模块，然后从弹出菜单中选择删除。将从 MIB 管理器 中删除选定的 MIB 模块。但是，不会删除为 MIB 导入提供的原始 MIB 源文件。

注： 只应删除含有错误或者存在新的替代版本的 MIB 模块。使用此方法删除 MIB 模块可能导致 MIB 依赖关系缺失。还可能在导出过程中生成不完整的规则文件，并且在日志文件中出现警告或错误消息。
