일반 환경 설정 설정

일반 환경 설정을 사용하여 MIB 가져오기 옵션을 지정합니다.

디렉토리 환경 설정을 설정하려면 환경 설정을 클릭하거나, 기본 메뉴에서 파일 > 환경 설정을 클릭한 후 환경 설정 창의 왼쪽 분할창에서 일반을 클릭하십시오.

  • 가져오기 경우 서브디렉토리 조사 선택란을 선택하면, 가져오기 프로세스 도중 모든 서브디렉토리를 검색하여 모든 MIB 파일을 찾습니다.
  • 파일 분할자 최대 스레드 수 필드에서는 가져오기 프로세스 도중 MIB를 발견하는 데 사용된 계산 상의 스레드 수를 지정합니다. 이 필드를 선택하면 많은 수의 MIB를 가져올 때 성능이 향상될 수 있습니다. 기본값은 5입니다.
  • MIB 구문 분석기 최대 스레드 수 필드에서는 가져오기 프로세스 도중 MIB 파일을 구문 분석하는 데 사용되는 계산 상의 스레드 수를 지정하십시오. 이 필드를 선택하면 많은 수의 MIB를 가져올 때 성능이 향상될 수 있습니다. 기본값은 10입니다.
  • MIB Manager 창에서 선택한 항목의 형광펜 색상을 변경하려면 을 클릭하고 색상표에서 색상을 선택한 다음 확인을 클릭하십시오.
