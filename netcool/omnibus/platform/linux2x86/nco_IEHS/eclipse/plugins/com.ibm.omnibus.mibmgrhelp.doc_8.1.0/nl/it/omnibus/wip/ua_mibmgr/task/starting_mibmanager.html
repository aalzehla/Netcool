IBM Tivoli Netcool/OMNIbus Versione 8.1

Avvio di MIB Manager

Avviare la GUI di MIB Manager dalla riga comandi. Non è progettato per essere eseguito sotto il controllo processi o come servizio Windows.

Informazioni su questa attività

Consultare la documentazione relativa ai sistemi operativi supportati per informazioni sui requisiti necessari per eseguire MIB Manager sui sistemi operativi AIX e Solaris.

Assicurarsi che la variabile di ambiente $DISPLAY sia impostata su un server X11 funzionante prima di avviare MIB Manager.

Procedura

  1. Utilizzare il seguente comando per avviare la GUI di MIB Manager:

    NCHOME/omnibus/bin/nco_mibmanager

  2. Utilizzare il seguente comando per avviare il programma di utilità di riga comandi MIB Manager:

    NCHOME/omnibus/bin/nco_mibmanager_batch [options]

  • Opzioni di riga comandi MIB Manager
    MIB Manager dispone di un programma di utilità di riga comandi che è possibile utilizzare per importare i dati MIB ed esportare i file delle regole. È possibile immettere i comandi manualmente o richiamarli in modo programmatico.
