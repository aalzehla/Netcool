검색 환경 설정 설정

이 절에서는 MIB 파일을 검색하는 데 사용되는 MIB Manager 환경 설정을 정의하는 방법을 설명합니다.

검색 환경 설정을 설정하려면, 환경 설정을 클릭하거나 기본 메뉴에서 파일 > 환경 설정을 클릭한 후 환경 설정 창의 왼쪽 분할창에서 검색을 클릭하십시오.

MIB 검색 경로 필드에서는 MIB 파일이 있는 위치로 검색 경로를 설정하십시오. 디렉토리는 세미콜론(;)으로 구분되어야 합니다. 모든 서브디렉토리를 검색하여 자동으로 종속 MIB 파일을 찾습니다.

기본 경로에는 MIB Manager와 함께 패키지로 제공되는 RFC MIB 파일과 기본 MIB 파일의 위치가 포함됩니다. 기본 파일 경로 끝에서 새 디렉토리를 지정할 수 있습니다. 그러면 기본 디렉토리에서 가장 일반적으로 사용되는 MIB가 있는 종속 항목을 검색한 후, rfc 디렉토리에서 이보다 덜 사용되는 MIB를 검색합니다.
