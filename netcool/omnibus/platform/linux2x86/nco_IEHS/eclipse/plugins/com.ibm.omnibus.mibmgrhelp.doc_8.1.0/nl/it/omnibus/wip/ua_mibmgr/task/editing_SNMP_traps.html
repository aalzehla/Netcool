Modifica di trap SNMP

Come modificare i trap SNMP.

Informazioni su questa attività

Nella vista Dettagli, è possibile modificare i seguenti campi di un trap SNMP:
  • @Severity
  • @Type
  • @ExpireTime
  • CODEBLOCK
Nota: SNMP non fornisce meccanismi standard per specificare informazioni set (@Type ) o clear (@Severity). È necessario specificare queste informazioni manualmente.

Nella vista Dettagli, fare doppio clic su un valore di campo per renderlo modificabile. Dopo aver apportato una modifica ad un campo, fare clic su per salvare la modifica oppure fare clic su per annullare la modifica.

Procedura

Per modificare un trap SNMP:

  1. Nella vista Moduli MIB o nella vista Struttura ad albero OID, selezionare un trap.

    Informazioni dettagliate sul trap vengono visualizzate nella vista Dettagli.

  2. Per modificare il campo @Severity, fare doppio clic sul valore del campo e scegliere un nuovo valore dall'elenco a discesa.
  3. Per modificare il campo @Type, fare doppio clic sul valore del campo e scegliere un nuovo valore dall'elenco a discesa.
  4. Per modificare il campo @ExpireTime, fare doppio clic sul valore del campo e immettere un nuovo periodo di tempo (in secondi, minuti, ore o giorni).
  5. Per modificare il campo CODEBLOCK, fare doppio clic sul valore del campo (anche se il campo è vuoto) e immettere un nuovo blocco di codice.
