Netcool MIB Manager

O Netcool MIB Manager é um aplicativo de GUI baseado no IBM® Eclipse que pode ser usado para analisar os arquivos Management Information Base (MIB) de Protocolo Simples de Gerenciamento de Rede (SNMP), a partir dos quais é possível gerar arquivos de regras do Netcool. É desejado como uma substituição para o utilitário. mib2rules

O Tivoli Netcool/OMNIbus também inclui um utilitário de linha de comandos do Netcool MIB Manager (nco_mibmanager_batch) que pode ser usado para emitir comandos manualmente ou programaticamente.

O Netcool MIB Manager possui os recursos a seguir:
  • Ele importa arquivos MIB SNMP e resolve dependências de MIB para construir uma árvore Identificadora de Objetos (OID).
  • Ele pode exportar alguns ou todos os objetos SNMP importados para arquivos de regras Netcool e outros formatos de arquivos.
  • Ele inclui as MIBs de base e as MIBs RFC mais comumente requeridas por outros arquivos MIB.
  • Ele inclui definições de dispositivo para possibilitar o agrupamento centrado no dispositivo de módulos MIB importados.
  • Ele pode filtrar e procurar Módulos MIB e a árvore OID por nome de objeto ou OID.
  • Pode gerar traps SNMP para enviar o SNMP Probe (nco_p_mttrapd) ou outros agentes SNMP.
  • Possui uma interface da linha de comandos que pode ser usada para importar dados MIB e arquivos de regras de exportação. É possível emitir comandos manualmente ou chamá-los programaticamente.
  • Iniciando o MIB Manager
    Inicie a GUI do MIB Manager a partir da linha de comandos. Não é projetado para ser executado sob um controle de processo ou como um serviço do Windows.
  • Usando o Netcool MIB Manager
    É possível usar o MIB Manager para analisar módulos MIB e criar arquivos de regras do Netcool.
  • Configurando preferências globais
    É possível configurar diversas preferências globais do MIB Manager a partir da janela Preferências.
