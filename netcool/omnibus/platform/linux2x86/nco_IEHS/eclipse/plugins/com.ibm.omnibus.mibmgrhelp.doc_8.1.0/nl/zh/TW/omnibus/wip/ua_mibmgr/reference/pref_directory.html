設定目錄喜好設定

使用目錄喜好設定來指定目錄，用來儲存匯入的 MIB、MIB 資料以及匯出的規則檔案。

若要設定目錄喜好設定，請按一下 喜好設定，或從功能表中按一下檔案 > 喜好設定，然後從「喜好設定」視窗的左窗格中，按一下目錄。

  • 在資料目錄欄位中，指定將儲存匯入的 MIB 作為 XML 檔的位置。預設位置為 is NCHOME/omnibus/var/mibmanager/data。
  • 在匯出目錄欄位中，指定產生匯出檔的位置。會建立基於匯出類型的子目錄。預設位置為 NCHOME/omnibus/var/mibmanager/export。
  • 在匯入目錄欄位中，指定匯入的 MIB 檔所在的位置。預設位置為 NCHOME/omnibus/var/mibmanager/mibs。
