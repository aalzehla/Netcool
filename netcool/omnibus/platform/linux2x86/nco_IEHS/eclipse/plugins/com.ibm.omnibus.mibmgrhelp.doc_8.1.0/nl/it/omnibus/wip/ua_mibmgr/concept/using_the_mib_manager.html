Utilizzo di Netcool MIB Manager

È possibile utilizzare MIB Manager per analizzare i moduli MIB e creare file delle regole Netcool.

MIB Manager include una serie di MIB di base e MIB RFC nella directory NCHOME/omnibus/var/mibmanager/mibs. Questi MIB vengono forniti per consentire di importare ulteriori MIB. MIB Manager utilizza automaticamente questi MIB forniti per risolvere le dipendenze durante l'importazione di nuovi MIB. I moduli MIB forniti sono visibili nella vista Moduli MIB quando si avvia MIB Manager per la prima volta.

MIB Manager include anche file di dati XML che rappresentano un'importazione dei MIB di base forniti. MIB Manager utilizza tali file XML per memorizzare i dati MIB importati dai file MIB di testo non elaborati.

Aggiungere i MIB ad una directory a cui è possibile accedere dal server da cui viene avviato MIB Manager. Tutti i MIB basati sugli standard devono essere installati in una directory separata che può essere inserita nel percorso di ricerca per una semplice risoluzione delle dipendenze. Per impostazione predefinita, i MIB SNMP di base vengono installati nella directory NCHOME/omnibus/var/mibmanager/mibs/base e tutti i MIB RFC vengono installati nella directory NCHOME/omnibus/var/mibmanager/mibs/rfc.

La finestra Netcool MIB Manager

Quando si avvia MIB Manager, i seguenti elementi vengono visualizzati nella finestra iniziale di MIB Manager: diversi elementi vengono visualizzati nella finestra iniziale di MIB Manager. La seguente figura mostra tali elementi.

Figura 1. Finestra iniziale di MIB Manager
 1  Barra degli strumenti
Fornisce l'accesso alle funzioni di MIB Manager utilizzate più frequentemente.
 2  Prospettive
È possibile visualizzare i dati MIB in una delle due prospettive: MIB Manager o mib2rules. La prospettiva MIB Manager è la prospettiva predefinita. La prospettiva mib2rules è incentrata sugli Identificativi oggetto (OID) e viene fornita per gli utenti che hanno familiarità con il programma di utilità mib2rules.
 3  Vista Dispositivo
Utilizzare questa vista per creare, aggiornare ed eliminare dispositivi MIB Manager.
 4  Vista Dettagli e vista Console
La vista Dettagli visualizza i dettagli degli elementi MIB selezionati. La vista Console mostra tutti i messaggi di log generati da MIB Manager mentre l'istanza corrente è in esecuzione.
 5  Moduli MIB
Utilizzare questa vista per ricercare ed eliminare moduli MIB e per ricercare valori MIB specifici.
 6  Vista Struttura OID
Questa vista fornisce una rappresentazione di tutti gli oggetti MIB presenti nei moduli MIB importati.
  • La vista Moduli MIB
    La vista Moduli MIB fornisce una rappresentazione di tutti i moduli MIB importati e consente di ricercare moduli MIB, di eliminare moduli MIB e di ricercare valori MIB specifici.
  • La vista Struttura ad albero OID
    La vista Struttura al albero OID (Object Identifier) fornisce una rappresentazione di tutti gli oggetti MIB situati nei moduli MIB importati.
  • Importazione di dati MIB
    Come importare dati MIB SNMP in MIB Manager.
  • Esportazione di dati MIB
    Come esportare i dati MIB SNMP MIB da MIB Manager in una varietà di formati.
  • Modifica di trap SNMP
    Come modificare i trap SNMP.
  • Generazione di trap SNMP
    Come generare i trap SNMP. Questa funzionalità può essere utilizzata anche per testare i file delle regole generati da MIB Manager.
  • Creazione ed eliminazione di dispositivi
    I dispositivi MIB Manager vengono utilizzati per raggruppare i moduli MIB associati e per definire tutti gli oggetti richiesti per l'esportazione. È possibile creare, aggiornare ed eliminare i dispositivi nella GUI di MIB Manager.
