Utilización de Netcool MIB Manager

Puede utilizar MIB Manager para analizar módulos MIB y crear archivos de reglas de Netcool.

MIB Manager incluye un conjunto de MIBs y MIBs de RFC, en el directorio NCHOME/omnibus/var/mibmanager/mibs. Estas MIB se proporcionan para permitir la importación de MIB adicionales. MIB Manager utiliza automáticamente estas MIB empaquetados para resolver dependencias durante la importación de nuevas MIB. Los módulos MIB empaquetados están visibles en la vista Módulos MIB cuando se inicia MIB Manager por primera vez.

MIB Manager también incluye archivos de datos XML que representan una importación de las MIB base empaquetadas. MIB Manager utiliza estos archivos XML para almacenar los datos de MIB que importa de los archivos de MIB de texto sin formato.

Añada las MIB a un directorio al que se pueda acceder desde el servidor desde el que se inicie MIB Manager. Todas las MIB base estándar deben estar instaladas en un directorio aparte que se puede colocar en la vía de acceso de búsqueda para realizar de modo fácil la resolución de dependencias. De forma predeterminada las MIBs de SNMP básicas están instaladas en el directorio NCHOME/omnibus/var/mibmanager/mibs/base y todas las MIBs de RFC MIBs están instaladas en el directorio NCHOME/omnibus/var/mibmanager/mibs/rfc.

Ventana de Netcool MIB Manager

Cuando se inicia MIB Manager, los elementos siguientes se muestran en la ventana inicial de MIB Manager: se muestran varios elementos en la ventana inicial de MIB Manager. En la figura siguiente se muestran estos elementos.

Figura 1. Ventana inicial de MIB Manager
 1  Barra de herramientas
Proporciona acceso a las funciones de MIB Manager que más se utilizan.
 2  Perspectivas
Puede ver los datos de MIB en una de las dos perspectivas, MIB Manager o mib2rules. La perspectiva MIB Manager es la perspectiva predeterminada. La perspectiva mib2rules se centra en identificadores de objeto (OID) y se proporciona para los usuarios que están familiarizados con el programa de utilidad mib2rules.
 3  Vista Dispositivo
Utilice esta vista para crear, actualizar y suprimir dispositivos de MIB Manager.
 4  Vista Detalles y vista Consola
La vista Detalles muestra los detalles de los elementos de MIB seleccionados. La vista Consola muestra todos los mensajes de registro que genera MIB Manager mientras se está ejecutando la instancia actual.
 5  Módulos MIB
Utilice esta vista para buscar y suprimir módulos de MIB y para buscar valores de MIB específicos.
 6  Vista Árbol de OID
Esta vista ofrece una representación de todos los objetos de MIB que hay en los módulos de MIB importados.
  • La vista Módulos MIB
    La vista Módulos MIB proporciona una representación de todos los módulos MIB importados y permite buscar módulos MIB, suprimir módulos MIB y buscar valores MIB concretos.
  • La vista Árbol de OID
    La vista Árbol de identificador de objeto (OID) proporciona una representación de todos los objetos MIB que se encuentran en los módulos MIB importados.
  • Importación de datos MIB
    Cómo importar datos MIB SNMP en MIB Manager.
  • Exportación de datos MIB
    Cómo exportar datos MIB de SNMP desde MIB Manager en una variedad de formatos.
  • Edición de interrupciones SNMP
    Como editar interrupciones SNMP.
  • Generación de interrupciones SNMP
    Cómo generar interrupciones SNMP. Esta funcionalidad también se puede utilizar para probar los archivos de reglas generados por MIB Manager.
  • Creación y supresión de dispositivos
    Los dispositivos de MIB Manager se utilizan para agrupar módulos MIB asociados y definir todos los objetos requeridos para la exportación. Puede crear, actualizar y suprimir dispositivos en la interfaz gráfica de usuario de MIB Manager.
