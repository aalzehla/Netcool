「OID ツリー」ビュー

「オブジェクト ID (OID) ツリー」ビューは、インポートされた MIB モジュールに 存在するすべての MIB オブジェクトを表示します。

「OID ツリー」ビュー内のデータは、 以下のオプションのいずれかを「ビュー」ドロップダウン・リストから選択することで フィルタリングできます。
  • すべて
  • トラップ/通知
  • オブジェクト
  • モジュール

特定の値を「OID ツリー」ビューで 検索するには、テキストを「検索」フィールドに入力して、 「Enter」を押すか (検索) をクリックします。

MIB マネージャーは、 選択されたアイテムから検索を開始するか、アイテムが現在選択されていない場合は、 リスト内の最初のアイテムから検索を開始します。一致する次のアイテムを リストから検索するには、続けて (検索) をクリックします。

アイテムが 見つかると、アイテムは自動的に強調表示され、 「OID ツリー」ビューに表示されます。

「検索」フィールドに、 有効な正規表現を指定するか、オブジェクト名のすべてまたは一部を表すプレーン・テキスト、 または OID を入力します。それぞれのブランチ・ノードに 表示される番号は、個々のオブジェクトの OID の 最後の桁です。
