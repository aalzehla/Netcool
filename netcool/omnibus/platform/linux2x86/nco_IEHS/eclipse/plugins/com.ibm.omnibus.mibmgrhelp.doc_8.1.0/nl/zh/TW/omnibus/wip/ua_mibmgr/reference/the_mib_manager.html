Netcool MIB Manager

Netcool MIB Manager 是基於 IBM® Eclipse 的 GUI 應用程式，您可以使用它來剖析可從中產生 Netcool 規則檔案的「簡易網路管理通訊協定 (SNMP)」管理資訊庫 (MIB) 檔案。它預期作為 mib2rules 公用程式的取代項目。

Tivoli Netcool/OMNIbus 還包含一個 Netcool MIB Manager 指令行公用程式 (nco_mibmanager_batch)，您可以使用該公用程式來手動或以程式化方式發出指令。

Netcool MIB Manager 具有下列功能：
  • 它可匯入 SNMP MIB 檔案並解析 MIB 相依關係以建置物件 ID (OID) 樹狀結構。
  • 它可以將部分或全部匯入的 SNMP 物件匯出到 Netcool 規則檔案及其他檔案格式。
  • 它包括其他 MIB 檔案最常要求的基本 MIB 及 RFC MIB。
  • 它包括裝置定義以啟用已匯入 MIB 模組的裝置中樞分組。
  • 它可以依物件名稱或 OID 來過濾及搜尋 MIB 模組和 OID 樹狀結構。
  • 它可以產生 SNMP 設陷以傳送至 SNMP 探測器 (nco_p_mttrapd) 或其他 SNMP 代理程式。
  • 它具有一個可用來匯入 MIB 資料及匯出規則檔案的指令行介面。您可以手動發出指令，或以程式化方式呼叫指令。
  • 啟動 MIB Manager
    從指令行啟動 MIB Manager GUI。它未設計為受程序控制執行或作為 Windows 服務執行。
  • 使用 Netcool MIB Manager
    您可以使用 MIB Manager 來剖析 MIB 模組及建立 Netcool 規則檔案。
  • 配置廣域喜好設定
    您可以從「喜好設定」視窗配置數個 MIB Manager 廣域喜好設定。
  • 關於 SNMP
    本節提供有關「簡易網路管理通訊協定 (SNMP)」架構、SNMP 管理資訊庫 (MIB) 及一些其他 MIB 概念的其他相關資訊。
