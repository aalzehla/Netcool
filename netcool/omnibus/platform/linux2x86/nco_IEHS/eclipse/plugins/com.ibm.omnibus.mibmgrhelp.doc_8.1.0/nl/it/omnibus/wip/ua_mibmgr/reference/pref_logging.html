Impostazione di preferenze di registrazione

Utilizzare le preferenze di registrazione per specificare la directory dei file di log MIB Manager e per impostare il livello dei messaggi predefinito.

Per impostare le preferenze di directory, fare clic su Preferenze oppure, dal menu principale, fare clic su File > Preferenze, quindi dal pannello di sinistra della finestra Preferenze, fare clic su Registrazione.

  • Nel campo Directory di log, specificare il percorso in cui viene scritto il file di log dell'applicazione. Il valore predefinito è NCHOME/omnibus/log.
  • Nel campo Livello messaggi predefinito, specificare il livello messaggi predefinito dell'applicazione. Per modificare il livello di registrazione messaggi predefinito, è possibile utilizzare anche il menu Modifica. Il livello messaggi è valido da quando l'applicazione viene avviata per la prima volta fino a quando l'applicazione viene chiusa, a meno che non venga modificato mediante il menu Modifica.
    Nota: la modifica del livello messaggi nel campo Livello messaggi predefinito non comporta la modifica del livello messaggi corrente dell'applicazione. Tale modifica influisce solo sul livello messaggi impostato inizialmente quando l'applicazione è stata avviata per la prima volta (il livello messaggi predefinito). Per evitare la generazione di file di log di grandi dimensioni, impostare il livello messaggi predefinito su warn.
  • Impostazione del livello di registrazione
    È possibile modificare il livello di registrazione per fornire file di log o di traccia più dettagliati al fine di semplificare il debug di MIB Manager. Le informazioni supplementari fornite aumentando i livelli di registrazione possono consentire il debug di problemi specifici durante l'analisi dei MIB.
