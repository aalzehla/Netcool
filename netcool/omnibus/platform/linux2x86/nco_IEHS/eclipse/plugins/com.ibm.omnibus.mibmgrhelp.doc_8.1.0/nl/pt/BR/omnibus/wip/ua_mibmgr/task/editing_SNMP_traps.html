Editando traps SNMP

Como editar traps SNMP.

Sobre Esta Tarefa

Na visualização Detalhes, é possível editar os campos a seguir de um trap SNMP:
  • @Severity
  • @Type
  • @ExpireTime
  • CODEBLOCK
Nota: O SNMP não fornece mecanismos padrão para especificar as informações da configuração (@Type ) ou limpar (@Severity). Você deverá especificar estas informações manualmente.

Na visualização Detalhes, clique duas vezes em um valor do campo para torná-lo editável. Após fazer uma alteração em um campo, clique em para salvar a alteração ou clique em para cancelar a alteração.

Procedimento

Para editar um trap SNMP:

  1. Na visualização MIB Modules ou na visualização OID Tree, selecione um trap.

    Informações detalhadas sobre o trap estão exibidas na visualização Detalhes.

  2. Para editar o campo @Severity, clique duas vezes no valor do campo e escolha um novo valor a partir da lista suspensa.
  3. Para editar o campo @Type, clique duas vezes no valor do campo e escolha um novo valor a partir da lista suspensa.
  4. Para editar o campo @ExpireTime, clique duas vezes no valor do campo e insira um novo período de tempo (em segundos, minutos, horas ou dias).
  5. Para editar o campo CODEBLOCK, clique duas vezes no valor do campo (mesmo se o campo estiver vazio) e insira um novo bloco de códigos.
