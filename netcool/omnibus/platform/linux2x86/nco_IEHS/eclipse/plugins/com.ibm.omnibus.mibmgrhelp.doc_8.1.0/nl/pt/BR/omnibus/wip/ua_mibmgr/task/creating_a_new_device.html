Criando e excluindo dispositivos

Um dispositivo do MIB Manager é usado para agrupar Módulos MIB associados e definir todos os objetos necessários para exportação. É possível criar, atualizar e excluir dispositivos na GUI do MIB Manager.

Sobre Esta Tarefa

Os dispositivos do MIB Manager são exibidos na visualização Dispositivo. Selecione um dispositivo da árvore Dispositivo para exibir o fabricante, o número do modelo e a versão OS do dispositivo.

Procedimento

Criando um dispositivo

  1. Para criar um novo dispositivo:
    1. Na visualização MIB Modules, selecione ao menos um MIB Module e em seguida clique no (Criar Dispositivo).
    2. No campo Fabricante, selecione um fabricante do dispositivo a partir da lista suspensa.

      Os campos Modelo e Versão OS serão preenchidos com as opções disponíveis.

    3. No campo Modelo, selecione uma opção a partir da lista suspensa.

      Se não houver modelos disponíveis, insira um nome de modelo descritivo.

    4. No campo Versão OS, selecione uma opção a partir da lista suspensa.

      Se não houver versões OS disponíveis, insira um nome de versão descritivo OS.

    5. No campo Módulos Mib Mapeados, selecione os módulos MIB que você deseja associar com o dispositivo novo.
    6. Clique em OK para criar o dispositivo.

      O dispositivo novo será exibido na visualização Dispositivo.

Atualizando um dispositivo

  1. Para atualizar um dispositivo:
    1. Na visualização Dispositivo, selecione um dispositivo e expanda seus subnós até que a versão do OS esteja visível.
    2. Selecione a versão do OS e em seguida clique no (Editar Dispositivo).
    3. No campo Fabricante, selecione um fabricante do dispositivo a partir da lista suspensa.

      Os campos Modelo e Versão OS serão preenchidos com as opções disponíveis.

    4. No campo Modelo, selecione uma opção a partir da lista suspensa.

      Se não houver modelos disponíveis, insira um nome de modelo descritivo.

    5. No campo Versão OS, selecione uma opção a partir da lista suspensa.

      Se não houver versões OS disponíveis, insira um nome de versão descritivo OS.

    6. No campo Módulos Mib Mapeados, selecione os módulos MIB que você deseja associar com o dispositivo novo.
    7. Clique em OK para atualizar os detalhes do dispositivo.

      Os detalhes atualizados do dispositivo serão exibidos na visualização Dispositivo.

Excluindo um Dispositivo

  1. Na visualização Dispositivo, selecione o dispositivo e em seguida clique no (Excluir Dispositivo).

    Para excluir diversos dispositivos, mantenha pressionada as teclas Ctrl+Alt, selecione os elementos usando o botão esquerdo do mouse e em seguida clique no (Excluir Dispositivo).
