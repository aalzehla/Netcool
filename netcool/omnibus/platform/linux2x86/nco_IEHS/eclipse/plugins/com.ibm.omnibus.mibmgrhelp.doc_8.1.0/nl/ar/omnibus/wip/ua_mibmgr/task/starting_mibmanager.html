IBM Tivoli Netcool/OMNIbus النسخة 8.1

بدء MIB Manager

قم ببدء MIB Manager GUI من سطر الأمر. لم يتم تصميمه لكى يتم تشغيله تحت سيطرة العملية أو كخدمة Windows.

نبذة عن هذه المهمة

ارجع الى المطبوعات الفنية لأنظمة التشغيل التى يتم دعمها للحصول على معلومات عن متطلبات أو تشغيل MIB Manager على أنظمة تشغيل AIX و Solaris.

تأكد أن متغير بيئة التشغيل $DISPLAY قد تم تحديده الى وحدة خدمة X11 تعمل قبل بدء MIB Manager.

اجراء

  1. قم باستخدام الأمر التالي لبدء MIB Manager GUI:

    NCHOME/omnibus/bin/nco_mibmanager

  2. قم باستخدام الأمر التالي لبدء وظيفة سطر أمر MIB Manager:

    NCHOME/omnibus/bin/nco_mibmanager_batch [options]

  • اختيارات سطر أمر MIB Manager
    MIB Managerبه وظيفة سطر الأمر التى يمكنك استخدامها لاستقبال بيانات MIB وارسال ملفات القواعد. يمكنك اصدار الأوامر يدويا أو استدعائها برمجيا.
