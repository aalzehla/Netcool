IBM Tivoli Netcool/OMNIbus, מהדורה 7.4

הפעלת MIB Manager

ניתן להפעיל את MIB Manager משורת הפקודה. הוא לא אמור לרוץ תחת בקרת תהליך או בתור שירות של Windows.

כדי להפעיל את MIB Manager, השתמשו בפקודה שלהלן:

$NCHOME/omnibus/bin/nco_mibmanager

%NCHOME%\omnibus\bin\nco_mibmanager.bat

הערה: עיינו בתיעוד של מערכות ההפעלה הנתמכות לקבלת מידע על הדרישות להרצת Tivoli Netcool/OMNIbus MIB Manager במערכות ההפעלה של AIX ו-Solaris.
נושא אב: Tivoli Netcool/OMNIbus MIB Manager
