Configurando as preferências de diretório

Use as preferências do diretório para especificar os diretórios usados para armazenar MIBs importadas, dados da MIB e arquivos de regras exportados.

Para configurar as preferências de diretório, clique no Preferências ou a partir do menu, clique em Arquivo > Preferências e em seguida a partir da área da janela esquerda da janela Preferências, clique em Diretório.

  • No campo Diretório de Dados, especifique onde os MIBs importados serão armazenados como arquivos XML. O local padrão é NCHOME/omnibus/var/mibmanager/data.
  • No campo Exportar Diretório, especifique onde os arquivos exportados serão gerados. Um subdiretório baseado no tipo de exportação será criado. O local padrão é NCHOME/omnibus/var/mibmanager/export.
  • No campo Importar Diretório, especifique onde os arquivos MIB importados estarão localizados. O local padrão é NCHOME/omnibus/var/mibmanager/mibs.
