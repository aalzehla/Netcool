Impostazione del livello di registrazione

È possibile modificare il livello di registrazione per fornire file di log o di traccia più dettagliati al fine di semplificare il debug di MIB Manager. Le informazioni supplementari fornite aumentando i livelli di registrazione possono consentire il debug di problemi specifici durante l'analisi dei MIB.

Per modificare il livello di registrazione, fare clic su Modifica > Debug dalla barra dei menu della finestra principale. I file di log presentano sei livelli di messaggi in cui è possibile spostarsi per aumentare il livello di dettagli acquisito: NONE, ERROR, WARN, INFO, DEBUG e VERBOSE.

Nella tabella che segue viene fornita una breve descrizione di ciascun livello di registrazione:

Tabella 1. Un elenco di livelli di registrazione con la descrizione associata
Livello log Descrizione
none

Specifica un livello log in cui non viene emesso nessun messaggio di debug.

Le prestazioni dell'applicazione vengono migliorate ma nessun messaggio di avvertenza o di errore viene visualizzato durante l'analisi.
error

Specifica un livello log in cui non vengono registrati solo gli errori che influiscono sui risultati.

Sono inclusi tutti gli errori identificati da MIB Manager che determinano la visualizzazione di dati errati nella finestra principale.
warn

Specifica un livello log in cui vengono registrati messaggi di avvertenza e di errore.

Un'avvertenza indica una condizione che potrebbe determinare dati errati e quindi deve essere verificata. Questa condizione, tuttavia, non determina il danneggiamento dei dati.
info

Specifica un livello log in cui vengono registrati messaggi informativi, di avvertenza e critici.

I messaggi informativi contengono i dettagli relativi all'avanzamento complessivo delle attività. Ad esempio, forniscono il testo completo relativo agli oggetti man mano che vengono analizzati.
debug
Specifica un livello log in cui vengono registrati tutti i messaggi.
Nota: Eseguire MIB Manager solo in modalità debug per tentare di risolvere una condizione critica. Il volume dei dati prodotti durante l'analisi di un numero moderato di MIB riempirà rapidamente il file system.
verbose

Specifica un livello log in cui vengono registrati tutti i messaggi informativi.

Un messaggio dettagliato include lo stato di variabili, array e hash man mano che ne viene modificato il valore e lo stato.
