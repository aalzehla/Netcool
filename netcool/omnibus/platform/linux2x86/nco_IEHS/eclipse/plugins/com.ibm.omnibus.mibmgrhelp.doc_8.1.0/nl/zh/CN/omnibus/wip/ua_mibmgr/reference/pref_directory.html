设置目录首选项

使用目录首选项可指定用于存储导入的 MIB、MIB 数据以及导出的规则文件的目录。

要设置目录首选项，请单击 首选项，或在主菜单中单击文件 > 首选项，然后在“首选项”窗口的左窗格中单击目录。

  • 在数据目录字段中，指定将用于存储导入的 MIB（以 XML 文件格式）的目录。缺省位置是 NCHOME/omnibus/var/mibmanager/data。
  • 在导出目录字段中，指定将导出的文件生成到哪个目录。将创建基于导出类型的子目录。缺省位置是 NCHOME/omnibus/var/mibmanager/export。
  • 在导入目录字段中，指定导入的 MIB 文件所在的目录。缺省位置是 NCHOME/omnibus/var/mibmanager/mibs。
