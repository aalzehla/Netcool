SNMP 트랩 편집

SNMP 트랩을 편집하는 방법입니다.

이 태스크 정보

세부사항 보기에서는 다음 SNMP 트랩 필드를 편집할 수 있습니다.
  • @Severity
  • @Type
  • @ExpireTime
  • CODEBLOCK
참고: SNMP는 set (@Type ) 또는 clear (@Severity) 정보를 지정하기 위한 표준 메커니즘을 제공하지 않습니다. 이 정보를 직접 지정해야 합니다.

세부사항 보기에서는 편집 가능하도록 작성하려는 필드 값을 두 번 클릭하십시오. 필드를 변경한 후에는 를 클릭하여 변경사항을 저장하거나 를 클릭하여 변경사항을 취소하십시오.

프로시저

SNMP 트랩을 편집하려면 다음을 수행하십시오.

  1. MIB 모듈 보기나 OID 트리 보기에서 트랩을 선택하십시오.

    트랩에 대한 세부사항 정보가 세부사항 보기에 표시됩니다.

  2. @Severity 필드를 편집하려면, 필드 값을 두 번 클릭한 후 드롭 다운 목록에서 새 값을 선택하십시오.
  3. @Type 필드를 편집하려면, 필드 값을 두 번 클릭한 후 드롭 다운 목록에서 새 값을 선택하십시오.
  4. @ExpireTime 필드를 편집하려면, 필드 값을 두 번 클릭한 후 새 기간(초, 분, 시 또는 일 단위)을 입력하십시오.
  5. CODEBLOCK 필드를 편집하려면, 필드 값(필드가 비어 있는 경우에도)을 두 번 클릭한 후 새 코드 블록을 입력하십시오.
