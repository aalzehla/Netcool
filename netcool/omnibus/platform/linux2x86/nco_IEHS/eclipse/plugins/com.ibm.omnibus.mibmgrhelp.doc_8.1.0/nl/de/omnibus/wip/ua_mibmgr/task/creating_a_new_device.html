IBM Tivoli Netcool/OMNIbus Version 8.1

Geräte erstellen und löschen

Ein MIB Manager-Gerät dient zum Gruppieren zugeordneter MIB-Module und zur Definition aller für den Export erforderlichen Objekte. Sie können Geräte in der MIB Manager-GUI erstellen, aktualisieren und löschen.

Informationen zu diesem Vorgang

MIB Manager-Geräte werden in der Sicht Gerät angezeigt. Wählen Sie aus der Baumstruktur Gerät ein Gerät aus, um den Hersteller, die Modellnummer sowie die Betriebssystemversion des Geräts anzuzeigen.

Vorgehensweise

Gerät erstellen

  1. So erstellen Sie ein neues Gerät:
    1. Wählen Sie in der Sicht MIB-Module mindestens ein MIB-Modul aus und klicken Sie dann auf ("Gerät erstellen").
    2. Wählen Sie im Feld Hersteller einen Gerätehersteller aus der Dropdown-Liste aus.

      In die Felder Modell und Betriebssystemversion werden die verfügbaren Optionen eingefügt.

    3. Wählen Sie im Feld Modell eine Option aus der Dropdown-Liste aus.

      Wenn keine Modelle verfügbar sind, geben Sie einen beschreibenden Modellnamen ein.

    4. Wählen Sie im Feld Betriebssystemversion eine Option aus der Dropdown-Liste aus.

      Wenn keine Betriebssystemversionen verfügbar sind, geben Sie für die Betriebssystemversion einen beschreibenden Namen ein.

    5. Wählen Sie im Feld Zugeordnete MIB-Module die MIB-Module aus, die dem neuen Gerät zugeordnet werden sollen.
    6. Klicken Sie auf OK, um das Gerät zu erstellen.

      Das neue Gerät wird in der Sicht Gerät angezeigt.

Gerät aktualisieren

  1. So aktualisieren Sie ein Gerät:
    1. Wählen Sie in der Sicht Gerät ein Gerät aus und erweitern Sie dessen Unterknoten, bis die Betriebssystemversion zu sehen ist.
    2. Wählen Sie die Betriebssystemversion aus und klicken Sie dann auf ("Gerät bearbeiten").
    3. Wählen Sie im Feld Hersteller einen Gerätehersteller aus der Dropdown-Liste aus.

      In die Felder Modell und Betriebssystemversion werden die verfügbaren Optionen eingefügt.

    4. Wählen Sie im Feld Modell eine Option aus der Dropdown-Liste aus.

      Wenn keine Modelle verfügbar sind, geben Sie einen beschreibenden Modellnamen ein.

    5. Wählen Sie im Feld Betriebssystemversion eine Option aus der Dropdown-Liste aus.

      Wenn keine Betriebssystemversionen verfügbar sind, geben Sie für die Betriebssystemversion einen beschreibenden Namen ein.

    6. Wählen Sie im Feld Zugeordnete MIB-Module die MIB-Module aus, die dem neuen Gerät zugeordnet werden sollen.
    7. Klicken Sie auf OK, um die Gerätedetails zu aktualisieren.

      Die aktualisierten Gerätedetails werden in der Sicht Gerät angezeigt.

Gerät löschen

  1. Wählen Sie in der Sicht Gerät das Gerät aus und klicken Sie dann auf ("Gerät löschen").

    Halten Sie zum Löschen mehrerer Geräte die Steuertaste und die Taste Alt gedrückt, wählen Sie mit der linken Maustaste die Elemente aus und klicken Sie dann auf ("Gerät löschen").
