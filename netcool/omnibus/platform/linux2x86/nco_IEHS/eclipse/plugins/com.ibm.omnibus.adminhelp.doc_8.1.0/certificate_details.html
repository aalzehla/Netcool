Certificate Details

Use this window to examine the contents of a digital server certificate that an ObjectServer or process agent presents to the Netcool/OMNIbus Administrator client in order to establish a Secure Sockets Layer (SSL) connection.

This window displays information about the certificate owner and certificate authority (CA), and includes the following details:

  • The X.509 version
  • The distinguished name of the server (including the common name)
  • The algorithm used for encrypting the signature, and the object identifier (OID)
  • The public key type, modulus, and public exponent
  • The date and time when the server certificate was issued. and the date and time when it expires
  • The distinguished name of the Certificate Authority (CA) that issued the server certificate
  • The serial number of the server certificate
  • The algorithm used for encrypting the public key
  • The digital signature of the certificate

Click OK to close the window after examining the certificate. You can then choose whether to accept or reject the certificate.
