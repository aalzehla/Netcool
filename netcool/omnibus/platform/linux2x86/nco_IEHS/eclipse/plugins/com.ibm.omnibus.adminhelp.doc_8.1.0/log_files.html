Log Files

Use this pane to view details of the ObjectServer log files that are set up in the selected ObjectServer, and to set up and maintain log files.

In the Log Files pane, the following details are shown for each log file:
  • The unique name assigned to the log file
  • The full path and file name of the physical file
  • The maximum number of ObjectServer files to create for storing the data
  • The maximum file size
  • An indication of whether the file is activated on the system so that data can be written to it
