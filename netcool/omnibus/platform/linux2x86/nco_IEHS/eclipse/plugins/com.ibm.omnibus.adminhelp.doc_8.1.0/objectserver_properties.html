ObjectServer Properties

Use this pane to view details of all ObjectServer properties that are defined for the selected ObjectServer, and to configure these properties.

In the ObjectServer Properties pane, the following details are shown for each property:
  • The unique name assigned to the property
  • The default value of the property
  • The property description
  • An indication of whether you can change the value of the property
  • An indication of whether changes to the property take immediate effect
