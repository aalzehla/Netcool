Column Visuals

Use this pane to configure column visual settings that determine the visual appearance of event list columns. You can view details of all column visuals that are set up in the selected ObjectServer, and to set up and maintain column visuals.

In the Column Visuals pane, the following details are shown for each column visual:
  • The unique name assigned to the event list column
  • The text used for the column heading in the event list
  • The default width of the column
  • The maximum width of the column
  • The justification (or alignment) of the column heading
  • The justification (or alignment) of data within the column
