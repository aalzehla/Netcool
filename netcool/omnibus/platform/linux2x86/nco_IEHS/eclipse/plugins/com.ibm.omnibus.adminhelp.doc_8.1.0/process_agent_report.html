IBM Tivoli Netcool/OMNIbus Version 8.1

Process Agent Report

Use this window to view details for the process agents within the process control system configuration. Details are shown for all process agents that were selected when the Import Connections Wizard was last run.

Tip: After you have started Netcool/OMNIbus Administrator, you can select File > Import at any time to import new server communication information that is specified in the Server Editor. This information facilitates communication between Tivoli Netcool/OMNIbus server components such as ObjectServers, gateways, process agents, and proxy servers.
In the Process Agent Report window, the following details are shown for each process agent:
  • The unique name assigned to the process agent
  • The host name of the computer on which the process agent is running
  • The port on which the process agent listens for unencrypted connections
  • An indication of whether SSL encryption is configured
  • The number of running processes controlled by the process agent

    You must be connected to the process agent to see this information; otherwise, a hyphen (-) is shown.

  • The number of pending processes controlled by the process agent

    You must be connected to the process agent to see this information; otherwise, a hyphen (-) is shown.

  • The number of dead processes controlled by the process agent

    You must be connected to the process agent to see this information; otherwise, a hyphen (-) is shown.

  • An indication of whether information needs to be saved

    This column shows a value of true if changes have been made to the configuration of the process agent, but the configuration file has not yet been saved.
