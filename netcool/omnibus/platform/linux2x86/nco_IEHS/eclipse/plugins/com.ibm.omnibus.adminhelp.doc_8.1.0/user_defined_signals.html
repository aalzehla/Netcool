User Defined Signals

Use this pane to view details of all user-defined signals that are set up in the selected ObjectServer, and to set up and maintain user-defined signals.

In the User Defined Signals pane, the following details are shown for each user-defined signal:
  • The unique name assigned to the user-defined signal
  • A descriptive comment about the signal
