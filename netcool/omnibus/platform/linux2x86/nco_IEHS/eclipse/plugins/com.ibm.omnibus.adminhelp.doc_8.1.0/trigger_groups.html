Trigger Groups

Use this pane to view details of all trigger groups that are set up in the selected ObjectServer, and to set up and maintain trigger groups.

In the Trigger Groups pane, the following details are shown for each trigger group:
  • The unique name assigned to the trigger group
  • An indication of whether the trigger group is activated on the system
