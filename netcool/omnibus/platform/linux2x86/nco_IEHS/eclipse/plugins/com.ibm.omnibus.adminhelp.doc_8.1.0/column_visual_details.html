Column Visual Details

Use this window to set up a new column visual or edit the details of an existing column visual. A column visual defines default display properties of a column in the alerts.status table, within event lists.

Complete this window as follows:
Column
If you are creating a new column visual, select the column for which you are adding the visual.
Title
Type the title that you want to display as the column header in Tivoli Netcool/OMNIbus event lists.
Default
Specify the default column width (in characters).
Maximum
Specify the maximum column width (in characters).
Title
Select the justification or alignment of the column title.
Column
Select the justification or alignment of the information in the column.

Saving your column visual details

After creating or editing a column visual, you can save or cancel the changes made by clicking the relevant button as follows:
OK
Click this button to save the column visual details and close the window. New column visuals are added to the Column Visuals pane.
Cancel
Click this button to close the window without saving your changes.
