IBM Tivoli Netcool/OMNIbus Version 8.1

Save Filter

Use this window to save a customized filter that you have created for the columns within the current pane.

Complete this window as follows:
Name
Type a name for the filter, or select a previously-saved filter to overwrite from the drop-down list. The name can contain only letters, numbers, underscores, and spaces.
OK
Click this button to save the filter and close the window.
Cancel
Click this button to close the window without saving the filter.
