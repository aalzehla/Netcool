Procedures

Use this pane to view details of all procedures that are set up in the selected ObjectServer, and to set up and maintain SQL procedures and external procedures.

In the Procedures pane, the following details are shown for each procedure:
  • The unique name assigned to the procedure
  • The type of procedure
