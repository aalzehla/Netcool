ObjectServer Security

Use this window to enter authentication credentials for connecting to an ObjectServer.

Complete this window as follows:
Username
Type the user name that is used to log into the ObjectServer.
Password
Type the password used to log into the ObjectServer.
Always use for this connection
Select this check box to indicate that the specified user name and password should be saved for automatic reuse on subsequent connection attempts to this ObjectServer. These settings last for the length of the application session.
Use as default
Select this check box if you want the values specified for the user name and password to be automatically filled in the next time this window is displayed.
Note: If you select both check boxes, the Always use for this connection user name and password values take precedence. These settings last for the length of the application session.

Verifying your authentication credentials

After specifying your authentication details, you can start or cancel the connection attempt by clicking the relevant button as follows:
OK
Click this button to verify the credentials for connection, and close the window.
Cancel
Click this button to close the ObjectServer Security window without attempting to connect to the ObjectServer. You return to the ObjectServer Report window.
