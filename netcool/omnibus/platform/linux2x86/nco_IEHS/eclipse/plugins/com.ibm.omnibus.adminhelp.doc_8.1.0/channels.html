IBM Tivoli Netcool/OMNIbus Version 8.1

Channels

Use this pane to view details of all channels that are set up in the current ObjectServer, and to set up and maintain channels for fast-track event notifications.

In the Channels pane, the following details are shown for each channel:
  • The unique name assigned to the channel
  • The description that is associated with the channel name
Related tasks:
Creating and editing channels
Copying and pasting channels
Deleting a channel
Sending messages to channel recipients
Disconnecting Accelerated Event Notification clients
Shutting down Accelerated Event Notification clients
