IBM Tivoli Netcool/OMNIbus Version 8.1

Tools

Use this pane to view details of all tools that are set up in the selected ObjectServer, and to configure tools for the Tivoli Netcool/OMNIbus desktop; that is, the event list and Conductor.

In the Tools pane, the following details are shown for each tool:
  • The unique name assigned to the tool
  • An indication of whether the tool is activated on the system for use by operators
  • An indication of whether SQL commands are enabled for the tool
  • An indication of whether external executable commands are enabled for the tool
  • An indication of whether the tool enforces a journal entry for an event

The tools shown within the Tools pane can be added to the event list and Conductor desktop menus to help with alert management.
