IBM Tivoli Netcool/OMNIbus Version 8.1

Filter

Use this window to apply a filter condition to a selected column in the current pane.

Complete this window as follows:
Column
This list contains the names of all the columns in the selected table. Select a column on which to filter.
Operation
Select a comparison operator to use in the filter condition.
Value
Type a value on which you want to filter.
Apply
Click this button to see the result of the filter without exiting the Filter window. You can preview the results of different filter conditions in this way. Note, however, that subsequent filters are applied only to the last set of results obtained in the current pane (and not to the full data set in the pane).
OK
Click this button to save the filter and close the window.
Cancel
Click this button to close the window without saving the filter.
