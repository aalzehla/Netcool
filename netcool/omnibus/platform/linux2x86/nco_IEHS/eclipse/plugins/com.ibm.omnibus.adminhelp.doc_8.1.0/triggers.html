Triggers

Use this pane to view details of all triggers that are set up in the selected ObjectServer, and to set up and maintain database, signal, and temporal triggers.

In the Triggers pane, the following details are shown for each trigger:
  • The unique name assigned to the trigger
  • The trigger group to which the trigger belongs
  • The type of trigger
  • The priority of execution of the trigger
  • An indication of whether debugging information is sent to the ObjectServer message log whenever the trigger fires
  • An indication of whether the trigger is activated on the system
  • An indication of whether the trigger group to which the trigger belongs is activated on the system
Tip: To view only one type of trigger, click Show Database Triggers Only, Show Temporal Triggers Only, or Show Signal Triggers Only in the toolbar.
