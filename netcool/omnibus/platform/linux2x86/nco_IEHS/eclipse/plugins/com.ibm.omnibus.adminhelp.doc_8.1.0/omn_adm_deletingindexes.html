Deleting indexes

If an index is no longer required, delete it from the catalog.indexes table.

Procedure

To delete column indexes:

  1. From the Netcool/OMNIbus Administrator window, select the System menu button.
  2. Click Databases. The Databases, Tables and Columns pane opens.
  3. To open the catalog.indexes table and view all indexes, click indexes and then click the Data View tab. Each index is represented as a row in the table. You can use the IndexName column to identify the required index.
  4. Right click the required row and click Indexes > Delete Column Index, then click Yes to confirm the deletion of the index.
