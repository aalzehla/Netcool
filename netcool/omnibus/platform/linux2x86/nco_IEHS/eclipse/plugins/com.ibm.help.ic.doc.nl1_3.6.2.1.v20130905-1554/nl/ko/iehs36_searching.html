Information Center 검색

Information Center에서 모든 주제를 검색하거나 개별 주제 또는 주제 콜렉션을 검색하거나 정보의 서브세트만 검색하도록 검색 범위를 작성할 수 있습니다. 검색 결과 보기에서는 관련성에 따라 정렬된 최대 500개의 결과를 표시할 수 있습니다.

이 태스크 정보

컨텐츠 보기와 검색 결과 보기 사이를 전환하려면 컨텐츠 탭() 또는 검색 결과 탭()을 클릭하십시오.

주: 검색 문자열은 대소문자를 구분하지 않습니다.

프로시저

  • Information Center에 포함된 모든 주제를 검색하려면 검색 필드에 조회할 사항을 입력하고 이동을 클릭하십시오. 또한 주제의 서브세트만 정기적으로 검색하도록 검색 범위를 작성할 수도 있습니다.
  • 특정 주제 또는 주제 세트를 검색하려면 탐색 트리에서 주제를 선택하고 컨텐츠 보기의 도구 모음에서 주제 검색()을 클릭하십시오. 선택된 주제 검색 또는 선택된 주제 및 모든 하위 주제 검색을 선택하고 검색 필드에 조회할 사항을 입력하고 확인을 클릭하십시오.

결과

Information Center에 협업 플러그인을 설치한 경우 검색 결과 보기에서는 검색 조회와 일치하는 주제 및 주석을 표시합니다. 두 개의 태그 주제 및 주석 위로 마우스를 이동하여 주제 및 주석의 검색 결과 사이를 전환할 수 있습니다. 검색 결과 목록에서 주제 또는 주석을 선택하여 보십시오.

협업 플러그인이 설치되어 있지 않은 경우, 검색 결과 보기에서는 조회와 일치하는 주제를 표시합니다. 검색 결과 목록에서 주제를 선택하여 보십시오.

동일한 제목과 동일한 요약의 주제는 유사한 페이지로 인식됩니다. 검색 결과의 주제 중 하나에 유사한 페이지가 있는 경우 유사한 페이지의 총계가 해당 주제의 맨 아래에 초록색 글꼴로 표시됩니다. 또한 유사한 페이지 태그를 클릭하여 모든 유사한 페이지를 펼쳐 표시할 수 있습니다.

다음에 수행할 작업

  1. 검색 결과 탭()을 클릭하여 검색 결과 보기를 보십시오. 기본적으로 검색 결과는 그룹화되지 않습니다.
  2. 도구 모음에서 결과 카테고리 표시()를 클릭하여 검색 결과를 그룹화하십시오.
도구 모음에서 아이콘을 사용하여 검색 결과를 제어할 수 있습니다.
  • 각 검색 결과 아래에서 요약을 표시하거나 숨기려면 결과 설명 표시()를 클릭하십시오.
  • 검색 강조표시를 사용하거나 사용하지 않으려면 강조표시 ()를 클릭하십시오.
  • 고급 검색 기술
    따옴표, 와일드카드 또는 부울 연산자로 고급 검색을 수행하여 더 작은 검색 결과 세트를 리턴할 수 있습니다.
  • 빠른 검색 수행
    빠른 검색 기능을 사용하여 탐색에서 주제 또는 주제 세트를 검색할 수 있습니다.
