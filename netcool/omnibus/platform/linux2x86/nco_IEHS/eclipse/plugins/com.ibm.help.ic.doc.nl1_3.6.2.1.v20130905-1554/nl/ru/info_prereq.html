Предварительные требования для работы со справочной системой

Перед установкой IBM Eclipse Help System необходимо убедиться в том, что рабочая станция, операционная система, браузер и аппаратное обеспечение отвечают требованиям, описанным в этом разделе.

Требования к браузеру

Для работы с IBM Eclipse Help System необходим браузер Microsoft Internet Explorer версии 6.0 или 7.0 либо Firefox версии 2.0 или 3.0. Для того чтобы работали все функции IBM Eclipse Help System, в браузере необходимо включить поддержку cookie и JavaScript и выключить функцию блокировки всплывающих окон.

Требования к среде операционной системы

Для работы с IBM Eclipse Help System необходимо установить среду выполнения Java (JRE) версии 6.0. Подробное описание требований приведено в следующей таблице.

Требования к аппаратному обеспечению

z/OS

Для применения IBM Eclipse Help System в системе z/OS необходимо убедиться в том, что в z/OS выполнены требования к объему физической памяти и размеру REGION.

  • Физическая память = <основная-память> + <размер-модуля> +<число-параллельных-пользователей> * <размер-буфера-ответов>
    • основная-память - это память, необходимая для запуска пакета IBM Eclipse Help System. В z/OS ее объем составляет 90 МБ.
    • размер-модуля - это размер модуля документации.
    • число-параллельных-пользователей - это число пользователей, одновременно работающих со справочной системой.
    • размер-буфера-ответов указан в файле preferences.ini, расположенном в каталоге eclipse\plugins\org.eclipse.help.base_3.0.1.
  • Размер REGION должен совпадать с размером физической памяти.

Другие операционные системы

Необходимо не менее 128 МБ оперативной памяти. Рекомендуемый объем памяти составляет не менее 256 МБ.

Прим.: Необходимо увеличить оценку объема памяти в соответствии с размером модуля документации. Например, если размер модуля документации составляет 100 МБ, то дополнительно потребуется по крайней мере 80 МБ оперативной памяти.

Поддерживаемые способы настройки

Справочную систему можно настроить с помощью файла org.eclipse.help.base_<версия>/preferences.ini, расположенного в каталоге eclipse/plugins, однако рекомендуется не изменять стандартную конфигурацию.

Во избежание сбоев в работе справочной системы не изменяйте следующие параметры:

always_external_browser
В операционных системах Microsoft Windows по умолчанию применяется параметр always_external_browser=true. При запуске справочной системы в автономном режиме этот параметр указывает, что должно быть открыто окно браузера по умолчанию, а не встроенного браузера справочной системы. Параметр always_external_browser=true не следует устанавливать в операционных системах, отличных от Windows.
