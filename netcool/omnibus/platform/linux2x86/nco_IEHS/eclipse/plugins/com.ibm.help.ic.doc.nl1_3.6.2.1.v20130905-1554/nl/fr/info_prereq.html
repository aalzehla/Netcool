Prérequis pour le Centre de documentation

Lorsque vous configurez le système d'aide IBM Eclipse, vous devez vous assurer que votre poste de travail, votre système d'exploitation, votre navigateur et votre matériel correspondent bien aux conditions décrites dans la présente rubrique.

Navigateur

Pour pouvoir utiliser IBM Eclipse Help System, vous devez installer Microsoft Internet Explorer version 6.0 ou 7.0, ou Firefox 2.0 ou 3.0. Pour permettre à toutes les fonctionnalités de cette version d'IBM Eclipse Help System d'être utilisables, vous devez activer les cookies et JavaScript dans le navigateur et désactiver dans ce dernier le blocage des fenêtres en incrustation.

Matériel

z/OS

Pour pouvoir exécuter IBM Eclipse Help System sur z/OS, vous devez vérifier que votre système z/OS réunit bien les conditions requises pour la taille de la mémoire effective et du paramètre REGION.

  • Mémoire réelle = <mémoire_base> + <taille_plug-in> +<utilisateurs_simultanés> * <taille_tampon_réponse>
    • mémoire_base est la mémoire nécessaire au démarrage du package IBM Eclipse Help System. Elle est de 90 Mo sur z/OS.
    • taille_plug-in est la taille du plug-in de documentation.
    • utilisateurs_simultanés est le nombre d'utilisateurs simultanés accédant au système d'aide.
    • La valeur de taille_tampon_réponse se trouve dans le fichier preferences.ini du répertoire eclipse\plugins\org.eclipse.help.base_3.0.1.
  • La taille de REGION doit être identique à celle de la mémoire réelle.

Autres systèmes d'exploitation

Le minimum requis est de 128 Mo de RAM. 256 Mo ou plus sont recommandés.

Remarque : Vous devez augmenter la mémoire selon la taille des plug-in de documentation. Par exemple, si la taille du plug-in de documentation est de 100 Mo, ajoutez au moins 80 Mo de RAM supplémentaire.

Personnalisation prise en charge

Vous pouvez configurer le système d'aide à l'aide du fichier org.eclipse.help.base_<n° version>/preferences.ini qui se trouve dans le répertoire eclipse/plugins, bien que nous vous conseillions d'utiliser la configuration proposée par défaut. Pour des explications détaillées sur la personnalisation du système d'aide à l'aide du fichier preferences.ini.

Ne modifiez pas les paramètres suivants ; le système d'aide risquerait de ne pas fonctionner correctement :

always_external_browser
Dans le cas des systèmes d'exploitation Microsoft Windows, la configuration par défaut de ce paramètre est always_external_browser=true. Lorsque vous démarrez le système d'aide en mode autonome, grâce à ce paramètre, c'est le navigateur par défaut du système d'exploitation qui s'ouvre, et non celui intégré au système d'aide. Il n'est pas nécessaire de définir ce paramètre comme always_external_browser=true pour les autres systèmes que Windows.
