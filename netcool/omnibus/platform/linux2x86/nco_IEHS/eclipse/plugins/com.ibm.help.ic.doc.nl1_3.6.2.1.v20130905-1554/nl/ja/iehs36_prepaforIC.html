インフォメーション・センターを使用するための準備

インフォメーション・センターを使用して最適な結果を得るために、以下の手順を実行します。

  • サポート対象のブラウザーを使用していることを確認します。

    このインフォメーション・センターを使用するには、サポート対象の Web ブラウザーをシステムにインストールする必要があります。 このインフォメーション・センターのサポート対象のブラウザーについては、インフォメーション・センターでサポートされているブラウザーを参照してください。

  • Web ブラウザーを希望の言語に設定します。

    Web ブラウザーの言語オプションを使用すれば、インフォメーション・センターのコンテンツを希望の言語で表示できます。詳細については、希望の言語で情報を表示するための操作を参照してください。

  • インフォメーション・センターでサポートされているブラウザー
    このインフォメーション・センターを使用するには、サポート対象の Web ブラウザーをシステムにインストールする必要があります。
  • 希望の言語で情報を表示するための操作
    Web ブラウザーの言語オプションを使用すれば、インフォメーション・センターのコンテンツを希望の言語で表示できます。
