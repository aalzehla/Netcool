Pode localizar um tópico mais facilmente definindo um marcador. Defina marcadores, utilizando as funções num navegador da Web ou utilizando a vista Marcadores.

Definir marcadores utilizando um navegador da Web

Num Information Center implementado num servidor e visualizado numa intranet ou na Internet, é possível guardar marcadores relativos a tópicos no menu Marcadores ou Favoritos do navegador da Web.

Procedimento

Para definir um marcador para o tópico actual no navegador da Web:
  • Se estiver a utilizar o Internet Explorer, faça clique em Marcador () para guardar o tópico na pasta Favoritos.
  • Se não estiver a utilizar o Internet Explorer, faça clique com o botão direito do rato em qualquer área em branco na página actual para abrir o menu. Seleccione a opção Adicionar aos favoritos ou Adicionar aos marcadores para guardar o tópico numa das pastas Favoritos.
Para visualizar os marcadores, abra o menu Favoritos do navegador da Web.

Para eliminar um marcador, abra o menu Favoritos do navegador da Web para localizar o marcador. Faça clique com o botão direito do rato e, em seguida, faça clique em Eliminar no menu.
