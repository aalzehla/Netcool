Information Center에 책갈피 설정

책갈피를 설정하여 주제를 좀더 쉽게 찾을 수 있습니다. 웹 브라우저의 기능 또는 책갈피 보기를 사용하여 책갈피를 설정합니다.

상위 주제: Information Center에서 정보 보기

웹 브라우저를 사용하여 책갈피 설정

Information Center를 서버에 배치하여 인트라넷 또는 인터넷을 통해 보는 경우 주제에 대한 책갈피를 웹 브라우저의 책갈피 또는 즐겨찾기 메뉴에 저장할 수 있습니다.

프로시저

웹 브라우저에서 현재 주제에 대한 책갈피를 설정하려면 다음을 수행하십시오.
  • Internet Explorer를 사용하는 경우 책갈피()를 클릭하여 즐겨찾기 폴더에 주제를 저장하십시오.
  • Internet Explorer를 사용하지 않는 경우 현재 페이지의 빈 영역에서 마우스 오른쪽 단추를 클릭하여 메뉴를 여십시오. 즐겨찾기에 추가 또는 책갈피에 추가 옵션을 선택하여 즐겨찾기 폴더 중 하나에 주제를 저장하십시오.
책갈피를 보려면 웹 브라우저의 즐겨찾기 메뉴를 여십시오.

책갈피를 삭제하려면 웹 브라우저의 즐겨찾기 메뉴를 열어 책갈피를 찾으십시오. 마우스 오른쪽 단추를 클릭하고 메뉴에서 삭제를 클릭하십시오.
