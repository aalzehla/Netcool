ブール演算子 (AND、OR、NOT) による検索

検索を効果的にするために、検索ストリングに AND、OR および NOT を含めたブール演算子を挿入することができます。

ブール演算子では、検索範囲を狭めたり広げたりすることで、 検索を効率化することができます。 以下は例です。

database AND "log file"
文書内のどこかに database という単語と log file という句の両方を含むトピックを対象とするように、検索の幅を狭めます。
database OR "log file"
文書内のどこかに database という単語または log file という句のいずれかを含むトピックを対象とするように、検索の幅を広げます。
database NOT "log file"
文書内のどこかに database を含み、log file という語句を含まないトピックを検索します。
database OR "data base" NOT "log file"
文書内のどこかに database という単語または data base という句を含み、かつ、log file という句を含まないトピックを検索します。
