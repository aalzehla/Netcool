Navegadores suportados para o Information Center

Para utilizar este Information Center, tem de ter um navegador da Web suportado instalado no seu sistema.

São suportados os seguintes navegadores da Web:

  • Microsoft® Internet Explorer Versão 7.0 ou Versão 8.0
  • Mozilla Firefox Versão 3.5 ou Versão 3.6.

Caso esteja a utilizar uma versão diferente dos navegadores da Web suportados ou outro navegador da Web, algumas funcionalidades poderão não estar disponíveis ou não funcionar do modo esperado.

Para assegurar que todas as funções do Information Center são utilizáveis, tem de activar cookies e JavaScript™ no navegador da Web e permitir que o navegador da Web apresente as janelas emergentes deste Information Center.
