Fonctions d'accessibilité du centre de documentation

Les fonctions d'accessibilité servent à permettre l'utilisation de produits informatiques aux personnes qui ont un handicap physique, comme une mobilité ou une vision limitée, ou d'autres besoins spéciaux.

Avec les fonctions d'accessibilité de ce centre de documentation, les utilisateurs peuvent :

  • Utiliser un logiciel lecteur d'écran et un synthétiseur vocal numérique pour faire lire ce qui est affiché à l'écran. Pour l'utilisation de technologies d'assistance avec les contenus HTML, consultez leur documentation produit.
  • Utiliser une loupe d'écran pour agrandir ce qui est affiché à l'écran.
  • Exécuter des fonctions spécifiques ou équivalentes en utilisant seulement le clavier.
  • Structure de cadres utilisée dans le centre de documentation
    Le centre de documentation utilise un certain nombre de cadres. Cette rubrique en fournit une description.
  • Raccourcis et accélérateurs clavier
    Certaines opérations qui s'effectuent avec la souris peuvent également être effectuées avec une touche ou une combinaison de touches.
