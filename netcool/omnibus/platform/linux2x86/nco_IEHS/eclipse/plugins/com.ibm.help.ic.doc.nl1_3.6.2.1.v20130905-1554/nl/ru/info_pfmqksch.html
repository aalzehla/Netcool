Быстрый поиск

В этом разделе приведены инструкции по быстрому выполнению поиска.

Для выполнения быстрого поиска выполните следующие действия на панели навигации:

  1. Нажмите кнопку Поиск разделов() на панели инструментов области навигации, чтобы открыть меню.
    • Для того чтобы выполнить поиск только в текущем разделе, выберите пункт Поиск в этом разделе.
    • Для того чтобы выполнить поиск в текущем разделе и всех вложенных разделах, выберите пункт Поиск в этом разделе и его подразделах.
      1. В окне поиска введите запрос в поле Поиск.
      2. Нажмите OK, чтобы начать поиск.
  2. Во всплывающем окне поиска введите запрос в поле Поиск и нажмите OK, чтобы начать поиск.

Если поиск выполняется только в том разделе, в котором было открыто меню быстрого поиска, то при обнаружении совпадений этот раздел будет автоматически открыт. При отсутствии совпадений будет показано сообщение Ничего не найдено.

Если поиск выполняется в разделе и подразделе, то на панели результатов поиска будет показано до 500 результатов поиска, наиболее точно соответствующих запросу. При выборе результата поиска будет показан текст соответствующего раздела документации. В этом тексте слова из строки поиска будут выделены. При отсутствии совпадений будет показано сообщение Ничего не найдено.

Примечания:
  • Окно Поиск в этом разделе будет автоматически закрыто независимо от наличия результатов.
  • Окно Поиск в этом разделе и его подразделах не закрывается автоматически. Необходимо нажать кнопку Отмена или закрыть окно диалога.
