Recherche rapide

Vous pouvez rechercher une rubrique ou un ensemble de rubriques dans la vue de navigation avec la fonction Recherche rapide.

Dans la vue de navigation, procédez comme suit pour effectuer une recherche rapide :

  1. Cliquez sur Rechercher dans une rubrique () dans la barre d'outils de la vue de navigation pour afficher le menu.
    1. Pour chercher seulement dans la rubrique que vous ouvrez, cliquez sur Rechercher dans la rubrique sélectionnée.
    2. Pour chercher dans la rubrique et dans toutes ses sous-rubriques, cliquez sur Rechercher dans la rubrique sélectionnée et toutes ses sous-rubriques.
  2. Dans la boîte de dialogue de recherche, entrez votre requête dans la zone Rechercher et cliquez sur OK pour lancer la recherche.

Si des rubriques correspondent à vos expressions de recherche, elles sont affichées dans la vue des résultats de recherche avec un message indiquant le nombre de résultats.

Si aucune rubrique ne correspond à vos expressions de recherche, vous obtenez un message vous indiquant que la recherche n'a donné aucun résultat.
