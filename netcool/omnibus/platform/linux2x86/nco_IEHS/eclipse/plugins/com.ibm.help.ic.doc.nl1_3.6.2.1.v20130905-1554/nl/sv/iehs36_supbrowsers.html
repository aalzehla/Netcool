Webbläsare som kan användas med informationscentret

Om du vill använda det här informationscentret måste du använda en webbläsare som kan användas.

Det går att använda följande webbläsare:

  • Microsoft Internet Explorer version 7.0 eller 8.0
  • Mozilla Firefox version 3.5 eller 3.6.

Om du använder någon annan version av de webbläsare som kan användas eller någon annan webbläsare kommer vissa funktioner eventuellt inte att vara tillgängliga eller fungera på rätt sätt.

Om du vill se till att det går att använda alla funktioner i informationscentret måste du aktivera cookier och JavaScript i webbläsaren, och dessutom måste du ange att fönster får visas för informationscentret.
