ข้อกำหนดสิ่งที่จำเป็นต้องมีสำหรับระบบวิธีใช้

เมื่อคุณติดตั้ง IBM Eclipse Help System คุณต้องแน่ใจว่า เวิร์กสเตชัน ระบบปฏิบัติการ บราวเซอร์ และฮาร์ดแวร์ของคุณ ตรงกับข้อกำหนดที่ได้กล่าวไว้ในหัวข้อนี้

ข้อกำหนดเกี่ยวกับบราวเซอร์

หากต้องการทำงานกับ IBM Eclipse Help System คุณจำเป็นต้องติดตั้ง Microsoft Internet Explorer เวอร์ชัน 6.0 หรือ 7.0 หรือ Firefox 2.0 หรือ 3.0 เพื่อให้แน่ใจว่า ฟังก์ชันทั้งหมดของเวอร์ชันนี้ของ IBM Eclipse Help System สามารถใช้งานได้ คุณจำเป็นต้องเปิดคุกกี้และ JavaScript ในบราวเซอร์และปิดใช้งานฟังก์ชันของการบล็อกหน้าต่างป๊อปอัพ

ข้อกำหนดเกี่ยวกับสภาวะแวดล้อมระบบปฏิบัติการ

หากต้องการทำงานกับ IBM Eclipse Help System คุณจำเป็นต้องติดตั้ง Java Runtime Environment (JRE) เวอร์ชัน 6.0 โปรดดูตารางต่อไปนี้สำหรับข้อกำหนดโดยละเอียด

ตารางที่ 1. ข้อกำหนดเกี่ยวกับสภาวะแวดล้อมระบบปฏิบัติการ
เวอร์ชันของ IBM Eclipse Help System ระบบปฏิบัติการ ข้อกำหนดเกี่ยวกับระบบปฏิบัติการและ JRE
IBM Eclipse Help System สำหรับ Microsoft Windows แบบ 32 บิต
  • Windows XP Professional Intel แบบ 32 บิต
  • Windows Vista Business Intel แบบ 32 บิต
  • IBM Eclipse Help System เวอร์ชันนี้ประกอบด้วย IBM JRE แบบฝัง คุณสามารถลบ JRE ออกจากแพ็กเกจได้ถ้าคุณไม่ต้องการเก็บ JRE แบบฝังไว้ คุณต้องมี JRE 6.0 ติดตั้งไว้บนระบบของคุณเพื่อรัน IBM Eclipse Help System
IBM Eclipse Help System สำหรับ Linux
  • Red Hat Linux Enterprise 5 Intel แบบ 32 บิต
  • IBM Eclipse Help System เวอร์ชันนี้ประกอบด้วย IBM JRE แบบฝัง คุณสามารถลบ JRE ออกจากแพ็กเกจได้ถ้าคุณไม่ต้องการเก็บ JRE แบบฝังไว้ คุณต้องมี JRE 6.0 ติดตั้งไว้บนระบบของคุณเพื่อรัน IBM Eclipse Help System
  • สำหรับระบบปฏิบัติการเหล่านี้ คุณสามารถเปิดระบบวิธีใช้ในโหมดพื้นฐานผ่านบราวเซอร์ Konqueror
  • หากต้องการแสดงอักขระแบบสองทิศทาง อักขระแบบสองไบต์ หรืออักขระแบบมัลติไบต์ ในระบบวิธีใช้ ให้ติดตั้งฟอนต์ที่สอดคล้องกันบนระบบ และตั้งค่าฟอนต์ให้เป็นฟอนต์สำหรับแสดงผลของบราวเซอร์
  • หากต้องการสนับสนุนหลายภาษา ให้ตั้งค่าโลแคลของระบบให้เป็น UTF-8 ก่อนเริ่มต้นระบบวิธีใช้
IBM Eclipse Help System สำหรับ AIX
  • AIX 5.3 PPC แบบ 64 บิต
  • IBM Eclipse Help System เวอร์ชันนี้ประกอบด้วย IBM JRE แบบฝัง คุณสามารถลบ JRE ออกจากแพ็กเกจได้ถ้าคุณไม่ต้องการเก็บ JRE แบบฝังไว้ คุณต้องมี JRE 6.0 ติดตั้งไว้บนระบบของคุณเพื่อรัน IBM Eclipse Help System

ข้อกำหนดเกี่ยวกับฮาร์ดแวร์

z/OS

หากต้องการรัน IBM Eclipse Help System บน z/OS คุณต้องแน่ใจว่า z/OS ตรงกับข้อกำหนดของขนาดหน่วยความจำจริงและขนาด REGION

  • หน่วยความจำจริง = <basememory> + <plugsize> +<concurrentuser> * <responseBufferSize>
    • basememory คือหน่วยความจำที่ต้องใช้เพื่อเริ่มต้นแพ็กเกจ IBM Eclipse Help System ซึ่งมีขนาด 90 MB สำหรับ z/OS
    • plugsize คือขนาดของปลั๊กอินเอกสารคู่มือของคุณ
    • concurrentuser คือจำนวนของผู้ใช้ที่ใช้งานพร้อมกัน ซึ่งจะเข้าถึงระบบวิธีใช้ของคุณ
    • responseBufferSize อาจพบได้ใน preferences.ini ในไดเร็กทอรี eclipse\plugins\org.eclipse.help.base_3.0.1
  • ขนาด REGION ควรตั้งค่าให้เป็นแบบเดียวกับขนาดหน่วยความจำจริง

ระบบปฏิบัติการอื่นๆ

ความต้องการหน่วยความจำต่ำสุดคือ 128 MB RAM 256 MB หรือมากกว่าที่แนะนำ

หมายเหตุ: คุณจำเป็นต้องเพิ่มหน่วยความจำตามขนาดของปลั๊กอินเอกสารคู่มือ ตัวอย่างเช่น ถ้าขนาดของปลั๊กอินเอกสารคือ 100 MB และเพิ่มอย่างน้อย 80 MB ของ RAM เพิ่มเติม

การปรับแต่งค่าที่ได้รับการสนับสนุน

คุณสามารถตั้งค่าระบบวิธีใช้โดยการแก้ไขไฟล์ org.eclipse.help.base_<version number>/preferences.ini ภายใต้ไดเร็กทอรี eclipse/plugins ซึ่งแนะนำให้คุณเก็บคอนฟิกูเรชันที่เป็นดีฟอลต์ไว้

ห้ามแก้ไขพารามิเตอร์ต่อไปนี้ มิฉะนั้นระบบวิธีใช้อาจทำงานไม่ถูกต้อง:

always_external_browser
สำหรับระบบปฏิบัติการ Microsoft Windows คอนฟิกูเรชันที่เป็นดีฟอลต์ของพารามิเตอร์นี้คือ always_external_browser=true เมื่อคุณเริ่มต้นระบบวิธีใช้ในโหมดสแตนด์อะโลน ค่าติดตั้งนี้ช่วยคุณเปิดดีฟอลต์ของบราวเซอร์ของระบบปฏิบัติการแทนบราวเซอร์ที่ฝังอยู่ในระบบวิธีใช้ คุณไม่จำเป็นต้องตั้งค่า always_external_browser=true สำหรับระบบปฏิบัติการที่ไม่ใช่ Windows
