鍵盤快速鍵

您可以使用按鍵或組合鍵取代滑鼠來執行作業。

下列表格列出了可在資訊中心使用的 Microsoft Internet Explorer 和 Mozilla Firefox Web 瀏覽器的鍵盤快速鍵。
註： 在 Internet Explorer 中，使用快速鍵可將焦點移至控制元素，並按 Enter 鍵加以啟動。 在 Firefox 中，使用快速鍵可移動焦點同時啟動控制元素。
表 1. 使用 Internet Explorer 導覽資訊中心時可用的鍵盤快速鍵
動作 Internet Explorer 快速鍵
從頁框（頁面）內移至下一個鏈結、按鈕或主題分支 按 Tab 鍵，然後按 Enter 鍵進入鏈結
移至內容視圖頁框 按 Alt+C，然後按 Enter 鍵和 Shift+F6
移至索引視圖頁框 按 Alt+I，然後按 Enter 鍵和 Shift+F6
移至搜尋結果視圖頁框 按 Alt+R，然後按 Enter 鍵和 Shift+F6
移至協同作業視圖頁框 按 Alt+V，然後按 Enter 鍵和 Shift+F6
移至主題視圖頁框 按 Alt+K，然後按 Enter 鍵
移至主題協同作業視圖頁框 按 Alt+O，然後按 Enter 鍵
展開或收合主題分支 右移鍵或左移鍵
移至下個主題分支 下移鍵或 Tab 鍵
移至上個主題分支 上移鍵或 Shift+Tab
捲動至頂端 Home 鍵
捲動至底端 End 鍵
下個頁框 F6 鍵
上個頁框 Shift+F6
列印目前頁面或作用中的頁框 Ctrl+P
表 2. 使用 Firefox 導覽資訊中心時可用的鍵盤快速鍵
動作 Firefox 快速鍵
從頁框（頁面）內移至下一個鏈結、按鈕或主題分支 按 Tab 鍵，然後按 Enter 鍵進入鏈結
移至內容視圖頁框 Shift+Alt+C 和 Shift+F6
移至索引視圖頁框 Shift+Alt+I 和 Shift+F6
移至搜尋結果視圖頁框 Shift+Alt+R 和 Shift+F6
移至協同作業視圖頁框 Shift+Alt+V 和 Shift+F6
移至主題視圖頁框 Firefox 中沒有直接的快速鍵可執行此動作。使用 Tab 鍵或 JAWS 中的導覽功能。
移至主題協同作業視圖頁框 Shift+Alt+O
展開或收合主題分支 右移鍵或左移鍵
移至下個主題分支 下移鍵或 Tab 鍵
移至上個主題分支 上移鍵或 Shift+Tab
捲動至頂端 Home 鍵
捲動至底端 End 鍵
下個頁框 F6 鍵
上個頁框 Shift+F6
