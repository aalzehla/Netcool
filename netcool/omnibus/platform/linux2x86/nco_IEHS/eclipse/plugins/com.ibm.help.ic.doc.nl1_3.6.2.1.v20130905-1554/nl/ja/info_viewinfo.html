インフォメーション・センターの情報の表示

IBM® Eclipse ヘルプ・システム (IEHS) がインフォメーション・センター・モードで稼働している場合、 IEHS に情報を表示することができます。

インフォメーション・センターによって、ユーザーはオンライン情報を表示、ブラウズおよび検索することができます。 インフォメーション・センターは、Eclipse プロジェクト (http://www.eclipse.org) で開発されたオープン・ソース・ソフトウェアをベースに構築されています。 これにより、Eclipse 文書プラグインとしてパッケージされた内容を表示することができます。

インフォメーション・センターは、 組み込みの Web アプリケーション・サーバーを使用してシステムのコンテンツ要求を処理します。 この組み込み Web サーバーが、ランダム・ポートを使用することによりアプリケーション間の競合が回避されます。

製品またはアプリケーション開発者は、 文書プラグインを作成してそれらをインフォメーション・センターに統合し、ユーザーに向けて表示することができます。 文書プラグインの作成およびパッケージ化について確認するには、 Eclipse プロジェクトの Web サイト (http://www.eclipse.org) を参照してください。

  • インフォメーション・センターのアクセシビリティーとキーボード・ショートカット
    アクセシビリティー機能は、運動障害または視覚障害など身体に障害を持つユーザーがソフトウェア製品を快適に使用できるようにサポートします。
  • ヘルプ・システムの前提条件
    IBM Eclipse ヘルプ・システムを設定する場合、 ワークステーション、オペレーション・システム、ブラウザー、およびハードウェアがこのトピックに記載された要件を満たしていることを確認する必要があります。
  • インフォメーション・センターのナビゲート
    インフォメーション・センターでは、グラフィカル・ユーザー・インターフェースで製品資料をブラウズすることができます。
  • インフォメーション・センター内の検索
    このトピックでは、IBM Eclipse ヘルプ・システムの検索方法を説明します。
  • 索引ビューを使用した情報の検索
    このトピックでは、ヘルプ・システムで索引ビューを使用してトピックを検索する方法を説明します。
  • インフォメーション・センターでのブックマークの設定
    ブックマークを設定することによって、トピックをより簡単に検索することができます。
  • さまざまな言語による情報の表示
    インフォメーション・センターのユーザー・インターフェースを別の言語で表示するには、 ご使用のオペレーティング・システムの地域設定を、 選択するロケールおよび言語に変更する必要があります。
  • インフォメーション・センターでの印刷
    IBM Eclipse ヘルプ・システムには、 インフォメーション・センター内のヘルプの内容を印刷するために、さまざまな方法が用意されています。
