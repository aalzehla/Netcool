Поиск информации с помощью панели индекса

Панель Индекс содержит главный индекс всего содержимого Information Center. Панель Индекс предназначена для поиска информации по конкретной тематике в Information Center.

Прежде чем начать

Если поставщик документации выключил панель индекса в Information Center, вкладка Индекс () будет недоступна.

Об этой задаче

Для поиска информации с помощью панели Индекс выполните следующие действия:

Процедура

  1. Выберите вкладку Индекс (), чтобы открыть панель Индекс.
  2. Введите ключевое слово в поле ввода. В списке терминов индекса будет показано ключевое слово, наиболее близкое введенному.
  3. Нажмите ключевое слово или заголовок раздела в списке индекса, чтобы увидеть раздел на панели Оглавление.
    Примечания:
    • Регистр символов в индексе не учитывается. На панели Индекс показываются одни и те же результаты вне зависимости от регистра символов введенного ключевого слова.
    • Если ключевое слово связано с несколькими разделами, в списке терминов индекса такое ключевое слово не показывается. Вместо него показываются заголовки страниц, содержащих это ключевое слово.
    • В дополнение поиску на панели Индекс под панелью есть ссылки Предыдущий и Следующий для прокрутки списка терминов индекса.
