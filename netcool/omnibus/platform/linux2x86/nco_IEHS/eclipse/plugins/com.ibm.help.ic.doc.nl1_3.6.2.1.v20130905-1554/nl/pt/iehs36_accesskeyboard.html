Atalhos de teclado e teclas de atalho

Pode utilizar teclas ou combinações de teclas para executar operações que também pode ser efectuadas utilizando um rato.

As seguintes tabelas apresentam os atalhos de teclado e as teclas de atalho que pode utilizar no Information Center, para os navegadores da Web Microsoft® Internet Explorer e Mozilla Firefox.
Nota: No Internet Explorer, utilize as teclas de atalho para mover o foco para o elemento de controlo e prima Enter para activar. No Firefox, utilize as teclas de atalho para mover o foco e activar o elemento de controlo simultaneamente.
Tabela 1. Atalhos de teclado e teclas de atalho para navegar no Information Center utilizando o Internet Explorer
Acção Atalho para o Internet Explorer
Avançar para a ligação, botão ou ramificação de tópicos seguinte a partir de uma moldura (página) Tab e, em seguida, premir Enter para abrir a ligação
Mover para a moldura da vista Índice Alt+C e, em seguida, premir Enter e Shift+F6
Mover para a moldura da vista Índice remissivo Alt+I e, em seguida, premir Enter e Shift+F6
Mover para a moldura da vista Resultados da procura Alt+R e, em seguida, premir Enter e Shift+F6
Mover para a moldura da vista Colaboração Alt+V e, em seguida, premir Enter e Shift+F6
Mover para a moldura da vista Tópico Alt+K e, em seguida, premir Enter
Mover para a moldura da vista Colaboração de tópicos Alt+O e, em seguida, premir Enter
Expandir ou contrair ramificação de tópicos Seta para a direita ou seta para a esquerda
Avançar para a ramificação de tópicos seguinte Seta para baixo ou Tab
Retroceder para a ramificação de tópicos anterior Seta para cima ou Shift+Tab
Deslocar o cursor para o início Home
Deslocar o cursor para o fim End
Moldura seguinte F6
Moldura anterior Shift+F6
Imprimir a página actual ou a moldura activa Ctrl+P

Mover para a moldura da vista Índice remissivo Shift+Alt+I e Shift+F6
Mover para a moldura da vista Resultados da procura Shift+Alt+R e Shift+F6
Mover para a moldura da vista Colaboração Shift+Alt+V e Shift+F6
Mover para a moldura da vista Tópico Não existe qualquer atalho directo para esta acção no Firefox. Utilize Tab ou a função de navegação no JAWS.
Mover para a moldura da vista Colaboração de tópicos Shift+Alt+O
Expandir ou contrair ramificação de tópicos Seta para a direita ou seta para a esquerda
Avançar para a ramificação de tópicos seguinte Seta para baixo ou Tab
Retroceder para a ramificação de tópicos anterior Seta para cima ou Shift+Tab

Deslocar o cursor para o fim End
