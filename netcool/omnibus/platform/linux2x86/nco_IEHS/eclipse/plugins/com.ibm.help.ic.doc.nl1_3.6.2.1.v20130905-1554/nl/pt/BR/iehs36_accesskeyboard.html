Atalhos e Aceleradores de Teclado

É possível utilizar teclas ou combinação de teclas para executar operações que também podem ser realizadas utilizando-se um mouse.

As tabelas a seguir listam os atalhos e aceleradores de teclado que podem ser utilizados no centro de informações para os navegadores da Web Microsoft Internet Explorer e Mozilla Firefox.
Nota: No Internet Explorer, utilize as teclas de atalho para mover o foco para o elemento de controle e pressione Enter para ativar. No Firefox, utilize as teclas de atalho para mover o foco e ativar o elemento de controle ao mesmo tempo.
Tabela 1. Os atalhos e aceleradores de teclado para navegação do centro de informações utilizando o Internet Explorer
Ação Atalho para Internet Explorer
Acesse o próximo link, botão ou ramificação de tópico a partir de dentro de um quadro (página) Utilize Tab para ir até o link e, em seguida, pressione Enter para segui-lo
Mover para o quadro Visualização Conteúdo Alt+C e, em seguida, pressione Enter e Shift+F6
Mover para o quadro Visualização Índice Alt+I e, em seguida, pressione Enter e Shift+F6
Mover para o quadro Visualização Resultados da Procura Alt+R e, em seguida, pressione Enter e Shift+F6
Mover para o quadro Visualização Colaboração Alt+V e, em seguida, pressione Enter e Shift+F6
Mover para o quadro Visualização Tópico Alt+K e, em seguida, pressione Enter
Mover para o quadro Visualização Colaboração de Tópico Alt+O e, em seguida, pressione Enter
Expandir ou reduzir uma ramificação de tópico Seta para Direita ou Seta para Esquerda
Mover para a próxima ramificação de tópico Seta para Baixo ou Tab
Mover para a ramificação de tópico anterior Seta para Cima ou Shift+Tab
Rolar para a parte superior Início
Rolar para a parte inferior Término
Próximo quadro F6
Quadro anterior Shift+F6
Imprimir a página atual ou quadro ativo Ctrl+P
Tabela 2. Atalhos e aceleradores de teclado para navegação no centro de informações utilizando o Firefox
Ação Atalho para Firefox
Acesse o próximo link, botão ou ramificação de tópico a partir de dentro de um quadro (página) Utilize Tab para ir até o link e, em seguida, pressione Enter para segui-lo
Mover para o quadro Visualização Conteúdo Shift+Alt+C e Shift+F6
Mover para o quadro Visualização Índice Shift+Alt+I e Shift+F6
Mover para o quadro Visualização Resultados da Procura Shift+Alt+R e Shift+F6
Mover para o quadro Visualização Colaboração Shift+Alt+V e Shift+F6
Mover para o quadro Visualização Tópico Não há atalho direto para esta ação no Firefox. Utilize Tab ou a função de navegação em JAWS.
Mover para o quadro Visualização Colaboração de Tópico Shift+Alt+O
Expandir ou reduzir uma ramificação de tópico Seta para Direita ou Seta para Esquerda
Mover para a próxima ramificação de tópico Seta para Baixo ou Tab
Mover para a ramificação de tópico anterior Seta para Cima ou Shift+Tab
Rolar para a parte superior Início
Rolar para a parte inferior Término
Próximo quadro F6
Quadro anterior Shift+F6
Imprimir a página atual ou quadro ativo Ctrl+P
