Προβολή πληροφοριών στο Κέντρο πληροφοριών

Μπορείτε να εμφανίσετε πληροφορίες στο IBM Eclipse Help System (IEHS), κατά την εκτέλεσή του σε κατάσταση κέντρου πληροφοριών.

Το κέντρο πληροφοριών επιτρέπει την προβολή, την εξερεύνηση και την αναζήτηση ηλεκτρονικών πληροφοριών. Το κέντρο πληροφοριών βασίζεται σε λογισμικό ανοιχτού κώδικα που αναπτύχθηκε στα πλαίσια του έργου Eclipse (http://www.eclipse.org). Επιτρέπει την παρουσίαση πληροφοριών που έχουν συσκευαστεί ως πρόσθετη λειτουργία τεκμηρίωσης (documentation plug-in) του Eclipse.

Το κέντρο πληροφοριών χρησιμοποιεί έναν ενσωματωμένο εξυπηρετητή διαδικτυακών εφαρμογών (web application server) που διεκπεραιώνει τις αιτήσεις για περιεχόμενο που υποβάλλονται στο σύστημα. Αυτός ο ενσωματωμένος εξυπηρετητής διαδικτύου χρησιμοποιεί μια τυχαία θύρα για την αποφυγή διενέξεων μεταξύ εφαρμογών για τη χρήση θυρών.

Οι προγραμματιστές προϊόντων ή εφαρμογών μπορούν να δημιουργήσουν τις δικές τους πρόσθετες λειτουργίες τεκμηρίωσης (documentation plug-ins), τις οποίες μπορούν να ενσωματώσουν στο κέντρο πληροφοριών ώστε να είναι δυνατή η προβολή τους από τους χρήστες αυτών των προϊόντων ή εφαρμογών. Για περισσότερες πληροφορίες σχετικά με τη δημιουργία και τη συσκευασία πρόσθετων λειτουργιών τεκμηρίωσης, ανατρέξτε στο δικτυακό τόπο του έργου Eclipse: http://www.eclipse.org

  • Λειτουργίες διευκόλυνσης πρόσβασης και συντομεύσεις πλήκτρων στο κέντρο πληροφοριών
    Οι λειτουργίες διευκόλυνσης πρόσβασης βοηθούν τους χρήστες περιορισμένης κινητικότητας ή όρασης στην αποτελεσματική χρήση προϊόντων λογισμικού.
  • Προαπαιτούμενος εξοπλισμός και λογισμικό για το σύστημα βοήθειας
    Πριν εγκαταστήσετε το IBM Eclipse Help System, βεβαιωθείτε ότι ο σταθμός εργασίας σας, το λειτουργικό σύστημα, το πρόγραμμα πλοήγησης και ο εξοπλισμός του συστήματός σας πληρούν τις προϋποθέσεις που περιγράφονται σε αυτό το θέμα.
  • Πλοήγηση στο κέντρο πληροφοριών
    Το κέντρο πληροφοριών παρέχει ένα γραφικό περιβάλλον χρήστη για την εξερεύνηση της τεκμηρίωσης ενός προϊόντος.
  • Αναζήτηση πληροφοριών στο κέντρο πληροφοριών
    Με αυτό το θέμα θα μάθετε πώς να κάνετε αναζήτηση στο IBM Eclipse Help System.
  • Εύρεση πληροφοριών από την προβολή Ευρετήριο
    Σε αυτό το θέμα περιγράφεται ο τρόπος χρήσης της προβολής Ευρετήριο για την εύρεση θεμάτων στο σύστημα βοήθειας.
  • Ορισμός σελιδοδεικτών στο κέντρο πληροφοριών
    Ορίζοντας σελιδοδείκτη για ένα θέμα διευκολύνεται η εύρεση του θέματος.
  • Προβολή πληροφοριών σε διαφορετικές γλώσσες
    Για να εμφανιστεί το περιβάλλον χρήστη του κέντρου πληροφοριών σε κάποια άλλη γλώσσα, θα πρέπει να αλλάξετε τις τοπικές ρυθμίσεις του λειτουργικού σας συστήματος καθορίζοντας τη γλώσσα και τις αντίστοιχες παραμέτρους της προτίμησής σας.
  • Εκτύπωση στο κέντρο πληροφοριών
    Το IBM Eclipse Help System σας παρέχει διάφορους εναλλακτικούς τρόπους για να εκτυπώσετε περιεχόμενα του κέντρου πληροφοριών.
