Schnellsuche durchführen

In diesem Thema wird die Vorgehensweise bei der Schnellsuche beschrieben.

Warum und wann dieser Vorgang ausgeführt wird

Führen Sie in der Navigationssicht die folgenden Schritte aus, um eine Schnellsuche durchzuführen:

  1. Klicken Sie in der Symbolleiste der Navigationssicht auf Themen durchsuchen(), um das Menü anzuzeigen.
    • Wenn Sie nur das Thema durchsuchen möchten, in dem Sie sich befinden, klicken Sie auf Thema durchsuchen.
    • Wenn Sie dieses Thema und alle seine Unterthemen durchsuchen möchten, klicken Sie auf Thema und alle Unterthemen durchsuchen.
      1. Geben Sie im Suchdialogfenster die Abfrage im Feld Suchen ein.
      2. Klicken Sie auf OK, um die Suche zu starten.
  2. Geben Sie im Popup-Suchdialogfenster die Abfrage im Feld Suchen ein und klicken Sie anschließend auf OK, um die Suche zu starten.
Ergebnisse

Wenn Sie nur das Thema durchsuchen, für das Sie das Schnellmenü öffnen, und Suchbegriffe übereinstimmen, wird das Thema automatisch angezeigt. Wurden in diesem Thema keine Übereinstimmungen gefunden, wird die Nachricht Nichts gefunden angezeigt.

Wenn Sie ein Thema und seine Unterthemen durchsuchen und Übereinstimmungen mit Ihren Suchbegriffen gefunden wurden, werden in den Suchergebnissen die ersten 500 Ergebnisse angezeigt. Sie können ein Suchergebnis auswählen, um das jeweilige Thema anzuzeigen. Die in Ihrer Abfrage verwendeten Wörter sind in dem angezeigten Thema hervorgehoben. Wurden in diesem Thema keine Übereinstimmungen gefunden, wird die Nachricht Nichts gefunden angezeigt.

Hinweise:
  • Das Dialogfenster Thema durchsuchen wird unabhängig von den Suchergebnissen automatisch geschlossen.
  • Das Dialogfenster Thema und alle Unterthemen durchsuchen wird erst geschlossen, wenn Sie auf Abbrechen klicken oder das Dialogfenster schließen.
