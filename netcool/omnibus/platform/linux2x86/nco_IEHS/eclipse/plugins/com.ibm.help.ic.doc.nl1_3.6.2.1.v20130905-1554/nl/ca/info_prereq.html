Requisits previs del sistema d'ajuda

Quan configureu el sistema d'ajuda d'IBM Eclipse, necessitareu comprovar que l'estació de treball, sistema operatiu i maquinari compleixen els requisits que es descriuen en aquesta secció.

Requisits del navegador

Per treballar amb el sistema d'ajuda d'IBM Eclipse, cal instal·lar la versió 6.0 o 7.0 de Microsoft Internet Explorer o la versió 2.0 o 3.0 de Firefox. Per tal d'assegurar-vos que totes les funcions d'aquesta versió del sistema d'ajuda d'IBM Eclipse resultin útils, heu d'habilitar les galetes i el JavaScript al navegador i inhabilitar la funció del navegador que bloqueja les finestres emergents.

Requisits de l'entorn del sistema operatiu

Per emprar el sistema d'ajuda d'IBM Eclipse, cal que instal·leu Java Runtime Environment (JRE), versió 6.0. Consulteu la taula següent per conèixer els requisits detallats.

Requisits de maquinari

z/OS

Per executar el sistema d'ajuda d'IBM Eclipse a z/OS, heu d'assegurar-vos que z/OS compleix els requisits de mida de la memòria i de mida de REGION.

  • Memòria real = <memòriabase> + <midaconnector> +<usuarisimultani> * <midamemòriaintermèdiaresposta>
    • memòriabase és la memòria necessària per iniciar el paquet del sistema d'ajuda d'IBM Eclipse. Són 90 MB a z/OS.
    • midaconnector és la mida del connector de documentació.
    • usuarisimultani és el nombre d'usuaris simultanis que accediran al sistema d'ajuda.
    • La midamemòriaintermèdiaresposta pot trobar-se a preferences.ini dins el directori eclipse\plugins\org.eclipse.help.base_3.0.1.
  • La mida de REGION s'ha d'establir com a mida de memòria real.

Altres sistemes operatius

La memòria mínima necessària és 128 MB de RAM. Es recomanen 256 MB o més.

Nota: Cal que incrementeu la memòria segons la mida dels connectors de la documentació. Per exemple, si la mida del connector de la documentació és 100 MB, afegiu-hi com a mínim 80 MB addicionals de RAM.

Personalització suportada

Podeu configurar el sistema d'ajuda editant el fitxer org.eclipse.help.base_<número versió>/preferences.ini que es troba al directori eclipse/plugins, tot i que es recomana que mantingueu la configuració per defecte.

No modifiqueu els paràmetres següents, o el sistema d'ajuda no funcionarà correctament:

always_external_browser
Als sistemes operatius de Microsoft Windows, la configuració per defecte d'aquest paràmetre és always_external_browser=true. Quan inicieu el sistema d'ajuda en la modalitat autònoma, aquest valor farà que s'obri el navegador per defecte del sistema operatiu en lloc del navegador incorporat al sistema d'ajuda. No cal que definiu always_external_browser=true en sistemes operatius que no siguin de Windows.
