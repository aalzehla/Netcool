Προαπαιτούμενος εξοπλισμός και λογισμικό για το σύστημα βοήθειας

Πριν εγκαταστήσετε το IBM Eclipse Help System, βεβαιωθείτε ότι ο σταθμός εργασίας σας, το λειτουργικό σύστημα, το πρόγραμμα πλοήγησης και ο εξοπλισμός του συστήματός σας πληρούν τις προϋποθέσεις που περιγράφονται σε αυτό το θέμα.

Απαιτήσεις για το πρόγραμμα πλοήγησης

Για να εργαστείτε με το IBM Eclipse Help System, θα πρέπει να εγκαταστήσετε το Microsoft Internet Explorer Έκδοση 6.0 ή 7.0, ή το Firefox 2.0 ή 3.0. Για να εξασφαλίσετε ότι είναι δυνατή η χρήση όλων των λειτουργιών της τρέχουσας έκδοσης του IBM Eclipse Help System, πρέπει να ενεργοποιήσετε τα ίχνη παρακολούθησης (cookies) και τη JavaScript στο πρόγραμμα πλοήγησης καθώς και να απενεργοποιήσετε τη λειτουργία φραγής αναδυόμενων παραθύρων του προγράμματος πλοήγησης.

Απαιτήσεις περιβάλλοντος λειτουργικού συστήματος

Για να χρησιμοποιήσετε το IBM Eclipse Help System, πρέπει να εγκαταστήσετε ένα περιβάλλον Java Runtime Environment (JRE) έκδοση 6.0. Ανατρέξτε στον παρακάτω πίνακα για τις λεπτομερείς απαιτήσεις.

Απαιτήσεις εξοπλισμού

z/OS

Για να εκτελέσετε το IBM Eclipse Help System σε z/OS, το z/OS πρέπει να ικανοποιεί τις απαιτήσεις μεγέθους πραγματικής μνήμης και του μεγέθους REGION.

  • Πραγματική μνήμη = <βασική_μνήμη> + <μέγεθος_πρόσθετης_λειτουργίας> +<ταυτόχρονοι_χρήστες> * <μέγεθος_ενδιάμεσης_μνήμης_αποκρίσεων>
    • βασική_μνήμη είναι η μνήμη που απαιτείται για την εκκίνηση του πακέτου του IBM Eclipse Help System. Στο z/OS είναι 90 ΜΒ.
    • μέγεθος_πρόσθετης_λειτουργίας είναι το μέγεθος της πρόσθετης λειτουργίας τεκμηρίωσης.
    • ταυτόχρονοι_χρήστες είναι ο αριθμό των ταυτόχρονων χρηστών που θα συνδεθούν στο σύστημα βοήθειας.
    • το μέγεθος_ενδιάμεσης_μνήμης_αποκρίσεων μπορείτε να το βρείτε στο αρχείο preferences.ini στον κατάλογο eclipse\plugins\org.eclipse.help.base_3.0.1.
  • Το μέγεθος REGION πρέπει να είναι ίδιο με το μέγεθος πραγματικής μνήμης.

Άλλα λειτουργικά συστήματα

Η ελάχιστη απαιτούμενη μνήμη είναι 128 MB RAM. Συνιστάται η ύπαρξη 256 MB ή περισσότερων.

Σημείωση: Πρέπει να αυξήσετε τη μνήμη ανάλογα με το μέγεθος των πρόσθετων λειτουργιών τεκμηρίωσης. Για παράδειγμα, εάν το μέγεθος της πρόσθετης λειτουργίας τεκμηρίωσης είναι 100 MB, προσθέστε τουλάχιστον 80 MB μνήμης RAM.

Υποστηριζόμενες προσαρμογές

Μπορείτε να ρυθμίσετε το σύστημα βοήθειας τροποποιώντας το αρχείο org.eclipse.help.base_<αριθμός_έκδοσης>/preferences.ini στον κατάλογο eclipse/plugins. Ωστόσο, συνιστάται να διατηρήσετε τις προεπιλεγμένες ρυθμίσεις.

Μην τροποποιήστε τις ακόλουθες παραμέτρους, διαφορετικά το σύστημα βοήθειας ίσως δεν λειτουργήσει σωστά:

always_external_browser
Σε λειτουργικά συστήματα Microsoft Windows, η προεπιλεγμένη ρύθμιση είναι η εξής: always_external_browser=true. Όταν εκτελέσετε το σύστημα βοήθειας σε κατάσταση ανεξάρτητης λειτουργίας, με αυτή τη θα ανοίξει το προεπιλεγμένο πρόγραμμα πλοήγησης του λειτουργικού συστήματος και όχι το ενσωματωμένο πρόγραμμα πλοήγησης του συστήματος βοήθειας. Δεν χρειάζεται να ορίσετε τη ρύθμιση always_external_browser=true για λειτουργικά συστήματα εκτός των Windows.
