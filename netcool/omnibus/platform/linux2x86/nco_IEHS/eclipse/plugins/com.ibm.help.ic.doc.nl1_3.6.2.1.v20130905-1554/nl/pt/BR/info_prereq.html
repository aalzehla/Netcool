Pré-requisitos para o sistema de ajuda

Quando você configura o IBM Eclipse Help System, é preciso assegurar que a estação de trabalho, o sistema operacional, o navegador e o hardware atendem os requisitos descritos neste tópico.

Requisitos de navegador

Para trabalhar com o IBM Eclipse Help System, você precisa instalar o Microsoft Internet Explorer, versão 6.0 ou 7.0, ou o Firefox 2.0 ou 3.0. Para assegurar que todas as funções desta versão do IBM Eclipse Help System são utilizáveis, você precisa habilitar os cookies e o JavaScript no navegador e desabilitar a função de bloqueio de pop-up do navegador.

Requisitos do ambiente do sistema operacional

Para trabalhar com IBM Eclipse Help System, você precisa instalar o Java Runtime Environment (JRE) versão 6.0. Consulte a seguinte tabela para requisitos detalhados.

Requisitos de hardware

z/SO

Para executar o IBM Eclipse Help System no z/OS, você precisa ter certeza de que o z/OS atende aos requisitos de tamanho real da memória e de tamanho da REGIÃO.

  • Memória real = <basememory> + <plugsize> +<concurrentuser> * <responseBufferSize>
    • basememory é a memória necessária para iniciar o pacote do IBM Eclipse System Help. Ela é de 90 MB no z/SO.
    • plugsize é o tamanho do seu plug-in de documentação.
    • concurrentuser é o número de usuários simultâneos que acessarão o Sistema de Ajuda.
    • responseBufferSize pode ser encontrado em preferences.ini no diretório eclipse\plugins\org.eclipse.help.base_3.0.1.
  • O tamanho da REGIÃO deve ser configurado igual ao tamanho da memória real.

Outros sistemas operacionais

A memória mínima necessária é de 128 MB RAM. Recomenda-se 256 MB ou mais.

Nota: Você precisa aumentar a memória de acordo com o tamanho dos plug-ins de documentação. Por exemplo, se o tamanho do plug-in de documentação para 100 MB, adicione pelo menos 80 MB de RAM.

Customização suportada

Você pode configurar o sistema de ajuda editando o arquivo org.eclipse.help.base_<número da versão>/preferences.ini no diretório eclipse/plugins, embora seja sugerido que você deva conservar a configuração padrão.

Não modifique os seguintes parâmetros, caso contrário o sistema de ajuda pode não funcionar corretamente:

always_external_browser
Para sistemas operacionais Microsoft Windows, a configuração padrão deste parâmetro é always_external_browser=true. Quando você inicia o sistema de ajuda no modo independente, essa configuração o ajuda a abrir o navegador padrão do sistema operacional ao invés do navegador integrado no sistema de ajuda. Não é necessário configurar always_external_browser=true para os sistemas operacionais diferentes do Windows.
