Εμφάνιση των αποτελεσμάτων αναζήτησης

Μπορείτε να εμφανίσετε τα αποτελέσματα αναζήτησης στο κέντρο πληροφοριών χρησιμοποιώντας διάφορα κριτήρια.

Εάν στα θέματα της τεκμηρίωσης έχουν οριστεί διάφορα κριτήρια, όπως αριθμός έκδοσης, πλατφόρμα ή κατηγορία, στα αποτελέσματα αναζήτησης μπορείτε να ομαδοποιήσετε τα θέματα.

  1. Πατήστε στην καρτέλα Αποτελέσματα αναζήτησης (). Προεπιλεγμένα, τα αποτελέσματα αναζήτησης δεν εμφανίζονται ομαδοποιημένα.
  2. Επιλέξτε Ομαδοποίηση αποτελεσμάτων αναζήτησης () στη γραμμή εργαλείων.
  3. Επιλέξτε ένα κριτήριο ομαδοποίησης από το μενού.
    • GroupByNone

      Αυτή είναι η προεπιλογή, με την οποία τα αποτελέσματα αναζήτησης δεν εμφανίζονται ομαδοποιημένα.

    • GroupByVersion
    • GroupByProdname
    • GroupByPlatform
    • Εμφάνιση κατηγοριών αποτελεσμάτων

      Ομαδοποιεί τα αποτελέσματα αναζήτησης σύμφωνα με τα ονόματά τους.

  4. Τα αποτελέσματα αναζήτησης θα εμφανιστούν σύμφωνα με το επιλεγμένο κριτήριο.
Σημείωση: Για να προσθέσετε ή να αφαιρέσετε τις περιγραφές των αποτελεσμάτων στην προβολή των αποτελεσμάτων αναζήτησης, επιλέξτε Εμφάνιση περιγραφών αποτελεσμάτων ().
