檢視資訊中心中的資訊

當 IBM® Eclipse Help System (IEHS) 以資訊中心模式執行時，便可檢視其中的資訊。

使用者可以利用資訊中心來檢視、瀏覽和搜尋線上資訊。資訊中心是建置在 Eclipse Project（http://www.eclipse.org）所開發的開放程式碼軟體之上。它可以顯示已套裝成 Eclipse 文件外掛程式的內容。

資訊中心搭配使用內嵌的 Web 應用程式伺服器來處理系統中的內容要求。這個內嵌的 Web 伺服器會使用隨機的埠，以避免各應用程式間的埠發生衝突。

產品或應用程式開發人員可以建立自己的文件外掛程式，並將它們整合在資訊中心內，供使用者檢視。要瞭解如何建立和包裝文件外掛程式， 請參閱 Eclipse Project 的網站：http://www.eclipse.org

  • 資訊中心的協助工具和鍵盤快速鍵
    協助工具特性，可協助行動不便或視障之類的殘疾使用者，順利使用軟體產品。
  • 說明系統的必備需求
    設定 IBM Eclipse Help System 時，請務必確定您的工作站、作業系統、瀏覽器及硬體皆符合本主題中所說明的需求。
  • 導覽資訊中心
    資訊中心提供一個用來瀏覽產品文件的圖形使用者介面。
  • 搜尋資訊中心
    在閱讀此主題後，您可以瞭解如何在 IBM Eclipse Help System 中進行搜尋。
  • 利用索引視圖尋找資訊
    本主題說明如何使用「索引」視圖來尋找說明系統中的主題。
  • 設定資訊中心的書籤
    您可以為主題設定書籤，便可更容易找到該主題。
  • 檢視不同語言的資訊
    如果要以另一種語言來檢視資訊中心使用者介面，您必須將作業系統的區域設定變更為所選擇的語言環境和語言。
  • 在資訊中心中列印
    IBM Eclipse Help System 提供不同的方法來列印資訊中心內的說明內容。
