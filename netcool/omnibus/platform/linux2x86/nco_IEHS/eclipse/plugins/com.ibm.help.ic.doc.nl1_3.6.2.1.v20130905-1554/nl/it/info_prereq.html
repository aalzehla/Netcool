Requisiti per il sistema della guida

Quando si imposta l'IBM Eclipse Help System, è necessario assicurarsi che la stazione di lavoro, il sistema operativo, il browser e l'hardware soddisfino i requisiti descritti in questo argomento.

Requisiti del browser

Per utilizzare IBM Eclipse Help System, è necessario installare Microsoft Internet Explorer Versione 6.0 o 7.0 oppure Firefox 2.0 o 3.0. Per assicurarsi che tutte le funzioni di questa versione dell'IBM Eclipse Help System siano utilizzabili, è necessario abilitare i cookie e il JavaScript nel browser e disabilitare la funzione del browser che blocca le finestre a comparsa (pop-up).

Requisiti hardware

z/OS

Per eseguire l'IBM Eclipse Help System su z/OS, è necessario assicurarsi che z/OS soddisfi i requisiti di dimensione della memoria fisica e di dimensione della REGIONE.

  • Memoria fisica = <basememory> + <plugsize> +<concurrentuser> * <responseBufferSize>
    • basememory è la memoria richiesta per avviare il pacchetto IBM Eclipse Help System. E' di 90 MB su z/OS.
    • plugsize è la dimensione del plugin della documentazione.
    • concurrentuser è il numero di utenti che accedono simultaneamente al Sistema della guida.
    • responseBufferSize si trova in preferences.ini nella directory eclipse\plugins\org.eclipse.help.base_3.0.1.
  • La dimensione della REGIONE deve avere lo stesso valore della dimensione della memoria fisica.

Altri sistemi operativi

E' richiesta almeno una dimensione della memoria pari a 128 MB. Si consiglia comunque di utilizzare un modulo di memoria da 256 MB o superiore.

Nota: E' necessario aumentare la memoria in base alla dimensione dei plugin della documentazione. Ad esempio, se la dimensione del plugin della documentazione è di 100 MB, si consiglia di aggiungere almeno 80 MB di memoria aggiuntiva.

Personalizzazione supportata

E' possibile configurare il sistema della guida modificando il file org.eclipse.help.base_<version number>/preferences.ini sotto la directory eclipse/plugins, anche se si consiglia di conservare la configurazione predefinita.

Non modificare i seguenti parametri; altrimenti è possibile che il sistema della guida non funzioni correttamente:

always_external_browser
Per i sistemi operativi Microsoft Windows, la configurazione predefinita di questo parametro è always_external_browser=true. Quando si avvia il sistema della guida in modalità stand-alone, questa impostazione aiuta l'utente ad aprire il browser predefinito del sistema operativo piuttosto che il browser incorporato nel sistema della guida. Non è necessario impostare always_external_browser=true per i sistemi operativi diversi da Windows.
