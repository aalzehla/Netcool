Yardım sistemine ilişkin önkoşul gereksinimleri

IBM Eclipse Help System uygulamasını kurarken, iş istasyonunuzun, işletim sisteminizin, tarayıcınızın ve donanımınızın bu konuda açıklanan gereksinimleri karşıladığından emin olmanız gerekir.

Tarayıcı gereksinimleri

IBM Eclipse Help System ile çalışmak için Microsoft Internet Explorer Sürüm 6.0 ya da 7.0 ya da Firefox 2.0 ya da 3.0 düzeyini kurmanız gerekir. IBM Eclipse Help System olanağının bu sürümünün tüm işlevlerini sağlamak için tarayıcınızda tanımlama bilgilerini ve JavaScript'i etkinleştirmeli ve tarayıcının açılan pencereleri engelleme işlevini devre dışı bırakmalısınız.

İşletim sistemi ortamı gereksinimleri

IBM Eclipse Help System ile çalışmak için Java Çalıştırma Ortamı (JRE) sürüm 6.0 düzeyini kurmanız gerekir. Ayrıntılı gereksinimler için aşağıdaki tabloya bakın.

Donanım gereksinimleri

z/OS

IBM Eclipse Help System olanağını z/OS işletim sisteminde çalıştırmak için z/OS'nin gerçek bellek boyutu ve REGION boyutu gereksinimlerini karşıladığından emin olmanız gerekir.

  • Gerçek bellek = <temelbellek> + <eklentiboyutu> +<eşzamanlıkullanıcı> * <yanıtAraBellekBoyutu>
    • temelbellek, IBM Eclipse Help System paketini başlatmak için gereken bellek miktarıdır. z/OS işletim sisteminde bu değer 90 MB'dir.
    • eklentiboyutu, belge eklentinizin boyutudur.
    • eşzamanlıkullanıcı, Yardım Sisteminize aynı anda erişecek kullanıcıların sayısıdır.
    • yanıtAraBellekBoyutu, eclipse\plugins\org.eclipse.help.base_3.0.1 dizinindeki preferences.ini dosyasında bulunabilir.
  • REGION boyutu, gerçek bellek boyutuyla aynı değere ayarlanmalıdır.

Diğer işletim sistemleri

En az 128 MB RAM gerekir. 256 MB ya da üstü önerilir.

Not: Belleği, belge eklentisinin boyutuna göre artırmanız gerekir. Örneğin, belge eklentisinin boyutu 100 MB ise, en az 80 MB RAM ekleyin.

Desteklenen özelleştirme

Yardım sistemini eclipse/plugins dizinindeki org.eclipse.help.base_<sürüm numarası>/preferences.ini dosyasını düzenleyerek yapılandırabilirsiniz; ancak, varsayılan yapılandırmayı tutmanız önerilir.

Aşağıdaki parametreleri değiştirmeyin; aksi takdirde, yardım sistemi düzgün çalışmayabilir:

always_external_browser
Microsoft Windows işletim sistemlerinde, bu parametrenin varsayılan yapılandırması always_external_browser=true değerindedir. Yardım sistemini bağımsız modda başlattığınızda, bu ayar, yardım sistemindeki yerleşik tarayıcı yerine işletim sisteminin varsayılan tarayıcısını açmanıza yardımcı olur. Windows dışındaki işletim sistemlerinde always_external_browser=true parametresini ayarlamanıza gerek yoktur.
