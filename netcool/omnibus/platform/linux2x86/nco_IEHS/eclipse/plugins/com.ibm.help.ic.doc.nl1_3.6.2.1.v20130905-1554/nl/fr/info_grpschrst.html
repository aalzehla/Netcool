Afficher les résultats d'une recherche

Vous pouvez afficher vos résultats de recherche dans le Centre de documentation en fonction de critères différents.

Pourquoi et quand exécuter cette tâche

Si les rubriques de la documentation ont été marquées avec des critères, vous pouvez afficher les résultats de la recherche en fonction de jeux de critères (catégorie, version ou nom de produit).

  1. Cliquez sur l'onglet Résultats de la recherche (). Par défaut, les résultats de la recherche seront regroupés par catégories.
  2. Dans la barre d'outils, cliquez sur Regrouper les résultats de la recherche ().
  3. Modifiez l'affichage par défaut des résultats (Afficher les catégories de résultats et choisissez un critère de regroupement : GroupByNone, GroupByVersion, GroupByProductName ou GroupByPlatform.
  4. Les résultats s'afficheront selon les critères que vous aurez retenus.
Remarque : Si vous cliquez sur le type de groupe GroupByNone, les résultats de la recherche seront présentés non regroupés.

Pour ajouter ou supprimer les descriptions des résultats dans la vue des résultats, cliquez sur Afficher les descriptions des résultats ().
