Синхронизация оглавления

В этом разделе приведена информация о синхронизации панели навигации и панели содержимого.

При открытии нового раздела по ссылке, приведенной в другом разделе, панель навигации не обновляется соответствующим образом.

Для того чтобы найти текущий раздел на панели навигации, нажмите кнопку Показать в оглавлении () на панели инструментов панели содержимого.

Для того чтобы панель навигации всегда была синхронизирована с панелью содержимого, нажмите кнопку Связать с оглавлением () на панели инструментов панели содержимого. Для выключения этой функции, нажмите ту же кнопку еще раз.

Прим.: Если на панели навигации применен фильтр, то попытка синхронизировать панель навигации для раздела, не отвечающего условиям фильтра, будет неудачной, так как соответствующий раздел исключен фильтром с панели навигации. Заголовок текущего раздела не будет выделен на панели навигации.
