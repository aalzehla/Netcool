設定資訊中心的書籤

您可以為主題設定書籤，便可更容易找到該主題。 您可以使用 Web 瀏覽器中的特性或使用「書籤」視圖設定書籤。

上層主題： 檢視資訊中心中的資訊

使用 Web 瀏覽器設定書籤

對於部署於伺服器並透過內部網路或網際網路檢視的資訊中心，您可以將主題的書籤儲存至 Web 瀏覽器的「書籤」或「我的最愛」功能表。

程序

如果要在 Web 瀏覽器中為目前的主題設定書籤：
  • 如果您使用 Internet Explorer，請按一下書籤 ()，將主題儲存在您的「我的最愛」資料夾。
  • 如果您使用的不是 Internet Explorer，請用滑鼠右鍵按一下目前頁面上的任何空白區域，以開啟功能表。選擇加到我的最愛或加到書籤選項，將主題儲存到其中一個「我的最愛」資料夾中。
如果要檢視您的書籤，請開啟 Web 瀏覽器的我的最愛功能表。

如果要刪除書籤，請開啟 Web 瀏覽器的我的最愛功能表以找出書籤。用滑鼠右鍵按一下書籤，然後在功能表中按刪除。
