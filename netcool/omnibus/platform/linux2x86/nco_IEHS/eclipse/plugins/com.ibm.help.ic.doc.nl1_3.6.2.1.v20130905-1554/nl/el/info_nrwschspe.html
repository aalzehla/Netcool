Περιορισμός εμβέλειας αναζήτησης

Μπορείτε να περιορίσετε την εμβέλεια αναζήτηση σε συγκεκριμένα θέματα.

Προεπιλεγμένα, κατά την αναζήτηση στο σύστημα βοήθειας, γίνεται αναζήτηση σε όλα τα θέματα που εμφανίζονται στην προβολή Πλοήγηση. Ωστόσο, μπορείτε να περιορίσετε την εμβέλεια αναζήτησης σε ένα συγκεκριμένο σύνολο θεμάτων, δημιουργώντας μια λίστα αναζήτησης και αποθηκεύοντάς την για μελλοντική χρήση.

Μπορείτε να δημιουργήσετε μια λίστα αναζήτησης χρησιμοποιώντας την ακόλουθη διαδικασία:

  1. Πατήστε στη διασύνδεση Εμβέλεια αναζήτησης δίπλα στο πεδίο Αναζήτηση. Εμφανίζεται το παράθυρο Επιλογή εμβέλειας αναζήτησης.
  2. Επιλέξτε Αναζήτηση μόνο στα ακόλουθα θέματα και πατήστε Δημιουργία. Εμφανίζεται το παράθυρο Νέα λίστα αναζήτησης.
  3. Στο πεδίο Όνομα λίστας, καθορίστε ένα όνομα για τη νέα λίστα αναζήτησης.
  4. Στη λίστα Θέματα για αναζήτηση, επιλέξτε τους κλάδους της προβολής Πλοήγηση που θέλετε να συμπεριλάβετε στην αναζήτηση. Μπορείτε να αναπτύξετε τους κλάδους για να επιλέξετε υποκατηγορίες θεμάτων.
  5. Πατήστε OK. Η νέα λίστα αναζήτησης εμφανίζεται τώρα στη λίστα Αναζήτηση μόνο στα ακόλουθα θέματα. Αυτή είναι η προεπιλογή. Μπορείτε να πατήσετε OK για να χρησιμοποιηθεί αυτή η λίστα αναζήτησης για τις επόμενες αναζητήσεις σας.
    Περιορισμός: Σε ορισμένα προγράμματα πλοήγησης (όπως το Konqueror και το Safari) η δυνατότητα αποθήκευσης λιστών αναζήτησης δεν υποστηρίζεται.

Η ενεργή λίστα αναζήτησης εμφανίζεται πάντα στο επάνω μέρος του παραθύρου του προγράμματος πλοήγησης, δίπλα στη διασύνδεση Εμβέλεια αναζήτησης.

Η τελευταία λίστα αναζήτησης που δημιουργήσατε παραμένει επιλεγμένη κάθε φορά που θα εκκινήσετε το σύστημα.
