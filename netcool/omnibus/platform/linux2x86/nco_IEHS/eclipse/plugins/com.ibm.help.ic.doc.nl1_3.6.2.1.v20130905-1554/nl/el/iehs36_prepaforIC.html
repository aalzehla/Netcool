Προετοιμασία για τη χρήση του κέντρου πληροφοριών

Εκτελέστε τα παρακάτω βήματα για να εξασφαλίσετε τα καλύτερα δυνατά αποτελέσματα από τη χρήση του κέντρου πληροφοριών.

  • Βεβαιωθείτε ότι υποστηρίζεται το πρόγραμμα πλοήγησης (browser) που χρησιμοποιείτε.

    Για να χρησιμοποιήσετε αυτό το κέντρο πληροφοριών, θα πρέπει να έχει εγκατασταθεί ένα υποστηριζόμενο πρόγραμμα πλοήγησης στο σύστημά σας. Μπορείτε να βρείτε μια λίστα των υποστηριζόμενων προγραμμάτων πλοήγησης για το κέντρο πληροφοριών στη διεύθυνση Υποστηριζόμενα προγράμματα πλοήγησης για το κέντρο πληροφοριών

  • Καθορίστε την προτιμώμενη γλώσσα σας στις ρυθμίσεις του προγράμματος πλοήγησης.

    Μπορείτε να χρησιμοποιήσετε την επιλογή γλώσσας του προγράμματος πλοήγησης για να ορίσετε την παρουσίαση του περιεχομένου του κέντρου πληροφοριών στη γλώσσα της προτίμησής σας. Βλ. Παρουσίαση πληροφοριών στη γλώσσα της προτίμησής σας για περισσότερες πληροφορίες.

  • Υποστηριζόμενα προγράμματα πλοήγησης για το κέντρο πληροφοριών
    Για να χρησιμοποιήσετε αυτό το κέντρο πληροφοριών, θα πρέπει να έχει εγκατασταθεί ένα υποστηριζόμενο πρόγραμμα πλοήγησης στο σύστημά σας.
  • Παρουσίαση πληροφοριών στη γλώσσα της προτίμησής σας
    Μπορείτε να χρησιμοποιήσετε την επιλογή γλώσσας του προγράμματος πλοήγησης για να ορίσετε την παρουσίαση του περιεχομένου του κέντρου πληροφοριών στη γλώσσα της προτίμησής σας.
