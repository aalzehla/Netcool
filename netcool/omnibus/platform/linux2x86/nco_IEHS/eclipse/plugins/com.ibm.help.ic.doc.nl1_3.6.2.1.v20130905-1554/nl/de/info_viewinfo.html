Informationen im Information Center anzeigen

Sie können Informationen in IBM Eclipse Help System (IEHS) anzeigen, wenn es im Information Center-Modus ausgeführt wird.

Warum und wann dieser Vorgang ausgeführt wird
Mit dem Information Center können Benutzer Onlineinformationen anzeigen, darin blättern und diese durchsuchen. Das Information Center basiert auf Open-Source-Software, die vom Eclipse-Projekt (http://www.eclipse.org) entwickelt wurde. Damit können Inhalte angezeigt werden, die als Eclipse-Dokumentations-Plug-in gepackt wurden.

Das Information Center verwendet einen eingebetteten Webanwendungsserver zum Bearbeiten der Inhaltsanforderungen im System. Dieser eingebettete Web-Server verwendet einen wahlfreien Port, um Portkonflikte zwischen Anwendungen zu vermeiden.

Produkt- bzw. Anwendungsentwickler können eigene Dokumentations-Plug-ins erstellen und zum Anzeigen für die Benutzer in das Information Center integrieren. Rufen Sie die Website des Eclipse-Projekts auf, um Informationen über das Erstellen und Packen von Dokumentations-Plug-ins zu erhalten: http://www.eclipse.org.

  • Eingabehilfen und Direktaufrufe über die Tastatur im Information Center
    Die Eingabehilfefunktion unterstützt Benutzer mit körperlichen Behinderungen wie z. B. eingeschränkter Beweglichkeit oder eingeschränktem Sehvermögen beim erfolgreichen Einsatz von Softwareprodukten.
  • Vorausgesetzte Anforderungen für die Hilfefunktion
    Beim Einrichten von IBM Eclipse Help System müssen Sie sicherstellen, dass Ihre Workstation, Ihr Betriebssystem, Ihr Browser sowie Ihre Hardware die in diesem Thema beschriebenen Anforderungen erfüllen.
  • Im Information Center navigieren
    Das Information Center stellt eine grafische Benutzerschnittstelle zum Durchsuchen der Produktdokumentation zur Verfügung.
  • Information Center durchsuchen
    In diesem Thema wird die Vorgehensweise beim Suchen in IBM Eclipse Help System beschrieben.
  • Informationen mit der Indexsicht suchen
    In diesem Thema wird das Arbeiten mit der Indexsicht zum Suchen nach Themen in der Hilfefunktion beschrieben.
  • Lesezeichen im Information Center setzen
    Sie können ein Thema schneller finden, wenn Sie ein Lesezeichen setzen.
  • Informationen in unterschiedlichen Sprachen anzeigen
    Wenn Sie die Benutzerschnittstelle des Information Centers in einer anderen Sprache anzeigen möchten, müssen Sie die Ländereinstellungen Ihres Betriebssystems in die gewünschte Ländereinstellung und Sprache ändern.
  • Im Information Center drucken
    IBM Eclipse Help System stellt Ihnen unterschiedliche Möglichkeiten zum Drucken des Inhalts der Hilfe im Information Center bereit.
