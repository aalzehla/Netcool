準備使用資訊中心

請執行下列步驟，以確保最佳的資訊中心使用結果：

  • 確認您使用的瀏覽器是受支援的瀏覽器。

    如果要使用這個資訊中心，您必須在系統上安裝支援的 Web 瀏覽器。您可以在這裡找到資訊中心的支援瀏覽器：資訊中心支援的瀏覽器

  • 將 Web 瀏覽器設為您偏好的語言。

    您可以設定 Web 瀏覽器的語言選項，以您偏好的語言顯示資訊中心內容。如需相關資訊，請參閱以偏好的語言顯示資訊。
