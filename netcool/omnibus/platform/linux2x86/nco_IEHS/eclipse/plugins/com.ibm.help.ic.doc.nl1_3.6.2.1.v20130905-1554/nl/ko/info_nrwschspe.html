검색 범위 좁히기

검색 범위를 특정 주제로 좁힐 수 있습니다.

기본적으로 도움말 시스템에서 검색을 수행하면 탐색 보기에 표시된 모든 주제를 검색합니다. 그러나, 검색 목록을 작성하여 검색 범위를 특정 주제 집합으로 좁히고 나중에 사용할 수 있도록 검색 목록을 저장할 수 있습니다.

검색 목록을 작성하려면 다음 단계를 수행하십시오.

  1. 검색 필드 옆의 검색 범위를 클릭하십시오. 검색 범위 선택 창이 표시됩니다.
  2. 다음 주제만 검색을 선택하고 새로 작성을 클릭하십시오. 새 검색 목록 창이 표시됩니다.
  3. 목록 이름 필드에 새 검색 목록 이름을 입력하십시오.
  4. 검색할 주제 목록에서 검색에 포함시킬 탐색 보기의 분기를 선택하십시오. 분기를 펼치면 표제의 두 번째 레벨에서 하위 범주를 선택할 수 있습니다.
  5. 확인을 클릭하십시오. 이제 새 검색 목록이 다음 주제만 검색 목록에 표시됩니다. 기본적으로 이 옵션은 자동으로 선택됩니다. 이 검색 목록을 다음 검색에 적용하려면 확인을 클릭합니다.
    제한사항: 일부 브라우저(예: Konqueror 및 Safari)에서는 검색 목록을 저장하는 기능이 지원되지 않습니다.

현재 사용 중인 검색 목록은 브라우저의 맨 위, 검색 범위 링크 옆에 항상 표시됩니다.

마지막으로 작성한 검색 목록이 시스템을 사용할 때마다 유지됩니다.
