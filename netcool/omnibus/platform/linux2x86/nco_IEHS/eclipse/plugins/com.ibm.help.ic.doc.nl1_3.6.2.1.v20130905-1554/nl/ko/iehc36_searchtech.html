고급 검색 기술

따옴표, 와일드카드 또는 부울 연산자로 고급 검색을 수행하여 더 작은 검색 결과 세트를 리턴할 수 있습니다.

  • Information Center에서 검색을 수행할 때 해당 항목을 큰따옴표로 묶으면 입력한 단어 또는 구만을 검색할 수 있습니다.

    예를 들어 검색 필드에 "로그 파일"을 입력하면 문자열 로그 파일의 모든 항목을 검색합니다. 큰따옴표를 입력하지 않으면 단어 로그 및 단어 파일 모두를 포함하는 주제가 리턴됩니다.

  • 검색에서 별표(*)와 물음표(?) 같은 와일드카드를 사용하여 검색 범위를 확장할 수 있습니다.
    • 검색 조회에서 알 수 없거나 변형되는 여러 문자에 대해서는 별표(*)를 사용합니다. 예를 들어 par*을 입력한 경우 partly, participate, partial 및 par로 시작하는 다른 단어를 포함하는 주제가 리턴됩니다.
      주: 검색 문자열에 대한 제한 사항이 한 가지 있습니다. 별표(*)를 포함한 검색 용어를 사용하는 경우 별표 앞에는 최소 3자 이상이 있어야 합니다. 또한 와일드카드가 없는 검색 용어에는 최소 3자 이상이 있어야 합니다.
    • 검색 조회에서 단일 문자에 대해서는 물음표를 사용합니다. 예를 들어 par?을 입력한 경우 partial 또는 partly가 아닌 part를 포함하는 주제가 리턴됩니다.
  • 부울 연산자 AND, OR 및 NOT를 사용하여 조회를 더 구체적으로 설정할 수 있습니다. 예를 들어, 다음과 같습니다.
    데이터베이스 AND "로그 파일"
    데이터베이스 단어와 로그 파일 구를 모두 포함한 주제로 검색 범위를 좁힙니다.
    데이터베이스 OR "로그 파일"
    데이터베이스 단어 또는 로그 파일 구 중 하나를 포함한 주제로 검색 범위를 확장합니다.
    database NOT "log file"
    데이터베이스 단어는 포함하고 로그 파일 구는 포함하지 않는 주제로 검색 범위를 좁힙니다.
    데이터베이스 OR "데이터 베이스" NOT "로그 파일"
    데이터베이스 단어 또는 데이터 베이스 구를 포함하지만 로그 파일 구는 포함하지 않는 주제를 검색합니다.
