Themen im Information Center drucken

Sie können das Thema drucken, das Sie derzeit anzeigen; es ist aber auch möglich, ein oder mehrere Themen in der Inhaltssicht zu drucken.

Vorgehensweise

  • Gehen Sie wie folgt vor, um das aktuell angezeigte Thema zu drucken:
    1. Klicken Sie in der Symbolleiste auf Seite drucken (). Das Fenster Print des Browsers wird angezeigt.
    2. Starten Sie den Druckvorgang, indem Sie im Fenster Print auf Print bzw. auf OK klicken.
  • Gehen Sie wie folgt vor, um ein Thema oder mehrere Themen in der Inhaltssicht zu drucken:
    1. Wählen Sie das Thema in der Navigation aus.
    2. Klicken Sie in der Symbolleiste auf Themen drucken ().
      • Wenn Sie lediglich das ausgewählte Thema drucken möchten, klicken Sie auf Ausgewähltes Thema drucken.
      • Wenn Sie das ausgewählte Thema mit allen seinen Unterthemen drucken möchten, klicken Sie auf Ausgewähltes Thema und alle Unterthemen drucken; anschließend müssen Sie festlegen, ob der Seitenumbruch zwischen jedem Thema in der Druckausgabe auftrten soll.
      Das Fenster Print des Browsers wird angezeigt.
    3. Starten Sie den Druckvorgang, indem Sie im Fenster 'Print' auf Print bzw. auf OK klicken.
