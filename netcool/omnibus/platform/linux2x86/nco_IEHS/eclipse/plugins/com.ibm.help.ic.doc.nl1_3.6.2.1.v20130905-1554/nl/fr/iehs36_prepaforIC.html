Préparation à l'utilisation du centre de documentation

Les étapes suivantes permettent de tirer profit au mieux du centre de documentation.

  • Assurez-vous que votre navigateur is supporté.

    Pour pouvoir utiliser le centre de documentation, votre système doit disposer d'un navigateur web supporté. Les navigateurs supportés sont indiqués à la rubrique Navigateurs supportés pour le centre de documentation.

  • Configurez votre navigateur web sur la langue de votre choix.

    Vous pouvez configurer la langue de votre navigateur web afin que les contenus du centre de documentation s'affichent dans la langue de votre choix. Pour plus d'informations, voir Affichage des informations dans la langue de votre choix.
