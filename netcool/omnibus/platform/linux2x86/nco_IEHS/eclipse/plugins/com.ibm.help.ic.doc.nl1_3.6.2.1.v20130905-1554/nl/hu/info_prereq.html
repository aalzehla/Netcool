Súgórendszer használatához szükséges előfeltételek

Amikor beállítja az IBM Eclipse Help System rendszert, akkor győződjön meg róla, hogy a munkaállomás, az operációs rendszer, a böngésző és a hardver megfelel az ebben a témakörben ismertetett követelményeknek.

Böngészőre vonatkozó követelmények

Az IBM Eclipse Help System kezeléséhez telepítse a Microsoft Internet Explorer 6.0 vagy 7.0 változatát, illetve a Firefox 2.0 vagy 3.0 változatát. Ahhoz, hogy az IBM Eclipse Help System ezen változatának minden funkciója használható legyen, engedélyezze a cookie-kat és a JavaScript-et a böngészőben, és tiltsa le az előugró ablakok blokkolása böngészőfunkciót.

Hardverre vonatkozó követelmények

z/OS

Az IBM Eclipse Help System futtatásához z/OS rendszeren gondoskodjék róla, hogy a z/OS kielégítse a valós memória méretére és a REGION méretre vonatkozó követelményeket.

  • Valós memória = <alapmemória> + <bedolgozóméret> +<párhuzamosfelhasználó> * <válaszpufferméret>
    • Az alapmemória az IBM Eclipse Help System csomag indításához szükséges memória. z/OS rendszeren ez 90 MB.
    • A bedolgozóméret a dokumentáció bedolgozó mérete.
    • A párhuzamosfelhasználó a súgórendszerhez hozzáférő párhuzamos felhasználók száma.
    • A válaszpufferméret a preferences.ini fájlban található az eclipse\plugins\org.eclipse.help.base_3.0.1 könyvtárban.
  • A REGION méretet ugyanannyira kell állítani, mint a valós memória mérete.

Egyéb operációs rendszerek

A minimálisan szükséges memória 128 MB RAM. 256 MB vagy több javasolt.

Megjegyzés: A dokumentáció bedolgozók méretének megfelelően növelni kell a memóriát. Ha például a dokumentáció bedolgozó mérete 100 MB, akkor vegyen fel további legalább 80 MB RAM memóriát.

Támogatott személyre szabás

A súgórendszer konfigurálható az eclipse/plugins könyvtárban található org.eclipse.help.base_<verziószám>/preferences.ini fájl szerkesztésével, habár ajánlott az alapértelmezett konfiguráció megtartása.

Az alábbi paramétereket ne módosítsa, különben elképzelhető, hogy a súgórendszer nem fog megfelelően működni:

always_external_browser
Microsoft Windows operációs rendszerek esetén a paraméter alapértelmezett beállítása always_external_browser=true. Amikor elindítja a súgórendszert az önálló módban, akkor a beállítás miatt az operációs rendszer alapértelmezett böngészőjét fogja megnyitni a súgórendszerbe beágyazott böngésző helyett. Az always_external_browser=true paramétert nem kell beállítani a Windows rendszertől eltérő operációs rendszerek esetén.
