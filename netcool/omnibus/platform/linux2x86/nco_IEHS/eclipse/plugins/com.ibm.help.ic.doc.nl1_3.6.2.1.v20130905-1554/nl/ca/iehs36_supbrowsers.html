Navegadors suportats per a l'Information Center

Per utilitzar aquest Information Center, cal que un navegador web suportat estigui instal·lat al vostre sistema.

Els següents navegadors web estan suportats:

  • Microsoft® Internet Explorer Versió 7.0 o Versió 8.0
  • Mozilla Firefox Versió 3.5 o Versió 3.6.

Si utilitzeu un versió diferent dels navegadors web suportats o un altre navegador web, és possible que algunes funcions no estiguin disponibles o no funcionin com és d'esperar.

Per assegurar-vos que totes les funcions de l'Information Center es poden fer servir, heu d'habilitar les galetes i el JavaScript™ al navegador web i heu de permetre que el navegador web mostri finestres emergents d'aquest Information Center.
