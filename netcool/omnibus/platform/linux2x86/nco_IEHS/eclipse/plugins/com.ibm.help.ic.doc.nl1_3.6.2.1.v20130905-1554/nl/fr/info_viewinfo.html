Consulter des informations dans le Centre de documentation

Vous pouvez consulter des informations dans IBM Eclipse Help System (IEHS) lorsque ce dernier s'exécute en mode Centre de documentation.

Pourquoi et quand exécuter cette tâche
Le Centre de documentation permet d'afficher et de parcourir la documentation et de lancer des recherches, le tout en ligne. Il s'appuie sur des logiciels Open Source développés dans le cadre du projet Eclipse (http://www.eclipse.org). Il permet d'afficher les données intégrées sous forme de plug-in à la documentation Eclipse.

Le Centre de documentation utilise un serveur d'applications Web intégré pour traiter au sein du système les demandes de contenu. Ce serveur Web fait appel à un port sélectionné de manière aléatoire pour éviter les conflits entre les ports utilisés par les applications.

Les développeurs de produits ou d'applications peuvent créer leurs propres plug-in de documentation et les intégrer au Centre de documentation pour qu'ils puissent être visualisés par les utilisateurs. Pour apprendre à créer et packager des plug-in de documentation, voir le site Web du projet Eclipse : http://www.eclipse.org

  • Accessibilité et raccourcis-clavier du Centre de documentation
    Les fonctions d'accessibilité permettent aux personnes présentant un handicap (mobilité ou vision limitée) d'utiliser des logiciels sans difficulté.
  • Prérequis pour le Centre de documentation
    Lorsque vous configurez le système d'aide IBM Eclipse, vous devez vous assurer que votre poste de travail, votre système d'exploitation, votre navigateur et votre matériel correspondent bien aux conditions décrites dans la présente rubrique.
  • Navigation dans le Centre de documentation
    Le Centre de documentation est doté d'une interface graphique permettant à l'utilisateur de parcourir la documentation des produits.
  • Recherche dans le Centre de documentation
    Cette rubrique vous apprendra à effectuer des recherches dans IBM Eclipse Help System.
  • Rechercher des informations à l'aide de la vue Index
    Comment utiliser la vue Index pour rechercher des rubriques du système d'aide.
  • Définir des signets dans le Centre de documentation
    Pour retrouver plus facilement une rubrique, vous pouvez la marquer par un signet.
  • Affichage des informations dans différentes langues
    Pour afficher dans une autre langue l'interface utilisateur du Centre de documentation, vous devez modifier les paramètres régionaux du système d'exploitation en indiquant l'environnement local et la langue souhaitée.
  • Impression dans le Centre de documentation
    IBM Eclipse Help System offre plusieurs manières différentes d'imprimer le contenu de l'aide à partir du Centre de documentation.
