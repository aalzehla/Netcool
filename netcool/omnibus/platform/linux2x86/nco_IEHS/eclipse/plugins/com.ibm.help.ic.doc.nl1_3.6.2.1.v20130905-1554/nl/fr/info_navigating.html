Navigation dans le Centre de documentation

Le Centre de documentation est doté d'une interface graphique permettant à l'utilisateur de parcourir la documentation des produits.

Pourquoi et quand exécuter cette tâche

Située dans la partie gauche de l'interface, la vue Sommaire () affiche les titres des rubriques proposées par la documentation. Les titres s'affichent dans une table des matières ou une arborescence de navigation. Pour afficher le contenu d'une rubrique, sélectionnez son titre dans l'arborescence.

Cliquez sur Précédent () et Suivant () pour vous déplacer dans l'historique des rubriques affichées.

Cliquez sur Accueil () pour ouvrir la page d'accueil.

Pour revenir à la vue de navigation après une recherche, cliquez sur l'onglet Sommaire () dans la partie inférieure de la vue.

Pour savoir comment naviguer à l'aide du clavier, voir Saisie au clavier et navigation.
