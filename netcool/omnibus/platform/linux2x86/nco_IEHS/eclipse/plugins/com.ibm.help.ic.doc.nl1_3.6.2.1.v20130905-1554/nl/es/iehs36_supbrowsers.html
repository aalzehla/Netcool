Navegadores soportados para el Information Center

Para utilizar este Information Center debe tener instalado en el sistema un navegador web soportado.

Estás soportados los siguientes navegadores web:

  • Microsoft® Internet Explorer Versión 7.0 o Versión 8.0
  • Mozilla Firefox Versión 3.5 o Versión 3.6.

Si utiliza una versión distinta de los navegadores web soportados u otro navegador web, es posible que algunas funciones no estén disponibles o no funcionen como se espera.

Para asegurarse de que se pueden utilizar todas las funciones del Information Center, debe habilitar las cookies y JavaScript™ en el navegador web y debe permitir que el navegador web visualiza ventanas emergentes de este Information Center.
