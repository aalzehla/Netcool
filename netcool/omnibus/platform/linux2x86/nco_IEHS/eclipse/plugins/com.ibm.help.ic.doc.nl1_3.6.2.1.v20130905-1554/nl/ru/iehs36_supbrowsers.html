Поддерживаемые браузеры для Information Center

Для использования этого Information Center в системе должен быть установлен поддерживаемый браузер.

Поддерживаются следующие браузеры:

  • Microsoft Internet Explorer версии 7.0 или 8.0
  • Mozilla Firefox версии 3.5 или 3.6.

При использовании другой версии поддерживаемых браузеров или другого браузера часть функций может быть недоступна или работать неправильно.

Для того чтобы работали все функции Information Center, в браузере должны быть включены cookie и JavaScript и разрешены всплывающие окна из этого Information Center.
