Găsirea informaţiei folosind vizualizarea index

Vizualizarea index oferă un index master al tuturor conţinuturilor din centrul de informare. Folosiţi vizualizarea Index pentru a găsi informaţii despre un subiect specific din centrul de informare.

Înainte de a începe

Dacă furnizorul de documentaţie a oprit vizualizarea index pentru centrul de informare, fila Index () nu este disponibilă.

Despre acest task

Pentru a găsi informaţii folosind vizualizarea Index:

Procedură

  1. Faceţi clic pe fişa Index tab () pentru a afişa vizualizarea Index.
  2. Tastaţi un cuvânt cheie în câmpul de intrare. Lista termenilor index este actualizată pentru a afişa cuvântul cheie care se potriveşte cel mai bine cu cuvântul cheie introdus.
  3. Faceţi clic pe cuvântul cheie sau titlul subiectului din lista index pentru a afişa subiectul în Vizualizare conţinut.
    Notă:
    • Indecşii nu sunt sensibili la majuscule. Acelaţi rezultate sunt afişate în vizualizarea Index indiferent dacă cuvântul cheie introdus este cu litere mici sau mari.
    • Dacă un cuvânt cheie este asociat cu mai mult de un subiect, cuvântul individual nu este afişat în lista cu termeni index. În schimb, sunt afişate titlurile paginilor care conţin acel cuvânt cheie.
    • În plus faţă de căutarea vizualizării Index, puteţi folosi legăturile Anterior şiUrmător din josul vizualizării Index pentru a defila prin lista cu termeni index.
