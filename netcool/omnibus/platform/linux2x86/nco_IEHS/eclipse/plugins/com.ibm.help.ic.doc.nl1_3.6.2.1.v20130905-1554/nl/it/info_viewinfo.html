Visualizzazione delle informazioni nell'Information Center

E' possibile visualizzare le informazioni nell'IBM Eclipse Help System (IEHS), quando è in esecuzione in modalità di information center.

Informazioni su questa attività
Con l'Information center, gli utenti possono visualizzare, sfogliare e ricercare le informazioni online. L'information center è basato su software open source sviluppato dal Progetto Eclipse (http://www.eclipse.org). E' in grado di visualizzare i contenuti che sono stati aggiunti ad un package come plugin della documentazione Eclipse.

L'information center utilizza un server di applicazioni web incorporato per gestire le richieste dei contenuti nel sistema. Tale server web incorporato, utilizza una porta casuale per evitare conflitti di porte tra le applicazioni.

Gli sviluppatori dell'applicazione o del prodotto possono creare i propri plugin della documentazione e integrarli all'information center per la visualizzazione da parte degli utenti. Per comprendere meglio il processo di creazione e di packaging dei plugin della documentazione, consultare il sito Web del Progetto Eclipse all'indirizzo:http://www.eclipse.org
