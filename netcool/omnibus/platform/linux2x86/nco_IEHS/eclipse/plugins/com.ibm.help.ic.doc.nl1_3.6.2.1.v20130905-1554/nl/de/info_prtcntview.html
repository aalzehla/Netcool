Inhaltssicht drucken

In diesem Thema wird die Vorgehensweise beim Drucken des Themas beschrieben, das Sie derzeit im Information Center anzeigen.

Warum und wann dieser Vorgang ausgeführt wird

Sie können die Inhaltssicht, die momentan angezeigt wird, drucken, indem Sie die folgenden Schritte ausführen:

  1. Klicken Sie in der Symbolleiste der Inhaltssicht auf Drucken (). Das Druckfenster wird geöffnet.
  2. Wenn der Druckjob an den Drucker gesendet werden kann, klicken Sie im Druckfenster auf Drucken, um die Druckausgabe zu starten.
