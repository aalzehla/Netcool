Localizando Informações com a Visualização do Índice

Este tópico descreve como trabalhar com a visualização do Índice para localizar tópicos no sistema de ajuda.

Para trabalhar com a visualização do Índice para localizar as informações que deseja, execute as seguintes etapas:

  1. Clique na guia Índice () para exibir a visualização do Índice.
  2. Digite uma palavra-chave no campo de entrada. A lista de índice rolará automaticamente até a palavra-chave que mais se aproxima.
    Nota:
    • Se uma palavra-chave se associar a mais de um tópico, ela não será exibida no Índice. Você pode encontrar os títulos dos tópicos exibidos na visualização do Índice.
  3. Clique no teclado ou no subteclado, ou em um título de tópico, para exibir o tópico na visualização de Conteúdo.
    Nota: Os índices não fazem distinção entre maiúsculas e minúsculas.

    Para alternar entre a visualização de navegação e de Índice, clique em Conteúdos () ou em Índice ().
