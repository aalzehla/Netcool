Localizando Informações Utilizando a Visualização Índice

A visualização Índice fornece um índice mestre de todo o conteúdo no centro de informações. Utilize a visualização Índice para localizar informações sobre tópicos específicos no centro de informações.

Antes de Iniciar

Se o provedor de documentação tiver desligado a visualização Índice para o centro de informações, a guia Índice () não estará disponível.

Sobre Esta Tarefa

Para localizar informações utilizando a visualização Índice:

Procedimento

  1. Clique na guia Índice () para exibir a visualização do Índice.
  2. Digite uma palavra-chave no campo de entrada. A lista de termos do índice é atualizada para exibir a palavra-chave que corresponda mais proximamente à palavra-chave que você inseriu.
  3. Clique na palavra-chave ou no título do tópico na lista de índices para exibir o tópico na visualização Conteúdo.
    Observações:
    • Os índices não fazem distinção entre maiúsculas e minúsculas. Os mesmos resultados são exibidos na visualização Índice independentemente de a palavra-chave estar em maiúsculas ou minúsculas.
    • Se uma palavra-chave estiver associada a mais de um tópico, a palavra-chave individual não é mostrada na lista de termos do índice. Em vez disso, os títulos das páginas que contêm essa palavra-chave é mostrada.
    • Além disso, para procurar a visualização de Índice, é possível utilizar os links Anterior e Avançar na parte inferior da visualização Índice para rolar pela lista de termos do índice.
