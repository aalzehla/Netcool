Προβολή πληροφοριών στο κέντρο πληροφοριών

Το κέντρο πληροφοριών επιτρέπει την προβολή, εξερεύνηση, αναζήτηση και εκτύπωση ηλεκτρονικών πληροφοριών. Το κέντρο πληροφοριών βασίζεται σε λογισμικό ανοιχτού κώδικα που αναπτύχθηκε στα πλαίσια του έργου Eclipse (www.eclipse.org).

  • Προετοιμασία για τη χρήση του κέντρου πληροφοριών
    Εκτελέστε τα παρακάτω βήματα για να εξασφαλίσετε τα καλύτερα δυνατά αποτελέσματα από τη χρήση του κέντρου πληροφοριών.
  • Πλοήγηση στο κέντρο πληροφοριών
    Χρησιμοποιήστε το γραφικό περιβάλλον χρήστη που παρέχεται από το κέντρο πληροφοριών για να αναζητήσετε και να εξετάσετε πληροφορίες τεκμηρίωσης για το προϊόν.
  • Αναζήτηση πληροφοριών στο κέντρο πληροφοριών
    Μπορείτε να αναζητήσετε πληροφορίες σε όλα τα θέματα του κέντρου πληροφοριών, σε ένα μεμονωμένο θέμα ή σε μια συλλογή θεμάτων, ή μπορείτε να δημιουργήσετε μια εμβέλεια αναζήτησης για να εκτελέσετε μια αναζήτηση σε ένα υποσύνολο των πληροφοριών. Στην προβολή Αποτελέσματα αναζήτησης μπορούν να παρουσιαστούν έως 500 αποτελέσματα, ταξινομημένα με βάση το βαθμό σχετικότητάς τους με τα κριτήρια αναζήτησης.
  • Ορισμός της κατάλληλης εμβέλειας
    Μπορείτε να περιορίσετε την εμβέλεια εργασίας σας ορίζοντας τα κατάλληλα κριτήρια φιλτραρίσματος.
  • Εύρεση πληροφοριών με χρήση της προβολής ευρετηρίου
    Η προβολή Ευρετήριο παρέχει ένα κύριο ευρετήριο όλου του περιεχομένου του κέντρου πληροφοριών. Χρησιμοποιήστε την προβολή Ευρετήριο για να βρείτε πληροφορίες για συγκεκριμένα θέματα στο κέντρο πληροφοριών.
  • Εκτύπωση θεμάτων στο κέντρο πληροφοριών
    Μπορείτε να εκτυπώσετε το εμφανιζόμενο θέμα ή να εκτυπώσετε ένα ή περισσότερα θέματα απευθείας από την προβολή Περιεχόμενα.
  • Ορισμός σελιδοδεικτών στο κέντρο πληροφοριών
    Ορίζοντας σελιδοδείκτη για ένα θέμα διευκολύνεται η εύρεση του θέματος. Μπορείτε να ορίσετε σελιδοδείκτες χρησιμοποιώντας τις λειτουργίες του προγράμματος πλοήγησης ή χρησιμοποιώντας την προβολή Σελιδοδείκτες.
  • Λειτουργίες διευκόλυνσης πρόσβασης στο κέντρο πληροφοριών
    Οι λειτουργίες διευκόλυνσης πρόσβασης βοηθούν τα άτομα που έχουν κάποια σωματική αναπηρία, όπως π.χ. περιορισμένη κινητικότητα ή όραση ή άλλες ειδικές ανάγκες, στην αποτελεσματική χρήση προϊόντων πληροφορικής.
