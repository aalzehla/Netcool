정확한 단어 또는 구문 검색

Information Center에서 입력한 단어 또는 구문만을 정확하게 검색할 수 있습니다.

Information Center에서 검색을 수행할 때 입력한 단어 또는 구문만을 검색하려면 해당 항목을 큰따옴표로 묶으십시오.

예를 들어, 검색 필드에 로그 파일 대신 "로그 파일"을 입력하면 로그 파일만을 검색합니다. 큰따옴표가 없으면 주제에서 로그 및 파일을 모두 검색합니다.

주: 영어 및 독일어의 경우에만 검색 엔진이 단일 검색 단어의 다른 형식도 "검색합니다". 예를 들어, 도전을 입력하면 도전적인도 함께 검색됩니다. 검색하려는 단어가 포함된 모든 단어를 검색 결과에서 제외시키려면 검색 필드에 해당 단어를 큰따옴표로 묶어서 입력하십시오.
