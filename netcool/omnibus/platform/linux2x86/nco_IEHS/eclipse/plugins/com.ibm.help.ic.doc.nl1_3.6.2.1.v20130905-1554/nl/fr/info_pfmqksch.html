Recherche rapide

Cette rubrique vous apprendra à effectuer une recherche rapide.

Pourquoi et quand exécuter cette tâche

Dans la vue de navigation, procédez comme suit pour effectuer une recherche rapide :

  1. Dans la barre d'outils de la vue de navigation, cliquez sur Rubriques de recherche() pour afficher le menu.
    • Pour limiter la recherche à la rubrique actuellement ouverte, cliquez sur Rechercher dans cette rubrique.
    • Pour effectuer la recherche dans la rubrique et dans toutes ses sous-rubriques, cliquez sur Rechercher dans cette rubrique et les sous-rubriques.
      1. Dans la boîte de dialogue de recherche, entrez la requête dans la zone Rechercher.
      2. Cliquez sur OK pour lancer la recherche.
  2. Dans la boîte de dialogue de recherche, entrez la requête dans la zone Rechercher et cliquez sur OK pour lancer la recherche.
Résultats

Si la recherche porte uniquement sur la rubrique ouverte dans le menu Raccourci et que des résultats correspondent aux termes recherchés, la rubrique s'affiche automatiquement. S'il n'y a pas d'élément correspondant dans la rubrique, un message indique Terme introuvable.

Si vous effectuez la recherche dans une rubrique et ses sous-rubriques et que des éléments correspondent à votre recherche, la vue Résultats de la recherche affiche les 500 premiers résultats. Sélectionnez un résultat pour afficher le rubrique associée. Les mots inclus dans votre recherche sont mis en évidence dans la rubrique affichée. S'il n'y a pas d'élément correspondant dans la rubrique, un message indique Terme introuvable.

Remarques :
  • La boîte de dialogue Rechercher dans cette rubrique se ferme automatiquement, quels que soient les résultats de la recherche.
  • La boîte de dialogue Rechercher dans cette rubrique et les sous-rubriques ne se fermera pas automatiquement et, pour ce faire, il vous faudra cliquer sur Annuler ou refermer la boîte de dialogue.
