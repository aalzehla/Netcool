مشاهدة معلومات في مركز المعلومات

يمكنك مشاهدة معلومات في IBM Eclipse Help System (IEHS)‏، عند تشغيله بنمط مركز المعلومات.

نبذة عن هذه المهمة
باستخدام مركز المعلومات، يستطيع المستخدمين مشاهدة واستعراض وبحث المعلومات المتاحة من خلال الشاشة. حيث تم تصميم مركز المعلومات بناءا على برنامج ذو مصدر مفتوح يتم تطويره بواسطة Eclipse Project (http://www.eclipse.org)‏. كما يمكن عرض المحتويات التي تم تجميعها باعتبارها Eclipse Documentation Plug-in.

مركز المعلومات يستخدم مع وحدة خدمة مدمجة للتعامل مع طلبات المحتويات في النظام. تقوم وحدة خدمة Web المدمجة باستخدام منفذ عشوائي لتجنب حدوث أي تعارض بين التطبيقات.

يستطيع مطوري البرامج أو التطبيقات تكوين البرامج المساعدة الخاصة بهم وتكاملها في مركز المعلومات ليتم مشاهدتهم بواسطة المستخدمين. لمعرفة المزيد عن تكوين وتجميع البرامج المساعدة، ارجع الى موقع Eclipse Project: http://www.eclipse.org

  • امكانية التوصل والمسارات المختصرة للوحة المفاتيح في information center
    تشرح خواص الاستخدام المتاحة للمستخدمين طريقة استخدام منتجات البرامج بسهولة وبنجاح.
  • المتطلبات الرئيسية لنظام المساعدة
    عند اعداد IBM Eclipse Help System، يجب التأكد من أن وحدة العمل ونظام التشغيل وبرنامج الاستعراض والجهاز متوافقين مع المتطلبات الموضحة في هذا الموضوع.
  • التحرك من خلال مركز المعلومات
    يقدم مركز المعلومات واجهة تعامل بيانية لاستعراض المطبوعات الفنية الخاصة بالمنتج.
  • البحث في مركز المعلومات
    يمكنك معرفة كيفية البحث في نظام مساعدة IBM Eclipse Help System بعد قراءة هذا الموضوع.
  • ايجاد معلومات باستخدام مشاهدة الفهرس
    يشرح هذا الموضوع كيفية التعامل مع مشاهدة الفهرس لايجاد موضوعات في نظام المساعدة.
  • تحديد علامات التوقف في مركز المعلومات
    يمكنك ايجاد موضوع بطريقة أسهل من خلال تحديد علامة توقف.
  • مشاهدة معلومات بلغات مختلفة
    لمشاهدة واجهة تعامل المستخدم لمركز المعلومات information center بلغة أخرى، يجب أن تقوم بتغيير المحددات الاقليمية الخاصة بنظام التشغيل الى اللغة ومحددات اللغة التي تريدها.
  • الطباعة في مركز المعلومات
    يقدم نظام مساعدة IBM Eclipse Help System طرق مختلفة لطباعة محتويات المساعدة التي توجد في مركز المعلومات.
