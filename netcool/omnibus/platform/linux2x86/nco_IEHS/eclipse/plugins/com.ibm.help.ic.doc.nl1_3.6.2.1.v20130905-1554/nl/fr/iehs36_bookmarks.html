Définir des signets dans le centre de documentation

Pour retrouver plus facilement une rubrique, vous pouvez la marquer par un signet. La définition des signets peut se faire avec les fonctions du navigateur web ou en utilisant la vue Signets.

Rubrique parent : Consulter les informations du centre de documentation

Définition des signets avec le navigateur web

Lorsque le centre de documentation est déployé sur un serveur et consulté via un réseau intranet ou Internet, vous pouvez sauvegarder des signets pour les rubriques dans le menu de favoris ou de signets de votre navigateur web.

Procédure

Pour définir un signet dans votre navigateur web pour la rubrique en cours :
  • Si vous utilisez Internet Explorer, cliquez sur Signet () pour sauvegarder la rubrique dans votre dossier Favoris.
  • Si vous n'utilisez pas Internet Explorer, cliquez avec le bouton droit de la souris sur n'importe quelle zone vide de la page en cours afin d'afficher le menu. Sélectionnez l'option Ajouter aux favoris ou Ajouter aux signets pour sauvegarder la rubrique dans l'un de vos dossiers de favoris.
Pour afficher vos signets, ouvrez le menu Favoris de votre navigateur web.

Pour supprimer un signet, ouvrez le menu Favoris de votre navigateur web et repérez le signet à supprimer. Cliquez dessus avec le bouton droit de la souris et cliquez sur Supprimer dans le menu.
