Υποστηριζόμενα προγράμματα πλοήγησης για το κέντρο πληροφοριών

Για να χρησιμοποιήσετε αυτό το κέντρο πληροφοριών, θα πρέπει να έχει εγκατασταθεί ένα υποστηριζόμενο πρόγραμμα πλοήγησης στο σύστημά σας.

Υποστηρίζονται τα ακόλουθα προγράμματα πλοήγησης:

  • Microsoft® Internet Explorer Έκδοση 7.0 ή Έκδοση 8.0
  • Mozilla Firefox Έκδοση 3.5 ή Έκδοση 3.6.

Εάν χρησιμοποιείτε διαφορετική έκδοση ενός υποστηριζόμενου προγράμματος πλοήγησης ή κάποιο άλλο πρόγραμμα πλοήγησης, μπορεί ορισμένες λειτουργίες να μην είναι διαθέσιμες ή να μη λειτουργούν όπως αναμένεται.

Για να εξασφαλίσετε ότι είναι δυνατή η χρήση όλων των λειτουργιών του κέντρου πληροφοριών, θα πρέπει να ενεργοποιήσετε τα ίχνη παρακολούθησης (cookies) και το JavaScript™ στο πρόγραμμα πλοήγησης και θα πρέπει να επιτρέψετε την παρουσίαση αναδυόμενων παραθύρων (pop-up windows) από αυτό το κέντρο πληροφοριών στο πρόγραμμα πλοήγησης.
