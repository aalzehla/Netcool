Ver la información en el Information Center

Puede visualizar información en el Sistema de ayuda de IBM Eclipse (IEHS) cuando ejecuta la modalidad del Information Center.

Con el Information Center, los usuarios pueden ver, examinar y realizar búsquedas en la información en línea. El Information Center tiene como fundamento el software de código abierto desarrollado por The Eclipse Project (http://www.eclipse.org). Puede mostrar contenido que se haya empaquetado como conector de la documentación de Eclipse.

El Information Center se utiliza con un servidor de aplicaciones web incorporado para manejar las peticiones de contenido del sistema. Este servidor web incluido emplea un puerto aleatorio para evitar que se produzcan conflictos de puertos entre las aplicaciones.

Los desarrolladores de productos o aplicaciones pueden crear sus propios conectores de documentación e integrarlos en el Information Center para que los usuarios puedan verlos. Para aprender a crear y empaquetar conectores de documentación, consulte el sitio web de The Eclipse Project: http://www.eclipse.org

    Las características de accesibilidad ayudan a los usuarios que tienen discapacidades físicas, como las de movilidad restringida o visión limitada, a utilizar los productos de software de manera satisfactoria.
