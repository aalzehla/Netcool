דרישות מקדימות של מערכת העזרה

בעת הגדרת מערכת העזרה IBM Eclipse, עליכם לוודא שתחנת העבודה, מערכת ההפעלה, הדפדפן והחומרה עומדים בדרישות המתוארות בנושא זה.

דרישות דפדפן

כדי לעבוד עם מערכת העזרה IBM Eclipse, עליכם להתקין את Microsoft Internet Explorer גירסה 6.0 או 7.0, או את Firefox 2.0 או 3.0. כדי להבטיח שכל הפונקציות של גירסה זו של IBM Eclipse Help System ניתנות לשימוש, עליכם להפעיל קובצי Cookie ו-JavaScript בדפדפן ולהשבית את פונקציית הדפדפן שחוסמת חלונות מוקפצים.

דרישות של סביבת מערכת ההפעלה

כדי לעבוד עם IBM Eclipse Help System, עליכם להתקין את Java Runtime Environment (JRE)‎ גירסה 6.0. עיינו בטבלה הבאה לקבלת דרישות מפורטות.

דרישות חומרה

z/OS

כדי להריץ את IBM Eclipse Help System ב-z/OS, עליכם לוודא ש-z/OS עומד בדרישות של גודל זיכרון ממשי וגודל REGION.

  • זיכרון ממשי = <basememory> + <plugsize> +<concurrentuser> * <responseBufferSize>
    • basememory הוא הזיכרון הדרוש להפעלת החבילה של IBM Eclipse Help System. ב-z/OS מדובר ב-90 MB.
    • plugsize הוא הגודל של תוסף התיעוד.
    • concurrentuser הוא מספר המשתמשים שייגשו למערכת העזרה בו-זמנית.
    • ניתן למצוא את responseBufferSize ב-preferences.ini במדריך eclipse\plugins\org.eclipse.help.base_3.0.1.
  • יש להגדיר את REGION לאותו גודל של הזיכרון הממשי.

מערכות הפעלה אחרות

קיבולת הזיכרון המינימלית הדרושה היא 128 MB RAM. מומלץ להשתמש ב-256 MB ומעלה.

הערה: יש להגדיל את קיבולת הזיכרון בהתאם לגודל תוספי התיעוד. לדוגמה, אם גודל תוסף התיעוד הוא 100 MB, הוסיפו לפחות 80 MB של זיכרון RAM נוסף.

התאמה נתמכת

ניתן להגדיר את התצורה של מערכת העזרה באמצעות הקובץ org.eclipse.help.base_<version number>/preferences.ini תחת המדריך eclipse/plugins, למרות שמומלץ להשתמש בתצורת ברירת המחדל.

אל תשנו את הפרמטרים הבאים; אחרת ייתכן שמערכת העזרה לא תפעל כראוי:

always_external_browser
עבור מערכות ההפעלה Microsoft Windows, תצורת ברירת המחדל של פרמטר זה היא always_external_browser=true. בעת הפעלת מערכת העזרה במצב עצמאי, הגדרה זו עוזרת לכם לפתוח את דפדפן ברירת המחדל של מערכת ההפעלה במקום הדפדפן המוטבע במערכת העזרה. הפרמטר always_external_browser=true אינו דרוש למערכות הפעלה שאינן ממשפחת Windows.
