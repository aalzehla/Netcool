การดำเนินการค้นหาแบบด่วน

คุณสามารถศึกษาวิธีการดำเนินการค้นหาแบบด่วนหลังได้ หลังจากอ่านหัวข้อนี้

ในมุมมองนำทาง ให้ทำตามขั้นตอนต่อไปนี้เพื่อดำเนินการค้นหาแบบด่วน:

  1. คลิก Search Topics() บนแถบเครื่องมือของมุมองนำทางเพื่อแสดงเมนู
    • หากต้องการค้นหาเฉพาะในหัวข้อที่คุณเปิดอยู่ ให้คลิก Search this topic
    • หากต้องการค้นหาผ่านหัวข้อและหัวข้อย้อยทั้งหมดของหัวข้อนั้น ให้คลิก Search this topic and all subtopics
      1. ในไดอะล็อกบ็อกซ์ค้นหา ให้ป้อนเคียวรีลงในฟิลด์ Search
      2. คลิก OK เพื่อเริ่มต้นการค้นหา
  2. ในไดอะล็อกบ็อกซ์ค้นหา ให้ป้อนเคียวรีในฟิลด์ Search และคลิก OK เพื่อเริ่มต้นการค้นหา

ถ้าคุณค้นหาเฉพาะในหัวที่คุณเปิดเมนู Quick และสตริงการค้นหาของคุณตรงกัน หัวข้อจะถูกแสดงโดยอัตโนมัติ ถ้าไม่พบหัวข้อที่ตรงกัน ข้อความ Nothing found จะปรากฏขึ้น

ถ้าคุณค้นหาหัวข้อและหัวข้อย่อยของหัวข้อนั้น และเงื่อนไขการค้นหาของคุณตรงกัน มุมมองผลลัพธ์การค้นหาจะแสดงผลลัพธ์ 500 ลำดับแรก คุณสามารถเลือกผลลัพธ์การค้นหาเพื่อแสดงหัวข้อนั้น คำที่มาจากเคียวรีของคุณจะถูกไฮไลต์ในหัวข้อที่แสดง ถ้าไม่พบหัวข้อที่ตรงกัน ข้อความ Nothing found จะปรากฏขึ้น

Notes:
  • ไดอะล็อกบ็อกซ์ Search this topic จะปิดโดยอัตโนมัติโดยไม่สนใจผลลัพธ์การค้นหา
  • ไดอะล็อกบ็อกซ์ Search this topic and all subtopics จะไม่ปิดจนกว่าคุณคลิก Cancel หรือปิดหน้าต่างไดอะล็อก
