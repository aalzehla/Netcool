Affichage des informations dans la langue de votre choix

Vous pouvez configurer la langue de votre navigateur web afin que les contenus du centre de documentation s'affichent dans la langue de votre choix.

Avant de commencer

L'ajout d'une langue à la liste de langues d'affichage du navigateur ne garantit pas que l'ordinateur dispose des polices nécessaires à l'affichage du contenu du centre de documentation dans cette langue.

Avec certaines combinaisons navigateur - système d'exploitation, vous devez également modifier les paramètres régionaux du système d'exploitation en spécifiant le lieu et la langue.

Procédure

  • Pour afficher le contenu du centre de documentation dans la langue de votre choix avec Internet Explorer :
    1. Ouvrez Internet Explorer et cliquez sur Outils > Options Internet > Langues. La fenêtre de préférences de langue s'affiche.
    2. Assurez-vous que la langue voulue soit indiquée en premier dans la liste de langues.
      • Pour ajouter une langue à la liste, cliquez sur Ajouter.
      • Pour amener une langue au début de la liste, sélectionnez-la et cliquez autant de fois que nécessaire sur Monter.
    3. Actualisez la page pour afficher le contenu du centre de documentation dans la langue choisie.
  • Pour afficher le contenu du centre de documentation dans la langue de votre choix avec Mozilla Firefox ou un autre navigateur basé sur Mozilla :
    1. Cliquez sur Choisir dans la section Langues de la boîte de dialogue Outils > Options > Contenu. Le panneau Langues s'affiche dans la fenêtre Préférences.
    2. Assurez-vous que la langue voulue soit indiquée en premier dans la liste de langues.
      • Pour ajouter une langue à la liste, cliquez sur Ajouter et sélectionnez-la dans la fenêtre Ajouter des langues.
      • Pour amener une langue au début de la liste, sélectionnez-la et cliquez autant de fois que nécessaire sur Monter.
    3. Actualisez la page pour afficher le contenu du centre de documentation dans la langue choisie.

Résultats

Le centre de documentation affiche son contenu dans la langue qui est spécifiée dans les préférences de votre navigateur. Si le contenu n'est pas traduit dans cette langue, il est affiché en anglais.
