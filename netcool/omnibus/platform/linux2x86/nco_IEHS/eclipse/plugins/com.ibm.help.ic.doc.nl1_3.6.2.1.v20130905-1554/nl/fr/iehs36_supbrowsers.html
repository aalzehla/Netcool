Navigateurs supportés pour le centre de documentation

Pour pouvoir utiliser le centre de documentation, votre système doit disposer d'un navigateur web supporté.

Les navigateurs web suivants sont supportés :

  • Microsoft Internet Explorer version 7.0 ou version 8.0
  • Mozilla Firefox version 3.5 ou version 3.6.

Avec une autre version d'un des navigateurs supportés ou un autre navigateur, certaines fonctions peuvent ne pas être disponibles ou ne pas fonctionner comme prévu.

Pour pouvoir utiliser toutes les fonctions du centre de documentation, vous devez autoriser dans votre navigateur les cookies, JavaScript et l'affichage des fenêtres en incrustation de ce centre de documentation.
