Software de requisito previo para el sistema de ayuda

Cuando configure el Sistema de ayuda de IBM Eclipse, será necesario que se asegure de que la estación de trabajo, el sistema operativo, el navegador y el hardware cumplen los requisitos que se describen en este tema.

Requisitos del navegador

Para trabajar con el Sistema de ayuda de IBM Eclipse, será necesario instalar Microsoft Internet Explorer Versión 6.0 ó 7.0, o bien Firefox 2.0 ó 3.0. Para asegurarse de que se pueden utilizar todas las funciones de esta versión del Sistema de ayuda de IBM Eclipse, es necesario que habilite las cookies y JavaScript en el navegador e inhabilitar la función de bloqueo de ventanas emergentes del navegador.

Requisitos del entorno del sistema operativo

Para trabajar con el Sistema de ayuda de IBM Eclipse, será necesario que instale Java Runtime Environment (JRE) versión 6.0. Consulte la tabla siguiente para ver los requisitos detallados.

Requisitos de hardware

z/OS

Para ejecutar el Sistema de ayuda de IBM Eclipse en z/OS, debe asegurarse de que z/OS cumple los requisitos de tamaño de memoria real y de tamaño de REGIÓN.

  • Memoria real = <memoria_base> + <tamaño_conector> +<usuario_concurrente> * <tamaño_almacenamiento-intermedio_respuesta>
    • memoria_base es la memoria necesaria para iniciar el paquete del Sistema de ayuda de IBM Eclipse. Es de 90 MB en z/OS.
    • tamaño_conector es el tamaño del conector de la documentación.
    • usuario_concurrente es el número de usuarios concurrentes que accederán al Sistema de ayuda.
    • tamaño_almacenamiento-intermedio_respuesta se puede encontrar en preferences.ini en el directorio eclipse\plugins\org.eclipse.help.base_3.0.1.
  • Tamaño de REGIÓN se debe establecer en el mismo que el tamaño de memoria real.

Otros sistemas operativos

La memoria mínima necesaria es 128 MB de RAM. Se recomiendan 256 MB o más.

Nota: Es necesario que aumente la memoria en función del tamaño de los conectores de la documentación. Por ejemplo, si el tamaño del conector de la documentación es de 100 MB, añada como mínimo 80 MB de RAM adicional.

Personalización soportada

Puede configurar el sistema de ayuda editando el archivo org.eclipse.help.base_<número de versión>/preferences.ini del directorio eclipse/plugins, aunque se sugiere mantener la configuración predeterminada.

No modifique los parámetros siguientes; de lo contrario es posible que el sistema de ayuda no funcione correctamente:

always_external_browser
Para los sistemas operativos Microsoft Windows, la configuración predeterminada de este parámetro es always_external_browser=true. Cuando inicia el sistema de ayuda en modalidad autónoma, este valor le ayuda a abrir el navegador predeterminado del sistema operativo en lugar del navegador incorporado en el sistema de ayuda. No es necesario que establezca always_external_browser=true para los sistemas operativos que no sean Windows.
