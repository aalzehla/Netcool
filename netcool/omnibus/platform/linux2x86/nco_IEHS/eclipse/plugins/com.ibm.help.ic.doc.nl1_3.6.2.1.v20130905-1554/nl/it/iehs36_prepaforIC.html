Preparazione all'utilizzo del Centro informazioni

Effettuare le seguenti operazioni per ottenere i risultati migliori dall'utilizzo del Centro informazioni.

  • Verificare che il proprio browser sia supportato.

    Per utilizzare questo Centro informazioni, è necessario che sul sistema sia installato un browser Web supportato. I browser supportati per il Centro informazioni sono disponibili in Browser supportati per il Centro informazioni

  • Impostare il browser Web sulla propria lingua preferita.

    È possibile impostare l'opzione della lingua del browser Web per visualizzare il contenuto del Centro informazioni nella lingua preferita dall'utente. Per ulteriori informazioni, consultare Visualizzazione delle informazioni nella lingua preferita dall'utente.
