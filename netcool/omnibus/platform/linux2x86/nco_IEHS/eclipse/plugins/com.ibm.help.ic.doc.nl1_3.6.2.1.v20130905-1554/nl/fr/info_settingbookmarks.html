Définir des signets dans le Centre de documentation

Pour retrouver plus facilement une rubrique, vous pouvez la marquer par un signet.

Pourquoi et quand exécuter cette tâche
Lorsque le Centre de documentation est déployé sur un serveur et consulté via un réseau intranet ou Internet, vous pouvez enregistrer des signets associés aux rubriques dans le menu de favoris ou de signets de votre afficheur.

Si le Centre de documentation est installé et affiché en local sur le système, les signets sont sauvegardés dans la vue Signets.

  1. Pour définir un signet pour la rubrique en cours :
    • Dans le Centre de documentation déployé sur un serveur, cliquez avec le bouton droit de la souris sur n'importe quel emplacement vide de la page en cours afin d'afficher le menu. Choisissez l'option Ajouter aux favoris ou Ajouter aux signets pour enregistrer la rubrique dans l'un de vos dossiers de favoris.
    • Dans le Centre de documentation installé sur votre machine, cliquez sur Signet () dans la barre d'outils au dessus du coin droit.
  2. Pour voir vos signets :
    • Dans le Centre de documentation déployé sur un serveur, l'icône Signet () ne s'affiche que lorsqu'on utilise Internet Explorer. Vous pouvez utiliser le menu Favoris d'Internet Explorer pour rechercher les signets que vous avez créés.
    • Dans le Centre de documentation installé sur votre machine, cliquez sur l'onglet Signets () dans la barre d'outils située au bas de la vue Signets.
  3. Pour supprimer un signet :
    • Dans le Centre de documentation déployé sur un serveur, ouvrez le menu Favoris d'Internet Explorer pour rechercher le signet. Cliquez dessus avec le bouton droit de la souris et choisissez Supprimer dans le menu.
    • Dans le Centre de documentation installé sur votre machine, cliquez sur l'onglet Signets (), sélectionnez le nom de la rubrique dans le volet des rubriques, et cliquez dans la barre d'outils sur Supprimer le signet sélectionné ().
  4. Vous pouvez également supprimer la totalité des signets dans le Centre de documentation qui s'exécute en local sur votre machine. Cliquez sur l'onglet Signets () et cliquez dans la barre d'outils sur Supprimer tous les signets ().
