การค้นหาคำหรือวลีที่แน่นอน

คุณสามารถค้นหาคำหรือวลีที่แน่นอนเหมือนกับสตริงที่แน่นอนใน information center

เมื่อคุณค้นหาข้อมูลใน information center คุณสามารถระบุคำหรือวลีที่เป็นสตริงที่แน่นอนได้โดยใส่เครื่องหมายอัญประกาศคู่

ตัวอย่างเช่น เมื่อคุณใส่ "log file" ลงในฟิลด์ Search แทน log file นั่นหมายความว่า คุณค้นหาสตริง log file หากไม่มีเครื่องหมายอัญประกาศคู่ เงื่อนไขการค้นหาจะถูกตีความให้ค้นหาทั้งคำว่า log และคำว่า file ในหัวข้อต่างๆ

หมายเหตุ: เอ็นจินการค้นหา จะค้นหา "รากศัพท์" ของคำเดี่ยวที่ใช้ในการค้นหา ซึ่งมีในภาษาอังกฤษและภาษาเยอรมันเท่านั้น ตัวอย่างเช่น การค้นหาคำว่า challenge จะค้นหาคำว่า challenging ด้วย ให้ใส่เครื่องหมายอัญประกาศคู่สำหรับเงื่อนไขที่คุณป้อนในฟิลด์ Search เมื่อคุณไม่ต้องการให้ผลลัพธ์การค้นหาของคุณมีเงื่อนไขที่นอกเหนือจากเงื่อนไขที่คุณต้องการ
