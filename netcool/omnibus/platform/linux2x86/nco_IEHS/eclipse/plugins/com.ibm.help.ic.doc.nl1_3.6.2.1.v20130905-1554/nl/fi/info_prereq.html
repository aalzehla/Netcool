Ohjejärjestelmän ennakkoedellytykset

Kun otat IBM Eclipse -ohjejärjestelmän käyttöön, varmista, että työasema, käyttöjärjestelmä, selain ja laitteisto vastaavat tässä aiheessa kuvattuja edellytyksiä.

Selainta koskevat edellytykset

IBM Eclipse -ohjejärjestelmän käyttö edellyttää, että asennettuna on Microsoft Internet Explorer -ohjelman versio 6.0 tai 7.0 tai Firefox-ohjelman versio 2.0 tai 3.0. Evästeet ja JavaScript on otettava käyttöön selaimessa ja kohoikkunoiden esto poistettava käytöstä, jotta IBM Eclipse -ohjejärjestelmän tämän version kaikki toiminnot ovat käytettävissä.

Käyttöjärjestelmää koskevat edellytykset

IBM Eclipse -ohjejärjestelmän käyttö edellyttää, että asennettuna on Java Runtime Environment (JRE) -käyttöympäristön versio 6.0. Katso tarkemmat vaatimukset seuraavasta taulukosta.

Laitteistovaatimukset

z/OS-käyttöjärjestelmä

IBM Eclipse -ohjejärjestelmän ajo z/OS-järjestelmässä edellyttää, että z/OS-järjestelmä vastaa todellista muistikokoa ja muistialueen kokoa koskevia vaatimuksia.

  • Todellinen muisti = <basememory> + <plugsize> +<concurrentuser> * <responseBufferSize>
    • basememory on muisti, joka tarvitaan IBM Eclipse -ohjejärjestelmäpaketin aloitusta varten. Perusmuistin koko on z/OS-järjestelmässä 90 megatavua.
    • plugsize on dokumentaatiolisäosan koko.
    • concurrentuser on ohjejärjestelmää samaan aikaan käyttävien käyttäjien määrä.
    • responseBufferSize on hakemiston eclipse\plugins\org.eclipse.help.base_3.0.1 tiedostossa preferences.ini.
  • Muistialueen koko on määritettävä samaksi kuin todellinen muistikoko.

Muut käyttöjärjestelmät

RAM-muistin tarvittava vähimmäiskoko on 128 megatavua. Suositeltava koko on vähintään 256 megatavua.

Huomautus: Muistia on lisättävä dokumentaatiolisäosien koon mukaan. Jos esimerkiksi dokumentaatiolisäosan koko on 100 megatavua, lisää vähintään 80 megatavua RAM-muistia.

Tuettu mukautus

Voit määrittää ohjejärjestelmän kokoonpanon muokkaamalla hakemistossa eclipse/plugins olevaa tiedostoa org.eclipse.help.base_<version numero>/preferences.ini, vaikka oletuskokoonpanon säilyttäminen on suositeltavaa.

Älä muuta seuraavia parametreja. Jos niitä muutetaan, ohjejärjestelmä ei ehkä toimi oikein:

always_external_browser
Microsoft Windows -käyttöjärjestelmissä tämän parametrin oletusasetus on always_external_browser=true. Kun ohjejärjestelmä aloitetaan erillistilassa, tämä asetus auttaa aloittamaan käyttöjärjestelmän oletusselaimen ohjejärjestelmässä valmiina olevan selaimen asemesta. Asetusta always_external_browser=true ei tarvitse määrittää muissa kuin Windows-käyttöjärjestelmissä.
