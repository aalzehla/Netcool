Impression de rubriques dans le centre de documentation

Vous pouvez imprimer la rubrique affichée ou une ou plusieurs rubriques de la vue Sommaire.

Procédure

  • Pour imprimer la rubrique affichée :
    1. Cliquez sur Imprimer la page () dans la barre d'outils. La fenêtre Imprimer du navigateur s'affiche.
    2. Pour lancer l'impression, cliquez sur Imprimer ou OK dans la fenêtre Imprimer.
  • Pour imprimer une rubrique ou un ensemble de rubriques de la vue Sommaire :
    1. Sélectionnez la rubrique voulue dans la zone de navigation.
    2. Cliquez sur Imprimer des rubriques () dans la barre d'outils.
      • Pour imprimer seulement la rubrique sélectionnée, sélectionnez Imprimer la rubrique sélectionnée.
      • Pour imprimer la rubrique sélectionnée et toutes ses sous-rubriques, cliquez sur Imprimer cette rubrique et toutes ses sous-rubriques, puis indiquez si vous souhaitez le saut de page entre les rubriques dans la sortie imprimée.
      La fenêtre Imprimer du navigateur s'affiche.
    3. Pour lancer l'impression, cliquez sur Imprimer ou OK dans la fenêtre Imprimer.
