Eingabe und Navigation über die Tastatur

Sie können mithilfe der Tastatur im Information Center navigieren.

Navigation über die Tastatur

Mit den folgenden Tastenkombinationen können Sie über die Tastatur in der Hilfefunktion navigieren:

  • Um im Thementeilfenster zum nächsten Link zu gelangen, drücken Sie die Tabulatortaste.
  • Um einen Knoten innerhalb der Navigationsstruktur ein- oder auszublenden, drücken Sie die Rechts- bzw. Linkspfeiltaste.
  • Um den nächsten Themenknoten auszuwählen, drücken Sie den Abwärtspfeil.
  • Um den vorhergehenden Themenknoten auszuwählen, drücken Sie den Aufwärtspfeil.
  • Um von einer der Sichten zum nächsten Link, zur nächsten Schaltfläche oder zum nächsten Themenknoten zu gelangen, drücken Sie die Tabulatortaste.
  • Um nach oben oder unten in einem Teilfenster zu blättern, drücken Sie die Taste Pos 1 oder Ende.
  • Um innerhalb der Seite zurückzugehen, drücken Sie die Tastenkombination Alt+Linkspfeil. Um innerhalb der Seite vorwärts zu gehen, drücken Sie die Tastenkombination Alt+Rechtspfeiltaste.
  • Um zum nächsten Teilfenster zu gelangen, drücken Sie die Taste F6.
  • Um zum vorherigen Teilfenster zu gelangen, drücken Sie die Tastenkombination Umschalttaste+F6.
  • Um das aktive Teilfenster zu drucken, drücken Sie die Tastenkombination Strg+P.
  • Um das Kontextmenü zu öffnen, drücken Sie die Tastenkombination Alt+M.
  • Um zur vorherigen Option im Kontextmenü zu gelangen, drücken Sie die Aufwärtspfeiltaste.
  • Um zur nächsten Option im Kontextmenü zu gelangen, drücken Sie die Abwärtspfeiltaste.
  • Um eine Option im Kontextmenü auszuwählen, wechseln Sie auf die Option, und drücken Sie die Eingabetaste.
  • Um das Kontextmenü zu schließen, drücken Sie die Taste Esc.
  • Um zur Inhaltssicht zu gelangen, drücken Sie die Tastenkombination Alt+C (drücken Sie anschließend die Eingabetaste, wenn Sie IE verwenden).
  • Um zur Sicht mit den Suchergebnissen zu gelangen, drücken Sie die Tastenkombination Alt+R (drücken Sie anschließend die Eingabetaste, wenn Sie IE verwenden).
  • Um zur Indexsicht zu gelangen, drücken Sie die Tastenkombination Alt+I (drücken Sie anschließend die Eingabetaste, wenn Sie IE verwenden).

Für Benutzer von JAWS 7

  • Um einen Knoten in der Navigationsstruktur einzublenden, drücken Sie die Tastenkombination Alt+O (Alt O in JAWS).
  • Um einen Knoten in der Navigationsstruktur auszublenden, drücken Sie die Tastenkombination Alt+B (Alt B in JAWS).
  • Um abwärts zum nächsten Themenknoten in der Baumstruktur zu gelangen, drücken Sie die Tastenkombination Strg+Umschalttaste+Abwärtspfeil.
  • Um aufwärts zum nächsten Themenknoten in der Baumstruktur zu gelangen, drücken Sie die Tastenkombination Strg+Umschalttaste+Aufwärtspfeil.
  • Um nach oben oder unten in einem Teilfenster zu blättern, drücken Sie die Taste 'Bild auf' bzw. 'Bild ab'.
  • Um zur vorherigen Option im Kontextmenü zu gelangen, drücken Sie die Tastenkombination Strg+Umschalttaste+Aufwärtspfeil.
  • Um zur nächsten Option im Kontextmenü zu gelangen, drücken Sie die Tastenkombination Strg+Umschalttaste+Abwärtspfeil.
  • Um eine Option im Kontextmenü auszuwählen, wechseln Sie auf die Option, und drücken Sie die Eingabetaste.
  • Um zwischen der Inhalts-, Suchergebnis-, Index- und Lesezeichensicht zu wechseln, verschieben Sie die Auswahl im Rahmen der Navigationssichten und verwenden Sie anschließend die im vorhergehenden Abschnitt beschriebenen Tastenkombinationen.

Verwenden Sie die anderen Tastenkombinationen so, wie sie im Abschnitt zur Navigation über die Tastatur dokumentiert sind.

Für Benutzer von Home Page Reader 3.4.1

  • Um einen Knoten in der Baumstruktur ein- und auszublenden, drücken Sie die Eingabetaste.
  • Um das Kontextmenü zu öffnen, drücken Sie die Tastenkombination Strg+L, drücken Sie dann die Taste Ende und anschließend die Eingabetaste.
  • Um zur vorherigen Option im Kontextmenü zu gelangen, drücken Sie Umschalttaste+Tabulatortaste.
  • Um zur nächsten Option im Kontextmenü zu gelangen, drücken Sie die Tabulatortaste.
  • Um eine Option im Kontextmenü auszuwählen, wechseln Sie auf die Option, und drücken Sie die Eingabetaste.
  • Um das Kontextmenü zu schließen, drücken Sie die Tastenkombination Alt+Umschalttaste+H.
  • Um zwischen der Inhalts-, Suchergebnis-, Index- und Lesezeichensicht zu wechseln, verschieben Sie die Auswahl im Rahmen der Navigationssichten und verwenden Sie anschließend die folgenden Tastenkombinationen:
    • Um zur Inhaltssicht zu gelangen, drücken Sie die Tastenkombination Alt+Umschalttaste+C.
    • Um zur Suchergebnissicht zu gelangen, drücken Sie die Tastenkombination Alt+Umschalttaste+R.
    • Um zur Indexsicht zu gelangen, drücken Sie die Tastenkombination Alt+Umschalttaste+I.

Verwenden Sie die anderen Tastenkombinationen so, wie sie im Abschnitt zur Navigation über die Tastatur dokumentiert sind.
