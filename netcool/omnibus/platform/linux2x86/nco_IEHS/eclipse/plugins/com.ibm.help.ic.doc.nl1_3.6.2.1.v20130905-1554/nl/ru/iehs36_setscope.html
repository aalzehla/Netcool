Настройка подходящей области

Рабочую область можно сузить, указав соответствующие критерии фильтрации.

Содержимое Information Center можно фильтровать по различных критериям, включая платформу, имя продукта и версию. Для работы с меньшим набором разделов можно отфильтровать содержимое.

Исключенные разделы, которые не удовлетворяют критериям или не принадлежат наборам разделов области, не показываются в дереве навигации при выполнении синхронизации.

  • Использование существующей области
    Существующую область можно использовать для сужения своей рабочей области.
  • Изменение существующей области
    Для удовлетворения новых требований исходную область можно изменить.
  • Создание новой области
    Для удовлетворения новых требований можно создать новую область.
