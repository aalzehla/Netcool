准备使用信息中心

执行以下步骤以使信息中心达到最佳使用效果。

  • 确保您的浏览器受支持。

    要使用该信息中心，必须在系统上安装受支持的 Web 浏览器。 您可以在信息中心的受支持浏览器中查找信息中心的受支持浏览器

  • 将 Web 浏览器设置为使用您的首选语言。

    您可以设置 Web 浏览器的语言选项，以使用首选语言来显示信息中心内容。请参阅使用首选语言显示信息，以获取更多信息。

  • 信息中心的受支持浏览器

  • 使用首选语言显示信息
    您可以设置 Web 浏览器的语言选项，以使用首选语言来显示信息中心内容。
