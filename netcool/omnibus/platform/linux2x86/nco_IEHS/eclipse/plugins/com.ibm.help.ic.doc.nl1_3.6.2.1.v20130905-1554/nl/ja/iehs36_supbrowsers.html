インフォメーション・センターでサポートされているブラウザー

このインフォメーション・センターを使用するには、サポート対象の Web ブラウザーをシステムにインストールする必要があります。

以下の Web ブラウザーがサポートされています。

  • Microsoft Internet Explorer バージョン 7.0 またはバージョン 8.0
  • Mozilla Firefox バージョン 3.5 またはバージョン 3.6

サポートされている Web ブラウザーの別バージョンや他の Web ブラウザーを使用する場合は、一部の機能が無効になったり、正しく動作しなくなったりする可能性があります。

インフォメーション・センターのすべての機能を使用するには、ブラウザーで Cookie と JavaScript を有効にし、このインフォメーション・センターからポップアップ・ウィンドウを表示できるように Web ブラウザーを設定することが必要です。
