Τροποποίηση της υπάρχουσας εμβέλειας

Μπορείτε να τροποποιήσετε την αρχική σας εμβέλεια έτσι ώστε να ανταποκρίνεται στις νέες σας απαιτήσεις.

Διαδικασία

  1. Επιλέξτε Εμβέλεια: στη γραμμή εργαλείων.
  2. Στο παράθυρο που εμφανίζεται, επιλέξτε την εμβέλεια από τη λίστα και πατήστε Τροποποίηση.
  3. Στο εμφανιζόμενο παράθυρο
    1. Μετονομάστε την εμβέλεια καθορίζοντας ένα νέο όνομα στο πεδίο Όνομα λίστας.
    2. Πατήστε () για να εμφανιστούν πρόσθετες επιλογές και να επιλέξετε κριτήρια εμβέλειας και σύνολα θεμάτων.
  4. Πατήστε OK για να αποθηκεύσετε τις αλλαγές σας.

Επόμενες ενέργειες
