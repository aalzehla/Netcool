Információk keresése az Index nézet segítségével

Az Index nézet egy főindexet biztosít az információs központ teljes tartalmához. Az Index nézet segítségével kereshet információkat adott témakörökről az információs központban.

Mielőtt elkezdené

Ha a dokumentáció szolgáltatója kikapcsolta az index nézetet az információs központban, akkor az Index lap () nem elérhető.

Erről a feladatról

Információk keresése az index nézettel:

Eljárás

  1. Kattintson a Index lapra () az Index nézet megjelenítéséhez.
  2. Írjon be egy kulcsszót a beviteli mezőbe. Az index kifejezések listája frissítésre kerül, és megjeleníti a megadott kulcsszóhoz legközelebb álló kulcsszót.
  3. Kattintson a kulcsszóra vagy a témakör címére az indexben a témakör megjelenítéséhez a Tartalom nézetben.
    Megjegyzések:
    • Az indexben a kis- és nagybetűk nincsenek megkülönböztetve. Ugyanazok az eredmények jelennek meg az Index nézetben, a bemeneti kulcsszó betűtípusától függetlenül.
    • Ha egy kulcsszó több témakörhöz is tartozik, akkor az egyéni kulcsszó nem jelenik meg az index kifejezések listájában. Ehelyett, a kulcsszót tartalmazó oldalak oldala jelenik meg.
    • Az Index nézet keresése mellett használhatja az Előző és a Következő hivatkozásokat az Index nézet alján az index kifejezések listájának végiggörgetéshez.
Szülőtéma: Információk megtekintése az információs központban
