Visualizando informações no Centro de Informações

Você pode visualizar informações no IBM Eclipse Help System (IEHS), quando estiver executando no modo Centro de Informações.

Com o centro de informações, os usuários podem visualizar, navegar e procurar informações on-line. O centro de informações é construído em software livre desenvolvido pelo Projeto Eclipse (http://www.eclipse.org). Ele pode exibir o conteúdo que foi compactado como um plug-in de documentação do Eclipse.

O Centro de Informações utiliza um servidor de aplicativos da Web integrado para tratar os pedidos de conteúdo no sistema. Esse servidor da Web incorporado utiliza uma porta aleatória para evitar conflitos de porta entre os aplicativos.

Os desenvolvedores do produto ou aplicativos podem criar seus próprios plug-ins de documentação e integrá-los ao centro de informações para a visualização de seus usuários. Para saber mais sobre a criação e pacote de plug-ins de documentação, veja o web site do Projeto Eclipse:http://www.eclipse.org

  • Acessibilidade e atalhos de teclado na central de informação
    Os recursos de acessibilidade ajudam os usuários que possuem deficiências físicas, como mobilidade restrita ou visão limitada, a utilizarem os produtos de software com sucesso.
  • Pré-requisitos para o sistema de ajuda
    Quando você configura o IBM Eclipse Help System, é preciso assegurar que a estação de trabalho, o sistema operacional, o navegador e o hardware atendem os requisitos descritos neste tópico.
  • Navegando no Centro de Informações
    O centro de informações oferece uma interface gráfica de usuário para que você navegue pela documentação do produto.
  • Procurando no Centro de Informações
    Você pode aprender como procurar no IBM Eclipse System Help após ler este tópico.
  • Localizando Informações com a Visualização do Índice
    Este tópico descreve como trabalhar com a visualização do Índice para localizar tópicos no sistema de ajuda.
  • Configuração de indicadores no centro de informações
    Você pode encontrar um tópico com mais facilidade marcando-o como indicador.
  • Visualizando Informações em Diferentes Idiomas
    Para visualizar a interface de usuário do centro de informações em outro idioma, você tem que alterar as configurações regionais de seu sistema operacional de acordo com o local e idioma de sua preferência.
  • Imprimindo no centro de informações
    O IBM Eclipse Help System oferece a você diferentes formas de imprimir os conteúdos da ajuda no centro de informações.
