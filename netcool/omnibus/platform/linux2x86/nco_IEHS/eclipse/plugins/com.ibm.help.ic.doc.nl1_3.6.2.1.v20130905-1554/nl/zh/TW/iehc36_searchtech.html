進階搜尋技術

您可以使用引號、萬用字元或布林運算子執行更進階的搜尋，以傳回為數較少的一組搜尋結果範圍

  • 當您在資訊中心搜尋時，您可以用雙引號括住單字或詞組以將其識別為完全相符的字串。

    例如，當您在搜尋欄位中輸入 "log file" 時，您搜尋的是所有出現 log file 字串的項目。如果未輸入雙引號，會傳回包含 log 和 file 這兩個字的實例主題。

  • 您可以在搜尋中使用星號 (*) 及問號 (?) 等萬用字元以擴大搜尋範圍。
    • 您可以在搜尋查詢中使用星號 (*) 代表多個不明或可變的字元。 例如，如果您輸入 par*，則會傳回包含 partly、participate、partial 和其他以 par 開頭之單字的主題。
      註： 搜尋字串有一項限制： 使用帶有星號的搜尋詞彙時，星號前必須至少有三個字元，同樣地，沒有萬用字元的搜尋詞彙必須至少包含三個字元。
    • 您可以在搜尋查詢中使用問號代表單一字元。 例如，如果您輸入 par?，則會傳回包含 part 的主題，但不會傳回包含 partial 或 partly 的主題。
  • 您可以使用布林運算子 AND、OR 和 NOT 來讓查詢更為明確。例如：
    database AND "log file"
    將搜尋結果範圍縮小為同時包含 database 詞彙和 log file 詞組的主題。
    database OR "log file"
    將搜尋結果範圍擴充為包含 database 詞彙或 log file 詞組的主題。
    database NOT "log file"
    將搜尋結果範圍縮小為包含 database 詞彙但不包含 log file 詞組的主題。
    database OR "data base" NOT "log file"
    搜尋包含 database 詞彙或 data base 詞組，但不包含 log file 詞組的主題。
