Lesezeichen im Information Center setzen

Sie können ein Thema schneller finden, wenn Sie ein Lesezeichen setzen.

Warum und wann dieser Vorgang ausgeführt wird
Bei einem Information Center, das auf einem Server implementiert ist und über ein Intranet oder das Internet angezeigt wird, können Lesezeichen für Themen im Menü für Lesezeichen oder Favoriten des Browsers gespeichert werden.

Bei einem auf Ihrer Maschine lokal installierten Information Center können Sie Lesezeichen für Themen in der Lesezeichensicht speichern.

  1. Gehen Sie wie folgt vor, um ein Lesezeichen für das aktuelle Thema zu setzen:
    • Klicken Sie im auf einem Server implementierten Information Center mit der rechten Maustaste auf einen beliebigen freien Bereich auf der aktuellen Seite, um das Menü zu öffnen. Wählen Sie die Option zum Hinzufügen zu den Favoriten oder die Option zum Hinzufügen zu den Lesezeichen aus, um das Thema in einem Ihrer Favoritenordner zu speichern.
    • Klicken Sie im auf Ihrer Maschine installierten Information Center in der Symbolleiste rechts oben auf Lesezeichen ().
  2. Gehen Sie wie folgt vor, um Ihre Lesezeichen anzuzeigen:
    • Im auf einem Server implementierten Information Center wird das Symbol Lesezeichen () nur angezeigt, wenn Sie den Internet Explorer verwenden. Mit dem Menü Favoriten des Internet Explorers können Sie nach den von Ihnen erstellten Lesezeichen suchen.
    • Klicken Sie im auf Ihrer Maschine installierten Information Center in der Symbolleiste der Lesezeichensicht unten auf die Registerkarte Lesezeichen ().
  3. Gehen Sie wie folgt vor, um ein Lesezeichen zu löschen:
    • Öffnen Sie im auf einem Server implementierten Information Center das Menü Favoriten des Internet Explorers, um nach dem Lesezeichen zu suchen. Klicken Sie mit der rechten Maustaste darauf und wählen Sie im Menü Löschen aus.
    • Klicken Sie im auf Ihrer Maschine installierten Information Center auf die Registerkarte Lesezeichen (), wählen Sie den Themennamen im Thementeilfenster aus und klicken Sie in der Symbolleiste auf Ausgewähltes Lesezeichen löschen ().
  4. Sie können auch alle Lesezeichen im Information Center löschen, das auf lokal auf Ihrer Maschine ausgeführt wird. Klicken Sie auf die Registerkarte Lesezeichen () und in der Symbolleiste auf Alle Lesezeichen löschen ().
