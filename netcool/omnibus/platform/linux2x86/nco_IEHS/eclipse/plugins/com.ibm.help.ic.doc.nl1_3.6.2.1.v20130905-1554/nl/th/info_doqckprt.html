การดำเนินการพิมพ์แบบด่วน

หัวข้อนี้แสดงวิธีการพิมพ์หัวข้อต่างๆ ใน information center

ในมุมมองนำทาง ให้ทำตามขั้นตอนต่อไปนี้เพื่อพิมพ์หัวข้อต่างๆ:

  1. เลือกหัวข้อในมุมมองนำทาง
  2. คลิก Print topics() บนแถบเครื่องมือเพื่อแสดงเมนู
    • หากต้องการพิมพ์เฉพาะหัวข้อที่เลือกไว้ ให้คลิก Print selected topic
    • หากต้องการพิมพ์หัวข้อที่เลือกไว้และหัวข้อย่อยทั้งหมดของหัวข้อนั้น ให้คลิก Print selected topic and all subtopics เพื่อเปิดหน้าต่างแสดงตัวอย่างก่อนพิมพ์
    • หากต้องการเปลี่ยนการตั้งค่าหน้ากระดาษก่อนพิมพ์ ให้เลือก File > Page Setup ในหน้าต่างแสดงตัวอย่างและแก้ไขค่าติดตั้ง
  3. คลิก Print ในหน้าต่างการพิมพ์
    ข้อจำกัด:

    MaxTopics และ maxConnections เป็นข้อจำกัดในการพิมพ์แบบเร็ว ตัวแปรดังกล่าวเป็นพารามิเตอร์ที่กำหนดไว้ใน preference สำหรับ resource optimization ก่อนที่ระบบจะเริ่มทำงาน

    ถ้าจำนวนหัวข้อที่คุณพิมพ์เกินค่าของ maxTopics ที่กำหนดไว้ คุณจะเห็นหน้าต่าง Alert
    รูปที่ 1. ตัวอย่างหน้าต่าง Alert
    • คลิก Yes เพื่อดำเนินการพิมพ์ ถ้าคุณเห็นด้วยกับค่าจำกัดของ maxTopics
    • คลิก No เพื่อยกเลิกการพิมพ์
    เมื่อจำนวนการเชื่อมต่อของพรินเตอร์เกินค่าของ maxConnections ที่กำหนดไว้ คุณจะเห็นหน้าต่าง Error
    รูปที่ 2. ตัวอย่างหน้าต่าง Error
    • คลิก OK และลองอีกครั้งในภายหลัง
หัวข้อต้นทาง: การพิมพ์ใน information center
