Acessibilidade e atalhos de teclado na central de informação

Os recursos de acessibilidade ajudam os usuários que possuem deficiências físicas, como mobilidade restrita ou visão limitada, a utilizarem os produtos de software com sucesso.

O IBM Eclipse Help System (IEHS) é um Componente Comum IBM. Os produtos da IBM adotam o IEHS para apresentar documentação on-line para clientes externos.

No Banco de Dados de Acessibilidade de Produto do Acessibility Center (AC) da IBM, o IEHS é categorizado como um Componente Reutilizável, pois é um componente reutilizado nos produtos IBM, ao invés de ser um componente vendido separadamente para clientes externos.
