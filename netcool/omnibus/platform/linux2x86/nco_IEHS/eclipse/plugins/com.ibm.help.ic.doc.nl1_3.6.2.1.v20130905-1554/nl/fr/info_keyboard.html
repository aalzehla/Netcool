Saisie au clavier et navigation

Le clavier permet de naviguer dans le Centre de documentation.

Déplacement à l'aide du clavier

Les combinaisons de touches suivantes permettent de naviguer dans le système d'aide à l'aide du clavier :

  • Dans la sous-fenêtre Rubrique, appuyez sur la touche Tabulation pour passer au lien suivant.
  • Pour développer et réduire un noeud dans l'arborescence de navigation, appuyez sur les touches Flèche gauche et Flèche droite.
  • Pour passer au noeud de rubriques suivant, appuyez sur la touche Flèche bas.
  • Pour passer au noeud de rubriques précédent, appuyez sur la touche Flèche haut.
  • Pour passer au lien, au bouton ou au noeud de rubriques suivant à l'intérieur d'une vue, appuyez sur la touche Tabulation.
  • Pour aller directement au début ou à la fin du document, appuyez respectivement sur la touche Origine ou sur la touche Fin.
  • Pour revenir en arrière, appuyez sur Alt+Flèche gauche ; pour passer à la section suivante, appuyez sur Alt+Flèche droite.
  • Pour passer à la sous-fenêtre suivante, appuyez sur F6.
  • Pour revenir à la sous-fenêtre précédente, appuyez sur Maj+F6.
  • Pour imprimer la sous-fenêtre active, appuyez sur Ctrl+P.
  • Pour ouvrir le menu contextuel, appuyez sur les touches Alt+M.
  • Pour passer à l'élément précédent du menu contextuel, appuyez sur la touche Flèche haut.
  • Pour passer à l'élément suivant du menu contextuel, appuyez sur la touche Flèche bas.
  • Pour sélectionner un élément du menu contextuel, passez à cet élément et appuyez sur la touche Entrée.
  • Pour fermer le menu contextuel, appuyez sur Echap.
  • Pour passer à la vue Sommaire, appuyez sur Alt+C (puis appuyez sur la touche Entrée si vous utilisez Internet Explorer).
  • Pour passer à la vue Résultats de la recherche, appuyez sur Alt+R (puis appuyez sur la touche Entrée si vous utilisez Internet Explorer).
  • Pour passer à la vue Index, appuyez sur Alt+I (puis appuyez sur la touche Entrée si vous utilisez Internet Explorer).

Pour les utilisateurs de JAWS 7

  • Pour développer un noeud dans l'arborescence de navigation, utilisez les touches Alt+O ("Alt O", selon JAWS).
  • Pour réduire un noeud dans l'arborescence de navigation, utilisez les touches Alt+B ("Alt B", selon JAWS).
  • Pour descendre au noeud de rubriques suivant dans l'arborescence, utilisez les touches Ctrl+Maj+ Flèche bas.
  • Pour remonter au noeud de rubriques précédent dans l'arborescence, utilisez les touches Ctrl+Maj+ Flèche haut.
  • Pour aller directement au début ou à la fin du document, appuyez respectivement sur la touche Origine ou sur la touche Fin.
  • Pour passer à l'élément précédent du menu contextuel, appuyez sur les touches Ctrl+Maj+Flèche haut.
  • Pour passer à l'élément suivant du menu contextuel, appuyez sur les touches Ctrl+Maj+Flèche bas.
  • Pour sélectionner un élément dans le menu contextuel, passez à cet élément et appuyez sur Ctrl+Entrée.
  • Pour basculer entre les différentes vues Sommaire, Résultats de la recherche, Index et Signets, allez au cadre Sélection des vues de navigation, puis utilisez les combinaisons de touches décrites dans la précédente section.

Utilisez les autres combinaisons de clavier décrites dans la section Navigation à l'aide du clavier.

A l'intention des utilisateurs de Home Page Reader 3.4

  • Pour développer et réduire un noeud dans l'arborescence, appuyez sur la touche Entrée.
  • Pour ouvrir le menu contextuel, appuyez sur les touches Ctrl+L, puis successivement sur les touches Fin et Entrée.
  • Pour passer à l'élément précédent du menu contextuel, appuyez sur les touches Maj+Tab.
  • Pour passer à l'élément suivant du menu contextuel, appuyez sur Tab.
  • Pour sélectionner un élément du menu contextuel, passez à cet élément et appuyez sur la touche Entrée.
  • Pour fermer le menu contextuel, appuyez sur les touches Alt+Maj+H.
  • Pour basculer entre les différentes vues Sommaire, Résultats de la recherche, Index et Signets, allez au cadre Sélection des vues de navigation, puis :
    • Pour passer à la vue Sommaire, appuyez sur Alt+Maj+C.
    • Pour passer à la vue Résultats de la Recherche, appuyez sur Alt+Maj+R.
    • Pour passer à la vue d'Index, appuyez sur Alt+Maj+I.

Utilisez les autres combinaisons de clavier décrites dans la section Navigation à l'aide du clavier.
