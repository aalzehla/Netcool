Browser supportati per il Centro informazioni

Per utilizzare questo Centro informazioni, è necessario che sul sistema sia installato un browser Web supportato.

Sono supportati i seguenti browser Web:

  • Microsoft® Internet Explorer Versione 7.0 o Versione 8.0
  • Mozilla Firefox Versione 3.5 o Versione 3.6.

Se si utilizza una versione diversa dei browser Web supportati o un altro browser Web, alcune funzioni potrebbero non essere disponibili o non funzionare come previsto.

Per assicurarsi che tutte le funzioni del Centro informazioni siano utilizzabili, è necessario abilitare i cookie e JavaScript™ nel browser Web ed è necessario consentire al browser Web di visualizzare le finestre a comparsa da questo Centro informazioni.
