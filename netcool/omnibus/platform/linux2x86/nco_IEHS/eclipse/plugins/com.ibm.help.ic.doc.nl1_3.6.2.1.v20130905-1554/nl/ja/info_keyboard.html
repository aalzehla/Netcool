キーボードによる入力とナビゲーション

キーボードを使用してインフォメーション・センターをナビゲートできます。

キーボードによるナビゲート

以下のキーの組み合わせを使用して、キーボードによるヘルプ・システムのナビゲートを行います。

  • 「トピック」ペインで次のリンクにジャンプするには、Tab を押します。
  • ナビゲーション・ツリーのノードを展開および縮小するには、右矢印および左矢印を押します。
  • 次のトピック・ノードに移動するには、下矢印を押します。
  • 前のトピック・ノードに移動するには、上矢印を押します。
  • ビューの中から、次のリンク、ボタン、またはトピック・ノードにジャンプするには、Tab を押します。
  • ペインの中で上端または下端までスクロールするには、Home または End を押します。
  • 戻るには Alt+ 左矢印を押し、先に進むには Alt+ 右矢印を押します。
  • 次のペインにジャンプするには、F6 を押します。
  • 前のペインにジャンプするには、Shift+F6 を押します。
  • アクティブになっているペインを印刷するには、Ctrl+P を押します。
  • ポップアップ・メニューを開くには、Alt+M を押します。
  • ポップアップ・メニューで前の項目に移動するには、上矢印を押します。
  • ポップアップ・メニューで次の項目に移動するには、下矢印を押します。
  • ポップアップ・メニューで項目を選択するには、希望の項目に移動して Enter を押します。
  • ポップアップ・メニューを閉じるには、Esc を押します。
  • 目次ビューに移動するには、Alt+C を押します (IE を使用している場合は、その後で Enter キーを押します)。
  • 検索結果ビューに移動するには、Alt+R を押します (IE を使用している場合は、その後で Enter キーを押します)。
  • 索引ビューに移動するには、Alt+I を押します (IE を使用している場合は、その後で Enter キーを押します)。

JAWS 7 をお使いの方へ

  • ナビゲーション・ツリーのノードを展開するには、Alt+O を押します。JAWS が "Alt O" と読み上げます。
  • ナビゲーション・ツリーのノードを縮小するには、Alt+B を押します。JAWS が "Alt B" と読み上げます。
  • 次のトピックを見るためにツリー内の下のノードに移動するには、Ctrl+Shift+ 下矢印を使用します。
  • 前のトピックを見るためにツリー内の上のノードに移動するには、Ctrl+Shift+ 上矢印を使用します。
  • ペインの中で上端または下端までスクロールするには、PageUp または PageDown を押します。
  • ポップアップ・メニューで前の項目に移動するには、Ctrl+Shift+ 上矢印を押します。
  • ポップアップ・メニューで次の項目に移動するには、Ctrl+Shift+ 下矢印を押します。
  • ポップアップ・メニューで項目を選択するには、希望の項目に移動して Enter を押します。
  • 目次ビュー、検索結果ビュー、索引ビュー、およびブックマーク・ビューの間で切り替えるには、 ナビゲーション・ビュー・フレームの選択に移動して、前のセクションに記述されているキーの組み合わせを使用します。

「キーボードからのナビゲート」に記述されているその他のキーボードの組み合わせも使用できます。

ホームページ・リーダー 3.4.1 をお使いの方へ

  • ツリー構造のノードを拡張および縮小するには、Enter を押します。
  • ポップアップ・メニューを開くには、Ctrl+L、End、次に Enter を押します。
  • ポップアップ・メニューで前の項目に移動するには、Shift+Tab を押します。
  • ポップアップ・メニューで次の項目に移動するには、Tab を押します。
  • ポップアップ・メニューで項目を選択するには、希望の項目に移動して Enter を押します。
  • ポップアップ・メニューを閉じるには、Alt+Shift+H を押します。
  • 目次ビュー、検索結果ビュー、索引ビュー、およびブックマーク・ビューの間で切り替えるには、 ナビゲーション・ビュー・フレームの選択に移動して、以下を行います。
    • 目次ビューに移動するには、Alt+Shift+C を押します。
    • 検索結果ビューに移動するには、Alt+Shift+R を押します。
    • 索引ビューに移動するには、Alt+Shift+I を押します。

「キーボードからのナビゲート」に記述されているその他のキーボードの組み合わせも使用できます。
