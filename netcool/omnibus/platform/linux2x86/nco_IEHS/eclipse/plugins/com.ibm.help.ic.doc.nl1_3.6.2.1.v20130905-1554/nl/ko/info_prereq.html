도움말 시스템용 전제조건 요구사항

IBM® Eclipse Help System을 설치할 때는 워크스테이션, 운영 체제, 브라우저 및 하드웨어가 이 주제에 설명된 요구사항을 충족하는지 확인해야 합니다.

브라우저 요구사항

IBM Eclipse Help System에 대해 작업하려면 Microsoft® Internet Explorer 버전 6.0 또는 7.0이나 Firefox 2.0 또는 3.0을 설치해야 합니다. 이 버전의 IBM Eclipse Help System의 모든 기능을 사용하려면 브라우저에서 쿠키 및 JavaScript™를 사용하고 팝업 창을 차단하는 브라우저의 기능을 사용하지 않아야 합니다.

운영 체제 환경 요구사항

IBM Eclipse Help System에 대해 작업하려면 Java™ Runtime Environment(JRE) 버전 6.0을 설치해야 합니다. 자세한 요구사항은 다음 표를 참조하십시오.

표 1. 운영 체제 환경 요구사항

  • Windows XP Professional Intel 32비트

  • Red Hat Linux Enterprise 5 Intel®32비트

하드웨어 요구사항

z/OS®

IBM Eclipse Help System을 z/OS에서 실행하려면 z/OS가 실제 메모리 크기와 영역 메모리 크기의 요구사항을 충족하는지 확인해야 합니다.

  • 실제 메모리 = <basememory> + <plugsize> +<concurrentuser> * <responseBufferSize>
    • basememory는 IBM Eclipse Help System 패키지를 시작하는 데 필요한 메모리입니다. z/OS에서는 90MB입니다.
    • plugsize는 문서 플러그인의 크기입니다.
    • concurrentuser는 도움말 시스템에 동시에 액세스할 수 있는 사용자의 수입니다.
    • responseBufferSize는 eclipse\plugins\org.eclipse.help.base_3.0.1 디렉토리의 preferences.ini에서 찾을 수 있습니다.
  • 영역 크기는 실제 메모리 크기와 동일하게 설정되어야 합니다.

기타 운영 체제

최소 필요한 메모리는 128MB RAM이지만, 256 MB 이상이 권장됩니다.

주: 문서 플러그인의 크기에 따라 메모리를 증가시켜야 합니다. 예를 들어, 문서 플러그인의 크기가 100MB이면 추가 RAM을 80MB 이상 추가하십시오.

지원되는 사용자 정의

eclipse/plugins 디렉토리에 있는 org.eclipse.help.base_<version number>/preferences.ini 파일을 편집하여 도움말 시스템을 구성할 수 있지만, 가능하면 기본 구성을 유지할 것을 권장합니다.

다음 매개변수는 수정하지 마십시오. 수정하면 도움말 시스템이 제대로 작동하지 않을 수 있습니다.

always_external_browser
Microsoft Windows 운영 체제에서 이 매개변수의 기본 구성은 always_external_browser=true입니다. 도움말 시스템을 독립형 모드로 시작하면 이 설정에서는 도움말 시스템에 임베드된 브라우저 대신 운영 체제의 기본 브라우저를 열 수 있습니다. Windows 이외의 운영 체제에는 always_external_browser=true를 설정할 필요가 없습니다.
