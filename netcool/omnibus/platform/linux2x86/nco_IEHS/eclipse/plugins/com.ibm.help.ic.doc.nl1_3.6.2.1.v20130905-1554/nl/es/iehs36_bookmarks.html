Establecer favoritos en el Information Center

Puede encontrar un tema más fácilmente estableciendo una marca de favorito. Establezca marcas de favoritos utilizando las características de un navegador web o utilizando la vista Favoritos.

Tema principal: Ver la información en el Information Center

Establecer marcas de favoritos mediante un navegador web

En el caso de que el Information Center esté desplegado en un servidor y se consulte a través de una intranet o Internet, podrá guardar las marcas de favoritos para temas en los favoritos del navegador o en el menú de favoritos.

Procedimiento

Para establecer un favorito para el tema actual en el navegador web:
  • Si está utilizando Internet Explorer, pulse Favorito () para guardar el tema en la carpeta Favoritos.
  • Si no está utilizando Internet Explorer, pulse con el botón derecho del ratón sobre cualquier área en blanco de la página actual para abrir el menú. Seleccione la opción de Añadir a favoritos o Añadir a marcadores para guardar el tema en una de las carpetas de Favoritos.
Para ver las marcas de favoritos, abra el menú Favoritos del navegador web.

Para suprimir una marca de favorito, abra el menú Favoritos del navegador web para buscar la marca de favorito. Pulse con el botón derecho del ratón y pulse Suprimir en el menú.
