Consulter les informations du centre de documentation

Le centre de documentation permet d'afficher et de parcourir la documentation en ligne, d'y effectuer des recherches et de l'imprimer. Il repose sur des logiciels Open Source développés dans le cadre du projet Eclipse (www.eclipse.org).

  • Préparation à l'utilisation du centre de documentation
    Les étapes suivantes permettent de tirer profit au mieux du centre de documentation.
  • Navigation dans le centre de documentation
    Utilisez l'interface utilisateur graphique du centre de documentation pour trouver et afficher de la documentation produit.
  • Recherche dans le centre de documentation
    Vous pouvez effectuer des recherches dans toutes les rubriques du centre de documentation, dans une rubrique particulière ou dans un ensemble de rubriques, ou créer une portée de recherche pour ne chercher que dans un sous-ensemble de la documentation. La vue Résultats de la recherche peut afficher jusqu'à 500 résultats, triés par pertinence.
  • Définition de la portée appropriée
    Vous pouvez réduire votre portée de travail en définissant les critères de filtrage appropriés.
  • Recherche d'informations à l'aide de la vue Index
    La vue Index fournit un index maître de tout le contenu du centre de documentation. Elle permet de chercher des informations sur des sujets spécifiques dans celui-ci.
  • Impression de rubriques dans le centre de documentation
    Vous pouvez imprimer la rubrique affichée ou une ou plusieurs rubriques de la vue Sommaire.
  • Définir des signets dans le centre de documentation
    Pour retrouver plus facilement une rubrique, vous pouvez la marquer par un signet. La définition des signets peut se faire avec les fonctions du navigateur web ou en utilisant la vue Signets.
  • Fonctions d'accessibilité du centre de documentation
    Les fonctions d'accessibilité servent à permettre l'utilisation de produits informatiques aux personnes qui ont un handicap physique, comme une mobilité ou une vision limitée, ou d'autres besoins spéciaux.
