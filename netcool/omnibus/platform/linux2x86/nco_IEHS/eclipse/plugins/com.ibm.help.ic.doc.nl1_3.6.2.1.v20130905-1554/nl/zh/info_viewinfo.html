查看信息中心内的信息

当 IBM® Eclipse Help System（IEHS）以信息中心方式运行时，您可以查看其中的信息。

使用信息中心，用户可以查看、浏览和搜索联机信息。 信息中心是在由 Eclipse Project（http://www.eclipse.org）开发的开放式源代码软件上构建的。它可以显示已经打包为 Eclipse 文档插件的内容。

信息中心使用嵌入式 Web 应用程序服务器来处理系统中的内容请求。 此嵌入式 Web 服务器使用随机端口来避免应用程序之间的端口冲突。

产品或应用程序开发者可以创建自己的文档插件，并将其集成到信息中心以供用户查看。要了解有关创建文档插件以及将其打包的信息，请访问 Eclipse Project 的 Web 站点：http://www.eclipse.org
