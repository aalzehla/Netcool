Im Information Center drucken

IBM Eclipse Help System stellt Ihnen unterschiedliche Möglichkeiten zum Drucken des Inhalts der Hilfe im Information Center bereit.

Warum und wann dieser Vorgang ausgeführt wird

Sie können das Thema drucken, das Sie derzeit in der Inhaltssicht anzeigen; es ist aber auch möglich, ein oder mehrere Themen in der Navigationsstruktur per Schnelldruck zu drucken.

  • Inhaltssicht drucken
    In diesem Thema wird die Vorgehensweise beim Drucken des Themas beschrieben, das Sie derzeit im Information Center anzeigen.
  • Schnelldruck durchführen
    In diesem Thema wird die Vorgehensweise beim Drucken von einem oder mehreren Themen im Information Center beschrieben.
