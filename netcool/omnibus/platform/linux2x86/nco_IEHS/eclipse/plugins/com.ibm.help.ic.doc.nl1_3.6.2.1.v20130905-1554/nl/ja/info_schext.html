正確な単語または句の検索

インフォメーション・センター内で正確な単語または句を正確なストリングとして検索することができます。

インフォメーション・センター内で検索する場合は、 単語または句を二重引用符で囲むことで正確なストリングとして指定できます。

例えば、「検索」フィールドに、log file ではなく "log file" と入力した場合は、log file というストリングを検索することになります。 二重引用符で囲まないと、この検索語は、トピック内で単語 log と file の両方のインスタンスを検索するものと解釈されます。

注: 英語とドイツ語についてのみ、検索エンジンは単一検索語の他の形の「語幹」になります。例えば、 単語 challenge が検索対象であると、単語 challenging も検出されます。このような語の変化型を検索結果に含めない場合は、 「検索」フィールドに入力する語を二重引用符で囲みます。
