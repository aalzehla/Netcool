Navegadores Suportados para o Centro de Informações

Para utilizar este centro de informações, você deve ter um navegador da Web suportado instalado em seu sistema.

Os seguintes navegadores da Web são suportados:

  • Microsoft Internet Explorer Versão 7.0 ou Versão 8.0
  • Mozilla Firefox Versão 3.5 ou Versão 3.6.

Se você utilizar uma versão diferente dos navegadores da Web suportados ou outro navegador da Web, algumas funcionalidades poderão não estar disponíveis ou não funcionar como esperado.

Para assegurar que todas as funções do centro de informações sejam utilizáveis, você deve ativar cookies e JavaScript no navegador da Web e deve permitir que o navegador da Web exiba janelas pop-up nesse centro de informações.
