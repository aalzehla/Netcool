在信息中心内设置书签

通过设置书签，可以更容易地找到主题。 使用 Web 浏览器中的功能部件或使用“书签”视图来设置书签

父主题： 查看信息中心内的信息

使用 Web 浏览器来设置书签

对于部署在服务器上并通过内部网或因特网查看的信息中心，可以将主题书签保存到 Web 浏览器的“书签”或“收藏夹”菜单中。

过程

要在 Web 浏览器中为当前主题设置书签：
  • 如果正在使用 Internet Explorer，请单击书签（）以将主题保存到“收藏夹”文件夹中。
  • 如果不在使用 Internet Explorer，请右键单击当前页面上的任何空白区域以打开菜单。选择选项添加到收藏夹或添加到书签以将主题保存到某个“收藏夹”文件夹中。
要查看书签，请打开 Web 浏览器的收藏夹菜单。

要删除书签，请打开 Web 浏览器的收藏夹菜单以查找该书签。进行右键单击，然后在菜单中单击删除。
