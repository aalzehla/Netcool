Wymagania wstępne centrum informacyjnego

Podczas konfigurowania systemu pomocy platformy IBM Eclipse należy upewnić się, że stacja robocza, system operacyjnym, przeglądarka i elementy sprzętowe spełniają wymagania opisane w tym temacie.

Wymagania dotyczące przeglądarki

Do pracy z systemem pomocy platformy IBM Eclipse niezbędna jest przeglądarka Microsoft Internet Explorer w wersji 6.0 lub 7.0 albo Firefox w wersji 2.0 lub 3.0. Aby wszystkie funkcje tej wersji systemu pomocy platformy IBM Eclipse działały poprawnie, należy w przeglądarce włączyć obsługę informacji cookie oraz JavaScript i wyłączyć funkcję blokowania elementów wywoływanych.

Do pracy z systemem pomocy platformy IBM Eclipse niezbędne jest środowisko Java Runtime Environment (JRE) w wersji 6.0. Szczegółowe wymagania są opisane w poniższej tabeli.

Wymagania sprzętowe

z/OS

Przed uruchomieniem systemu pomocy platformy IBM Eclipse w systemie z/OS należy upewnić się, że system z/OS spełnia wymagania dotyczące rozmiaru pamięci rzeczywistej i rozmiaru REGION.

  • Pamięć rzeczywista = <pamięć_podstawowa + <rozmiar_wtyczki> +<liczba_użytkowników_jednoczesnych> * <rozmiar_buforu_odpowiedzi>
    • pamięć_podstawowa to pamięć wymagana do uruchomienia pakietu systemu pomocy platformy IBM Eclipse. W systemie z/OS jest to 90 MB.
    • rozmiar_wtyczki to rozmiar wtyczki dokumentacji.
    • liczba_użytkowników_jednoczesnych to liczba jednocześnie pracujących użytkowników, którzy korzystają z systemu pomocy.
    • rozmiar_buforu_odpowiedzi można znaleźć w pliku preferences.ini w katalogu eclipse\plugins\org.eclipse.help.base_3.0.1.
  • Rozmiar REGION należy ustawić na taką samą wartość jak rozmiar pamięci rzeczywistej.

Inne systemy operacyjne

Wymagane jest przynajmniej 128 MB pamięci RAM. Zalecany rozmiar pamięci to 256 MB lub więcej.

Uwaga: Może być wymagane więcej pamięci w zależności od rozmiaru wtyczek dokumentacji. Jeśli na przykład rozmiar wtyczki dokumentacji wynosi 100 MB, należy dodać przynajmniej 80 MB pamięci RAM.

Obsługiwane dostosowania

Chociaż system pomocy można skonfigurować przy użyciu pliku org.eclipse.help.base_<numer_wersji>/preferences.ini znajdującego się w katalogu eclipse/plugins, to zaleca się zachowanie konfiguracji domyślnej. Szczegółowe informacje na temat dostosowywania systemu pomocy przy użyciu pliku preferences.ini.

Nie należy modyfikować następujących parametrów, ponieważ mogłoby to spowodować nieprawidłowe działanie systemu pomocy:

always_external_browser
W przypadku systemów operacyjnych Microsoft Windows domyślna konfiguracja tego parametru to always_external_browser=true. Jeśli system pomocy zostanie uruchomiony w trybie autonomicznym, to ustawienie przyczynia się do otwarcia domyślnej przeglądarki systemu operacyjnego zamiast przeglądarki wbudowanej w system pomocy. W systemach operacyjnych innych niż Windows nie trzeba ustawiać parametru always_external_browser=true.
