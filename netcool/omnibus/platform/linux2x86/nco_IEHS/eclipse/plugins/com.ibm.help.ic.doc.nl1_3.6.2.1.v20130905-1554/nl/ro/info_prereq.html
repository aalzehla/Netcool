Cerinţele preliminare pentru sistemul de ajutor

Când setaţi IBM Eclipse Help System, trebuie să vă asiguraţi că staţia de lucru, sistemul de operare, browser-ul şi hardware-ul îndeplinesc cerinţele prezentate în acest subiect.

Cerinţele de browser

Pentru a lucra cu IBM Eclipse Help System, trebuie să instalaţi Microsoft Internet Explorer versiunea 6.0 sau 7.0 sau Firefox 2.0 sau 3.0. Pentru a vă asigura că pot fi folosite toate funcţiile acestei versiuni de IBM Eclipse Help System, trebuie să activaţi cookie-urile şi JavaScript în browser şi să dezactivaţi funcţia browser-ului de blocare a ferestrelor pop-up.

Cerinţele mediului sistemului de operare

Pentru a lucra cu IBM Eclipse Help System, trebuie să instalaţi Java Runtime Environment (JRE) versiunea 6.0. Consultaţi următoarea tabelă pentru cerinţe detaliate.

Cerinţele de hardware

z/OS

Pentru a rula IBM Eclipse Help System pe z/OS, trebuie să vă asiguraţi că z/OS îndeplineşte cerinţele privind mărimea memoriei reale şi dimensiunea REGION.

  • Memorie reală = <basememory + <plugsize> +<concurrentuser> * <responseBufferSize>
    • basememory este memoria necesară pentru a porni pachetul sistem de ajutor IBM Eclipse. Este 90 MB pe z/OS.
    • plugsize este dimensiunea plugin-ului de documentaţie.
    • concurrentuser este numărul de utilizatori concurenţi care vor accesa sistemul de ajutor.
    • responseBufferSize poate fi găsit în preferences.ini, în directorul eclipse\plugins\org.eclipse.help.base_3.0.1.
  • Dimensiunea REGION trebuie setată la fel ca dimensiunea memoriei reale.

Alte sisteme de operare

Memoria minimă necesară este 128 MB RAM. Se recomandă 256 MB sau mai mult.

Notă: Trebuie să creşteţi memoria corespunzător dimensiunii plugin-urilor de documentaţie. De exemplu, dacă dimensiunea plugin-ului de documentaţie este 100 MB, adăugaţi cel puţin 80 MB de RAM suplimentar.

Personalizarea suportată

Puteţi să configuraţi sistemul de ajutor prin editarea fişierului org.eclipse.help.base_<număr versiune>/preferences.ini din directorul eclipse/plugins, dar se recomandă să păstraţi configuraţia implicită.

Nu modificaţi următorii parametri; în caz contrar este posibil ca sistemul de ajutor să nu funcţioneze corect:

always_external_browser
Pentru sistemele de operare Microsoft Windows, configuraţia implicită a acestui parametru este always_external_browser=true. Când porniţi sistemul de ajutor în mod independent, această setare vă ajută să deschideţi browser-ul implicit al sistemului de operare în locul browser-ului înglobat din sistemul de ajutor. Nu este necesar să setaţi always_external_browser=true pentru alt sistem de operare în afară de Windows.
