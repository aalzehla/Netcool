搜尋完全相符的單字或詞組

您可以在資訊中心中搜尋與確切字串相符的確切單字或片語。

當您在資訊中心中搜尋時，您可以用雙引號括住一些單字或片語，將其識別成確切字串。

例如，在搜尋欄位中輸入 "log file" 時，您搜尋的是 log file 字串，而非 log file。當沒有雙引號時，搜尋詞彙會解譯為在主題中搜尋同時含有 log 和 file 這兩個字的實例。

註: （僅限英文和德文）搜尋引擎會「衍生」單一搜尋字組的其他形式。 比方說，搜尋 challenge 字組也會尋找 challenging 字組。 當您不想要搜尋結果包含這樣的詞彙變化時，請用雙引號括住您在搜尋欄位中輸入的詞彙。
