IBM Tivoli Netcool/OMNIbus, 버전 7.4

이벤트 목록 시작

이벤트 목록에서 경보를 모니터하고 관리하려면 올바른 인증 신임 정보를 사용하여 실행 중인 ObjectServer에 연결해야 합니다. 컨덕터, 명령행 또는 Windows 시작 메뉴에서 이벤트 목록을 시작할 수 있습니다.

이벤트 목록을 시작하려면 다음을 수행하십시오.
  1. 운영 체제별로 적절한 조치를 수행하십시오.
    옵션 설명
    UNIX
    • 컨덕터 창의 단추 막대에서 이벤트 목록을 클릭하십시오.
    • 명령 프롬프트에서 다음을 입력하십시오.

      $NCHOME/omnibus/bin/nco_event &
    Windows
    • 컨덕터 메뉴에서 이벤트 목록을 선택하십시오.
    • Windows 시작 메뉴에서 프로그램 > Netcool 스위트 > 이벤트 목록을 선택하십시오.
    • 명령 프롬프트에서 다음을 입력하십시오.

      %NCHOME%\omnibus\desktop\NCOEvent.exe
    명령행에서 이벤트 목록을 시작하는 경우, nco_event 및 NCOEvent.exe 명령을 사용하여 명령행 옵션을 추가로 지정할 수 있습니다.
  2. 명령행 또는 Windows 시작 메뉴에서 이벤트 목록을 시작하고 이전에 컨덕터를 시작하지 않았으면, 실행 중인 ObjectServer에 연결하기 위해 인증 세부사항을 제공하도록 프롬프트가 표시됩니다. 다음과 같이 이벤트 목록의 로그인 창을 완료해야 합니다.
    사용자 이름
    사용자 이름을 입력하십시오.
    비밀번호
    비밀번호를 입력하십시오. 사용자가 입력한 텍스트의 위치에 별표(*) 문자가 나타납니다.
    참고: 사용자 이름 및 암호가 외부 인증 소스(ObjectServer가 아니라)와 대조하여 검증되고 있는 경우 이 소스가 멀티바이트 문자를 지원하는지 여부를 확인해야 합니다. 멀티바이트 문자가 지원되지 않는 경우 ASCII 문자를 사용하여 사용자 이름과 비밀번호를 지정해야 합니다.
    서버
    ObjectServer 이름이 NCOMS(Netcool/OMNIbus Master Server)의 기본값입니다. 환경에 여러 ObjectServer가 구성된 경우 이 드롭 다운 목록에서 ObjectServer가 모두 사용 가능합니다. 연결하려는 ObjectServer를 선택하십시오.
    확인
    로그인 세부사항을 인증하려면 이 단추를 클릭하십시오.
    참고: 로그인 시도는 3번까지 가능합니다. 각 시도에서 Password 필드는 지워집니다. 3번째 시도에서 올바르게 로그인하지 못하면 프로그램이 종결됩니다.
    취소
    로그인 시도를 취소하려면 이 단추를 클릭하십시오.

결과

이벤트 목록 인스턴스(nco_event)가 실행 중이면, nco_event 명령이 새 이벤트 목록 창을 엽니다. 실행 중인 nco_event 인스턴스가 없으면, nco_event 명령이 이를 시작하고 모니터 상자 창을 표시합니다.

이벤트 목록을 시작하면 이벤트 목록 모니터 상자 창이 열립니다. 이 창에서 ObjectServer가 이벤트, 알람, 메시지 또는 데이터 항목을 수신할 때 생성되는 모든 경보를 모니터하고 관리할 수 있습니다.

일반적으로 이벤트 목록을 시작하면 기본 이벤트 목록 구성(default.elc)이 이벤트 목록 모니터 상자 창에 표시됩니다. 사용자 정의 이벤트 목록 구성 파일을 열도록 선택할 수도 있습니다. 명령행에서 또는 파일 > 열기 창을 사용하여 사용자 정의 구성을 지정할 수 있습니다. 이 사용자 정의 구성은 파일 시스템 또는 HTTP나 FTP를 사용하여 액세스할 수 있는 원격 서버에 있습니다.

장애 조치 로그인: 이벤트 목록이 기본 ObjectServer와의 연결을 유실하고 보조 또는 장애 조치 ObjectServer가 있으면, 보조 ObjectServer에 로그인하도록 프롬프트가 표시됩니다. 기본 ObjectServer를 다시 사용할 수 있게 되면 이벤트 목록이 기본 ObjectServer에 다시 연결 중임을 표시하는 메시지가 나타납니다. 기본 ObjectServer에 다시 로그인하지 않아도 됩니다.
  • UNIX 이벤트 목록 명령행 옵션
    UNIX에서 nco_event 명령을 사용하여 이벤트 목록을 시작하는 경우, 다수의 명령행 옵션을 지정하여 구성을 수정할 수 있습니다. $NCHOME/omnibus/bin 디렉토리에서 이 명령을 실행할 수 있습니다.
  • Windows 이벤트 목록 명령행 옵션
    Windows에서 NCOEvent.exe 명령을 사용하여 이벤트 목록을 시작하는 경우, 다수의 명령행 옵션을 지정하여 구성을 수정할 수 있습니다. %NCHOME%\omnibus\desktop 디렉토리에서 이 명령을 실행할 수 있습니다.
