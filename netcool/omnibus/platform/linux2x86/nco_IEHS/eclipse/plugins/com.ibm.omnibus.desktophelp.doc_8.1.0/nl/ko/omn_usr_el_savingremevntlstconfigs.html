IBM Tivoli Netcool/OMNIbus, 버전 7.4

원격 이벤트 목록 구성 저장

네트워크의 원격 서버에서 이벤트 목록 구성을 연 경우 해당 구성을 해당 서버에 저장하도록 허가되지 않습니다. 파일을 다른 위치에 저장해야 합니다. 서버는 HTTP 또는 FTP를 사용하여 액세스 가능해야 합니다.

원격 이벤트 목록 구성을 저장하려면 다음을 수행하십시오.
  1. 이벤트 목록 모니터 상자 창에서 파일 > 다른 이름으로 저장을 클릭하십시오. 구성 저장 창이 열립니다.
  2. 이벤트 목록 구성 파일을 저장하려는 위치를 지정한 다음 파일을 .elc 파일 확장자로 저장하십시오. 파일을 저장할 기본 위치는 다음과 같습니다.
    • UNIX: 이벤트 목록을 시작한 작업 중인 디렉토리
    • Windows: 임시 인터넷 파일 폴더

다음에 수행할 작업

참고: 파일 > 저장을 사용하여 이벤트 목록 구성 파일을 저장하려고 시도하는 경우 오류 메시지가 표시됩니다. 그런 다음 로컬 사본을 저장하도록 허용하는 구성 저장 창이 열립니다.
