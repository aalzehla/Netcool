Adding events to the task list

You can use the task list to identify events that require special attention. Adding events to the task list works in conjunction with the Task List event list filter.

About this task

To add events to the task list, select the events in the event list, and then click Alerts > Task List > Add to Task List. The events are shown in the Task List monitor box.

You can also remove events from the task list. Select the events and then click Alerts > Task List > Remove from Task List.
