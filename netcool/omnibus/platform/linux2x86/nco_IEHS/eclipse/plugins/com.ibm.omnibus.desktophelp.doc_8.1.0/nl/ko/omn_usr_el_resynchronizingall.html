이벤트 목록 모두 재동기화

시스템 관리자가 구성 변경사항을 작성한 경우 이벤트 목록을 재동기화해야 합니다. 이벤트 목록을 모든 오브젝트에 대한 구성과 재동기화하거나 개별 오브젝트(예: 컬럼 비주얼, 경보 클래스, 사용자, 그룹, 메뉴 또는 서비스)를 선택할 수 있습니다.

이벤트 목록 모니터 상자 창에서 사용 가능한 이벤트 목록을 재동기화하려면 파일 > 재동기화를 클릭한 다음 필수 재동기화 옵션을 선택하십시오.

재동기화 프로세스는 UNIX 및 Windows 이벤트 목록에서 서로 다르게 작동합니다. UNIX 이벤트 목록에서 현재 선택되어 있는 보기는 재동기화되지 않지만 보기 빌더를 사용하여 선택된 보기에 추가된 모든 컬럼은 재동기화된 값을 반영합니다.

Windows 이벤트 목록에서 모니터 상자에 현재 선택되어 있는 보기만 재동기화됩니다. 재동기화 후 모니터 상자를 사용하여 다른 보기를 선택하는 경우 해당 보기가 재동기화되지 않습니다.

상위 주제: 이벤트 목록에 대한 공동 조치 수행
