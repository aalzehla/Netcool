IBM Tivoli Netcool/OMNIbus, 버전 7.4

임시 이벤트 목록에 대한 UNIX 명령행 옵션

nco_elct 명령을 실행하여 UNIX에서 임시 이벤트 목록을 시작하는 경우 이벤트 목록을 사용자 정의하기 위해 다수의 명령행 옵션을 지정해야 합니다. $NCHOME/omnibus/bin 디렉토리에서 이 명령을 실행할 수 있습니다.

다음 표는 UNIX에서 임시 이벤트 목록을 시작하기 위한 명령행 옵션을 설명합니다.

표 1. UNIX 임시 이벤트 목록(nco_elct) 명령행 옵션
명령행 옵션 설명
-cmap 개인용 색상 맵을 지정합니다. 이 명령행 옵션은 줄일 수 없습니다.
-diagnostic 확장된 진단 메시지를 표시합니다.
-dualwrite 1 | 0 데스크탑 ObjectServer 아키텍처에서 작동할 때 이중 쓰기 모드를 사용할지 또는 사용하지 않을지 여부를 표시합니다. 올바른 옵션은 다음과 같습니다.
  • 0: 이중 쓰기 모드 사용 안함
  • 1: 이중 쓰기 모드 사용

-dualwrite 명령행 옵션은 데스크탑 ObjectServer의 master.national 테이블에 있는 DualWrite 필드 항목을 겹쳐씁니다.

-ffile string | URL 임시 이벤트 목록에 사용할 필터 파일을 지정합니다. 이벤트 목록 필터는 .elf 확장자를 가지고 있습니다.
참고: 또한 HTTP 및 FTP를 사용하여 원격 .elf 파일을 검색할 수 있습니다. UNIX 이벤트 목록을 시작하기 위해 nco_event 명령을 실행할 때 -config 명령행 옵션을 지정하여 이 작업을 수행합니다.
-fmetric string 임시 이벤트 목록의 필터 지표를 지정합니다. -ffile 명령행 옵션을 사용하여 기존 이벤트 목록 필터 파일을 지정한 경우 이 설정이 무시됩니다.
-fname string 필터의 이름을 지정합니다. -ffile 명령행 옵션을 사용하여 기존 이벤트 목록 필터 파일을 지정한 경우 이 설정이 무시됩니다.
-ftext string 이벤트 필터링에 사용할 SQL 문을 지정합니다. -ffile 명령행 옵션을 사용하여 기존 이벤트 목록 필터 파일을 지정한 경우 이 설정이 무시됩니다.
-help 이 명령에 대한 도움말을 표시하고 종료합니다.
-masterserver string 데스크탑 ObjectServer 아키텍처에서 사용할 마스터 ObjectServer를 표시합니다.

-masterserver 명령행 옵션은 데스크탑 ObjectServer의 master.national 테이블에서 MasterServer 필드 항목을 겹쳐씁니다.

-networktimeout integer 네트워크 장애가 발생할 경우 ObjectServer와의 연결 또는 로그인 시도가 제한시간 초과되는 시간(초)을 지정합니다. 지정된 제한시간 기간이 지난 후 이벤트 목록은 ObjectServer에 다시 연결하려고 시도합니다. 두 번째 제한시간 기간 동안 연결에 실패하면, 이벤트 목록이 백업 ObjectServer(사용 가능한 경우)에 연결하려고 시도합니다. 기본값은 20초입니다.
-password string 선택한 사용자 이름에 대한 비밀번호를 지정합니다. nco_elct 유틸리티는 비밀번호의 영숫자 문자만을 채택합니다. 특수 문자(예: #, /, >, @ 및 !)는 비밀번호에 사용할 수 없습니다.
-server string 연결 중인 ObjectServer 이름을 지정합니다.
-username string 이벤트 목록에 로그인하기 위해 사용할 사용자 이름을 지정합니다.
-version 버전 정보를 표시하고 종료됩니다.
-vfile string | URL 지정된 이벤트 목록 보기를 사용하여 임시 이벤트 목록을 엽니다. 이벤트 목록 보기는 .elv 파일 확장자를 가지고 있습니다.
참고: 또한 HTTP 및 FTP를 사용하여 원격 .elv 파일을 검색할 수 있습니다. UNIX 이벤트 목록을 시작하기 위해 nco_event 명령을 실행할 때 -config 명령행 옵션을 지정하여 이 작업을 수행합니다.
팁: 임시 이벤트 목록에 대한 명령행 옵션은 고유 약어로 단축할 수 있습니다. 예를 들어, -ffile은 -ff로 단축될 수 있습니다.
상위 주제: 명령행에서 임시 이벤트 목록 시작
관련 개념:
데스크탑 ObjectServer 아키텍처
관련 참조:
