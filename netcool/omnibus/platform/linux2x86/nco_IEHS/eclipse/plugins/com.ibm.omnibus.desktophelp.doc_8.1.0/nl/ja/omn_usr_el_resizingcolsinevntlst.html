イベント・リスト内の列のサイズ変更

UNIX と Windows の イベント・リストで列のサイズを変更できます。

イベント・リスト内の列をサイズ変更するには、以下のようにします。
  1. UNIX の場合のみ、Shift+Ctrl キーを押したままにします。
  2. 列の仕切りをマウス・ボタンでクリックします。
  3. 内側または外側にマウスをドラッグして、列の幅を変更します。
