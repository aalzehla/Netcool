IBM Tivoli Netcool/OMNIbus, 버전 7.4

이벤트 목록의 인터넷 서비스 모니터

IBM Tivoli Composite Application Manager for Internet Service Monitoring은 Tivoli Netcool/OMNIbus 제품 슈트의 나머지 부분과 연계하여 작동합니다. 모니터 및 프로브 모두에 생성된 이벤트는 동일한 ObjectServer에 전달될 수 있고 동일한 데스크탑을 사용하여 이 이벤트를 조작할 수 있습니다.

Composite Application Manager for Internet Service Monitoring이 설치되면 새 단추 두 개와 드롭 다운 목록이 이벤트 목록에 추가됩니다. 다음과 같이 단추와 드롭 다운 목록을 사용할 수 있습니다.
  • 이벤트 목록에서 서비스 경보를 표시하려면 서비스 보기 단추()를 클릭하십시오. 서비스 드롭 다운 목록을 사용하여 서비스 유형을 변경할 수 있습니다.
  • 이벤트 목록에서 서비스 창을 시작하려면 서비스 표시 단추()를 클릭하십시오.
  • 서비스 드롭 다운 목록()을 사용하여 이벤트 목록에 표시되어 있는 서비스 상태 이벤트의 유형을 변경하십시오. 예를 들어, 목록에서 모니터 유형을 선택하면 해당 모니터 유형에서 서비스 상태 이벤트가 모두 표시됩니다.
  • 이벤트 목록에서 서비스 보기
    IBM Tivoli Composite Application Manager for Internet Service Monitoring이 설치되어 있는 경우 인터넷 서비스 모니터에서 서비스 경보를 보고 조작할 수 있습니다.
