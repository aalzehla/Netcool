监视事件列表中的影响服务事件

影响服务事件（SAE）是一种警报，它警告操作员一个关键客户服务已受到一个或多个网络事件的影响。影响服务事件在 IBM Tivoli Network Manager IP Edition 内生成。

如果您的系统已配置为支持该功能，那么可以通过 Tivoli Netcool/OMNIbus 事件列表监视影响服务事件。

从 Tivoli Netcool/OMNIbus 事件列表，您能以如下方式监视影响服务事件：

  • 要显示与影响服务事件关联的底层事件（例如 linkDown），请选择影响服务事件，右键单击该事件，然后从弹出菜单中单击显示 SAE 相关事件。将在新窗口中显示与选定事件关联的所有事件。
  • 要显示与某个事件（例如 linkDown）关联的影响服务事件，请选择该事件，右键单击该事件，然后从弹出菜单中单击显示 SAE 相关服务。将在新窗口中显示所有相关影响服务事件的列表。例如：如果在事件列表中选择了 linkDown 事件，那么该窗口将显示受 linkDown 事件影响的所有服务。
