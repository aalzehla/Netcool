從指令行、Script 及工具執行暫時性事件清單

您可以直接從指令行、在 Script 中或作為事件清單工具的一部分執行自訂的暫時性事件清單。

例如，您可以開啟事件清單，以套用過濾器來檢視特定 ObjectServer 的所有重要警示。

  • 從指令行啟動暫時性事件清單
    使用 nco_elct 指令，可以從 UNIX 指令行開啟暫時性事件清單，且使用 NCOelct.exe 指令，可以從 Windows 指令行開啟暫時性事件清單。

    這些範例會顯示如何在 UNIX 及 Windows 上，直接從指令行或從 Script 執行暫時性事件清單。
