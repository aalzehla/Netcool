檢視事件清單中的服務

如果您已安裝 IBM Tivoli Composite Application Manager for Internet Service Monitoring，則可以從網際網路服務監視器檢視及操作服務警示。

若要檢視服務資訊，請執行下列任何動作：
  • 從「管理者」中，開啟事件清單。 當已安裝 Composite Application Manager for Internet Service Monitoring 時，每當您從「管理者」存取監視器框視窗時，就會自動開啟「服務」視窗。
  • 從「事件清單」監視器框視窗中，按一下視窗 > 服務。
  • 從事件清單中，按一下工具列中的顯示服務視窗。

結果

即會開啟「服務」視窗。

「服務」視窗中的每列都會指出具有下列資訊的不同服務：
顏色符號
每列左側的顏色符號指出 ObjectServer 從該服務接收的最後一個警示的狀態。 符號以顏色編碼，如下所示：
  • 紅色：不符合服務水準協定。
  • 琥珀色：服務發生一些問題。
  • 綠色：服務沒有問題。
  • 黑色：服務狀態不明。
服務
此直欄會顯示受監視的服務名稱。
前次變更
此直欄會顯示前次變更服務狀態的日期及時間。
前次正確
此直欄會顯示前次服務良好的日期及時間。
前次錯誤
此直欄會顯示前次服務不正確的日期及時間。
前次最低限度
此直欄會顯示前次服務不重要的日期及時間。
前次報告
此直欄會顯示前次產生服務報告的日期及時間。

按兩下任何列，以開啟事件清單視窗，其中包含該服務類型的警示清單。 您可以依照從探測器接收警示的相同方法，來操作事件清單中的服務警示。

按一下關閉，以關閉「服務」視窗。
