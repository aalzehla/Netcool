IBM Tivoli Netcool/OMNIbus 7.4 版

存取桌面工具

您可以使用「管理者」，以管理 Tivoli Netcool/OMNIbus 桌面工具 (或用戶端應用程式)。

您可以使用「管理者」，以執行下列作業：
  • 執行所有桌面工具，而無需在每次啟動工具時登入。
  • 設定所有桌面工具的喜好設定。
  • 透過使用「伺服器編輯器」，設定 Tivoli Netcool/OMNIbus 伺服器元件的通訊。
  • 變更密碼。
  • 維護儲存常用 Tivoli Netcool/OMNIbus 檔案 (如事件清單配置、過濾器及視圖) 的個人檔案庫。
  • 瀏覽目錄以選取要新增至個人檔案庫或「管理者」按鈕列的檔案。
  • 存取系統管理者定義的工具，以管理事件清單中的事件。
  • 啟動管理者
    「管理者」會提供對所有 Tivoli Netcool/OMNIbus 桌面工具的單一登入存取，您可以使用這些工具來檢視及管理警示。
  • 變更 ObjectServer 上的密碼
    如果針對 ObjectServer 以外的任何外部來源對您進行鑑別，則您必須始終要求外部來源的系統管理者變更密碼。
  • 設定 UNIX 管理者的廣域喜好設定
    從「UNIX 管理者」中，您可以設定「管理者」視窗、事件清單及事件通知的廣域喜好設定。
  • 設定 Windows 管理者的廣域喜好設定
    從「Windows 管理者」中，您可以設定事件清單及警示通知的廣域喜好設定。
  • 在 UNIX 上存取線上說明
    線上說明使用「IBM® Eclipse 說明系統 (IEHS)」（其為 Web 應用程式）進行部署。基於安裝及配置設定，您可以獨立模式或資訊中心模式存取線上說明。
  • 在 Windows 上存取線上說明
    線上說明使用「IBM Eclipse 說明系統 (IEHS)」（其為 Web 應用程式）進行部署。基於安裝及配置設定，您可以獨立模式或資訊中心模式存取線上說明。
  • 停止管理者
    當在 UNIX 或 Windows 上關閉「管理者」時，任何使用「管理者」啟動的工具都會保持開啟狀態。當所有其他開啟的視窗不再需要時，您必須手動關閉它們。
