IBM Tivoli Netcool/OMNIbus 7.4 版

配置內容、伺服器通訊及設定

您可以從「UNIX 管理者」及「Windows 管理者」配置 Tivoli Netcool/OMNIbus 元件的內容，以及伺服器元件的通訊資訊。在 Windows 上，您可以在設定視窗內的合併作業中執行這兩項功能。在 UNIX 上，您也可以變更事件清單中的日期格式。

  • 配置內容
    您可以編輯 ObjectServer、Proxy 伺服器、資料庫起始設定公用程式 (nco_dbinit)、Confpack 公用程式 (nco_confpack)、「Netcool/OMNIbus 管理者」(nco_config)、探測器及閘道之內容檔中所含的資訊。 內容檔的副檔名為 .props。
  • 配置伺服器通訊資訊
    您必須維護 Tivoli Netcool/OMNIbus 伺服器元件的伺服器通訊資訊。 您可以使用「伺服器編輯器」，以建立及修改伺服器元件的通訊明細、測試伺服器活動，以及新增備份 (或失效接手) 伺服器與接聽器。
  • 在 Windows 上配置 Tivoli Netcool/OMNIbus 設定
    在單一作業中，您可以配置 Windows 上 Tivoli Netcool/OMNIbus 的內容及伺服器通訊資訊。
  • 變更 UNIX 上事件清單中的日期格式
    事件清單中的日期格式由 NCO_TIME 環境變數定義。 預設格式為 mm/dd/yy hh:mm:ss；例如，11/12/03 20:13:36。 此格式由 POSIX strptime 函數決定。
