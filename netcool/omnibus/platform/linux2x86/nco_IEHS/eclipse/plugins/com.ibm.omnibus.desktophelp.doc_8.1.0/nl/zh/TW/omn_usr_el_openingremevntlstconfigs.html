開啟遠端事件清單配置

您可以開啟遠端事件清單配置檔案，以開啟位於遠端伺服器上的一或多個事件清單。 伺服器必須可以使用 HTTP 或 FTP 存取。

若要開啟遠端事件清單配置檔案：
  1. 從「事件清單」監視器框視窗中，按一下檔案 > 開啟。 即會開啟「開啟配置」視窗。
  2. 如下所示完成此視窗：在檔名欄位中，輸入遠端檔案的網址，然後按一下確定。 您必須使用下列其中一種網址格式：

    http://servername[:port]/path/filename.elc

    http://username:password@servername[:port]/path/filename.elc

    ftp://servername[:port]/path/filename.elc

    ftp://username:password@servername[:port]/path/filename.elc

    僅當伺服器未使用預設埠時，才需要指定選用埠號 :port。 若為 HTTP，Web 伺服器必須正在遠端主機上執行。

    您可以針對 servername 值選擇性地指定 IPv4 或 IPv6 位址。 當指定 IPv6 位址時，請將位址含括在方括弧 [ ] 內；例如：

    http://[2003:13f8:206:190:204:baef:fe6a:6af0]:2880/mydir/myfile.elc

    若為 FTP，則路徑是從起始目錄開始的相對路徑；例如，/mydir/myconfig.elc。

    遠端檔案的副本會寫入本端快取。 在 Windows 系統上，當您指定先前已開啟的遠端檔案時，會使用已快取的副本。如果無法擷取遠端檔案，則會顯示錯誤訊息。

結果

如果擷取成功，則「事件清單」監視器框視窗會重新整理，以顯示該配置內的監視器框，且所選的配置檔案名稱會顯示在事件清單的標題列中。
