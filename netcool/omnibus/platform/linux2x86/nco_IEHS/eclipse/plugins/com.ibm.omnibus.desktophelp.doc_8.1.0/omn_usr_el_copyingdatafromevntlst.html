Copying data from the event list

You can copy event data from the event list to the clipboard for use in other applications.

About this task

To copy data:

Procedure

  1. From the event list, select an event field. A black box surrounds the field.
  2. Click Edit > Copy to Clipboard.

What to do next

You can then paste this information into another application.
Tip: You can also copy alert information from the Event Information window.
