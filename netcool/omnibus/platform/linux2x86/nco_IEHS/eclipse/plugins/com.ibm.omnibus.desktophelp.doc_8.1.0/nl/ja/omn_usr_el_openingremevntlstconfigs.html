リモート・イベント・リスト構成を開く

リモート・サーバーに配置された 1 つ以上のイベント・リストを開くために、リモート・イベント・リスト構成ファイルを開くことができます。 HTTP または FTP を使用してサーバーにアクセス可能である必要があります。

リモート・イベント・リスト構成ファイルを開くには、以下のようにします。
  1. 「イベント・リスト」モニター・ボックス・ウィンドウから、「ファイル」 > 「開く」をクリックします。 「構成を開く」ウィンドウが開きます。
  2. このウィンドウに以下のように入力します。「ファイル名」フィールドにリモート・ファイルの Web アドレスを入力し、「OK」をクリックします。 Web アドレスのフォーマットには、以下のいずれかを使用する必要があります。

    http://servername[:port]/path/filename.elc

    http://username:password@servername[:port]/path/filename.elc

    ftp://servername[:port]/path/filename.elc

    ftp://username:password@servername[:port]/path/filename.elc

    サーバーがデフォルトのポートを使用していない場合にのみ、オプションのポート番号 :port を 指定する必要があります。 HTTP の場合、 Web サーバーはリモート・ホスト上で稼働している必要があります。

    オプションで、servername 値の IPv4 または IPv6 アドレスを指定することもできます。 IPv6 アドレスを指定する場合は、例えば以下のように、アドレスを大括弧 [] で囲みます。

    http://[2003:13f8:206:190:204:baef:fe6a:6af0]:2880/mydir/myfile.elc

    FTP を使用する場合、パスはホーム・ディレクトリーからの相対パスである必要があります。例えば、/mydir/myconfig.elc です。

    リモート・ファイルのコピーが、ローカル・キャッシュに 書き込まれます。 Windows システムの場合、以前に開いたリモート・ファイルを指定すると、キャッシュされたコピーが使用されます。 リモート・ファイルが検索できない場合は、 エラー・メッセージが表示されます。

タスクの結果

検索が成功した場合は「イベント・リスト」モニター・ボックス・ウィンドウが最新表示され、 その構成内にモニター・ボックスが表示されます。さらに、イベント・リストのタイトル・バーに選択された構成ファイル名が表示されます。
