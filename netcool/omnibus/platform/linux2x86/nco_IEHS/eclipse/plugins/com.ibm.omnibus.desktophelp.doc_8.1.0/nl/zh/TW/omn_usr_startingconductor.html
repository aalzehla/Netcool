IBM Tivoli Netcool/OMNIbus 7.4 版

啟動管理者

「管理者」會提供對所有 Tivoli Netcool/OMNIbus 桌面工具的單一登入存取，您可以使用這些工具來檢視及管理警示。

若要使用桌面工具，您必須透過使用有效的鑑別認證，連接到執行中的 ObjectServer。

提示： 或者，您可以從 UNIX 中的指令行或從 Windows 開始功能表，個別啟動桌面工具。然而，每次您以此方法啟動桌面工具時，都需要輸入使用者名稱及密碼，以進行鑑別並連接到選取的 ObjectServer。

若要啟動「管理者」：

  1. 針對作業系統執行適當的動作：
    選項 敘述
    UNIX 從命令提示字元中，輸入：$NCHOME/omnibus/bin/nco &
    Windows
    • 從 Windows 開始功能表中，按一下 Netcool 管理者。
    • 從命令提示字元中，輸入：

      %NCHOME%\omnibus\desktop\NCOConductor.exe
    即會開啟「管理者」的「登入」視窗。
  2. 如下所示完成此視窗：
    使用者名稱
    輸入您的使用者名稱。
    密碼
    輸入您的密碼。 會在您輸入文字的位置顯示星號 (*) 字元。
    註： 如果是針對外部鑑別來源（而不是 ObjectServer）驗證您的使用者名稱及密碼，請檢查此來源是否支援多位元組字元。如果不支援多位元組字元，您必須使用 ASCII 字元指定使用者名稱及密碼。
    伺服器
    ObjectServer 名稱預設為 NCOMS（Netcool/OMNIbus 主要伺服器）。 如果已針對您的環境配置多個 ObjectServer，則所有可用的 ObjectServer 都會顯示在此下拉清單中。選取您要連接至的 ObjectServer。
    確定
    按一下此按鈕，以鑑別您的登入詳細資料。
    註： 允許您嘗試登入三次。 每次嘗試時，都會清除密碼欄位。 如果您在第三次嘗試中仍無法正確登入，程式即會終止。
    取消
    按一下此按鈕，以取消登入嘗試。

結果

在成功登入時：
  • 「UNIX 管理者」以一個視窗開啟。
  • 「Windows 管理者」會在 Windows 系統匣中呈現為匯流排圖示。用滑鼠右鍵按一下此圖示，以檢視 Windows 管理者功能表。

    您可以針對每個執行中的 ObjectServer 啟動「管理者」。 在 Windows 上，每個「管理者」都會在系統匣中配置一個圖示。
