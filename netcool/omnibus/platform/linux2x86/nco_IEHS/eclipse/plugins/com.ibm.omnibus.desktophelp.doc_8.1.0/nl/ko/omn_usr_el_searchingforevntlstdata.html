IBM Tivoli Netcool/OMNIbus, 버전 7.4

이벤트 목록 데이터 검색

여러 방법으로 이벤트 목록 데이터를 검색할 수 있습니다. 검색할 특정 텍스트를 입력하고, 필터링 기능을 사용하여 데이터 발생과 일치하는 것을 신속하게 찾고, 심각도에 따라 이벤트 목록 데이터를 필터링할 수 있습니다.

  • 이벤트 목록에서 텍스트 검색
    이벤트 목록에서 사용자가 입력하는 특정 값과 일치하는 이벤트 데이터를 검색할 수 있습니다. 특정 컬럼에 있는 데이터를 검색하고 검색을 좁히는 옵션을 지정할 수 있습니다.
  • 신속하게 이벤트 필터링
    선택한 기준과 일치하는 이벤트를 빨리 표시하는 방법으로 빠른 필터링 기능을 사용할 수 있습니다. 예를 들어, 선택한 이벤트와 동시에 또는 선택한 이벤트 이전에 발생된 이벤트만 신속히 표시할 수 있습니다.
  • 심각도별 이벤트 필터링
    특정 심각도와 일치하는 이벤트만 표시하기 위해 이벤트 목록 데이터를 필터링할 수 있습니다.
