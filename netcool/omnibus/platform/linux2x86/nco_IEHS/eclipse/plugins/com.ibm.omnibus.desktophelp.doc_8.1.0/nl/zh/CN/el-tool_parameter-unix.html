工具参数

使用此窗口可向主机工作站发出 Ping 命令，或者可使用 Telnet 实用程序连接到远程主机工作站。 如果系统管理员已创建用于事件管理的工具，那么您还可以从该窗口运行其中一些工具。

如果您在运行 Ping 或 Telnet 工具，请如下所示完成此窗口。如果您在运行定制工具，此窗口上的字段将有所不同。
主机名称
输入要访问的工作站的 IP 地址或主机名。
确定
单击此按钮可运行命令。
如果运行了 Ping 工具，将在“工具输出”窗口中返回节点状态。 如果运行 Telnet 工具，将连接到远程主机工作站。
取消
单击此按钮可关闭该窗口而不发出命令。
相关任务:
发出 Ping 命令
发出 Telnet 命令
