IBM Tivoli Netcool/OMNIbus, 버전 7.4

Ping 명령 실행

Ping 도구는 alerts.status 테이블의 노드 컬럼 값에 대해 Ping 명령을 실행합니다.

Ping 명령을 실행하기 위해 이벤트 목록에서 다음 조치 중 하나를 수행하십시오.
  • ping하려는 노드를 포함하는 필수 행을 선택하십시오. 그 다음 경보 > 도구 > Ping을 클릭하거나 마우스 오른쪽 단추를 클릭한 다음 도구 > Ping을 클릭하십시오. 노드의 상태는 도구 출력 창 안에서 리턴됩니다.
  • 특정 호스트에 대해 ping 명령을 실행하려면 도구 > Ping을 클릭하십시오. 도구 매개변수 창(UNIX) 또는 내부 명령 매개변수 창(Windows)이 열립니다. 다음과 같이 이 창을 완료하십시오.
    호스트 이름
    액세스하려는 워크스테이션의 IP 주소 또는 호스트 이름을 입력하십시오.
    확인
    명령을 실행하려면 이 단추를 클릭하십시오.
    Ping 도구를 실행한 경우 도구 출력 창에서 노드의 상태를 리턴합니다. Telnet 도구를 실행한 경우 원격 호스트 워크스테이션에 연결됩니다.
    취소
    명령을 실행하지 않고 이 창을 닫으려면 이 단추를 클릭하십시오.
