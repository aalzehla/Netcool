IBM Tivoli Netcool/OMNIbus バージョン 7.4

イベント・リスト内のインターネット・サービス・モニター

IBM Tivoli Composite Application Manager for Internet Service Monitoring は、その他の Tivoli Netcool/OMNIbus 製品スイートと連動して作業を行います。 モニターとプローブの両方で生成されたイベントは、同じ ObjectServer に転送されます。 これらのイベントの操作は、同じデスクトップを使用して行うことができます。

Composite Application Manager for Internet Service Monitoring がインストールされると、2 つの新規ボタンとドロップダウン・リストがイベント・リストに追加されます。 このボタンおよびドロップダウン・リストは、以下のように使用します。
  • 「サービス・ビュー」ボタン をクリックすると、イベント・リストにサービス・アラートが表示されます。 「サービス」ドロップダウン・リストを使用して、サービス・タイプを変更できます。
  • イベント・リストから「サービス」ウィンドウを始動するには、「サービスを表示」ボタン をクリックします。
  • 「サービス」ドロップダウン・リスト は、イベント・リストに表示されるサービス状況イベントのタイプを変更するために使用します。 例えば、リストからモニター・タイプを選択すると、そのモニター・タイプのすべてのサービス状況イベントが表示されます。
  • イベント・リスト内のサービスの表示
    IBM Tivoli Composite Application Manager for Internet Service Monitoring がインストールされている場合は、インターネット・サービス・モニターからサービス・アラートを表示して操作できます。
