Freezing an event list

You can freeze any open event list so that updates from the ObjectServer are not displayed.

About this task

To freeze an event list, click View > Freeze from that event list window.

To restart automatic updates from the ObjectServer, click View > Freeze again.

Tip: You can also freeze and unfreeze all open event lists, by clicking View > Freeze All from the Event List monitor box window.
