IBM Tivoli Netcool/OMNIbus, 버전 7.4

Windows에서 이벤트 목록 구성 작성 및 편집

이벤트 목록 모니터 상자 창에 표시되는 모니터 상자에 대한 필터 및 보기 정의를 보유하는 이벤트 목록 구성을 작성하거나 편집할 수 있습니다.

이벤트 목록 모니터 상자 창에 표시되어야 하는 모니터 상자 컬럼 수를 지정하고 연관된 이벤트 목록에 적용해야 하는 필터 및 보기를 지정하며, 필터 및 보기 표시 순서를 지정할 수 있습니다.

Windows에서 이벤트 목록 구성을 작성하거나 편집하려면 다음을 수행하십시오.

  1. 이벤트 목록 모니터 상자 창에서 다음 두 조치 중 하나를 수행하십시오.
    • 새 이벤트 목록 구성을 작성 중인 경우, 현재 이벤트 목록 구성을 지우고 구성을 기본 설정으로 설정하십시오. 이를 수행하려면 파일 > 새로 작성을 클릭하십시오.
    • 이벤트 목록 모니터 상자 창에 표시되는 기존 이벤트 목록 구성을 편집 중인 경우, 2단계로 이동하십시오.
  2. 창 > 구성을 클릭하십시오. 이벤트 목록 구성 창이 열립니다.
  3. 다음과 같이 이 창을 완료하십시오.
    이름
    새 이벤트 목록 구성을 작성하는 경우 이벤트 목록 구성에 대한 고유 이름을 입력하십시오.
    팁: 도구 모음의 스핀 상자를 사용하여 이벤트 목록 모니터 상자 창에서 허용할 수 있는 모니터 상자 컬럼 수를 지정하십시오.
    필터 탭
    필터를 이 이벤트 목록 구성의 파트로 지정하려면 이 탭을 클릭하십시오. 선택한 필터가 이 탭의 목록에 표시됩니다. 새로 작성 단추를 사용하여 이벤트 목록 구성에 새 필터를 추가할 수 있고, 열기 단추를 사용하여 기존 필터를 이벤트 목록 구성에 로드할 수 있으며, 편집 단추를 사용하여 이벤트 목록 구성의 기존 필터를 편집할 수 있습니다. 또한 삭제 단추를 사용하여 필터를 제거할 수 있습니다.
    참고: 탭에 표시되는 필터의 순서는 이벤트 목록 팝업 메뉴 및 드롭 다운 목록에 필터 목록이 표시되는 순서를 결정합니다. 맨 위, 위로, 아래로 및 맨 아래 화살표 단추를 사용하여 이 순서를 변경할 수 있습니다.
    보기 탭
    이 이벤트 목록 구성의 파트인 보기를 지정하려면 이 탭을 클릭하십시오. 선택한 보기가 이 탭의 목록에 표시됩니다. 새로 작성 단추를 사용하여 이벤트 목록 구성에 새 보기를 추가할 수 있고, 열기 단추를 사용하여 기존 보기를 이벤트 목록 구성에 로드할 수 있으며, 편집 단추를 사용하여 이벤트 목록 구성의 기존 보기를 편집할 수 있습니다. 또한 삭제 단추를 사용하여 보기를 제거할 수 있습니다.
    참고: 탭에 표시되는 보기의 순서는 이벤트 목록 팝업 메뉴 및 드롭 다운 목록에 필터 목록이 표시되는 순서를 결정합니다. 맨 위, 위로, 아래로 및 맨 아래 화살표 단추를 사용하여 이 순서를 변경할 수 있습니다.
    새로 작성
    필터 또는 보기를 새로 작성하려면 도구 모음으로 이동하여 이 단추를 클릭해 필터 빌더 또는 보기 빌더를 여십시오.
    주의: 필터 빌더 또는 보기 빌더 내에서 이벤트 목록 구성에 이미 있는 필터 또는 보기 이름을 지정하는 경우 기존 필터 또는 보기가 바뀝니다.
    열기
    이벤트 목록 구성에 기존 필터 또는 보기를 로드하려면 도구 모음으로 이동하여 이 단추를 클릭하십시오. 결과 창에서 필터 또는 보기가 저장되어 있는 위치를 탐색하고 필터 또는 보기를 선택한 다음 확인을 클릭하십시오. 필터 또는 보기가 이벤트 목록 구성 창의 관련 탭에 추가됩니다.
    팁: 필터는 .elf 확장자를 가지고 있으며, 보기는 .elv 확장자를 가지고 있습니다.
    편집
    필터 또는 보기를 편집하려면 탭 목록에서 해당 필터 또는 보기를 선택한 다음 도구 모음으로 이동하고 이 단추를 클릭해 선택한 필터 또는 보기에 대한 필터 빌더 또는 보기 빌더를 여십시오. 필터 또는 보기의 이름을 이벤트 목록 구성에 없는 이름으로 변경하는 경우 필터 또는 보기가 새로 작성됩니다. 이미 있는 이름으로 변경하는 경우 새 필터 또는 보기가 이벤트 목록 구성의 기존 필터 또는 보기와 바뀝니다.
    삭제
    이벤트 목록 구성에서 필터 또는 보기를 삭제하려면 탭 목록에서 해당 필터 또는 보기를 선택한 다음 이 단추를 클릭해 선택한 필터 또는 보기를 삭제하십시오. 삭제를 확인하는 프롬프트가 없으므로 삭제를 클릭하기 전 반드시 해당 필터 또는 보기를 선택했는지 확인하십시오.
    맨 위
    필터 또는 보기를 목록의 맨 위로 이동하려면 인접 목록에서 해당 필터 또는 보기를 선택한 다음 이 단추를 클릭하십시오.
    위로
    필터 또는 보기를 목록에서 한 위치 위로 이동하려면 인접 목록에서 해당 필터 또는 보기를 선택한 다음 이 단추를 클릭하십시오.
    아래로
    필터 또는 보기를 목록에서 한 위치 위로 이동하려면 인접 목록에서 해당 필터 또는 보기를 선택한 다음 이 단추를 클릭하십시오.
    맨 아래
    필터 또는 보기를 목록의 맨 아래로 이동하려면 인접 목록에서 해당 필터 또는 보기를 선택한 다음 이 단추를 클릭하십시오.
    닫기
    이벤트 목록 구성 창을 닫고 이벤트 목록 구성을 현재 이벤트 목록에 적용하려면 이 단추를 클릭하십시오.
  4. 이벤트 목록 구성을 저장하려면, 모니터 상자 창에서 파일 > 다른 이름으로 저장을 클릭하십시오. 구성 저장 창이 열립니다.
  5. 다음과 같이 이 창을 완료하십시오.
    필터
    열려는 파일 유형에 대한 검색 조회를 지정하려면 이 필드를 사용하십시오. 이 필드는 필터 단추 및 파일 목록과 함께 작동합니다. 필터 필드에 현재 디렉토리 경로와 파일 유형을 입력한 후 필터를 클릭하십시오. 일치하는 파일이 모두 파일 목록에 표시됩니다.
    예를 들어, /myfiles/filters 디렉토리의 모든 필터 파일을 표시하려면 /myfiles/filters/*.elf를 입력하고 필터를 클릭하여 일치하는 파일을 모두 확인하십시오.
    디렉토리
    이 목록은 현재 디렉토리의 모든 서브디렉토리를 표시합니다. 디렉토리를 새로 고치려면 '.' 항목을 두 번 클릭하십시오. 디렉토리 트리를 하나의 디렉토리로 이동시키려면 '..' 항목을 클릭하십시오.
    파일
    이 목록은 현재 디렉토리의 파일을 표시하며, 이는 필터 필드의 선택사항과 일치합니다. 열려는 파일 이름을 선택하십시오. 겹쳐쓰려는 기존 파일도 선택할 수 있습니다.
    선택사항
    저장하거나 여는 중인 파일의 경로와 이름은 여기에 표시됩니다. 다른 파일을 열거나, 파일을 다른 위치에서 저장하거나, 다른 파일 이름을 사용하여 파일을 저장하려면 경로와 파일 이름을 편집할 수 있습니다.
    보기를 저장 중이면 파일 이름의 .elv 확장자를 사용하십시오. 필터를 저장 중이면 파일 이름의 .elf 확장자를 사용하십시오. 이벤트 목록 구성을 저장 중이면 파일 이름의 .elc 확장자를 사용하십시오.
    백업 기존 파일(UNIX만 해당)
    저장 중인 파일 이름이 이미 존재하는 경우 기존 보기 파일의 백업을 작성하려면 이 선택란을 선택하십시오. 이 선택란은 사용자가 필터, 보기 또는 이벤트 목록 구성을 저장하는 경우에만 표시됩니다.
    확인
    파일을 저장하거나 열려면 이 단추를 클릭합니다.
    필터
    이 단추는 필터 필드 및 파일 목록과 함께 작동합니다. 필터 필드의 선택사항을 기반으로 현재 디렉토리의 파일을 표시하려면 이 단추를 클릭하십시오. 파일 목록에 파일이 나타납니다.
    취소
    파일을 열거나 저장하지 않고 창을 닫으려면 이 단추를 클릭하십시오.
