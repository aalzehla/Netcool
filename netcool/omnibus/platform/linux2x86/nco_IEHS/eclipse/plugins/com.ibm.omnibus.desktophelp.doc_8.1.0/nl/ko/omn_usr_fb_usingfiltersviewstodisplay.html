필터 및 보기를 사용한 사용자 정의 이벤트 정보 표시

경보 데이터베이스 테이블은 ObjectServer에 전달되는 이벤트 정보를 보유합니다. 필터 및 보기를 사용하여 이 이벤트 정보 중 이벤트 목록에 표시되는 이벤트 정보를 정의할 수 있습니다.

필터는 볼 수 있는 경보(또는 데이터베이스 레코드)를 제어하고 보기는 해당 경보에 대해 볼 수 있는 컬럼(또는 데이터베이스 필드)을 제어합니다. SQL(Structured Query Language)을 사용하여 데이터베이스 테이블에 대한 정보 요청을 수행할 수 있습니다.

이벤트 목록 안에 경보를 표시하려면 alerts.status 테이블의 SELECT 문을 수행해야 합니다. SQL 문은 세 개의 개념 파트로 구분할 수 있습니다.
  • 보기 빌더에서 작성된 보기를 사용하여 alerts.status 테이블에서 보려는 컬럼을 선택하십시오.
  • 필터 빌더에서 작성된 필터를 사용하여 해당 컬럼 중 하나 이상의 컬럼에 특정 데이터 값을 포함하는 이벤트를 선택하십시오.
  • 보기 빌더에서 작성된 보기를 사용하여 이벤트를 표시하기 위한 순서를 선택하십시오.

다음 예제는 alerts.status 테이블에서 수행되는 SELECT 문을 표시합니다. 

select Node as NODE, Summary as Detail from alerts.status
where Node ='link1'

SELECT 절은 보기를 작성 및 편집하기 위해 사용되는 보기 빌더 안에 구성되어 있습니다. WHERE 절은 필터를 작성하고 편집하기 위해 사용되는 필터 빌더 안에 구성되어 있습니다.

  • 이벤트 목록 데이터에 대한 필터 설정
    필터는 모니터 상자 및 이벤트 목록에 사용자 정의 경보 정보를 표시하기 위해 ObjectServer alerts 데이터베이스 테이블에서 실행할 수 있는 SQL 조회를 빌드하는 데 사용됩니다.
  • 이벤트 목록에 대한 보기 설정
    보기는 이벤트 목록에 표시되는 컬럼의 선택사항과 이 컬럼 안에 있는 정보의 포맷팅을 정의합니다.
