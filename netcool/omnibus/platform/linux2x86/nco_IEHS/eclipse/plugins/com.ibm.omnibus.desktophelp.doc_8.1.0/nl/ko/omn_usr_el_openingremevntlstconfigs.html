원격 이벤트 목록 구성 열기

원격 서버에 있는 하나 이상의 이벤트 목록을 열기 위해 원격 이벤트 목록 구성 파일을 열 수 있습니다. HTTP 또는 FTP를 사용하여 서버에 액세스할 수 있어야 합니다.

원격 이벤트 목록 구성 파일을 열려면 다음을 수행하십시오.
  1. 이벤트 목록 모니터 상자 창에서 파일 > 열기를 클릭하십시오. 구성 열기 창이 열립니다.
  2. 다음과 같이 이 창을 완료하십시오. 파일 이름 필드에 원격 파일의 웹 주소를 입력하고 확인을 클릭하십시오. 웹 주소에 다음 형식 중 하나를 사용해야 합니다.

    http://servername[:port]/path/filename.elc

    http://username:password@servername[:port]/path/filename.elc

    ftp://servername[:port]/path/filename.elc

    ftp://username:password@servername[:port]/path/filename.elc

    서버가 기본 포트를 사용하고 있지 않은 경우에만 선택적 포트 번호(:port)를 지정해야 합니다. HTTP의 경우, 웹 서버가 원격 호스트에서 실행 중이어야 합니다.

    servername 값에 대해 IPv4 또는 IPv6 주소를 선택하여 지정할 수 있습니다. IPv6 주소를 지정하면 주소가 대괄호([]) 안에 묶입니다. 예제는 다음과 같습니다.

    http://[2003:13f8:206:190:204:baef:fe6a:6af0]:2880/mydir/myfile.elc

    FTP의 경우, 경로는 홈 디렉토리의 상대 경로입니다(예: /mydir/myconfig.elc).

    원격 파일 사본이 로컬 캐시에 기록됩니다. Windows 시스템에서는 이전에 열었던 원격 파일을 지정하면 캐시된 사본이 사용됩니다. 원격 파일을 검색할 수 없으면 오류 메시지가 표시됩니다.

결과

검색에 성공하면 이벤트 목록 모니터 상자 창이 새로 고쳐져 해당 구성 내의 모니터 상자를 표시하며, 선택된 구성 파일 이름이 이벤트 목록 제목 표시줄에 표시됩니다.
