使用過濾器及視圖來顯示自訂事件資訊

警示資料庫表格會保留轉遞到 ObjectServer 的事件資訊。 您可以使用過濾器及視圖，以定義在事件清單中顯示的事件資訊。

過濾器會控制可以查看的警示 (或資料庫記錄)，而視圖會控制那些警示可見的直欄 (或資料庫欄位)。 您可以使用「結構化查詢語言 (SQL)」對資料庫表格執行資訊要求。

若要在事件清單中顯示警示，則必須對 alerts.status 表格執行 SELECT 陳述式。 SQL 陳述式可分為三個概念性部分：
  • 透過使用「視圖建置器」中建立的視圖，選取 alerts.status 表格中您要檢視的直欄。
  • 透過使用「過濾器建置器」中建立的過濾器，選取在一或多個那些直欄中包含特定資料值的事件。
  • 透過使用「視圖建置器」中建立的視圖，選取顯示事件的順序。

下列範例會顯示在 alerts.status 表格上執行的 SELECT 陳述式： 

select Node as NODE, Summary as Detail from alerts.status
where Node ='link1'

SELECT 子句在「視圖建置器」中建構，可用於建立及編輯視圖。 WHERE 子句在「過濾器建置器」中建構，可用於建立及編輯過濾器。

  • 設定事件清單資料的過濾器
    過濾器用於建置可在 ObjectServer 警示資料庫表格上執行的 SQL 查詢，以便在監視器框及事件清單中顯示自訂警示資訊。
  • 設定事件清單的視圖
    視圖會定義事件清單中顯示的直欄選項，以及這些直欄中資訊的格式。
