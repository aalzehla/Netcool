IBM Tivoli Netcool/OMNIbus V7.4

访问桌面工具

您可使用 Conductor 来管理 Tivoli Netcool/OMNIbus 桌面 工具（或客户机应用程序）。

您可使用 Conductor 执行以下任务：
  • 运行所有桌面工具，而无需每次运行时都登录。
  • 为所有桌面工具设置首选项。
  • 使用服务器编辑器为 Tivoli Netcool/OMNIbus 服务器组件设置通信。
  • 更改密码
  • 维护用于存储常用 Tivoli Netcool/OMNIbus 文件（如事件列表配置、过滤器和视图）的个人库。
  • 浏览目录以选择要添加到个人库或 Conductor 按钮栏中的文件。
  • 访问系统管理员定义用于管理事件列表中各事件的工具。
  • 启动 Conductor
    Conductor 向所有 Tivoli Netcool/OMNIbus 桌面工具提供了单点登录访问，以用于查看和管理警报。
  • 更改 ObjectServer 上的密码
    如果针对外部源而非 ObjectServer 进行了认证，那么就必须一直从外部源的系统管理员处请求密码更改。
  • 设置 UNIX Conductor 的全局首选项
    从 UNIX Conductor 可以设置 Conductor 窗口、事件列表以及事件通知的全局首选项。
  • 设置 Windows Conductor 的全局首选项
    从 Windows Conductor 可以设置事件列表和警报通知的全局首选项。
  • 在 UNIX 上访问联机帮助
    联机帮助是使用 IBM® Eclipse 帮助系统（IEHS）（一种 Web 应用程序）进行部署的。根据您的安装和配置设置，您能以单机方式或信息中心方式访问联机帮助。
  • 在 Windows 上访问联机帮助
    联机帮助是使用 IBM Eclipse 帮助系统（IEHS）（一种 Web 应用程序）进行部署的。根据您的安装和配置设置，您能以单机或信息中心方式访问联机帮助。
  • 停止 Conductor
    当关闭 UNIX 或 Windows 上的 Conductor 时，任何以 Conductor 启动的工具都将保持打开状态。当其他已打开的窗口不再需要时，必须手动将它们关闭。
