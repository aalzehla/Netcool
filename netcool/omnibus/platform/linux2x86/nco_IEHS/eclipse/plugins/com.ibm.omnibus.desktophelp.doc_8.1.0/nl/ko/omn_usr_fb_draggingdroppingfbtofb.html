IBM Tivoli Netcool/OMNIbus, 버전 7.4

한 필터 빌더에서 다른 필터 빌더로 끌어놓기

UNIX에서 필터를 작성하는 빠른 방법은 한 열린 필터 빌더에서 다른 열린 필터 빌더로 요소를 끌어놓는 것입니다.

한 필터 빌더에서 다른 필터 빌더로 끌어놓으려면 다음을 수행하십시오.
  1. 커서를 편집 가능 선택란의 오른쪽에 위치한 끌기 위치 아이콘 위로 이동하십시오.

  2. 아이콘 위에서 마우스 가운데 단추를 클릭한 채로 유지하십시오.
  3. 마우스를 다른 필터 빌더로 끌어 이동하십시오. 커서가 올바른 놓기 영역에 들어오면 색상이 녹색으로 변경됩니다.
  4. 마우스 단추를 해제하여 필터를 놓으십시오.

    필터가 올바르지 않거나 완료되지 않은 경우 끌기 위치 아이콘에 빨간 십자가가 표시되고 끌어놓을 수 없습니다.
