更改 ObjectServer 上的密码

如果针对外部源而非 ObjectServer 进行了认证，那么就必须一直从外部源的系统管理员处请求密码更改。

rev="v731r1"ObjectServer 服务器用户的密码不能以任何下列字符开头：连字符 (-)、正斜杠 (/) 或双引号 (")。

要更改 ObjectServer 上的密码：

  1. 为操作系统执行相应的操作：
    选项 描述
    UNIX 从 UNIX 上的“Conductor”窗口，单击文件 > 更改密码。
    注： 在 UNIX 上，如果您的组织使用外部授权，那么更改密码菜单选项不可用。 这是因为 ObjectServer 内未存储外部认证密码。
    Windows 从 Windows 上的 Conductor 菜单，单击密码。
    此时将打开“更改密码”窗口。
  2. 如下所示完成此窗口：
    Current®密码
    输入当前的密码。 输入的任何文本都将替换为星号（*）字符。
    新密码
    输入新密码。
    验证密码
    重新输入新密码。
    确定
    单击该按钮可保存新密码。
    注： 如果您输入了无效的当前密码，或者新密码和验证密码字段的输入不匹配，那么会显示错误消息。 在两种情况下，您都可以关闭错误消息窗口并更正错误。
    取消
    单击该按钮可取消任何密码更改而保留您的旧密码。
