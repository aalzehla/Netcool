IBM Tivoli Netcool/OMNIbus, 버전 7.4

심각도별 이벤트 필터링

특정 심각도와 일치하는 이벤트만 표시하기 위해 이벤트 목록 데이터를 필터링할 수 있습니다.

특정 심각도에 따라 필터링하려면 이벤트 목록 분배 상태 표시줄에서 필수 심각도 색상을 클릭하십시오. 예를 들어, 심각도 레벨이 낮음(노란색과 연관됨)인 이벤트만 확인하려면 분배 상태 표시줄에서 노란색 단추를 클릭하십시오.

이벤트 목록은 새로 고쳐져 심각도 레벨이 minor인 경보만 표시합니다.

심각도 필터링을 제거하고 이벤트 목록을 모든 이벤트의 원래 보기로 복원하려면 분배 상태 표시줄의 모든 이벤트 단추를 클릭하십시오. 또는 분배 상태 표시줄에서 심각도 색상을 다시 클릭하십시오.

그림 1. 배포 상태 표시줄의 모든 Event 단추
참고: 분배 상태 표시줄에서 모든 이벤트 단추는 빠른 필터링 또는 심각도 필터링 사용 시에만 사용 가능합니다. UNIX에서 이 단추가 깜빡이면 필터링이 사용 중임을 표시합니다.
