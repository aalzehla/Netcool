명령행, 스크립트, 도구에서 임시 이벤트 목록 실행

명령행에서 직접, 스크립트에서 또는 이벤트 목록 도구의 일부로 사용자 정의된 임시 이벤트 목록을 실행할 수 있습니다.

예를 들어 필터를 적용하는 이벤트 목록을 열어 특정 ObjectServer의 모든 위험 경보를 볼 수 있습니다.

  • 명령행에서 임시 이벤트 목록 시작
    UNIX 명령행에서 임시 이벤트 목록을 열려면 nco_elct 명령을 사용하고, Windows 명령행에서 임시 이벤트 목록을 열려면 NCOelct.exe 명령을 사용하십시오.
  • 예제: 명령행 또는 스크립트에서 임시 이벤트 목록 시작
    이 예제는 UNIX 및 Windows에서 모두 명령행 또는 스크립트에서 직접 임시 이벤트 목록을 실행하는 방법을 표시합니다.
  • 예제: 도구에서 임시 이벤트 목록 시작
    이 예제는 UNIX 및 Windows 모두 도구에서 임시 이벤트 목록을 실행하는 방법을 표시합니다.
상위 주제: 이벤트 목록의 이벤트 모니터링 및 관리
