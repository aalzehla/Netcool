IBM Tivoli Netcool/OMNIbus, 버전 7.4

ObjectServer에 대한 비밀번호 변경

ObjectServer 이외의 특정 외부 소스에 대해 인증되는 경우 항상 외부 소스의 시스템 관리자로부터 비밀번호 변경을 요청해야 합니다.

rev="v731r1"ObjectServer 서버 사용자의 비밀번호는 하이픈(-), 슬래시(/) 또는 큰따옴표(")로 시작되면 안됩니다.

ObjectServer에서 비밀번호를 변경하려면 다음을 수행하십시오.

  1. 운영 체제별로 적절한 조치를 수행하십시오.
    옵션 설명
    UNIX UNIX의 컨덕터 창에서 파일 > 비밀번호 변경을 클릭하십시오.
    참고: UNIX에서 비밀번호 변경 메뉴는 조직에서 외부 인증을 사용하는 경우 사용 불가능합니다. 외부적으로 인증되는 비밀번호는 ObjectServer에 저장되지 않기 때문입니다.
    Windows Windows의 컨덕터 메뉴에서 비밀번호를 클릭하십시오.
    비밀번호 변경 창이 열립니다.
  2. 다음과 같이 이 창을 완료하십시오.
    현재 비밀번호
    현재 비밀번호를 입력하십시오. 입력한 텍스트는 별표(*) 문자로 대체됩니다.
    새 비밀번호
    새 비밀번호를 입력하십시오.
    비밀번호 확인
    새 비밀번호를 다시 입력하십시오.
    확인
    새 비밀번호를 저장하려면 이 단추를 클릭하십시오.
    참고: 올바르지 않은 현재 비밀번호를 입력하는 경우 또는 새 비밀번호 및 비밀번호 확인 필드의 항목이 일치하지 않는 경우 오류 메시지가 표시됩니다. 두 가지 경우 모두, 오류 메시지 창을 닫고 오류를 수정할 수 있습니다.
    취소
    변경된 비밀번호를 취소하고 이전 비밀번호를 유지하려면 이 단추를 클릭하십시오.
