イベント・リストのビュー設定

ビューでは、イベント・リストに表示される列の選択およびこれらの列内の情報のフォーマットが定義されます。

各ビューでは、以下の設定を指定することができます。

  • イベント・リストに組み込まれる列、およびそれらの列を表示する順序
  • 列の代わりのタイトルまたは別名
  • 列の幅、列見出しとデータの位置合わせなどの列フォーマット
  • イベント・リストを水平にスクロールする際に、特定の列を常にビュー内に残すかどうか
  • 列のソート順およびソート優先順位
  • イベント・リストに表示する行数を制限するかどうか

ビュー・ビルダーを使用して、ビューの作成および編集を行うことができます。

  • ビュー・ビルダーの開始
    ビュー・ビルダーには、ビューを構成するためのインターフェースが用意されています。
  • UNIX でのビューの作成または編集
    ビュー・ビルダーを使用して、イベント・リストにどの列を表示するかを定義します。
  • Windows でのビューの作成または編集
    ビュー・ビルダーを使用して、イベント・リストにどの列を表示するかを定義します。
  • UNIX でのビューのドラッグ・アンド・ドロップ
    UNIX では、ドラッグ・アンド・ドロップ機能を使用してビューを操作できます。
  • ビューを開く
    保存されているビューを開いて、ビュー定義を表示したり編集したりできます。
