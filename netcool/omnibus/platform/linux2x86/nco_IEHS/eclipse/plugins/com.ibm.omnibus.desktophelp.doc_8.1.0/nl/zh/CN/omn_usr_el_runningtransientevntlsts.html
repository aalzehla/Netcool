从命令行、脚本和工具运行瞬时事件列表

您可直接从命令行、脚本或部分事件列表工具运行定制的瞬时事件列表。

例如，您可打开应用过滤器来查看特定 ObjectServer 中所有紧急警报的事件列表。

  • 从命令行启动瞬时事件列表
    从 UNIX 命令行使用 nco_elct 命令可打开瞬时事件列表，从 Windows 命令行使用 NCOelct.exe 命令可打开瞬时事件列表。

    这些示例显示在 UNIX 和 Windows 上如何直接从命令行或脚本运行瞬时事件列表。

    这些示例显示在 UNIX 和 Windows 上如何从工具运行瞬时事件列表。
