Freezing all event lists

You can freeze all open event lists so that updates from the ObjectServer are not displayed.

About this task

To freeze all open event lists, click View > Freeze All from the Event List monitor box window.

To restart automatic updates from the ObjectServer, click View > Freeze All again.

Tip: You can also individually freeze and unfreeze any open event list, by clicking View > Freeze from that event list window.
