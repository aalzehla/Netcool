IBM Tivoli Netcool/OMNIbus, 버전 7.4

Tivoli Netcool/OMNIbus 데스크탑 도움말

Tivoli Netcool/OMNIbus는 많은 다른 네트워크 데이터 소스에서 엔터프라이즈 범위 이벤트 정보를 수집하고 운영자 및 관리자에게 이 정보의 통합된 보기를 제공하는 SLM(Service Level Management) 시스템입니다.

이 정보는 다음과 같은 방식으로 사용됩니다.
  • 운영자에게 지정
  • 헬프데스크 시스템에 전달
  • 데이터베이스에 로깅
  • 원격 Tivoli Netcool/OMNIbus 시스템으로 복제
  • 특정 이벤트에 대한 자동 응답 트리거에 사용

데스크탑은 이벤트를 보고 관리하며 이벤트 정보의 표시 방법을 구성하는 데 사용할 수 있는 통합 그래픽 도구 Suite입니다. 데스크탑 도구는 UNIX와 Windows 플랫폼 모두에서 사용 가능합니다.

이벤트 정보는 네트워크에서의 서비스 가용성을 보다 신속하게 판별할 수 있는 형식으로 전달됩니다. 이벤트 원인이 식별된 경우 데스크탑 도구를 사용하여 문제점을 신속히 분석할 수 있습니다.
