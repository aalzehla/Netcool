IBM Tivoli Netcool/OMNIbus, 버전 7.4

개인용 라이브러리에서 애플리케이션 시작

개인용 라이브러리에서 필터 빌더, 보기 빌더 및 이벤트 목록을 시작할 수 있습니다. 또한 기호 링크로 표시되는 파일에 대한 작업을 수행할 수 있습니다.

다음 한 방법 중 하나로 애플리케이션을 시작하십시오.
  • 필터가 로드된 필터 빌더를 시작하려면 개인용 라이브러리에서 필터 아이콘(.elf 파일)을 두 번 클릭하거나 아이콘을 마우스 오른쪽 단추로 클릭하고 팝업 메뉴에서 필터 빌더 시작을 클릭하십시오.
  • 보기가 로드된 보기 빌더를 시작하려면 개인용 라이브러리에서 보기 아이콘(.elv 파일)을 두 번 클릭하거나 아이콘을 마우스 오른쪽 단추로 클릭하고 팝업 메뉴에서 보기 빌더 시작을 클릭하십시오.
  • 특정 구성을 사용하여 이벤트 목록을 시작하려면 개인용 라이브러리에서 이벤트 목록 구성 아이콘(.elc 파일)을 두 번 클릭하거나 아이콘을 마우스 오른쪽 단추로 클릭하고 팝업 메뉴에서 이벤트 목록 시작을 클릭하십시오.
  • 기호 링크로 표시되는 파일을 시작하고 해당 작업을 수행하려면 해당 기호 링크를 두 번 클릭하십시오.
