尋找

使用此視窗，可以在事件清單的資料中搜尋特定文字。

如下所示完成此視窗：
直欄清單/直欄
選取您要搜尋的直欄。
值
輸入您要符合的搜尋值。 您可以輸入要搜尋的確切值，或輸入正規表示式。
選項
指定所需的相符類型：
  • 如果您想要僅尋找完全符合搜尋值大小寫的出現項目，請選取區分大小寫。 此選項僅在 Windows 上可用。
  • 按一下完全相符，以尋找所選直欄中資料與指定搜尋值完全相符的列。
  • 按一下正規表示式，以尋找所選直欄中資料與指定正規表示式相符的列。
  • 按一下子字串，以尋找所選直欄中資料某處包含指定值的列。

    您僅可以從完全相符、正規表示式及子字串中選擇一個選項。

尋找
按一下此按鈕，以尋找第一個相符的出現項目。 如果在事件清單中找到相符列，則會取消選取目前選取的任何列並選取該相符列。 「尋找」視窗仍會保持開啟狀態，以便您可以檢視任何其他相符的出現項目。
下一個
按一下此按鈕，以顯示事件清單中下一個相符項及後續的相符項。
關閉
按一下此按鈕，以關閉此視窗。
提示： 您可以快速定義搜尋準則，方法是將事件資訊從事件清單拖放到「尋找」視窗。 在 UNIX 上，在事件清單的資料格上按一下滑鼠的中鍵，並將其拖曳到「尋找」視窗的直欄清單中。 依預設，完全相符選項會變成已選取狀態。 在 Windows 上，使用左滑鼠按鈕來按住並拖曳。
