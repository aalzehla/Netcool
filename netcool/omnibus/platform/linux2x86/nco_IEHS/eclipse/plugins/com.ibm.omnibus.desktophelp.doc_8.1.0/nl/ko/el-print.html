IBM Tivoli Netcool/OMNIbus, 버전 7.4

인쇄

UNIX에서 이벤트 목록의 현재 보기를 인쇄하려면 이 창을 사용하십시오. 인쇄 옵션을 지정하여 결과물의 포맷팅을 제어할 수 있습니다.

모든 경보를 인쇄하거나 선택된 경보를 인쇄하는 방법 중에서 선택할 수 있습니다. 선택된 경보를 인쇄하려면 인쇄 창이 열려 있는 동안 선택할 경보를 결정하고 이벤트 목록으로 클릭하여 경보를 선택할 수 있습니다.

다음과 같이 이 창을 완료하십시오.
프린터
프린터 옵션은 인쇄 결과물을 프린터로 보냅니다.
파일
파일 옵션은 인쇄 결과물을 파일로 보냅니다.
프린터
프린터 옵션을 선택한 경우 프린터의 이름을 입력하십시오.
파일
파일 옵션을 선택한 경우 결과물을 보내려는 파일의 경로 및 이름을 입력하십시오.
파일 단추를 클릭하여 파일 선택사항 창에서 디렉토리 및 파일 이름을 탐색할 수도 있습니다. 파일로 인쇄 창에서 파일 이름을 지정한 다음 확인을 클릭하여 인쇄 창으로 돌아가십시오.
보기의 모든 경보
현재 이벤트 목록의 경보를 모두 인쇄하려면 이 옵션을 선택하십시오.
선택된 경보
선택된 경보만 인쇄하려면 이 옵션을 선택하십시오.
행 구분 기호
행 구분 기호를 결과물에 삽입하려면 이 선택란을 선택하십시오(예: 각 행 사이의 대시 라인(-)).
행 수
각 행 결과물에 행 수를 추가하려면 이 선택란을 체크하십시오.
컬럼 구분 기호
결과물에 컬럼 구분 기호를 삽입하려면 이 선택란을 체크하십시오(예: 필드 시작 및 끝 부분에 파이프(|) 문자).
컬럼 표제
출력 파일의 맨 위에 컬럼 표제를 추가하려면 이 선택란을 선택하십시오.
확인
선택한 인쇄 설정을 사용하여 이벤트 목록을 인쇄하려면 이 단추를 클릭하십시오.
취소
인쇄 조치를 취소하려면 이 단추를 클릭하십시오.
관련 태스크:
이벤트 목록의 내용 인쇄
