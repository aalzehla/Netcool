イベント・リスト内のサービスに影響を与えるイベントのモニター

サービスに影響を与えるイベント (SAE) とは、重要なカスタマー・サービスが 1 つ以上のネットワーク・イベントによって影響を受けていることをオペレーターに警告するためのアラートです。 サービスに影響を与えるイベントは IBM Tivoli Network Manager IP Edition 内で生成されます。

システムがこの機能をサポートするように構成されている場合は、Tivoli Netcool/OMNIbus イベント・リストのサービスに影響を与えるイベントをモニターすることができます。

Tivoli Netcool/OMNIbus イベント・リストから、次のようにしてサービスに影響を与えるイベントをモニターできます。

  • サービスに影響を与えるイベントに関連付けられている基本イベント (例えば、linkDown) を表示するには、サービスに影響を与えるイベントを選択して右クリックし、ポップアップ・メニューから「SAE 関連イベントの表示」をクリックします。 選択したイベントに関連付けられているすべてのイベントが新規ウィンドウに表示されます。
  • あるイベント (例えば linkDown) が関連しているサービスに影響を与えるイベントを表示するには、そのイベントを選択して右クリックし、次にポップアップ・メニューから「SAE 関連イベントの表示」をクリックします。 関連しているサービスに影響を与えるすべてのイベントのリストが新規ウィンドウに表示されます。 例えば、イベント・リスト内の linkDown イベントを選択した場合は、その linkDown イベントによって影響を受けるすべてのサービスがこのウィンドウに表示されます。
