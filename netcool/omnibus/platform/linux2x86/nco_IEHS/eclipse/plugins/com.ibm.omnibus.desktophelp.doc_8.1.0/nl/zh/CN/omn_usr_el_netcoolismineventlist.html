IBM Tivoli Netcool/OMNIbus V7.4

事件列表中的因特网服务监视器

IBM Tivoli Composite Application Manager for Internet Service Monitoring 与其余 Tivoli Netcool/OMNIbus 产品套件配合使用。 监视器和探测器生成的事件可发送到同一个 ObjectServer，并且您可使用同一个桌面处理这些事件。

安装 Composite Application Manager for Internet Service Monitoring 时，将向事件列表添加两个新按钮和一个下拉列表。 您可按以下步骤使用按钮和下拉列表：
  • 单击服务视图按钮 可显示事件列表中的服务警报。 您可使用服务下拉列表更改服务类型。
  • 单击显示服务按钮 可从事件列表启动服务窗口。
  • 使用服务下拉列表 可更改事件列表中显示的服务状态事件类型。 例如，从列表中选择监视器类型可显示该类监视器中的所有服务状态事件。
  • 查看事件列表内的服务
    如果安装了 IBM Tivoli Composite Application Manager for Internet Service Monitoring，那么您就可以查看和处理来自因特网服务监视器的服务警报。
