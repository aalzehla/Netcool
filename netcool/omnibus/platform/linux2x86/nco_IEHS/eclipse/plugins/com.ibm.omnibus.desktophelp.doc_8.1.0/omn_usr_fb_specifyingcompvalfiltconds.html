Specifying comparison values for filter conditions

When defining simple or complex conditions within the Filter Builder, you are required to select a database column to be used in the comparison, a comparison operator, and one or more comparison values.

About this task

For example: 
Severity=Critical
The database columns used in the comparison can be associated with one of three data types:
  • String
  • Integer
  • Time

Each of these data types requires a different set of entries for the comparison values.

These entries are required when constructing simple conditions.
