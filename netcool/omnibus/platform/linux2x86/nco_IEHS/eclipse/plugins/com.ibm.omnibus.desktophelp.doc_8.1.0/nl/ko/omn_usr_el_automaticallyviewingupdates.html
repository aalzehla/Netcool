IBM Tivoli Netcool/OMNIbus, 버전 7.4

이벤트 목록의 업데이트 자동 보기

업데이트가 일어날 때 이벤트 목록의 스크롤 정도를 제어할 수 있습니다. 맨 위로 이동 기능은 업데이트된 이벤트를 빨리 검토하는 데 도움이 될 수 있습니다.

맨 위로 이동 기능을 사용하면 새 이벤트가 이벤트 목록에 추가될 때마다 이벤트 목록이 맨 위 행으로 자동 스크롤됩니다. 맨 위로 이동 기능을 사용하지 않을 경우 이벤트 목록은 업데이트가 발생할 때 현재 상태를 유지합니다.

맨 위로 이동 기능을 사용하려면 보기 > 맨 위로 이동을 클릭하십시오. 이 메뉴 옵션은 전환 효과를 나타내므로, 맨 위로 이동 옵션을 다시 클릭하면 맨 위로 이동 기능이 사용되지 않습니다.

팁: 이벤트 목록을 열 때마다 맨 위로 이동 기능이 사용되는지 확인할 수 있습니다. 이벤트 목록 모니터 상자 창에서 Windows > 환경 설정을 클릭하여 이벤트 목록 환경 설정 창을 여십시오. 이 창의 기타 탭에서 기본값으로의 이동 단추 옵션을 선택하십시오.
