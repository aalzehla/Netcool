IBM Tivoli Netcool/OMNIbus 7.4 版

在 UNIX 上存取線上說明

線上說明使用「IBM® Eclipse 說明系統 (IEHS)」（其為 Web 應用程式）進行部署。基於安裝及配置設定，您可以獨立模式或資訊中心模式存取線上說明。

若要配置及存取線上說明，請依下表所述開始進行。

表 1. 說明配置設定
設定 說明
瀏覽器環境變數 (UNIX) 將瀏覽器目錄位置新增至下列環境變數 (如果尚未新增的話)：
  • PATH
  • LD_LIBRARY_PATH（Solaris 及 Linux）、LIBPATH (AIX®) 或 SHLIB_PATH (HP-UX)
  • 瀏覽器特定環境變數；例如，MOZILLA_FIVE_HOME
獨立模式 若要以獨立模式使用線上說明，則在安裝 Tivoli Netcool/OMNIbus 期間，您必須已選取本端說明系統功能。
安裝之後，僅當另一個本端服務正在使用預設 IEHS 埠號 8888 時，才必須進行配置。 如果是這種情況，請編輯 IEHS 配置檔 $NCHOME/omnibus/etc/nco_IEHS.cfg，如下所示：
  • IEHSMode：0
  • IEHSHost：空白
  • IEHSPort：未用的埠號
註： 變更配置檔中的埠號之後，您必須關閉本端 IEHS 伺服器，變更才會生效。 執行 $NCHOME/omnibus/bin/help_end 指令。 (如果您變更瀏覽器的環境變數設定，則必須同時關閉 IEHS 伺服器，變更才會生效)。
資訊中心模式 在資訊中心模式中，您的系統管理者會配置設定，以便所有使用者都可以共用線上說明系統的同一 IEHS Web 伺服器，以減輕本端主機或工作站上的負載。

如果您的系統管理者已通知您必須以資訊中心模式存取線上說明，則您必須配置本端 IEHS 配置檔 $NCHOME/omnibus/etc/nco_IEHS.cfg，如下所示：

  • IEHSMode：1
  • IEHSHost：IEHS 伺服器的 IP 位址或主機名稱
  • IEHSPort：執行 IEHS 伺服器的埠號

您的系統管理者會提供 IEHSHost 及 IEHSPort 設定。

當您以獨立模式存取線上說明時，您的預設說明瀏覽器視窗會自動開啟並顯示相關文字。 本端 IEHS 伺服器開始於第一個說明要求，並繼續執行直到您關閉它為止。 若要關閉本端 IEHS 伺服器，請執行下列指令：

$NCHOME/omnibus/bin/help_end

通常，僅在電腦重新開機之前才會執行 help_end 指令。

在您可以使用資訊中心模式存取線上說明之前，您的系統管理者必須啟動指定的遠端 IEHS 伺服器。

註： 在獨立模式中，只有在本端電腦上執行的瀏覽器才可以顯示說明主題。 如果另一個主機上正在執行開啟的瀏覽器，則說明系統會嘗試使用該瀏覽器來顯示線上說明。 這會導致下列錯誤訊息：
嘗試連接至 localhost:name 時，連線遭到拒絕
請關閉該視窗，以在正確的瀏覽器中顯示說明。
