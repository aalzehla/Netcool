管理者功能表

使用「管理者」作為 Tivoli Netcool/OMNIbus 的桌面進入點。 您可以使用「管理者」來啟動其他桌面工具或用戶端應用程式，讓您可以檢視及管理警示。 您也可以變更密碼，並可以存取由系統管理者定義的工具，以管理事件清單中的警示。

在 Windows 上，「管理者」呈現為蹦現功能表，您可以用滑鼠右鍵按一下 Windows 系統匣中的紅色匯流排圖示進行存取。 「管理者」功能表提供的選項可讓您存取所有已安裝的桌面工具，以及基於您的存取權限可用的桌面工具。

透過將游標移至匯流排圖示上，您可以顯示「管理者」登入明細的相關資訊，以在「工具提示」中查看使用者名稱及 ObjectServer 詳細資料。 若要從「管理者」執行工具，請從功能表選取選項。 該工具會自動啟動；您不需要輸入使用者名稱或密碼。

使用「管理者」功能表選項，如下所示：
事件清單
按一下此選項，以啟動具有預設配置的事件清單；亦即，具有預設視圖及預設過濾器。
伺服器
按一下此選項，以啟動「伺服器編輯器」，其用於設定可讓 Tivoli Netcool/OMNIbus 伺服器元件彼此進行通訊的通訊明細。
內容
按一下此選項，以啟動「內容編輯器」，您可用它來編輯 ObjectServer、Proxy 伺服器、程序代理程式、資料庫起始設定公用程式 (nco_dbinit)、探測器及閘道之內容檔中的配置資訊。
工具
按一下此選項，以啟動已由系統管理者配置的工具。
喜好設定
按一下此選項，以配置 Tivoli Netcool/OMNIbus 的廣域喜好設定。
密碼
按一下此選項，以變更您的密碼。
說明
按一下此選項，以存取使用桌面工具的相關說明。
關於管理者
按一下此選項，以顯示「管理者」版本。
結束
按一下此選項，以關閉「管理者」。
