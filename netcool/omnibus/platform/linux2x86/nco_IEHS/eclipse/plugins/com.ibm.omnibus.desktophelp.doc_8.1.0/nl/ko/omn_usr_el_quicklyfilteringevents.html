IBM Tivoli Netcool/OMNIbus, 버전 7.4

신속하게 이벤트 필터링

선택한 기준과 일치하는 이벤트를 빨리 표시하는 방법으로 빠른 필터링 기능을 사용할 수 있습니다. 예를 들어, 선택한 이벤트와 동시에 또는 선택한 이벤트 이전에 발생된 이벤트만 신속히 표시할 수 있습니다.

이벤트를 빨리 필터링하려면 다음을 수행하십시오.
  1. 이벤트 목록에서 빠른 필터의 기반이 될 값을 포함하는 셀을 선택하십시오.
  2. 경보 > 빠른 필터를 클릭한 후, 하위 메뉴에서 다음 옵션 중 하나를 선택하십시오.
    • 같음
    • 같지 않음
    • 초과
    • 이상
    • 미만
    • 이하
    • 유사
    • 유사하지 않음
    이벤트 목록은 새로 고쳐져 지정된 필터 기준과 일치하는 이벤트만을 표시합니다.

다음에 수행할 작업

빠른 필터링을 제거하고 이벤트 목록을 모든 이벤트의 원래 보기로 복원하려면 경보 > 빠른 필터 > 설정 해제를 클릭하십시오. 다른 방법으로, 다음 그림에 보이는 분배 상태 표시줄에서 모든 이벤트 단추를 클릭하십시오.

그림 1. 배포 상태 표시줄의 모든 Event 단추
참고: 분배 상태 표시줄의 모든 이벤트 단추는 빠른 필터링 또는 심각도 필터링을 사용할 때에만 사용 가능합니다. UNIX에서 이 단추가 깜빡이면 필터링이 사용 중임을 표시합니다.
