IBM Tivoli Netcool/OMNIbus, 버전 7.4

필터 저장

이 창에서 현재 필터를 저장할 수 있습니다.

이 창은 필터, 보기 및 이벤트 목록 구성의 저장 또는 열기 작업과 연관되어 있습니다. 따라서 필드가 모두 현재 조치에 해당되지 않을 수도 있습니다.

다음과 같이 이 창을 완료하십시오.

필터
열려는 파일 유형에 대한 검색 조회를 지정하려면 이 필드를 사용하십시오. 이 필드는 필터 단추 및 파일 목록과 함께 작동합니다. 필터 필드에 현재 디렉토리 경로와 파일 유형을 입력한 후 필터를 클릭하십시오. 일치하는 파일이 모두 파일 목록에 표시됩니다.
예를 들어, /myfiles/filters 디렉토리의 모든 필터 파일을 표시하려면 /myfiles/filters/*.elf를 입력하고 필터를 클릭하여 일치하는 파일을 모두 확인하십시오.
디렉토리
이 목록은 현재 디렉토리의 모든 서브디렉토리를 표시합니다. 디렉토리를 새로 고치려면 '.' 항목을 두 번 클릭하십시오. 디렉토리 트리를 하나의 디렉토리로 이동시키려면 '..' 항목을 클릭하십시오.
파일
이 목록은 현재 디렉토리의 파일을 표시하며, 이는 필터 필드의 선택사항과 일치합니다. 열려는 파일 이름을 선택하십시오. 겹쳐쓰려는 기존 파일도 선택할 수 있습니다.
선택사항
저장하거나 여는 중인 파일의 경로와 이름은 여기에 표시됩니다. 다른 파일을 열거나, 파일을 다른 위치에서 저장하거나, 다른 파일 이름을 사용하여 파일을 저장하려면 경로와 파일 이름을 편집할 수 있습니다.
보기를 저장 중이면 파일 이름의 .elv 확장자를 사용하십시오. 필터를 저장 중이면 파일 이름의 .elf 확장자를 사용하십시오. 이벤트 목록 구성을 저장 중이면 파일 이름의 .elc 확장자를 사용하십시오.
백업 기존 파일(UNIX만 해당)
저장 중인 파일 이름이 이미 존재하는 경우 기존 보기 파일의 백업을 작성하려면 이 선택란을 선택하십시오. 이 선택란은 사용자가 필터, 보기 또는 이벤트 목록 구성을 저장하는 경우에만 표시됩니다.
확인
파일을 저장하거나 열려면 이 단추를 클릭합니다.
필터
이 단추는 필터 필드 및 파일 목록과 함께 작동합니다. 필터 필드의 선택사항을 기반으로 현재 디렉토리의 파일을 표시하려면 이 단추를 클릭하십시오. 파일 목록에 파일이 나타납니다.
취소
파일을 열거나 저장하지 않고 창을 닫으려면 이 단추를 클릭하십시오.
