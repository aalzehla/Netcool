IBM Tivoli Netcool/OMNIbus Version 8.1

Event list

Tivoli Netcool/OMNIbus uses event lists to present information about the event data that is forwarded to the ObjectServer.

The event list displays a filtered view of color-coded events in a scrolling list. You can set up multiple event list clients, each with different priorities and filters. You can also use the event list to reprioritize alerts, and to assign or reassign problems to other users. When an alert has been assigned to a user, the user can acknowledge it, or deacknowledge it so that it can be picked up by another user.
