範例：從指令行或 Script 啟動暫時性事件清單

這些範例會顯示如何在 UNIX 及 Windows 上，直接從指令行或從 Script 執行暫時性事件清單。

範例：UNIX 上的 nco_elct 指令

下列指令會透過使用 default.elv 視圖檔案，從 NCOMS ObjectServer 建立暫時性事件清單。 只有節點 wombat 的事件會顯示在事件清單中。

$NCHOME/omnibus/bin/nco_elct -server NCOMS -username root -password "" 
-vfile "$NCHOME/omnibus/desktop/default.elv" -ftext "( Node = \'wombat\' )"

範例：Windows 上的 NCOelct 指令

下列指令會透過使用 lview.elv 視圖檔案及 tool.elf 過濾器檔案，從 NYC ObjectServer 建立暫時性事件清單。

%NCHOME%\omnibus\desktop\NCOelct.exe -server NYC -username root -password "" 
-elv "%NCHOME%\omnibus\ini\lview.elv" -elf "%NCHOME%\omnibus\ini\tool.elf" 
-params "wombat"

之前的 NCOelct 指令中參照的 tool.elf 檔案會具有下列內容：

filter_name = 'ToolFilter';
filter_text = '( Node = \'@NodeName\')';
filter_metric = 'avg(Severity)';
# 檔案結尾

在執行指令或 Script 時，-params 選項之後的字串 wombat 會取代 tool.elf 檔案中的 @NodeName 位置保留元，所以只有節點 wombat 的事件會顯示在事件清單中。
