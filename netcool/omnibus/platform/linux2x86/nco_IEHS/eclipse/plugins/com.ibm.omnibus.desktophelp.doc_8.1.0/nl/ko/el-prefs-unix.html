이벤트 목록 환경 설정

이벤트 목록의 환경 설정을 구성하려면 이 창을 사용하십시오.

이벤트 목록 환경 설정 창에는 다음 탭이 있습니다.
  • 모니터 탭
  • 새로 고치기 탭
  • 알림 탭
  • 플래시 탭
  • 기타 탭

모니터 탭

이벤트 목록 모니터 상자 창에서 모니터 상자가 표시하는 정보의 유형을 지정하려면 이 탭을 사용하십시오. 모니터 탭을 완료하려면 다음을 참조하십시오.
수 표시
모니터 상자에서 필터와 일치하는 경보 수를 표시하려면 이 선택란을 선택합니다.
최상위 심각도 표시
모니터 상자에 필터와 일치하는 경보 중에서 최고 심각도를 표시하려면 이 선택란을 선택하십시오.
최하위 심각도 표시
모니터 상자에 필터와 일치하는 경보 중에서 최저 심각도를 표시하려면 이 선택란을 선택하십시오.
지표 표시
모니터 상자에 선택된 필터 지표 값을 표시하려면 이 선택란을 선택하십시오.
심각도 경계 표시
모니터 상자 주위에 색상 경계를 표시하려면 이 선택란을 선택하십시오. 경계 색상은 필터와 일치하는 경보 중 가장 높은 심각도에 따라 변경됩니다.
라바램프 표시
모니터 상자에서 분포 표시기를 수평 막대로 표시하려면 이 옵션을 클릭하십시오.
히스토그램 표시
모니터 상자에서 분포 표시기를 막대 그래프로 표시하려면 이 옵션을 클릭하십시오.

새로 고치기 탭

이벤트 목록의 업데이트 간격을 구성하려면 이 탭을 사용하십시오. 다음과 같이 새로 고치기 탭을 완료하십시오.
시간 제한 새로 고치기
이 필드를 사용하여 경보 정보를 지정된 시간 간격으로 업데이트할지 또는 ObjectServer로부터 업데이트 알림을 대기할지 지정하십시오. 업데이트 빈도를 지정하려면 시간 제한 새로 고치기 선택란을 선택한 후 옆의 필드에 시간(초)을 입력하십시오.
ObjectServer의 모든 최신 데이터가 지정된 빈도로 이벤트 목록에 전송됩니다.
팁: Timed Refresh를 설정하면 이벤트 목록이 모든 해당 데이터를 설정 간격대로 다시 로드합니다. ObjectServer 성능 및 네트워크 트래픽에 영향을 미치므로 새로 고치기를 낮은 값(예: 60초 미만)으로 설정하지 마십시오.
참고: 시스템 관리자가 AllowTimedRefresh ObjectServer 특성을 FALSE로 설정한 경우, 새로 고침 타이머를 사용할 수 없으며 ObjectServer의 알림에 따라 업데이트가 발생합니다.

알림 탭

이벤트 목록이 최소화될 때 경보 알림의 환경 설정을 구성하려면 이 탭을 사용하십시오. 다음과 같이 알림 탭을 완료하십시오.
이벤트 변경 시 통지
이벤트 목록이 최소화될 때 새로 작성, 변경 또는 삭제된 경보에 대한 알림을 받으려면 이 선택란을 선택하십시오. 이벤트 목록이 최소화된 경우 아이콘이 표시된 데스크탑 환경에만 알림을 받습니다. 시기 및 방법 옵션을 사용하여 알림 방법을 설정하십시오.
시기
다음과 같이 알림을 받으려면 각 선택란을 선택하십시오.
  • 새 경고가 이벤트 목록에 추가될 때 알림을 수신하려면 새로 작성을 선택하십시오.
  • 기존 경보가 이벤트 목록에서 변경될 때 알림을 수신하려면 변경을 선택하십시오.
  • 기존 경보가 이벤트 목록에서 삭제될 때 알림을 수신하려면 삭제를 선택하십시오.
방법
알림이 발생하는 방법을 표시하려면 각 선택란을 선택하십시오.
  • 벨 울림을 선택하여 워크스테이션 또는 터미널 벨의 소리를 한 번 들으십시오.
  • 이벤트 목록 아이콘(UNIX) 위의 느낌표를 반짝이거나 최소화된 이벤트 목록(Windows)을 반짝이게 하려면 Alert Icon을 선택하십시오.
  • 화면에 이벤트 목록을 열려면 창 열기를 선택하십시오. 창 관리자를 대화식으로 사용하는 경우 이벤트 목록이 활성 창으로 열립니다.
  • 명령을 실행하려면 외부 명령 실행을 선택하십시오. 이 선택란을 선택한 경우, 명령어를 입력해야 합니다. 이 기능은 데스크탑 알림이 한 개인 경우에만 사용해야 하며 페이징 애플리케이션 핸들링을 위한 것입니다.

플래시 탭

새 경보를 수신할 때 깜빡이는 이벤트 목록 환경 설정을 지정하려면 이 탭을 사용하십시오. 다음과 같이 플래시 탭을 완료하십시오.
플래시 사용
이벤트 목록 플래시를 사용하려면 이 선택란을 선택하십시오. 이벤트 목록이 깜빡이는 속도를 표시하려면 속도 슬라이더를 사용하십시오. 깜빡거림의 밝은 정도를 표시하려면 밝기 슬라이더를 사용하십시오.

기타 탭

기타 이벤트 목록 환경 설정을 설정하려면 이 탭을 사용하십시오. 다음과 같이 기타 탭을 완료하십시오.
이벤트 목록 색상 표시
이벤트 목록에서 경보 심각도를 나타내는 색상을 사용하려면 이 선택란을 선택하십시오. 모든 이벤트 목록 행에 회색 배경을 사용하려면 이 선택란을 선택 취소하십시오.
분배 요약 표시
이벤트 목록에 분배 요약 표시줄을 표시하려면 이 선택란을 선택하십시오. 이 표시줄은 각 심각도 색상에 일치하는 경보 수를 표시합니다.
도구 모음 표시
이벤트 목록에서 도구 모음을 사용 가능하게 하려면 이 선택란을 선택하십시오.
기본으로 이동 단추 설정
모든 이벤트 목록에서 기본적으로 맨 위로 이동 기능을 사용하려면 이 선택란을 선택하십시오. 맨 위로 이동 기능을 사용하면 새 이벤트가 추가될 때마다 이벤트 목록의 맨 위로 스크롤되어 업데이트된 이벤트를 신속하게 검토할 수 있습니다.
팁: 이벤트 목록의 업데이트를 자동으로 볼 수 있도록 개별 이벤트 목록에 맨 위로 이동 기능을 사용하거나 개별 이벤트 목록에 이 기능을 사용하지 않을 수도 있습니다.
정보 세부사항 정렬
이벤트 정보 창의 세부사항 필드를 필드 이름의 영문자 순으로 정렬하려면 이 선택란을 선택하십시오. 이 선택란을 선택 취소하면 ObjectServer에서 판별된 순서대로 세부사항 필드가 표시됩니다.

환경 설정 적용 및 저장

적용
변경사항을 저장하지 않고 적용하려면 이 단추를 클릭하십시오. 변경사항을 데스크탑에 적용하는 방법을 테스트하려면 이 단추를 사용하십시오.
저장
환경 설정에서 발생한 모든 변경사항을 저장하려면 이 단추를 클릭하십시오.
팁: 이벤트 목록의 환경 설정에서 발생한 모든 변경사항을 보려면 열려 있는 이벤트 목록을 모두 닫고 다시 열어야 합니다.
닫기
변경사항을 저장하지 않고 창을 닫으려면 이 단추를 클릭하십시오.
실행 취소
각 탭에서 발생한 모든 변경사항을 실행 취소하려면 이 단추를 클릭하십시오.
