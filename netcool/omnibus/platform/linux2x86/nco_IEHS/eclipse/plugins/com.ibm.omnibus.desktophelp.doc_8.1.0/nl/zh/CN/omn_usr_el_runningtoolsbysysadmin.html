运行系统管理员配置的工具

如果系统管理员已创建用于事件管理的工具，那么就可以对事件列表内的事件运行这些工具。

如果已配置了工具，那么可以使用多种方法来运行工具：

  • 从 UNIX 上的“Conductor”窗口，单击工具菜单内的工具。
  • 从 Windows 上的 Conductor 菜单，单击工具，然后单击子菜单内的工具。
  • 从“事件列表”监视器框窗口单击工具菜单内的工具。
  • 从事件列表单击警报 > 工具，然后单击子菜单内的必需工具。 或者，可以右键单击，然后单击弹出菜单内的警报 > 工具。
