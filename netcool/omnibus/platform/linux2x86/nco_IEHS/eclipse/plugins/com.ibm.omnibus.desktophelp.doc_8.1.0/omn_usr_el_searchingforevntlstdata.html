Searching for event list data

You can search for event list data in a number of ways. You can enter specific text to search for, use a filtering facility to quickly find matching occurrences of data, and filter event list data by severity.

  • Searching for text in the event list
    You can search the event list for event data that matches a specific value that you enter. You can search for data within a specific column and specify options to narrow down the search.
  • Quickly filtering events
    You can use the quick filtering facility as a fast way of displaying events that match a selected criteria. For example, you can quickly display only those events that occurred at the same time as, or before, the selected event.
  • Filtering events by severity
    You can filter the event list data to display only those events that match a particular severity.
