Starting the Filter Builder

The Filter Builder provides a graphical interface within which you can build and test the SQL statements that comprise a filter.

About this task

You can start the Filter Builder in any of the following ways:

Procedure

  • From the Event List monitor box window, click Launch Filter Builder on the toolbar and select a previously-defined filter.
  • From a monitor box, click the filter name button.
  • From the event list, click Filter Builder on the toolbar, or select Edit > Edit Filter.
  • From the Event List Configuration window, click the New or Edit button.
  • From the UNIX Conductor, click Filter Builder in the button bar, or double-click a saved filter (.elf) file in the personal library.
