IBM Tivoli Netcool/OMNIbus, 버전 7.4

찾기

이벤트 목록의 데이터에서 특정 텍스트를 검색하려면 이 창을 사용하십시오.

다음과 같이 이 창을 완료하십시오.
컬럼 목록/컬럼
검색하려는 컬럼을 선택하십시오.
값
일치시킬 검색 값을 입력하십시오. 값 또는 정규식을 입력할 수 있습니다.
옵션
일치 유형을 지정하십시오.
  • 대소문자가 정확히 일치하는 검색값을 찾으려면 대소문자 구분 옵션을 선택하십시오. 이 옵션은 Windows에서만 사용 가능합니다.
  • 선택한 컬럼의 데이터가 지정된 검색 값과 정확히 일치하는 행을 찾으려면 일치를 클릭하십시오.
  • 선택한 컬럼의 데이터가 지정된 정규식과 일치하는 행을 찾으려면 정규식을 클릭하십시오.
  • 선택한 컬럼의 데이터 어딘가에 지정된 값을 포함하는 행을 찾으려면 하위 문자열을 클릭하십시오.

    일치, 정규식 및 하위 문자열에서 하나의 옵션만 선택할 수 있습니다.

찾기
첫 번째 일치하는 항목을 찾으려면 이 단추를 클릭하십시오. 일치하는 행이 이벤트 목록에서 발견되는 경우 현재 선택된 모든 행이 선택 취소되고 일치하는 행이 선택됩니다. 찾기 창은 계속 열려있으므로 추가로 일치하는 항목을 모두 볼 수 있습니다.
다음
이벤트 목록에서 다음 일치 및 후속 일치를 표시하려면 이 단추를 클릭하십시오.
닫기
이 창을 닫으려면 이 단추를 클릭하십시오.
팁: 이벤트 목록의 이벤트 정보를 찾기 창에 끌어놓으면 검색 기준을 빨리 정의할 수 있습니다. UNIX에서 이벤트 목록의 셀을 마우스 가운데 단추로 클릭하고 해당 셀을 찾기 창의 컬럼 목록에 끌어놓으십시오. 일치 옵션은 기본적으로 선택됩니다. Windows에서는 마우스 왼쪽 단추를 사용하여 클릭하고 끌어놓으십시오.
