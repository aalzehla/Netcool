IBM Tivoli Netcool/OMNIbus, 버전 7.4

UNIX에서 보기 작성 또는 편집

보기 빌더를 사용하면 이벤트 목록에 표시할 컬럼을 정의할 수 있습니다.

보기를 작성 또는 편집하려면 다음을 수행하십시오.
  1. 보기 빌더를 시작하십시오.
  2. 다음과 같이 보기 설정 영역을 완료하십시오.
    보기 이름
    보기를 새로 작성할 경우 현재 이름을 보기의 고유 이름으로 겹쳐쓰기합니다. 이 이름은 이벤트 목록에 사용됩니다. 기존 보기를 편집할 경우 이 필드에 항목을 작성할 필요가 없습니다.
    편집 가능
    해당 관리자 권한이 있는 경우 기타 사용자가 이 보기를 편집하도록 허용하려면 이 선택란을 체크하고 보기가 수정되지 못하게 하려면 이 선택란을 체크하지 마십시오.
    제한사항: 이 설정은 보기가 이벤트 목록 구성(.elc 파일)의 일부로 저장된 경우에만 적용됩니다.

    이제 포함할 컬럼을 선택하고 이벤트 목록에서 해당 순서 및 형식을 지정하여 보기를 정의할 수 있습니다. 또한 이벤트 목록에서 행의 정렬 순서 및 우선순위를 구성하고 표시된 행의 수를 제한할 수도 있습니다.

  3. 컬럼 표시 영역에서 다음과 같이 컬럼을 추가, 제거 및 재배치하십시오.
    이벤트 목록 행렬
    이 행렬은 이벤트 목록 보기에 표시하기 위해 선택되는 모든 컬럼과 해당 컬럼이 표시되는 순서를 표시합니다. 실제 컬럼 이름은 맨 위에 표시되고 아래의 행은 각 컬럼의 데이터 유형을 표시합니다. 행에서 더미는 문자열 컬럼을 식별하는 데 사용되고 숫자 1은 정수 컬럼을, 01/01/70 01:00:01은 시간 컬럼을 각각 식별합니다.
    이벤트 목록 행렬은 제거, 첫 번째, 왼쪽, 오른쪽 및 마지막 단추와 연계하여 사용됩니다. 이 행렬에서 임의의 컬럼에 대해 조치를 수행하려면 먼저 해당 컬럼과 연관되어 있는 셀을 클릭하십시오. 이 방법으로 컬럼을 선택하면 해당 컬럼에 대해 현재 정의되어 있는 세부사항도 컬럼 세부사항 영역에 표시됩니다.
    사용 가능 컬럼 목록
    컬럼 세부사항 영역에 인접한 이 목록은 이벤트 목록에 더 추가될 수 있는 컬럼을 모두 표시합니다. 사용 가능 컬럼 목록은 추가 단추와 연계하여 사용됩니다.
    이벤트 목록에 추가하려는 컬럼을 선택하십시오.
    팁: 이 목록의 컬럼 이름은 알파벳순으로 표시됩니다.
    추가
    사용 가능 컬럼 목록에 현재 선택되어 있는 컬럼을 이벤트 목록에 추가하려면 이 단추를 클릭하십시오. 컬럼은 이벤트 목록 행렬의 가장 오른쪽 항목으로 추가됩니다.
    팁: 끌어놓기 기법을 사용하여 컬럼을 추가할 수 있습니다. 사용 가능 컬럼 목록에서 추가하려는 컬럼을 선택하십시오. 이 컬럼에서 마우스 가운데 단추를 누른 상태로 커서를 이벤트 목록 행렬 안에 끌어놓으십시오. 컬럼을 놓을 수 있는 영역 안에 들어오면 커서가 변경되어 녹색 틱이 표시됩니다. 마우스 단추를 해제하십시오. 컬럼은 이벤트 목록 행렬의 가장 오른쪽 항목으로 추가됩니다.
    제거
    이벤트 목록 행렬에서 현재 선택된 컬럼을 사용 가능 컬럼 목록으로 다시 이동하려면 이 단추를 클릭하십시오. 이 컬럼은 더 이상 이벤트 목록에 표시되지 않습니다.
    팁: 끌어놓기 기법을 사용하여 컬럼을 제거할 수 있습니다. 이벤트 목록 행렬에서 제거하려는 컬럼의 셀을 선택하십시오. 이 셀에서 마우스 가운데 단추를 누른 상태로 커서를 사용 가능 컬럼 목록 안에 끌어놓으십시오. 컬럼을 놓을 수 있는 영역 안에 들어오면 커서가 변경되어 녹색 틱이 표시됩니다. 마우스 단추를 해제하십시오. 컬럼이 사용 가능 컬럼 목록에 추가됩니다.
    처음
    이벤트 목록 행렬에서 현재 선택된 컬럼을 이벤트 목록의 첫 번째(가장 왼쪽) 컬럼으로 위치를 변경하려면 이 단추를 클릭하십시오.
    왼쪽
    이벤트 목록 행렬에서 현재 선택된 컬럼을 왼쪽으로 한 위치 이동하려면 이 단추를 클릭하십시오.
    오른쪽
    이벤트 목록 행렬에서 현재 선택된 컬럼을 오른쪽으로 한 위치 이동하려면 이 단추를 클릭하십시오.
    마지막
    이벤트 목록 행렬에서 현재 선택된 컬럼을 이벤트 목록의 마지막(가장 오른쪽) 컬럼으로 위치를 변경하려면 이 단추를 클릭하십시오.
  4. 컬럼 표시 영역에서 다음과 같이 컬럼을 형식화하십시오.
    컬럼 세부사항
    이 영역을 사용하여 이벤트 목록 행렬에서 현재 선택되어 있는 컬럼을 포맷하십시오.
    컬럼/정렬
    시스템 정의된 컬럼 이름은 컬럼 필드에 읽기 전용 값으로 표시됩니다. 인접 정렬 목록에서 왼쪽, 가운데 또는 오른쪽을 선택하여 이벤트 목록의 컬럼 데이터 맞추기를 설정하십시오.
    제목/정렬
    컬럼이 시스템 정의된 이름 이외의 다른 제목으로 표시되게 하려면 제목 필드에 이 제목을 입력하십시오. 인접 정렬 목록에서 왼쪽, 가운데 또는 오른쪽을 선택하여 이벤트 목록의 컬럼 제목 맞추기를 설정하십시오.
    너비
    이벤트 목록의 컬럼 너비를 설정하려면 슬라이더 상자를 왼쪽 또는 오른쪽으로 끌어놓으십시오.
    고정 컬럼
    이벤트 목록의 왼쪽 측면에 다수의 고정 컬럼을 둘 수 있습니다. 보기의 컬럼을 가로 스크롤하더라도 고정 컬럼이 화면 밖으로 이동하지 않습니다. 이 슬라이더를 사용하여 보기에서 총 컬럼 수보다 최대 하나 더 적게 수정하십시오.
  5. 컬럼 정렬 영역에서 다음과 같이 이벤트 목록 컬럼의 정렬 순서 및 정렬 우선순위를 지정하십시오.
    사용 가능 정렬 목록
    이 목록(왼쪽)은 이벤트 데이터를 정렬할 수 있는 사용 가능 컬럼을 모두 표시합니다. 사용 가능 정렬 목록은 추가 단추와 연계하여 사용됩니다.
    이벤트 목록 행을 정렬할 컬럼을 선택하십시오.
    팁: 이 목록의 컬럼 이름은 알파벳순으로 표시됩니다.
    추가
    사용 가능 정렬 목록에서 선택된 컬럼을 인접 정렬 순서 정의 목록에 추가하려면 이 단추를 클릭하십시오. 컬럼이 정렬 목록의 맨 아래에 추가됩니다.
    팁: 끌어놓기 기법을 사용하여 컬럼을 추가할 수 있습니다. 사용 가능 정렬 목록에서 정렬 순서 정의 목록에 추가하려는 컬럼을 선택하십시오. 이 컬럼 이름에서 마우스 가운데 단추를 누른 상태로 커서를 정렬 순서 정의 목록 안에 끌어놓으십시오. 컬럼을 놓을 수 있는 영역 안에 들어오면 커서가 변경되어 녹색 틱이 표시됩니다. 마우스 단추를 해제하십시오. 컬럼이 정렬 순서 정의 목록의 맨 아래에 추가됩니다.
    정렬 순서 정의 목록
    이 목록(오른쪽)은 이벤트 데이터를 정렬하기 위해 사용될 컬럼을 모두 표시합니다. 이 목록을 사용하여 이벤트 데이터의 정렬 정의를 다음과 같이 지정하십시오.
    • 각 컬럼의 정렬 순서를 정의하십시오. 목록의 각 컬럼 이름 왼쪽으로 화살표가 표시됩니다. 기본적으로 이 화살표는 위쪽을 향하고 컬럼이 영문자 오름차순으로 정렬됨을 표시합니다. 내림차순으로 변경하려면 화살표를 두 번 클릭하십시오. 정렬 방향을 변경하기 위해 선택한 컬럼 이름에서 마우스 오른쪽 단추를 클릭한 다음 팝업 메뉴에서 변경을 선택할 수도 있습니다.
    • 모든 컬럼의 정렬 우선순위를 정의하십시오. 정렬 목록의 맨 위에 있는 필드는 정렬 시 최고 우선순위를 갖습니다. 첫 번째 필드에 다수 항목의 동일한 값이 포함된 경우 목록의 두 번째 필드가 해당 항목을 정렬하는 데 사용됩니다. 컬럼의 정렬 우선순위를 높이거나 낮추려면 컬럼 이름을 선택한 다음 아래에 설명한 대로 맨 위, 위로, 아래로 및 맨 아래 단추를 사용하십시오.
    또한 제거 단추와 연계하여 정렬 순서 정의 목록을 사용할 수도 있습니다.
    제거
    정렬 순서 정의 목록에서 현재 선택된 정렬 컬럼을 사용 가능 정렬 목록으로 다시 옮기려면 이 단추를 클릭하십시오. 또한 선택한 컬럼 이름에서 마우스 오른쪽 단추를 클릭한 다음 팝업 메뉴에서 제거를 선택할 수도 있습니다.
    팁: 끌어놓기 기법을 사용하여 컬럼을 제거할 수 있습니다. 정렬 순서 정의 목록에서 제거하려는 컬럼을 선택하십시오. 이 컬럼 이름에서 마우스 가운데 단추를 누른 상태로 커서를 사용 가능 정렬 목록 안에 끌어놓으십시오. 컬럼을 놓을 수 있는 영역 안에 들어오면 커서가 변경되어 녹색 틱이 표시됩니다. 마우스 단추를 해제하십시오. 컬럼이 사용 가능 정렬 목록에 추가됩니다.
    맨 위
    선택한 컬럼을 목록의 맨 위로 이동하려면 이 단추를 클릭하십시오. 또한 선택한 컬럼 이름에서 마우스 오른쪽 단추를 클릭한 다음 팝업 메뉴에서 맨 위를 클릭할 수도 있습니다.
    위로
    선택한 컬럼을 목록의 한 위치 위로 이동하려면 이 단추를 클릭하십시오. 또한 선택한 컬럼 이름에서 마우스 오른쪽 단추를 클릭한 다음 팝업 메뉴에서 위로를 클릭할 수도 있습니다.
    아래로
    선택한 컬럼을 목록의 한 위치 아래로 이동하려면 이 단추를 클릭하십시오. 또한 선택한 컬럼 이름에서 마우스 오른쪽 단추를 클릭한 다음 팝업 메뉴에서 아래로를 클릭할 수도 있습니다.
    맨 아래
    선택한 컬럼을 목록의 맨 아래로 이동하려면 이 단추를 클릭하십시오. 또한 선택한 컬럼 이름에서 마우스 오른쪽 단추를 클릭한 다음 팝업 메뉴에서 맨 아래를 클릭할 수도 있습니다.
  6. 행 제한 영역에서 다음과 같이 이벤트 목록의 디스플레이 행 수를 제한하십시오.
    행 제한[1-100]
    이벤트 목록에 표시되는 행 수를 1에서 100 사이로 제한하려면 이 선택란을 선택하십시오. 최소, 최대 및 기본값 필드가 사용 가능합니다.
    참고: 행 제한[1-100] 선택란을 체크하고 구성을 .elv 파일로 저장하면 .elv 파일은 Netcool/OMNIbus V3.6과 호환되지 않습니다.
    최소값
    표시할 최소 행 수를 입력하거나 화살표를 사용하여 값을 늘리거나 줄이십시오.
    최대값
    표시할 최대 행 수를 입력하거나 화살표를 사용하여 값을 늘리거나 줄이십시오.
    기본값
    표시할 기본 행 수를 입력하거나 화살표를 사용하여 값을 늘리거나 줄이십시오.
    이벤트 목록에서 설정
    이벤트 목록 사용자가 이벤트 목록 보기에 표시할 이벤트 수를 선택할 수 있게 하려면 이 선택란을 선택하십시오. 이 선택란을 체크하는 경우 사용자가 이벤트 목록 제어를 사용해서 선택하여 볼 수 있는 이벤트 수가 보기 빌더에 지정된 최소 및 최대 값에 의해 제한됩니다. 이 선택란을 지우는 경우 표시되는 이벤트 수는 항상 지정된 기본값이 됩니다.
  7. 보기 정의 후 다음과 같이 보기를 저장하거나 변경을 취소하십시오.
    적용
    변경사항을 저장하지 않고 보기에 적용하려면 이 단추를 클릭하십시오.
    닫기
    창을 닫고 변경사항을 버리려면 이 단추를 클릭하십시오.

결과

보기는 .elv 파일 확장자로 저장해야 합니다.

참고: 컨덕터에서 UNIX 보기 빌더를 실행할 때는 적용 단추가 표시되지 않습니다. 이 경우에는 UNIX 보기 빌더에서 파일 > 저장 또는 파일 > 다른 이름으로 저장을 선택하여 보기를 모두 저장해야 합니다.
