IBM Tivoli Netcool/OMNIbus, 버전 7.4

명령행에서 임시 이벤트 목록 시작

UNIX 명령행에서 임시 이벤트 목록을 열려면 nco_elct 명령을 사용하고, Windows 명령행에서 임시 이벤트 목록을 열려면 NCOelct.exe 명령을 사용하십시오.

임시 이벤트 목록을 시작하려면 다음을 수행하십시오.
운영 체제별로 적절한 조치를 수행하십시오.
옵션 설명
UNIX 명령 프롬프트에서 다음을 입력하십시오.

$NCHOME/omnibus/bin/nco_elct option
Windows 명령 프롬프트에서 다음을 입력하십시오.

%NCHOME%\omnibus\desktop\NCOelct.exe option
위의 표에 표시된 UNIX 및 Windows 명령에서 option은 하나 이상의 명령행 옵션을 표시합니다.

결과

이벤트 목록의 인스턴스(nco_event)가 실행 중인 경우, nco_elct 유틸리티가 새 이벤트 목록 창을 엽니다. nco_event에 실행 중인 인스턴스가 없는 경우, nco_elct 유틸리티는 인스턴스를 시작하고 이벤트 목록 모니터 상자 창과 명령행에 지정된 필터를 사용하는 이벤트 목록 창을 표시합니다.

  • 임시 이벤트 목록에 대한 UNIX 명령행 옵션
    nco_elct 명령을 실행하여 UNIX에서 임시 이벤트 목록을 시작하는 경우 이벤트 목록을 사용자 정의하기 위해 다수의 명령행 옵션을 지정해야 합니다. $NCHOME/omnibus/bin 디렉토리에서 이 명령을 실행할 수 있습니다.
  • 임시 이벤트 목록에 대한 Windows 명령행 옵션
    NCOelct.exe 명령을 실행하여 Windows에서 임시 이벤트 목록을 시작하는 경우 이벤트 목록을 사용자 정의하기 위해 다수의 명령행 옵션을 지정해야 합니다. %NCHOME%\omnibus\desktop 디렉토리에서 이 명령을 실행할 수 있습니다.
