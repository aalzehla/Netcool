도구 매개변수

이 창에서 호스트 워크스테이션에 Ping 명령을 실행하거나 Telnet 유틸리티를 사용하여 원격 호스트 워크스테이션에 연결하십시오. 시스템 관리자가 이벤트 관리에 사용할 도구를 작성한 경우에는 사용자도 이 창에서 일부 도구를 실행할 수 있습니다.

Ping이나 Telnet 도구를 실행 중이면 다음과 같이 이 창을 완료하십시오. 사용자 정의한 도구를 실행 중인 경우 이 창의 필드는 다양합니다.
호스트 이름
액세스하려는 워크스테이션의 IP 주소 또는 호스트 이름을 입력하십시오.
확인
명령을 실행하려면 이 단추를 클릭하십시오.
Ping 도구를 실행한 경우 도구 출력 창에서 노드의 상태를 리턴합니다. Telnet 도구를 실행한 경우 원격 호스트 워크스테이션에 연결됩니다.
취소
명령을 실행하지 않고 이 창을 닫으려면 이 단추를 클릭하십시오.

Ping 명령 실행
