Resizing columns in the event list

You can resize the columns in the UNIX and Windows event lists.

About this task

To resize the columns in the event list:

Procedure

  1. On UNIX only, press and hold down Shift+Ctrl.
  2. Click the mouse button on a column divider.
  3. Drag the mouse inwards or outwards to change the width of the column.
