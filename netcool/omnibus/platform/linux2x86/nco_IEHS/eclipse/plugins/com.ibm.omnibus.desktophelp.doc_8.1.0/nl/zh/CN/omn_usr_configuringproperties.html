IBM Tivoli Netcool/OMNIbus V7.4

配置属性

您可以编辑以下各项的属性文件中所含的信息：ObjectServer、代理服务器、数据库初始化实用程序（nco_dbinit）、Confpack 实用程序（nco_confpack）、Netcool/OMNIbus Administrator（nco_config）、探测器和网关。属性文件具有 .props 扩展名。

要编辑属性：
  1. 执行以下任一操作：
    • 从 UNIX 上的“Conductor”窗口，单击按钮栏内的属性。
    • 从 Windows 上的 Conductor 菜单，单击属性。

      此时将打开“属性编辑器”窗口。

  2. 如下所示完成此窗口：
    类型
    在 UNIX 上，选择以下任一选项：
    • 选择普通可编辑除探测器之外的组件的属性文件。 这些文件将出现在文件下拉列表中。
    • 选择探测器可编辑探测器属性文件。 这些文件将出现在文件下拉列表中。
    在 Windows 上，选择以下任一选项：
    • 选择 ObjectServer 可编辑类似 ObjectServer 和代理服务器的组件的属性文件。 这些文件将出现在名称下拉列表中。 这些文件通常存储在 %NCHOME%\omnibus\etc 目录中。
    • 选择探测器可编辑探测器属性文件。 这些文件将出现在名称下拉列表中。 这些文件通常存储在 %NCHOME%\omnibus\probes\win32 目录中。
    • 选择服务器可添加新的属性文件。 这些文件保存在 %NCHOME%\omnibus\ini 目录中。 名称下拉列表将使用文本 <New File> 进行填充。
    文件（UNIX）/名称 （Windows）
    从下拉列表中选择要编辑的属性文件。 显示的列表取决于从类型下拉列表中所作的选择。 缺省值为：
    • nco_dbinit.props：数据库初始化开始时（运行 nco_dbinit 以初始化数据库）要读取的属性文件。
    • NCO_PROXY.props：代理服务器属性文件（代理服务器可用于减少 ObjectServer 上的探测器连接数量）。
    • nco_confpack.props：Confpack 属性文件（nco_confpack 用于导入和导出 ObjectServer 配置）。
    • nco_config.props： Netcool/OMNIbus Administrator 属性文件
    • NCOMS.props：ObjectServer 属性文件
    在 Windows 上，不会显示 .props 文件扩展名，且在名称列表中选择文件时，将使用可用属性自动填充属性列表。
    打开（仅限 UNIX）
    在 UNIX 上，选择类型和文件名后，单击该按钮可打开属性文件。
    属性列表
    此矩阵显示添加到属性文件的任何属性的详细信息。 要更改某个属性，可以在此列表中选择该属性。 已为探测器属性文件定义了一组缺省属性，用户不能除去这些属性。 这些属性将以灰色显示在属性列表中。
    名称
    此字段显示了您从属性列表中选定进行编辑的属性的名称。 不能更改已有属性的名称。 在 Windows 上，还可以使用下拉列表来选择要编辑的属性。
    如果要向选定的属性文件添加新的属性，请在此输入属性的名称。
    如果您从类型列表选择了服务器以新建属性文件，请在此输入新属性的名称。
    值
    如果要使用现有的选定属性文件，请编辑选定属性的值，或为要添加的新属性输入值。
    如果您从类型列表选择了服务器以新建属性文件，请输入新属性的值。
    添加/更新
    此按钮将根据您要添加还是编辑现有属性而发生变化。 如果输入了新的属性名和值，单击添加可向属性列表添加该值。 如果更改了属性列表中选择的属性的值，单击更新可更新属性列表中的该值。
    如果您从类型列表选择了服务器以新建属性文件，单击添加可添加新的属性。
    除去
    单击此按钮可从属性文件中除去所选属性。 如果选择了缺省属性，那么不能使用此按钮。
    保存
    单击此按钮可保存属性文件的更改。
    如果您从类型列表选择了服务器来新建属性文件，请在向属性列表添加新的属性后单击保存 。 在保存新属性窗口中输入属性文件的名称，但请略去 .props 扩展名；系统将自动为您添加该扩展名。 单击保存将文件保存到 %NCHOME%\omnibus\ini 目录中。 属性文件名将显示在与类型字段相连的名称字段中，您可以继续向该文件添加属性。 请注意，将要求您编辑此文件以向您为属性指定的字符串值添加引号。
    关闭（仅限 UNIX）
    单击此按钮可在保存更改后关闭本窗口，或关闭本窗口而不保存更改。
    确定（仅限 Windows）
    单击此按钮可在保存更改后关闭本窗口。
    取消（仅限 Windows）
    单击此按钮可关闭窗口，而不保存更改。
