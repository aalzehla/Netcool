IBM Tivoli Netcool/OMNIbus Version 8.1

Internal Command Parameters

Use this window to issue a Ping command to a host workstation, or to use the Telnet utility to connect to a remote host workstation. If your system administrator has created tools to use in event management, you can also run some of these tools from this window.

If you are running the Ping or Telnet tool, complete this window as follows. If you are running a customized tool, the fields on this window will vary.
Name of the host
Type the IP address or host name of the workstation that you want to access.
OK
Click this button to run the command.
If you ran the Ping tool, it returns the status of the node within the Tool Output window. If you ran the Telnet tool, it connects to the remote host workstation.
Cancel
Click this button to close this window without issuing the command.
