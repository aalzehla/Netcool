示例：从命令行或脚本启动瞬时事件列表

这些示例显示在 UNIX 和 Windows 上如何直接从命令行或脚本运行瞬时事件列表。

示例：nco_elct 命令（UNIX 上）

下面的命令通过使用 default.elv 视图文件从 NCOMS ObjectServer 创建瞬时事件列表。 只有来自 wombat 节点的事件才能出现在此事件列表中。

$NCHOME/omnibus/bin/nco_elct -server NCOMS -username root -password "" 
-vfile "$NCHOME/omnibus/desktop/default.elv" -ftext "( Node = \'wombat\' )"

示例：NCOelct 命令（Windows 上）

下面的命令通过使用 lview.elv 视图文件和 tool.elf 过滤器文件从 NYC ObjectServer 创建瞬时事件列表。

%NCHOME%\omnibus\desktop\NCOelct.exe -server NYC -username root -password "" 
-elv "%NCHOME%\omnibus\ini\lview.elv" -elf "%NCHOME%\omnibus\ini\tool.elf" 
-params "wombat"

tool.elf 文件（在前述 NCOelct 命令中引用） 具有以下内容：

filter_name = 'ToolFilter';
filter_text = '( Node = \'@NodeName\')';
filter_metric = 'avg(Severity)';
# End of file

运行命令或脚本时，-params 选项后的wombat 字符串将替换 tool.elf 文件中的 @NodeName 占位符，因此事件列表中只显示 wombat 节点中的事件。
