IBM Tivoli Netcool/OMNIbus 7.4 版

桌面工具概觀

許多標準桌面工具隨 Tivoli Netcool/OMNIbus 一起安裝，可讓您用來操作儲存在 ObjectServer 中的事件。

這些桌面工具包括下列應用程式：
  • 管理者
  • 事件清單
  • 過濾器建置器
  • 視圖建置器

這些應用程式可從 ObjectServer 擷取資訊，並容許您檢視資料庫的現行狀態，從而反映受管理系統的狀態。

註： 在 Linux on System z® 或 HP-UX Integrity 系統上不支援桌面工具。您可以安裝及配置 Web GUI 元件，然後透過與桌面工具類似的方式，使用 Web GUI 用戶端來檢視、管理及配置事件。
  • 管理者
    「管理者」是 Tivoli Netcool/OMNIbus 的桌面進入點。
  • 事件清單
    Tivoli Netcool/OMNIbus 使用事件清單來呈現轉遞至 ObjectServer 的事件資料相關資訊。
  • 過濾器建置器
    您可以使用過濾器，來選擇要在事件清單中顯示的事件。 「過濾器建置器」可讓您建置簡式或複式事件過濾器。
  • 視圖建置器
    您可以使用「視圖建置器」來建立事件清單視圖。 視圖可讓您選擇要在事件清單中顯示的欄位。
  • 多文化支援
    Tivoli Netcool/OMNIbus 支援各種單位元組及多位元組字元編碼，以用於不同的語言環境。
