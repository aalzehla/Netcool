IBM Tivoli Netcool/OMNIbus, 버전 7.4

컨덕터 시작

컨덕터는 경보의 열람 및 관리가 가능한 모든 Tivoli Netcool/OMNIbus 데스크탑 도구에 대해 싱글 사인온 액세스를 제공합니다.

데스크탑 도구를 사용하려면 올바른 인증 신임 정보를 사용하여 실행 중인 ObjectServer에 연결해야 합니다.

팁: 선택에 따라 UNIX의 명령행 또는 Windows 시작 메뉴에서 데스크탑 도구를 개별적으로 시작할 수 있습니다. 하지만 이 방법으로 데스크탑 도구를 시작할 때마다 선택된 ObjectServer에 대한 인증 및 연결을 위해 사용자 이름과 비밀번호를 입력해야 합니다.

컨덕터를 시작하려면 다음을 수행하십시오.

  1. 운영 체제별로 적절한 조치를 수행하십시오.
    옵션 설명
    UNIX 명령 프롬프트에서 다음을 입력하십시오. $NCHOME/omnibus/bin/nco &
    Windows
    • Windows 시작 메뉴에서 Netcool Conductor를 클릭하십시오.
    • 명령 프롬프트에서 다음을 입력하십시오.

      %NCHOME%\omnibus\desktop\NCOConductor.exe
    컨덕터의 로그인 창이 열립니다.
  2. 다음과 같이 이 창을 완료하십시오.
    사용자 이름
    사용자 이름을 입력하십시오.
    비밀번호
    비밀번호를 입력하십시오. 사용자가 입력한 텍스트의 위치에 별표(*) 문자가 나타납니다.
    참고: 사용자 이름 및 암호가 외부 인증 소스(ObjectServer가 아니라)와 대조하여 검증되고 있는 경우 이 소스가 멀티바이트 문자를 지원하는지 여부를 확인해야 합니다. 멀티바이트 문자가 지원되지 않는 경우 ASCII 문자를 사용하여 사용자 이름과 비밀번호를 지정해야 합니다.
    서버
    ObjectServer 이름이 NCOMS(Netcool/OMNIbus Master Server)의 기본값입니다. 환경에 여러 ObjectServer가 구성된 경우 이 드롭 다운 목록에서 ObjectServer가 모두 사용 가능합니다. 연결하려는 ObjectServer를 선택하십시오.
    확인
    로그인 세부사항을 인증하려면 이 단추를 클릭하십시오.
    참고: 로그인 시도는 3번까지 가능합니다. 각 시도에서 Password 필드는 지워집니다. 3번째 시도에서 올바르게 로그인하지 못하면 프로그램이 종결됩니다.
    취소
    로그인 시도를 취소하려면 이 단추를 클릭하십시오.

결과

로그인에 성공하면,
  • UNIX 컨덕터가 창에 열립니다.
  • Windows 컨덕터가 Windows 시스템 트레이에 버스 아이콘으로 표시됩니다. Windows 컨덕터 메뉴를 보려면 이 아이콘을 마우스 오른쪽 단추로 클릭하십시오.

    실행 중인 각 ObjectServer에 대해 컨덕터를 시작할 수 있습니다. Windows에서 각 컨덕터는 시스템 트레이에 아이콘을 할당받습니다.

상위 주제: 데스크탑 도구 액세스
