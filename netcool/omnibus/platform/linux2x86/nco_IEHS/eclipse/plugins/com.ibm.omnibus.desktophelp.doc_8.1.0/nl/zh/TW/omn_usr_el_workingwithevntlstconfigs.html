使用事件清單配置

您可以儲存針對「事件清單」監視器框視窗上所有監視器框定義的過濾器、視圖及其他事件配置。 此資訊儲存在事件清單配置檔案 (.elc) 中。

您可以根據需要的數量建立不同的配置檔，且您可以隨時將不同配置檔載入到「事件清單」監視器框視窗中。

您可以開啟在本端儲存或在遠端伺服器上保留的事件清單配置。 您還可以在本端儲存事件清單配置，或將其儲存到可使用 HTTP 或 FTP 存取的遠端伺服器。

  • 儲存本端事件清單配置
    從「事件清單」監視器框視窗中，您可以儲存針對現行監視器框群組定義的過濾器、視圖及其他警示配置，然後將此配置套用至事件清單。 如果您已修改針對現行監視器框配置的任何過濾器，則您會發現儲存是非常有用的。
  • 開啟本端事件清單配置
    如果您先前已在本端儲存一或多個事件清單配置，則可以開啟任何相關聯的事件清單配置檔案，以監視及管理符合已定義過濾器及視圖的事件。
  • 開啟遠端事件清單配置
    您可以開啟遠端事件清單配置檔案，以開啟位於遠端伺服器上的一或多個事件清單。 伺服器必須可以使用 HTTP 或 FTP 存取。
  • 儲存遠端事件清單配置
    如果您已開啟網路上遠端伺服器的事件清單配置，則系統不會允許您將其儲存到該伺服器中。 您必須將檔案儲存到不同位置。 伺服器必須可以使用 HTTP 或 FTP 存取。
  • 在 UNIX 上建立及編輯事件清單配置
    您可以建立或編輯事件清單配置，其中保留「事件清單」監視器框視窗中出現之監視器框的過濾器及視圖定義。
  • 在 Windows 上建立及編輯事件清單配置
    您可以建立或編輯事件清單配置，其中保留「事件清單」監視器框視窗中出現之監視器框的過濾器及視圖定義。
