過濾器建置器概觀

使用「過濾器建置器」，可以建構事件清單的過濾器。

下圖會顯示「UNIX 過濾器建置器」。

圖 1. UNIX 上的過濾器建置器

「過濾器建置器」包含下列區域：

 1  過濾器設定區域
使用此區域，可以定義過濾器的名稱並設定其許可權。
 2  按鈕列
使用這些按鈕，可以定義過濾器元素。
 3  圖形顯示區域
此區域會以顯示元素相互關係的圖形樹狀結構，來顯示現行過濾器的元素定義。 透過使用按鈕列中的按鈕，您可以將元素新增至樹狀結構。 目前選取的元素會在其周圍顯示紅色邊框。 編輯與現行元素相關。 當您選取此區域內的元素時，會顯示其詳細資料，且可在元素定義區域中進行編輯。
 4  SQL 顯示區域
SQL 顯示區域會在建置過濾器時顯示 SQL。 您可以按一下按鈕列中的編輯 SQL，以在「過濾器 SQL 編輯」視窗中直接編輯 SQL。
 5  元素定義區域
此區域會顯示目前選取之元素的詳細資料，並可讓您指定一組不同的值。 選項會反映在圖形顯示區域及 SQL 顯示區域中。
 6  度量區域
使用度量按鈕，可以將測量套用至過濾器，以便它計算有用的數字。 例如，測量總和度量會提供 ObjectServer 中非重複警示的總數。
