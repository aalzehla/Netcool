從指令行啟動暫時性事件清單

使用 nco_elct 指令，可以從 UNIX 指令行開啟暫時性事件清單，且使用 NCOelct.exe 指令，可以從 Windows 指令行開啟暫時性事件清單。

若要啟動暫時性事件清單：
針對作業系統執行適當的動作：
選項 敘述
UNIX 從命令提示字元中，輸入：

$NCHOME/omnibus/bin/nco_elct option
Windows 從命令提示字元中，輸入：

%NCHOME%\omnibus\desktop\NCOelct.exe option
在之前的表格中顯示的 UNIX 及 Windows 指令中，option 代表一個以上指令行選項。

結果

如果事件清單 (nco_event) 的實例正在執行中，則 nco_elct 公用程式會開啟新的事件清單視窗。 如果沒有執行中的 nco_event 實例，則 nco_elct 公用程式會啟動一個實例，並會使用指令行中指定的過濾器顯示「事件清單」監視器框視窗及事件清單視窗。

  • 暫時性事件清單的 UNIX 指令行選項
    當您執行 nco_elct 指令，在 UNIX 上啟動暫時性事件清單時，必須指定一些指令行選項來自訂事件清單。您可以從 $NCHOME/omnibus/bin 目錄執行此指令。
  • 暫時性事件清單的 Windows 指令行選項
    當您執行 NCOelct.exe 指令，在 Windows 上啟動暫時性事件清單時，必須指定一些指令行選項來自訂事件清單。您可以從 %NCHOME%\omnibus\desktop 目錄執行此指令。
