在 Windows 上建立及編輯事件清單配置

您可以建立或編輯事件清單配置，其中保留「事件清單」監視器框視窗中出現之監視器框的過濾器及視圖定義。

您可以指定應在「事件清單」監視器框視窗上顯示的監視器框直欄數目，指定應套用至相關聯事件清單的過濾器及視圖，以及指定過濾器及視圖應顯示的順序。

若要在 Windows 上建立或編輯事件清單配置：

  1. 從「事件清單」監視器框視窗中，執行下列任一個動作：
    • 如果您正在建立新的事件清單配置，請清除現行事件清單配置，並將配置設定為預設值。若要這麼做，請按一下檔案 > 新建。
    • 如果您正在編輯「事件清單」監視器框視窗中顯示的現有事件清單配置，請跳至步驟 2。
  2. 按一下視窗 > 配置。 即會開啟「事件清單配置」視窗。
  3. 如下所示完成此視窗：
    名稱
    如果您正在建立新的事件清單配置，請輸入事件清單配置的唯一名稱。
    提示： 使用工具列中的調節框，以指定「事件清單」監視器框視窗中容許的監視器框直欄數目。
    過濾器標籤
    按一下此標籤，以指定屬於此事件清單配置一部分的過濾器。 所選過濾器會顯示在此標籤上的清單中。您可以使用新建按鈕將新過濾器新增至事件清單配置，使用開啟按鈕將現有過濾器載入事件清單配置，或是使用編輯按鈕來編輯事件清單配置中的現有過濾器。您也可以使用刪除按鈕來移除過濾器。
    註： 過濾器在標籤中的顯示順序，會決定事件清單蹦現功能表及下拉清單顯示過濾器清單的順序。 您可以使用頂端、上移、下移及底端方向鈕來變更此順序。
    視圖標籤
    按一下此標籤，以指定屬於此事件清單配置一部分的視圖。 所選視圖會顯示在此標籤上的清單中。 您可以使用新建按鈕將新視圖新增至事件清單配置，使用開啟按鈕將現有視圖載入事件清單配置，或是使用編輯按鈕來編輯事件清單配置中的現有視圖。您也可以使用刪除按鈕來移除視圖。
    註： 視圖在標籤中的顯示順序，會決定事件清單蹦現功能表及下拉清單顯示視圖清單的順序。 您可以使用頂端、上移、下移及底端方向鈕來變更此順序。
    新建
    若要建立新的過濾器或視圖，請跳至工具列，並按一下此按鈕，以開啟「過濾器建置器」或「視圖建置器」。
    小心： 在「過濾器建置器」或「視圖建置器」內，如果您指定事件清單配置中已存在的過濾器或視圖名稱，則會取代現有的過濾器或視圖。
    開啟
    若要將現有的過濾器或視圖載入事件清單配置中，請跳至工具列並按一下此按鈕。 從所產生的視窗中，導覽至儲存過濾器或視圖的位置，選取它，然後按一下確定。 過濾器或視圖會新增至「事件清單配置」視窗中的相關標籤。
    提示： 過濾器的副檔名為 .elf。 視圖的副檔名為 .elv。
    編輯
    若要編輯過濾器或視圖，請從標籤清單中選取相關過濾器或視圖，然後跳至工具列，並按一下此按鈕以開啟所選過濾器或視圖的「過濾器建置器」或「視圖建置器」。 如果您將過濾器或視圖名稱變更為事件清單配置中不存在的名稱，則會建立新的過濾器或視圖。 如果您將名稱變更為已存在的名稱，則新的過濾器或視圖會取代事件清單配置中現有的過濾器或視圖。
    刪除
    若要從事件清單配置刪除過濾器或視圖，請從標籤清單中選取相關過濾器或視圖，然後按一下此按鈕，以刪除所選過濾器或視圖。 因為不會提示您確認刪除，所以請確保已選取適當的過濾器或視圖，然後再按一下刪除。
    頂端
    若要將過濾器或視圖移至清單頂端，請從相鄰清單中選取相關過濾器或視圖，然後按一下此按鈕。
    上移
    若要將過濾器或視圖在清單中向上移動一個位置，請從相鄰清單中選取相關過濾器或視圖，然後按一下此按鈕。
    下移
    若要將過濾器或視圖在清單中向下移動一個位置，請從相鄰清單中選取相關過濾器或視圖，然後按一下此按鈕。
    底端
    若要將過濾器或視圖移至清單底端，請從相鄰清單中選取相關過濾器或視圖，然後按一下此按鈕。
    關閉
    按一下此按鈕，以關閉「事件清單配置」視窗，並將事件清單配置套用至現行事件清單。
  4. 若要儲存事件清單配置，請從監視器框視窗中按一下檔案 > 另存新檔。 即會開啟「儲存配置」視窗。
  5. 如下所示完成此視窗：
    過濾器
    使用此欄位可以指定您要開啟之檔案類型的搜尋查詢。 此欄位與過濾按鈕及檔案清單一起使用。 請在過濾器欄位中輸入現行目錄路徑及檔案類型，然後按一下過濾。 所有相符檔案都會顯示在檔案清單中。
    例如，若要在 /myfiles/filters 目錄中顯示所有過濾器檔案，請輸入 /myfiles/filters/*.elf，再按一下過濾以檢視所有相符檔案。
    目錄
    此清單顯示現行目錄中的所有子目錄。 若要重新整理目錄，請按兩下 '.' 項目。 若要將該目錄樹狀結構向上移動一層目錄，請按兩下 '..' 項目。
    檔案
    此清單顯示現行目錄中的檔案，這些檔案符合過濾器欄位中的選項。 選取您要開啟的檔案名稱。 您也可以選取要改寫的現有檔案。
    選項
    此處顯示您要儲存或開啟之檔案的路徑及名稱。 您可以編輯路徑及檔名，以開啟不同檔案、將檔案儲存至其他位置，或使用不同檔名儲存檔案。
    如果您要儲存視圖，請在檔名中使用 .elv 副檔名。 如果您要儲存過濾器，請在檔名中使用 .elf 副檔名。 如果您要儲存事件清單配置，請在檔名中使用 .elc 副檔名。
    備份現有的檔案（僅限 UNIX）
    如果您要儲存的檔名已經存在，則選取此勾選框，以建立現有視圖檔案的備份。 僅當您要儲存過濾器、視圖或事件清單配置時，才會顯示此勾選框。
    確定
    按一下此按鈕，以開啟或儲存檔案。
    過濾器
    此按鈕與過濾器欄位及檔案清單一起使用。 按一下此按鈕，以根據過濾器欄位中的選項顯示現行目錄中的檔案。 檔案顯示在檔案清單中。
    取消
    按一下此按鈕，以關閉此視窗而不開啟或儲存檔案。
