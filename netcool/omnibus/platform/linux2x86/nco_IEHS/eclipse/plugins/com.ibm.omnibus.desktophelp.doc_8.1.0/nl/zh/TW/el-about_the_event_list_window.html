事件清單

使用此視窗，可以監視及管理警示。 警示的相關資訊會根據過濾器及視圖顯示在事件清單中。 過濾器可讓您根據特定的準則顯示警示子集。 視圖可讓您選擇要顯示的警示欄位。

事件清單包含下列區域：
  • 警示區域
  • 配送摘要列
  • 狀態列
提示： 事件清單的標題列會顯示此事件清單目前正在使用的過濾器及視圖名稱。 標題列還會指出您是否在使用雙重伺服器桌面 (DSD)。

警示區域

警示區域由表格中的列及直欄組成：
  • 每一列都對應於符合目前所選之過濾器的警示。
  • 直欄是警示欄位。 選取的直欄由目前所選的視圖控制。

配送摘要列

配送摘要列說明符合每個嚴重性顏色的警示數目。 此項與「事件清單」監視器框視窗中監視器框之配送指示器上顯示的資訊相同。

您可以按一下其中一個嚴重性層次顏色，以便僅顯示符合所選嚴重性的那些警示。

狀態列

狀態列會顯示：
  • 事件清單的相關訊息，如符合現行過濾條件或由作業處理的項目數目
  • 現行日期及時間
  • 用於登入階段作業的現行使用者名稱
  • 事件清單連接的 ObjectServer 名稱，以及它是主要 (PRI) 還是備份 (BAK) ObjectServer
