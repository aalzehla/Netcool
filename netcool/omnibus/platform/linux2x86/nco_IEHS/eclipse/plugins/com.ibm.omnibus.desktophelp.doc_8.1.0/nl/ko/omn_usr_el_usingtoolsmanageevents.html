IBM Tivoli Netcool/OMNIbus, 버전 7.4

도구를 사용한 이벤트 관리

시스템 관리자는 이벤트 목록의 이벤트를 조작하기 위해 사용할 도구를 정의하고 이 도구를 Tivoli Netcool/OMNIbus 데스크탑 메뉴에 추가할 수 있습니다. 이러한 도구를 사용하여 ObjectServer에 대한 SQL 명령을 실행하고 외부 명령(예: 로컬 애플리케이션 배치 파일 또는 스크립트)을 실행할 수 있습니다.

많은 표준 도구에는 이벤트 목록이 제공됩니다. 이 도구는 Ping 및 Telnet 도구를 포함합니다. 이 도구는 이벤트에 관계없이 실행될 수 있습니다.

  • Ping 명령 실행
    Ping 도구는 alerts.status 테이블의 노드 컬럼 값에 대해 Ping 명령을 실행합니다.
  • Telnet 명령 실행
    Telnet 도구는 alerts.status 테이블의 노드 컬럼에 보유된 값에 대해 Telnet 세션을 실행합니다.
  • 시스템 관리자에 의해 구성된 도구 실행
    시스템 관리자가 이벤트 관리에 사용할 도구를 작성한 경우 이벤트 목록의 이벤트에 대해 이 도구를 실행할 수 있습니다.
