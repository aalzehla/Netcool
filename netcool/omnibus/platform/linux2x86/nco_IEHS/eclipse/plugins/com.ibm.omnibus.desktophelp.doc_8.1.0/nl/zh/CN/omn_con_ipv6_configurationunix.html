IBM Tivoli Netcool/OMNIbus V7.4

IPv6 配置

Tivoli Netcool/OMNIbus 提供对 IPv4 和 IPv6 的支持。 这些组件可在仅支持 IPv4、仅支持 IPv6，或支持 IPv4 和 IPv6 双配置的网络上运行和共存。

Tivoli Netcool/OMNIbus 服务器组件在 IPv6 和 IPv4 环境中运行，如下所示：
  • ObjectServer 可以处理源自 IPv4 和 IPv6 网络的事件。 当 ObjectServer 运行于双堆栈主机上时，响应命令而返回给客户机的主机名是对应于客户机正在运行的 IP 版本的服务器的主机名。 例如，运行 IPv4 的客户机接收 ObjectServer 的 IPv4 主机名，而运行 IPv6 的客户机接收 ObjectServer 的 IPv6 主机名。
  • 在 IPv4 和 IPv6 双环境中，单向和双向 ObjectServer 网关可以侦听通信套接字上的这两种接口。
  • 代理服务器可以支持运行于 IPv4 和 IPv6 主机上的探测器及 ObjectServer 之间的连接。
  • 在 IPv4 和 IPv6 双环境中，进程代理程序可以侦听通信套接字上的这两种接口。
支持以下 IPv6 地址格式：
  • 八组十六进制字符（每组四个字符），以冒号分隔；例如：ABCD:EF01:2345:6789:ABCD:EF01:2345:6789
  • 若所有 16 位均为 0，该段可由冒号（:）替换。

    例如，地址 1010:0000:0000:0000:ABCD:EF01:2345:6789 可写作 1010::ABCD:EF01:2345:6789。

  • IPv4 地址可以表示为 IPv6 地址。 For example:
    • 0:0:0:0:0:0:192.101.50.5
    • 0:0:0:0:0:FFFF:103.27.35.8

      这些地址也可以表示为如下形式：

    • ::192.101.50.5
    • ::FFFF:103.27.35.8
