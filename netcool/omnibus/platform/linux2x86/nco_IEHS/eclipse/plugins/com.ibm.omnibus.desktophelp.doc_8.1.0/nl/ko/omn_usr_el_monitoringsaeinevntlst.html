IBM Tivoli Netcool/OMNIbus, 버전 7.4

이벤트 목록에서 서비스 영향 이벤트 모니터링

서비스 관련 이벤트(SAE)는 하나 이상의 네트워크 이벤트에 의해 중요한 고객 서비스가 영향을 받고 있음을 운영자에게 경고하는 경보입니다. 서비스 관련 이벤트는 IBM Tivoli Network Manager IP Edition에서 생성됩니다.

시스템이 이 기능을 지원하도록 구성되어 있는 경우 Tivoli Netcool/OMNIbus 이벤트 목록에서 서비스 영향 이벤트를 모니터할 수 있습니다.

Tivoli Netcool/OMNIbus 이벤트 목록에서 다음과 같이 서비스 관련 이벤트를 모니터할 수 있습니다.

  • 서비스 영향 이벤트와 연관된 기본적인 이벤트(예: linkDowns)를 표시하려면 서비스 영향 이벤트를 선택하고 마우스 오른쪽 단추로 클릭한 후 팝업 메뉴에서 SAE 관련 이벤트 표시를 클릭하십시오. 선택한 이벤트와 연관된 모든 이벤트가 새 창에 표시됩니다.
  • 관련된 이벤트(예: linkDown)에 대해 서비스 영향 이벤트를 표시하려면 이벤트를 선택하고 마우스 오른쪽 단추로 클릭한 후 팝업 메뉴에서 SAE 관련 서비스 표시를 클릭하십시오. 모든 관련 서비스 영향 이벤트의 목록은 새 창에 표시됩니다. 예를 들어, 이벤트 목록에서 linkDown을 선택한 경우, 이 창에 해당 linkDown 이벤트에 의해 영향을 받는 모든 서비스가 표시됩니다.
