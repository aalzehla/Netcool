イベント・リスト

このウィンドウを使用して、アラートをモニターおよび管理します。 アラートに関する情報は、フィルターとビューに基づいてイベント・リスト内に表示されます。 フィルターを使用すると、特定の基準に基づいてアラートのサブセットを表示することができます。 ビューを使用すると、表示するアラート・フィールドを選択することができます。

イベント・リストには、以下の領域が含まれています。
  • アラート領域
  • 配布サマリー・バー
  • ステータス・バー
ヒント: イベント・リストのタイトル・バーには、そのイベント・リストで現在使用中のフィルターとビューの名前が表示されます。 タイトル・バーには、デュアル・サーバー・デスクトップ (DSD) を使用しているかどうかも示されます。

アラート領域

アラート領域は、表内の行と列から以下のように構成されます。
  • 各行は、現在選択されているフィルターに一致するアラートに対応する。
  • 列はアラート・フィールドである。 列の選択は、現在選択されているビューによって制御されます。

配布サマリー・バー

配布サマリー・バーには、各重大度の色に一致するアラートの数が示されます。 これは、「イベント・リスト」モニター・ボックス・ウィンドウ内のモニター・ボックスの配布インディケーターで表示される情報と同じものです。

重大度レベルの色のいずれかをクリックして、選択した重大度に一致するアラートのみを表示することができます。

ステータス・バー

ステータス・バーには、次の情報が表示されます。
  • 現在のフィルター条件に一致した項目数や、操作で処理された項目数などのイベント・リストに関するメッセージ
  • 現在の日時
  • セッションへのログインに使用した現在のユーザー名
  • イベント・リストが接続されている ObjectServer の名前と、これが 1 次 (PRI) ObjectServer かバックアップ (BAK) ObjectServer か
