IBM Tivoli Netcool/OMNIbus, 버전 7.4

임시 이벤트 목록에 대한 Windows 명령행 옵션

NCOelct.exe 명령을 실행하여 Windows에서 임시 이벤트 목록을 시작하는 경우 이벤트 목록을 사용자 정의하기 위해 다수의 명령행 옵션을 지정해야 합니다. %NCHOME%\omnibus\desktop 디렉토리에서 이 명령을 실행할 수 있습니다.

다음 표는 Windows에서 임시 이벤트 목록을 시작하기 위한 명령행 옵션을 설명합니다.

표 1. Windows 임시 이벤트 목록(NCOelct.exe) 명령행 옵션
명령행 옵션 설명
-diagnostic 확장된 진단 메시지를 표시합니다.
-dualwrite 0 | 1 데스크탑 ObjectServer 아키텍처에서 작동할 때 이중 쓰기 모드를 사용할지 또는 사용하지 않을지 여부를 표시합니다. 올바른 옵션은 다음과 같습니다.
  • 0: 이중 쓰기 모드 사용 안함
  • 1: 이중 쓰기 모드 사용

-dualwrite 명령행 옵션은 데스크탑 ObjectServer의 master.national 테이블에 있는 DualWrite 필드 항목을 겹쳐씁니다.

-elf string | URL

이 옵션은 필수입니다.

임시 이벤트 목록에 사용할 필터 파일을 지정합니다. 여기서

string은 .elf 파일에 대한 경로를 지정합니다.

URL은 HTTP 또는 FTP를 사용하여 액세스할 수 있는 원격 서버의 .elf 파일을 지정합니다. 다음 형식 중 하나를 사용하여 URL을 지정하십시오.

  • http://servername[:port]/path/filename.elf
  • http://username:password@servername[:port]/path/filename.elf
  • ftp://servername[:port]/path/filename.elf
  • ftp://username:password@servername[:port]/path/filename.elf

서버가 기본 포트를 사용하고 있지 않은 경우에만 선택적 포트 번호(:port)를 지정해야 합니다.

FTP를 사용 중인 경우 경로는 루트 디렉토리로부터의 전체 경로여야 합니다. 예를 들어, /home/user/mydir/myfilter.elf입니다.

-params 명령행 옵션을 사용하여 .elf 파일에 지정된 필터 매개변수 외에 필터 매개변수를 추가로 지정할 수 있습니다. 사용법 세부사항은 이 테이블에서 -params 설명을 참조하십시오.

-elv string | URL 임시 이벤트 목록에 사용할 이벤트 목록 보기 파일을 지정합니다. 여기서

string은 .elv 파일에 대한 경로를 지정합니다.

URL은 HTTP 또는 FTP를 사용하여 액세스할 수 있는 원격 서버의 .elv 파일을 지정합니다. 다음 형식 중 하나를 사용하여 URL을 지정하십시오.

  • http://servername[:port]/path/filename.elv
  • http://username:password@servername[:port]/path/filename.elv
  • ftp://servername[:port]/path/filename.elv
  • ftp://username:password@servername[:port]/path/filename.elv

서버가 기본 포트를 사용하고 있지 않은 경우에만 선택적 포트 번호(:port)를 지정해야 합니다.

FTP를 사용 중인 경우 경로는 루트 디렉토리로부터의 전체 경로여야 합니다. 예를 들어, /home/user/mydir/myview.elv입니다.

-help 이 명령에 대한 도움말을 표시하고 종료합니다.
-masterserver string 데스크탑 ObjectServer 아키텍처에서 사용할 마스터 ObjectServer를 표시합니다.

-masterserver 명령행 옵션은 데스크탑 ObjectServer의 master.national 테이블에서 MasterServer 필드 항목을 겹쳐씁니다.

-networktimeout integer 네트워크 장애가 발생할 경우 ObjectServer와의 연결 또는 로그인 시도가 제한시간 초과되는 시간(초)을 지정합니다. 지정된 제한시간 기간이 지난 후 이벤트 목록은 ObjectServer에 다시 연결하려고 시도합니다. 두 번째 제한시간 기간 동안 연결에 실패하면, 이벤트 목록이 백업 ObjectServer(사용 가능한 경우)에 연결하려고 시도합니다. 기본값은 20초입니다.
-params string 필터 파일의 filter_text 매개변수에서 단일 위치 표시기를 바꾸기 위해 사용할 문자열을 지정합니다. 이 위치 표시기는 @ 기호로 시작해야 합니다(예: @FilterText).
-password string 선택한 사용자 이름에 대한 비밀번호를 지정합니다. NCDelct 유틸리티는 비밀번호의 영숫자 문자만을 채택합니다. 특수 문자(예: #, /, >, @ 및 !)는 비밀번호에 사용할 수 없습니다.
-server string 연결 중인 ObjectServer 이름을 지정합니다.
-username string 이벤트 목록에 로그인하기 위해 사용할 사용자 이름을 지정합니다.
-version 버전 정보를 표시하고 종료됩니다.
상위 주제: 명령행에서 임시 이벤트 목록 시작
관련 개념:
데스크탑 ObjectServer 아키텍처
관련 참조:
