コマンド行、スクリプト、およびツールからの一時的イベント・リストの実行

カスタマイズした一時的イベント・リストは、コマンド行からや、スクリプトの中で、またはイベント・リスト・ツールの一部として直接実行することができます。

例えば、フィルターを適用するイベント・リストを開いて、特定の ObjectServer からのすべての重大なアラートを表示することができます。

  • コマンド行から一時的イベント・リストを開始する
    UNIX のコマンド行から一時的イベント・リストを開く場合は nco_elct コマンドを使用し、Windows のコマンド行から 一時的イベント・リストを開く場合は NCOelct.exe コマンドを使用します。
  • 例: コマンド行またはスクリプトからの一時的イベント・リストの開始
    これらの例では、UNIX および Windows の両方でコマンド行またはスクリプトから直接、一時的イベント・リストを実行する方法を示します。
  • 例: ツールからの一時的イベント・リストの開始
    これらの例では、UNIX および Windows 上でツールから一時的イベント・リストを実行する方法を示します。
