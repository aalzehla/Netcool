IBM Tivoli Netcool/OMNIbus バージョン 7.4

ツール・パラメーター

このウィンドウを使用して、Ping コマンドをホスト・ワークステーションに発行するか、または Telnet ユーティリティーを使用してリモート・ホスト・ワークステーションに接続します。 システム管理者がイベント管理用のツールを作成した場合は、ユーザーも、このウィンドウからそれらのツールの一部を実行することができます。

Ping または Telnet のツールを実行している場合、以下のようにしてこのウィンドウを完了します。 カスタマイズされたツールを実行している場合、このウィンドウに表示されるフィールドは異なります。
ホスト名
アクセスするワークステーションの IP アドレスまたはホスト名を入力します。
OK
このボタンをクリックして、コマンドを実行します。
Ping ツールを実行した場合は、「ツールの出力」ウィンドウ内のノードの状況が返されます。 Telnet ツールを実行した場合は、リモート・ホスト・ワークステーションに接続されます。
キャンセル
このボタンをクリックして、コマンドを発行せずにウィンドウを閉じます。
