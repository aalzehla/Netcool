IBM Tivoli Netcool/OMNIbus V7.4

桌面工具概述

大量标准的桌面工具与 Tivoli Netcool/OMNIbus 一起安装，以使您能够操作存储在 ObjectServer 中的事件。

这些桌面工具包含以下应用程序：
  • 执行器
  • 事件列表
  • 过滤器构建器
  • 视图构建器

这些应用程序检索来自 ObjectServer 的信息，并使您能够查看数据库的当前状态，该状态进而反映受管系统的状态。

注： 桌面工具在 Linux on System z® 或 HP-UX Integrity 系统上不受支持。您可以安装和配置 Web GUI 组件，然后使用 Web GUI 客户机来查看、管理和配置事件，其方法类似于桌面工具。
  • 执行器
    Conductor 是 Tivoli Netcool/OMNIbus 的桌面入口点。
  • 事件列表
    Tivoli Netcool/OMNIbus 使用事件列表来显示有关转发至 ObjectServer 的事件数据的信息。
  • 过滤器构建器
    您可使用过滤器选择要在事件列表中显示的事件。 过滤器构建器使您能够构建简单或复杂事件过滤器。
  • 视图构建器
    你可使用视图构建器来创建事件列表视图。 视图使您能够选择要在事件列表中显示的字段。
  • 多文化支持
    Tivoli Netcool/OMNIbus支持可用于不同语言环境的各种单字节和多字节字符编码。
