从命令行启动瞬时事件列表

从 UNIX 命令行使用 nco_elct 命令可打开瞬时事件列表，从 Windows 命令行使用 NCOelct.exe 命令可打开瞬时事件列表。

要启动瞬时事件列表：
为操作系统执行相应的操作：
选项 描述
UNIX 从命令提示符输入：

$NCHOME/omnibus/bin/nco_elct option
Windows 从命令提示符输入：

%NCHOME%\omnibus\desktop\NCOelct.exe option
在上表内显示的 UNIX 和 Windows 命令中，option 代表一个或多个命令行选项。

结果

如果事件列表的实例（nco_event）正在运行，nco_elct 实用程序将打开一个新事件列表窗口。 如果没有 nco_event 的实例正在运行，nco_elct 实用程序将启动一个实例，然后显示“事件列表”监视器框窗口以及使用命令行内指定的过滤器的事件列表窗口。

  • 瞬时事件列表的 UNIX 命令行选项
    在 UNIX 上运行 nco_elct 命令启动瞬时事件列表时，必须指定若干命令行选项来定制事件列表。您可从 $NCHOME/omnibus/bin 目录运行此命令。
  • 瞬时事件列表的 Windows 命令行选项
    在 Windows 上运行 NCOelct.exe 命令启动瞬时事件列表时，必须指定若干命令行选项来定制事件列表。可以从 %NCHOME%\omnibus\desktop 目录运行此命令。
