IBM Tivoli Netcool/OMNIbus, 버전 7.4

필터 조건에 대한 연산자 지정

필터 빌더에서 단순 또는 복합 조건을 정의할 때 연산자를 사용하여 비교를 정의합니다.

사용 가능 연산자의 목록은 다음을 참조하십시오.
  • 같음
  • 같지 않음
  • 초과
  • 미만
  • 이상
  • 이하
  • 유사
  • 유사하지 않음
  • 속함
  • 속하지 않음

이 연산자는 속함과 속하지 않음을 제외하고 모두 단일 값과 비교하기 위해 사용됩니다.

유사와 유사하지 않음 연산자는 필요한 경우 정규식을 채택하고 하위 문자열 비교를 수행합니다.
