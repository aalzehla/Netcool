Dragging and dropping views on UNIX

You can use the drag-and-drop feature with views on UNIX.

About this task

You can drag and drop from:
  • One View Builder to another
  • The View Builder to an Event List Configuration window

To drag and drop a view:

Procedure

  1. From the View Builder, move the cursor over the drag point icon (to the right of the Editable check box).

  2. Click and hold the middle mouse button.
  3. Drag the mouse to another View Builder or an Event List Configuration window. When the cursor is over a valid drop area, its color changes to green.
  4. Release the mouse button to drop the view.
