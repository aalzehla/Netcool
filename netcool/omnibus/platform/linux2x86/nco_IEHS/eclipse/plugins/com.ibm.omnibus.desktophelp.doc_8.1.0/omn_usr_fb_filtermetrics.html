Filter metrics

All filters can have a metric associated with them. A metric is a measurement that you can apply to a filter so that it calculates a useful figure.

For example, a metric could be an average, count, or sum of all the alerts. When you are creating a filter, you can select the type of metric to use. Metrics are displayed in monitor boxes.
