Hiding events

You can hide events that you no longer want to view in the event list. Typically, hidden events are not displayed in any monitor boxes or event lists, except for those using the All Events filter.

About this task

You can hide events only if you have permission to do so.

To hide selected events in the event list, click Alerts > Prioritize > Suppress/Escalate > Hidden. This sets the value of the SuppressEscl field to 5 in the alerts.status table. When you hide an event, it is hidden in all operator event lists except those where the filter allows for a match of SuppressEscl=5.

To remove the hidden status from events, open an event list that uses the All Events filter, select the alerts, and then click Alerts > Prioritize > Suppress/Escalate > Normal.
