選取及取消選取事件

若要使用事件清單中的一或多個事件，您必須首先選取事件。 然後，您可以使用警示功能表中的選項來操作事件。 您還可以從已選取事件清單中取消選取一或多個事件。

註： 您可以用滑鼠右鍵按一下事件，以存取警示蹦現功能表。 警示功能表是由您的系統管理者配置。

您可以使用下列方式來選取單一或多個事件：

  • 若要選取一個事件並取消選取所有其他事件，請按一下事件列。
  • 若要選取多個不連續的事件，請按住 Ctrl 鍵並按一下每個事件。
  • 若要在 UNIX 上選取多個連續的事件，請按一下以選取第一個事件，按住滑鼠按鈕並在清單中向上或向下拖曳游標。若要在 Windows 上選取多個連續的事件，請按住 Shift 鍵，然後按一下以選取第一個事件。然後，按住滑鼠按鈕並在清單中向上或向下拖曳游標。
  • 若要選取所有事件，請按一下編輯 > 全選。
  • 若要在 UNIX 上選取符合特定欄位中值的所有事件（也稱為智慧型相符），請按住 Shift 鍵，並在要與之相符的欄位上按一下左滑鼠按鈕。若要在 Windows 上選取符合特定欄位中值的所有事件，請按住 Ctrl+Alt 鍵，並在要與之相符的欄位上按一下左滑鼠按鈕。 例如，若要選取 UNIX 事件清單中「狀態」欄位已關閉的所有事件，請尋找其「狀態」欄位顯示為關閉的事件。將游標移至該欄位上，按住 Shift 鍵並在該欄位上按一下左滑鼠按鈕。

下一步

您可以使用下列方式來取消選取單一或多個事件：

  • 若要在事件清單中取消選取所有事件，請按一下編輯 > 取消全選。
  • 若要取消選取所選多個列中的事件，請按住 Ctrl 鍵並按一下該事件。
