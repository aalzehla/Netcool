개인용 라이브러리의 기호 링크 작성

로컬 UNIX 시스템의 파일에 대해 개인용 라이브러리에서 기호 링크를 작성할 수 있습니다.

기호 링크를 작성하려면 다음을 수행하십시오.
디렉토리 브라우저 창에서 Shift+Ctrl을 누름과 동시에 마우스 가운데 단추를 사용하여 기호 링크를 작성하려는 파일을 개인용 라이브러리에 끌어놓으십시오. (이 조치는 ln -s 명령을 사용하는 것과 같습니다.) 기호 링크는 개인용 라이브러리 안에 작성됩니다. 기호 링크에 대한 아이콘은 아이콘 위에 체인으로 표시됩니다.

다음에 수행할 작업

팁: 또한 개인용 라이브러리의 파일을 디렉토리 브라우저 안에 복사할 수 있습니다. 환경 설정 아이콘을 제외한 모든 파일을 이동 또는 복사할 수 있습니다.
상위 주제: 자주 사용되는 파일의 개인용 라이브러리 유지보수
