필터의 구성요소

필터는 요소의 콜렉션으로 구성되어 있습니다. 사용 가능 요소 유형은 조건, 논리 및 하위 조회 요소를 포함합니다.

  • 조건 요소
    필터 구성 시 조건 요소를 사용하여 데이터베이스 컬럼과 값을 비교할 수 있습니다. UNIX에서도 추가 고급 조건을 지정할 수 있습니다.
  • 논리 요소
    필터에 단일 조건 요소 사용 시 단순 비교로 제한됩니다. 논리 요소를 사용하면 추가 복합 비교를 위해 두 개의 요소를 결합할 수 있습니다.
  • 하위 조회 요소
    필터 구성 시 하위 조회 요소를 사용하여 조건의 조회를 작성할 수 있습니다.
