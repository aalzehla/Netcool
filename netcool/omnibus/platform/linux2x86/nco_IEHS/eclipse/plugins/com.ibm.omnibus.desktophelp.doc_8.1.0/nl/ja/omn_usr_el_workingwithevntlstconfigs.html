イベント・リスト構成の操作

「イベント・リスト」モニター・ボックス・ウィンドウのすべてのモニター・ボックスで定義したフィルターやビューや他のイベント構成を保存できます。 その情報の保存先は、イベント・リスト構成 (.elc) ファイルになります。

さまざまな構成ファイルを必要な数だけ作成できます。また、「イベント・リスト」モニター・ボックス・ウィンドウには、いつでも別の構成ファイルをロードできます。

ローカルに保存されているイベント・リスト構成も、リモート・サーバーに保存されているイベント・リスト構成も開くことができます。 さらに、イベント・リスト構成をローカルに保存することも、HTTP または FTP でアクセスできるリモート・サーバーに保存することも可能です。

  • ローカル・イベント・リスト構成の保存
    「イベント・リスト」モニター・ボックス・ウィンドウでは、現在のモニター・ボックス・グループで定義されているフィルターやビューや他のアラート構成を保存して、後からその構成をイベント・リストに適用できます。 現在のモニター・ボックスで構成されているいずれかのフィルターを変更した場合は、そのフィルターの変更を保存しておくと便利です。
  • ローカル・イベント・リスト構成を開く
    以前にローカルに 1 つ以上のイベント・リスト構成を保存している場合、 定義されたフィルターおよびビューに一致するイベントをモニターして管理するために、 関連付けられたイベント・リスト構成ファイルのいずれかを開くことができます。
  • リモート・イベント・リスト構成を開く
    リモート・サーバーに配置された 1 つ以上のイベント・リストを開くために、リモート・イベント・リスト構成ファイルを開くことができます。 HTTP または FTP を使用してサーバーにアクセス可能である必要があります。
  • リモート・イベント・リスト構成の保存
    ネットワーク上のリモート・サーバーからイベント・リスト構成を開いた場合は、そのサーバーに構成内容を保存できません。 別のロケーションにファイルを保存する必要があります。 そのサーバーは、HTTP または FTP によってアクセスできる状態になっていなければなりません。
  • UNIX 上でのイベント・リスト構成の作成および編集
    「イベント・リスト」モニター・ボックス・ウィンドウ内に表示されるモニター・ボックスで、フィルターを保持して定義を表示するイベント・リスト構成を作成または編集することができます。
  • Windows 上でのイベント・リスト構成の作成および編集
    「イベント・リスト」モニター・ボックス・ウィンドウ内に表示されるモニター・ボックスで、フィルターを保持して定義を表示するイベント・リスト構成を作成または編集することができます。
