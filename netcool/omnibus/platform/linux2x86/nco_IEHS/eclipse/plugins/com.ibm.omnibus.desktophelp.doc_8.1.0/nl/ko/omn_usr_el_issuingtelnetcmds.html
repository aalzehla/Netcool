Telnet 명령 실행

Telnet 도구는 alerts.status 테이블의 노드 컬럼에 보유된 값에 대해 Telnet 세션을 실행합니다.

Telnet 도구를 실행하려면 다음을 수행하십시오.
  1. 이벤트 목록에서 도구 > Telnet을 클릭하십시오. 도구 매개변수 창이 열립니다.
  2. 다음과 같이 이 창을 완료하십시오.
    호스트 이름
    액세스하려는 워크스테이션의 IP 주소 또는 호스트 이름을 입력하십시오.
    확인
    명령을 실행하려면 이 단추를 클릭하십시오.
    Ping 도구를 실행한 경우 도구 출력 창에서 노드의 상태를 리턴합니다. Telnet 도구를 실행한 경우 원격 호스트 워크스테이션에 연결됩니다.
    취소
    명령을 실행하지 않고 이 창을 닫으려면 이 단추를 클릭하십시오.
상위 주제: 도구를 사용한 이벤트 관리
