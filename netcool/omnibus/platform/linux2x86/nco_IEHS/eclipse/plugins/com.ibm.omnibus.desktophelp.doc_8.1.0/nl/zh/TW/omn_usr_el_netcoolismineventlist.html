IBM Tivoli Netcool/OMNIbus 7.4 版

事件清單中的網際網路服務監視器

IBM Tivoli Composite Application Manager for Internet Service Monitoring 可與其餘 Tivoli Netcool/OMNIbus 產品組合一起使用。 監視器及探測器產生的事件都可以轉遞到相同的 ObjectServer，且您可以使用相同的桌面操作這些事件。

安裝 Composite Application Manager for Internet Service Monitoring 之後，會將兩個新按鈕及一個下拉清單新增至事件清單。 您可以使用按鈕及下拉清單，如下所示：
  • 按一下服務視圖按鈕 ，以在事件清單中顯示服務警示。 透過使用服務下拉清單，您可以變更服務類型。
  • 按一下顯示服務按鈕 ，以從事件清單啟動「服務」視窗。
  • 使用服務下拉清單 ，以變更事件清單中顯示之服務狀態事件的類型。 例如，從清單中選取監視器類型會顯示該類型監視器的所有服務狀態事件。
  • 檢視事件清單中的服務
    如果您已安裝 IBM Tivoli Composite Application Manager for Internet Service Monitoring，則可以從網際網路服務監視器檢視及操作服務警示。
