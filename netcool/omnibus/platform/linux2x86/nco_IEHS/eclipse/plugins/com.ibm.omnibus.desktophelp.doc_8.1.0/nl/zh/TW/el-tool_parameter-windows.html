內部指令參數

使用此視窗，可以向主機工作站發出 Ping 指令，或使用 Telnet 公用程式連接到遠端主機工作站。 如果系統管理者已建立要在事件管理中使用的工具，則您也可以從此視窗執行其中的部分工具。

如果您正在執行 Ping 或 Telnet 工具，請按如下所述完成此視窗。如果您正在執行自訂工具，此視窗中的欄位將會不同。
主機名稱
輸入您要存取之工作站的 IP 位址或主機名稱。
確定
按一下此按鈕，以執行指令。
如果已執行 Ping 工具，則它會在「工具輸出」視窗內傳回節點的狀態。 如果已執行 Telnet 工具，則它會連接到遠端主機工作站。
取消
按一下此按鈕，以關閉此視窗而不發出指令。
相關工作:
發出 Ping 指令
發出 Telnet 指令
