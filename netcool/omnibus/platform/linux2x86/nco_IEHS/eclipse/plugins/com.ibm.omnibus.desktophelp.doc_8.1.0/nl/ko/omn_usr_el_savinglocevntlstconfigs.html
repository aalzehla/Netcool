IBM Tivoli Netcool/OMNIbus, 버전 7.4

로컬 이벤트 목록 구성 저장

이벤트 목록 모니터 상자 창에서 모니터 상자의 현재 그룹에 대해 정의되는 필터, 보기 및 기타 경보 구성을 저장할 수 있고 추후 이 구성을 이벤트 목록에 적용할 수 있습니다. 현재 모니터 상자에 대해 구성되는 필터 중 임의의 필터를 수정한 경우 저장하는 것이 유용하다는 것을 알 수 있습니다.

동일한 파일 또는 다른 파일에 대한 구성 변경사항을 저장할 수 있습니다. 이벤트 목록 모니터 상자 창의 제목 표시줄은 일반적으로 현재 이벤트 목록 구성의 파일 이름을 포함합니다. 이벤트 목록 모니터 상자 창의 제목 표시줄이 파일 이름 대신에 [없음]을 표시하는 경우 이는 구성이 아직 저장되지 않았음을 의미합니다. 이벤트 목록 구성 파일은 .elc 확장자로 저장해야 합니다.

이벤트 목록 모니터 상자 창의 제목 표시줄에 표시되는 이벤트 목록 구성 파일의 변경사항을 저장하려면 파일 > 저장을 클릭하십시오. 구성 저장 창이 열립니다. 파일을 변경사항으로 겹쳐쓰려면 확인을 클릭하십시오.

저장하지 않은 이벤트 목록 구성을 저장하거나 현재 이벤트 목록 구성을 다른 파일 이름으로 저장하려면 파일 > 다른 이름으로 저장을 클릭하십시오. 구성 저장 창이 열립니다. 다른 디렉토리 및 파일 이름을 선택하고 확인을 클릭하십시오. 이벤트 목록 모니터 상자 창이 새로 고쳐져 이벤트 목록의 제목 표시줄에 구성 파일 이름을 표시합니다.
