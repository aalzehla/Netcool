Running tools configured by your system administrator

If your system administrator has created tools to use in event management, you can run these tools on events in the event list.

About this task

If configured, you can run tools in a number of ways:

Procedure

  • From the Conductor window on UNIX, click the tool in the Tools menu.
  • From the Conductor menu on Windows, click Tools and then click the tool in the submenu.
  • From the Event List monitor box window, click the tool in the Tools menu.
  • From the event list, click Alerts > Tools and then click the required tool in the submenu. Alternatively, you can right-click and then click Alerts > Tools in the pop-up menu.
