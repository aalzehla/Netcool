IBM Tivoli Netcool/OMNIbus 7.4 版

使用工具管理事件

您的系統管理者可以定義要用於操作事件清單中事件的工具，並可將這些工具新增至 Tivoli Netcool/OMNIbus 桌面功能表以供您使用。 您可以使用此類工具在 ObjectServer 上執行 SQL 指令，以及執行外部指令，如本端應用程式批次檔或 Script。

事件清單有提供一些標準工具。 其中包括 Ping 及 Telnet 工具。 這些工具可獨立於事件執行。

  • 發出 Ping 指令
    Ping 工具會對 alerts.status 表格之「節點」直欄中的值執行 Ping 指令。
  • 發出 Telnet 指令
    Telnet 工具會對 alerts.status 表格之「節點」直欄中保留的值執行 Telnet 階段作業。
  • 執行系統管理者配置的工具
    如果系統管理者已建立要在事件管理中使用的工具，則您可以在事件清單中的事件上執行這些工具。
