IBM Tivoli Netcool/OMNIbus 7.4 版

在 Windows 上存取線上說明

線上說明使用「IBM® Eclipse 說明系統 (IEHS)」（其為 Web 應用程式）進行部署。基於安裝及配置設定，您可以獨立模式或資訊中心模式存取線上說明。

若要配置及存取線上說明，請依下表所述開始進行。

表 1. 說明配置設定
設定 說明
獨立模式 若要以獨立模式使用線上說明，則在安裝 Tivoli Netcool/OMNIbus 期間，您必須已選取本端說明系統功能。
安裝之後，僅當另一個本端服務正在使用預設 IEHS 埠號 8888 時，才必須進行配置。 如果是這種情況，請編輯 IEHS 配置檔 %NCHOME%\omnibus\ini\nco_IEHS.cfg，如下所示：
  • IEHSMode：0
  • IEHSHost：空白
  • IEHSPort：未用的埠號
註： 變更配置檔中的埠號之後，您必須關閉本端 IEHS 伺服器，變更才會生效。 執行 %NCHOME%\omnibus\bin\help_end.bat 指令。
資訊中心模式 在資訊中心模式中，您的系統管理者會配置設定，以便所有使用者都可以共用線上說明系統的同一 IEHS Web 伺服器，以減輕本端主機或工作站上的負載。

如果您的系統管理者已通知您必須以資訊中心模式存取線上說明，則您必須配置本端 IEHS 配置檔 %NCHOME%\omnibus\ini\nco_IEHS.cfg，如下所示：

  • IEHSMode：1
  • IEHSHost：IEHS 伺服器的 IP 位址或主機名稱
  • IEHSPort：執行 IEHS 伺服器的埠號

您的系統管理者會提供 IEHSHost 及 IEHSPort 設定。

當您以獨立模式存取線上說明時，您的預設說明瀏覽器視窗會自動開啟並顯示相關文字。 本端 IEHS 伺服器開始於第一個說明要求，並繼續執行直到您關閉它為止。 若要關閉本端 IEHS 伺服器，請執行下列指令：

%NCHOME%\omnibus\bin\help_end.bat

通常，僅在重新開啟電腦之前才會執行 help_end.bat 指令。

在您可以使用資訊中心模式存取線上說明之前，您的系統管理者必須啟動指定的遠端 IEHS 伺服器。

註： 在 Windows 上，如果您沒有首先關閉 IEHS 伺服器，就重新啟動系統或停止 IEHS 的 javaw 程序（使用「工作管理員」），則不會移除部分由 IEHS 伺服器產生的暫存檔。這可能會導致存取線上說明時出現困難。 在獨立模式及資訊中心模式中，您都必須執行 help_end.bat 或 IC_end.bat，以在重新啟動系統之前先關閉 IEHS 伺服器，且不得手動停止 IEHS javaw 程序。 如果您無法以獨立模式存取說明，或無法以資訊中心模式啟動 IEHS 伺服器，請執行 help_end.bat 或 IC_end.bat 以清除暫存檔，然後重試。
