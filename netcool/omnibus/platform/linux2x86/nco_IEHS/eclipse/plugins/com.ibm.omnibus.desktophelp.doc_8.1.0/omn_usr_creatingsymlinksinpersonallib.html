Creating symbolic links in the personal library

You can create a symbolic link in your personal library, to a file in your local UNIX system.

About this task

To create symbolic links:

Procedure

From the Directory Browser window, press Shift+Ctrl and then simultaneously use the middle mouse button to drag the file for which you want to create a symbolic link, to the personal library. (This is equivalent to using the ln -s command.) A symbolic link is created within the personal library. The icon for the symbolic link is shown with a chain over the icon image.

What to do next

Tip: You can also copy files from the personal library into the directory browser. You can move or copy all files except the prefs icon.
