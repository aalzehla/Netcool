IBM Tivoli Netcool/OMNIbus Version 8.1

Multicultural support

Tivoli Netcool/OMNIbus supports a variety of single-byte and multi-byte character encodings for use in different locales.

Note: If your user name and password are being verified against an external authentication source, you must check whether this source also supports multi-byte characters. If multi-byte characters are not supported, you must specify user names and passwords with ASCII characters.
