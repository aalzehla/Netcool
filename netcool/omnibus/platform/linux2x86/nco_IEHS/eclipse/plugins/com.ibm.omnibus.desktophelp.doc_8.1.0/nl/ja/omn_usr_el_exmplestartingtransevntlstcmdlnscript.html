例: コマンド行またはスクリプトからの一時的イベント・リストの開始

これらの例では、UNIX および Windows の両方でコマンド行またはスクリプトから直接、一時的イベント・リストを実行する方法を示します。

例: UNIX の nco_elct コマンド

以下のコマンドは、 default.elv ビュー・ファイルを使用して、NCOMS ObjectServer の一時的イベント・リストを作成します。 イベント・リストには、ノード wombat のイベントのみが表示されます。

$NCHOME/omnibus/bin/nco_elct -server NCOMS -username root -password "" 
-vfile "$NCHOME/omnibus/desktop/default.elv" -ftext "( Node = ¥'wombat¥' )"

例: Windows の NCOelct コマンド

以下のコマンドは、 lview.elv ビュー・ファイルおよび tool.elf フィルター・ファイルを使用して、NYC ObjectServer の一時的イベント・リストを作成します。

%NCHOME%¥omnibus¥desktop¥NCOelct.exe -server NYC -username root -password "" 
-elv "%NCHOME%¥omnibus¥ini¥lview.elv" -elf "%NCHOME%¥omnibus¥ini¥tool.elf" 
-params "wombat"

tool.elf ファイルは前の NCOelct コマンドで参照され、 以下の内容を持ちます。

filter_name = 'ToolFilter';
filter_text = '( Node = ¥'@NodeName¥')';
filter_metric = 'avg(Severity)';
# End of file

コマンドまたはスクリプトを実行する場合、tool.elf ファイル内で -params オプションに続くストリング wombat により @NodeName プレースホルダーが置き換えられるため、イベント・リストにはノード wombat のイベントのみが表示されます。
