Viewing the full contents of the personal library

On occasion, you might want to refresh or update the personal library to show certain files that are currently held in this location, but which are, by default, not shown.

About this task

From the Conductor, perform any of the following actions:

Procedure

  • To refresh the contents of the personal library, click View > Refresh.
  • To view hidden files, which have names beginning with a period, click View > Show Hidden Files.
  • To view backup files, which have names ending in .bak, click View > Show Backup Files.
