イベント・リストの内容の印刷

イベント・リストの現在の表示を印刷することができます。 UNIX および Windows では、別の印刷オプションが使用可能です。 UNIX の場合、出力のフォーマットを制御することができます。

すべてのアラートまたは選択されたアラートを印刷することができます。 選択されたアラートを印刷する場合は、「印刷」ウィンドウを開いているときに、イベント・リストをクリックしてアラートを選択することによって、選択されたアラートを印刷できます。

Windows 上でイベント・リストを印刷するには、以下のようにします。
  1. イベント・リストから、「ファイル」 > 「印刷」をクリックします。 Windows の標準の「印刷」ウィンドウが開きます。
  2. ウィンドウ・オプションに入力し「OK」をクリックして、印刷します。

UNIX 上でイベント・リストを印刷するには、以下のようにします。

  1. イベント・リストから、「ファイル」 > 「印刷」をクリックします。 「印刷」ウィンドウが開きます。
  2. このウィンドウに以下のように入力します。
    プリンター
    「プリンター」オプションをクリックして、プリント出力をプリンターに送信します。
    ファイル
    「ファイル」オプションをクリックして、プリント出力をファイルに送信します。
    プリンター
    「プリンター」オプションを選択した場合は、プリンターの名前を入力します。
    ファイル
    「ファイル」オプションを選択した場合は、その出力を送信するファイルのパスと名前を入力します。
    または、「ファイル」ボタンをクリックしてファイル選択ウィンドウを表示し、そのディレクトリーとファイル名にナビゲートします。 「ファイルに出力」ウィンドウでファイル名を指定した後、「OK」をクリックして「印刷」ウィンドウに戻ります。
    ビュー内のすべてのアラート
    このオプションを選択して、現在のイベント・リスト内にあるすべてのアラートを印刷します。
    選択されたアラート
    このオプションを選択して、選択されたアラートのみを印刷します。
    行の分離文字
    このチェック・ボックスを選択して、行の分離文字を出力に挿入します (例えば、各行の間に破線を挿入する)。
    行番号
    このチェック・ボックスを選択して、行番号を各行の出力に追加します。
    列の分離文字
    このチェック・ボックスを選択して、列の分離文字を出力に挿入します (例えば、フィールドの開始とフィールドの末尾にパイプ ( | ) 文字を挿入する)。
    列見出し
    このチェック・ボックスを選択して、列見出しを出力ファイルの先頭に追加します。
    OK
    このボタンをクリックして、イベント・リストを選択した印刷設定を使用して印刷します。
    キャンセル
    このボタンをクリックして、印刷操作をキャンセルします。
