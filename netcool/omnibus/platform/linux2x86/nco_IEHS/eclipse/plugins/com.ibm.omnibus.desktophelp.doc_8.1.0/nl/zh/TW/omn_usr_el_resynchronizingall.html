重新同步化所有事件清單

如果系統管理者進行了任何配置變更，則您必須重新同步化事件清單。 您可以使用所有物件的配置重新同步化事件清單，或者您可以選擇個別物件，例如，直欄視覺化、警示類別、使用者、群組、功能表或服務。

若要重新同步化「事件清單」監視器框視窗的可用事件清單，請按一下檔案 > 重新同步化，然後選取必要的重新同步化選項。

重新同步化程序在 UNIX 及 Windows 事件清單上的運作方式不同。在 UNIX 事件清單上，目前選取的視圖未重新同步化；然而，使用「視圖建置器」新增至所選視圖的任何直欄都會反映重新同步化的值。

在 Windows 事件清單中，只會重新同步化監視器框中目前選取的視圖。如果使用監視器框，以在重新同步化之後選取不同的視圖，則不會重新同步化該視圖。
