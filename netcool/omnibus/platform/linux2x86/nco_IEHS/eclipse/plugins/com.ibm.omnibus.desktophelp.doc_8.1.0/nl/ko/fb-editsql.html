IBM Tivoli Netcool/OMNIbus, 버전 7.4

필터 SQL 편집

이 창에서 필터를 작성하거나 편집할 때 필터 빌더에 삽입한 SQL 텍스트를 수동으로 편집할 수 있습니다.

올바른 SQL 구문을 사용하여 필터 조회를 입력하고 확인을 눌러서 입력을 저장한 후 필터 빌더로 돌아가십시오. 필터 빌더는 SQL 텍스트를 구문 분석하여 필터 트리를 빌드하려고 시도합니다. SQL이 올바르지 않은 경우 오류 메시지가 표시되며, 계속하려면 우선 필터 SQL 편집 창에서 오류를 정정해야 합니다.
