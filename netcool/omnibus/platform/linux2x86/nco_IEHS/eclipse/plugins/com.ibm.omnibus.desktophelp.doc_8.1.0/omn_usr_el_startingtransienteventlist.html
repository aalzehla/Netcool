IBM Tivoli Netcool/OMNIbus Version 8.1

Starting the transient event list from the command line

Use the nco_elct command to open the transient event list from the UNIX command line, and use the NCOelct.exe command to open the transient event list from the Windows command line.

About this task

To start the transient event list:

Procedure

Perform the appropriate action for your operating system:
Option Description
UNIX From a command prompt, enter:

$NCHOME/omnibus/bin/nco_elct option
Windows From a command prompt, enter:

%NCHOME%\omnibus\desktop\NCOelct.exe option
In the UNIX and Windows commands shown in the preceding table, option represents one or more command-line options.

Results

If an instance of the event list (nco_event) is running, the nco_elct utility opens a new event list window. If there is no running instance of nco_event, the nco_elct utility starts one, and displays the Event List monitor box window and an event list window using the filter specified at the command line.

  • UNIX command-line options for the transient event list
    When you run the nco_elct command to start the transient event list on UNIX, you must specify a number of command-line options to customize the event list. You can run this command from the $NCHOME/omnibus/bin directory.
  • Windows command-line options for the transient event list
    When you run the NCOelct.exe command to start the transient event list on Windows, you must specify a number of command-line options to customize the event list. You can run this command from the %NCHOME%\omnibus\desktop directory.
