Filter SQL Edit

Use this window to manually edit the SQL text that is inserted into the Filter Builder when you are creating or editing filters.

Type the filter query by using valid SQL syntax, and click OK to save your entry and return to the Filter Builder. The Filter Builder attempts to parse the SQL text and build a filter tree. If the SQL is invalid, an error message is displayed, and you must correct the errors in the Filter SQL Edit window before you can continue.
