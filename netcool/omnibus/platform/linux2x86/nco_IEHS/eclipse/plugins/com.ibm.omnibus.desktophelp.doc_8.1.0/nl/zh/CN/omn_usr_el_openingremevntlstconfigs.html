打开远程事件列表配置

可以打开远程事件列表配置文件，以便打开一个或多个位于远程服务器上的事件列表。 服务器必须可以使用 HTTP 或 FTP 进行访问。

要打开远程事件列表配置文件：
  1. 从“事件列表”监视器框窗口单击文件 > 打开。 此时将打开“打开配置”窗口。
  2. 按以下方法完成此窗口：在文件名字段内，输入远程文件的 Web 地址，然后单击确定。 必须对 Web 地址使用以下某个格式：

    http://servername[:port]/path/filename.elc

    http://username:password@servername[:port]/path/filename.elc

    ftp://servername[:port]/path/filename.elc

    ftp://username:password@servername[:port]/path/filename.elc

    仅当服务器未使用缺省端口时才需要指定可选端口号 :port。 对于 HTTP，Web 服务器必须在远程主机上运行。

    可以选择为 servername 值指定 IPv4 或 IPv6 地址。 指定 IPv6 地址时，使用方括号 [] 括起地址；例如：

    http://[2003:13f8:206:190:204:baef:fe6a:6af0]:2880/mydir/myfile.elc

    对于 FTP，路径是从主目录开始的相对路径；例如，/mydir/myconfig.elc。

    远程文件的副本将被写入本地高速缓存。 在 Windows 系统上，指定之前已打开的远程文件时，将使用高速缓存的副本。如果无法检索远程文件，将显示错误消息。

结果

如果检索已成功，将刷新“事件列表”监视器框窗口以显示该配置内的监视器框，而且事件列表的标题栏内将显示选定配置文件名。
