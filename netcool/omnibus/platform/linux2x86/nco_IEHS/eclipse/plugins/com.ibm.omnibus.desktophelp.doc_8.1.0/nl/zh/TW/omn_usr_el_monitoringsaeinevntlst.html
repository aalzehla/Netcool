IBM Tivoli Netcool/OMNIbus 7.4 版

監視事件清單中影響服務的事件

影響服務的事件 (SAE) 是一則警示，警告操作員已有一個以上網路事件影響到重要客戶服務。 影響服務的事件是在 IBM Tivoli Network Manager IP Edition 內產生。

如果系統已配置為支援此功能，則您可以從 Tivoli Netcool/OMNIbus 事件清單監視影響服務的事件。

從 Tivoli Netcool/OMNIbus 事件清單中，您可以監視影響服務的事件，如下所示：

  • 若要顯示與影響服務的事件相關聯的基礎事件（例如 linkDown）， 請選取影響服務的事件，用滑鼠右鍵按一下它，然後從蹦現功能表中按一下顯示 SAE 相關的事件。 與選定事件相關聯的所有事件都會顯示在新視窗中。
  • 若要顯示與事件（例如 linkDown）相關的影響服務的事件，請選取該事件，用滑鼠右鍵按一下它，然後從蹦現功能表中按一下顯示 SAE 相關的服務。 所有相關影響服務的事件的清單即會顯示在新視窗中。 例如，如果您在事件清單中選取了 linkDown 事件，則此視窗中會顯示受到該 linkDown 事件影響的所有服務。
