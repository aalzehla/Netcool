IBM Tivoli Netcool/OMNIbus V7.4

使用工具来管理事件

系统管理员可以定义用于处理事件列表中事件的工具，并可将这些工具添加到 Tivoli Netcool/OMNIbus 桌面菜单以供您使用。 可以使用这些工具对 ObjectServer 运行 SQL 命令，并可运行诸如本地应用程序批处理文件或脚本之类的外部命令。

随事件列表提供了多种标准工具。 其中包括 Ping 和 Telnet 工具。 这些工具可独立于事件运行。

  • 发出 Ping 命令
    Ping 工具可对 alerts.status 表的“节点”列内的值运行 Ping 命令。
  • 发出 Telnet 命令
    Telnet 工具对 alerts.status 表的“节点”列内保存的值运行 Telnet 会话。
  • 运行系统管理员配置的工具
    如果系统管理员已创建用于事件管理的工具，那么就可以对事件列表内的事件运行这些工具。
